#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Aby instalować regularne aktualizacje za pośrednictwem klasycznego portalu platformy Azure
1. Na stronie **Urządzenia** wybierz urządzenie, na którym chcesz zainstalować aktualizacje.
2. Przejdź do**pozycji Konserwacja** **urządzeń** > i przewiń w dół do **pozycji Aktualizacje oprogramowania**.
3. Aby sprawdzić dostępność aktualizacji, kliknij pozycję **Sprawdź aktualizacje** u dołu strony.
4. Jeśli dostępne są aktualizacje oprogramowania, zostanie wyświetlony komunikat. Kliknij **pozycję Zainstaluj aktualizacje,** aby rozpocząć aktualizowanie urządzenia.
   
    Zostaniesz powiadomiony, gdy aktualizacja zostanie pomyślnie zainstalowana.
