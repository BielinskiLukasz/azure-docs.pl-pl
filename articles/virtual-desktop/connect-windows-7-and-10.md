# <a name="connect-with-windows-7-and-windows-10-preview"></a>Połącz się z Windows 7 i Windows 10 (wersja zapoznawcza)

> Dotyczy Windows 7 i Windows 10.

Dostępny jest dostępny do pobrania klient, który zapewnia dostęp do zasobów pulpitu wirtualnego Windows (wersja zapoznawcza) z urządzeń z systemem Windows 7 i Windows 10.

## <a name="install-the-client"></a>Instalowanie klienta

[Pobierz](https://go.microsoft.com/fwlink/?linkid=2068602) i instalowanie klienta na komputerze lokalnym. Wymaga to uprawnień administratora.

## <a name="subscribe-to-a-feed"></a>Subskrybować źródła danych

Pobierz listę zasobów dostępnych dla Ciebie z komputera lokalnego, Subskrybuj źródło danych podane przez administratora.

Aby subskrybować źródła danych:

1. Uruchom klienta z listę wszystkich aplikacji, wyszukaj pulpitu zdalnego.
2. Wybierz konto służbowe lub szkolne na stronie głównej. Spowoduje to subskrypcji przy użyciu domyślnego adres URL źródła danych.
   - Możesz też podać inny adres URL, jeśli to konieczne z **zamiast tego użyj adresu URL** opcji.
3. Zaloguj się przy użyciu konta użytkownika.

> [!NOTE]
> Jeśli administrator skonfigurował odnajdywania poczty e-mail, klient automatycznie przeszukuje Twoje źródło danych adresu URL, jeśli wprowadź swój adres e-mail.

Po pomyślnym uwierzytelnieniu powinny teraz wyświetlić listę dostępnych zasobów.

Zasoby można uruchomić za pomocą jednej z dwóch metod.

- Przejdź do strony głównej klienta, a następnie wybierz zasób.
- Uruchom zasobu, tak jak zwykle innych aplikacji w Start Menu.
  -  Możesz również wyszukać aplikacje na pasku wyszukiwania.

## <a name="update-the-client"></a>Aktualizacja klienta programu

Po udostępnieniu nowej wersji klienta zostanie wyświetlone powiadomienie przez klienta i Centrum akcji Windows. Wybierz powiadomienie, aby rozpocząć proces aktualizacji.
