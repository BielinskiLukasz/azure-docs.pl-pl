# <a name="azure-network-security"></a>Zabezpieczenia sieci na platformie Azure
## <a name="abstract"></a>Abstrakcyjny
Usługi sieci platformy Azure Maksymalizuj elastyczność, dostępności, odporności, bezpieczeństwa i integralności zgodnie z projektem. Ten dokument zawiera szczegółowe informacje dotyczące funkcji sieciowych platformy Azure. Opisano również, jak klienci mogą używać funkcji natywnych zabezpieczeń w Azure w celu ochrony swoimi zasobami informacyjnymi.
Obejmują zamierzonych odbiorców dla ten oficjalny dokument:
- Menedżerów technicznego, Administratorzy sieci i deweloperów poszukujących rozwiązań zabezpieczeń, które są dostępne, obsługiwane na platformie Azure.
- MSP lub firm procesu dyrektorzy, którzy mają ogólne omówienie technologii platformy Azure i usług, które odnoszą się do zabezpieczenia sieci w chmurze publicznej Azure.
