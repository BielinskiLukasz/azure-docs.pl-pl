Można wykluczać mowę z tekstu przy użyciu zestawu Speech SDK dla SWIFT i celu języka C.

## <a name="prerequisites"></a>Wymagania wstępne

W poniższych przykładach założono, że masz konto platformy Azure i subskrypcję usługi mowy. Jeśli nie masz konta i subskrypcji, [Wypróbuj usługę mowy bezpłatnie](../../../overview.md#try-the-speech-service-for-free).

## <a name="install-speech-sdk-and-samples"></a>Instalowanie zestawu Speech SDK i przykładów

[Zestaw SDK mowy Cognitive Services](https://github.com/Azure-Samples/cognitive-services-speech-sdk) zawiera przykłady w postaci SWIFT i celu dla systemów iOS i Mac. Kliknij link, aby zobaczyć instrukcje dotyczące instalacji poszczególnych próbek:

* [Funkcja syntezy mowy w języku C na macOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/macos/text-to-speech)
* [Funkcja syntezy mowy w programie SWIFT w systemie macOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/macos/text-to-speech)
* [Funkcja syntezy mowy w celu języka C w systemie iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/ios/text-to-speech)
* [Synteza mowy w programie SWIFT w systemie iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/ios/text-to-speech)
* [Dodatkowe przykłady dla celu języka C w systemie iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/samples/objective-c/ios)

Udostępniamy również [zestaw SDK mowy online dla odwołania do języka C](/objectivec/cognitive-services/speech/).
