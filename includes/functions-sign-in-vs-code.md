## <a name="sign-in-to-azure"></a>Logowanie do platformy Azure

Zanim będzie można opublikować aplikację, należy zalogować się do platformy Azure. 

1. Jeśli nie jesteś jeszcze zalogowany, wybierz ikonę platformy Azure na pasku aktywności, a następnie w obszarze **Azure: Functions** wybierz pozycję **Zaloguj się na platformie Azure...**. Jeśli jeszcze go nie masz, możesz **utworzyć bezpłatne konto platformy Azure**. 

    ![Odpowiedź hosta localhost funkcji wyświetlona w przeglądarce](./media/functions-sign-in-vs-code/functions-sign-into-azure.png)

    Jeśli jesteś już zalogowany, przejdź do następnej sekcji.

1. Po wyświetleniu monitu w przeglądarce wybierz swoje konto platformy Azure i zaloguj się przy użyciu poświadczeń konta platformy Azure. 

1. Po pomyślnym zalogowaniu się możesz zamknąć nowe okno przeglądarki. Subskrypcje, które należą do konta platformy Azure są wyświetlane na pasku bocznym.
