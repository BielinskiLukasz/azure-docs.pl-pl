# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Aktualizacja narzędzia do modelowania zagrożeń wydanie 7.1.60702.1 - 7/2/2019

Wersja 7.1.60702.1 narzędzia Microsoft Threat Modeling Tool (TMT) została wydana 2 lipca 2019 r. i zawiera następujące zmiany:

- Usprawnienia w zakresie ułatwień dostępu
- Poprawki błędów
- Włączenie społeczności open source pod warunkiem wzornik zestaw

## <a name="feature-changes"></a>Zmiany funkcji

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Dostępny jest nowy wzornik urządzeń medycznych dostarczany przez społeczność open source

Społeczność open source wniósł szablon do modelowania urządzeń medycznych. Po aktualizacji nowy zestaw wzorników pojawi się w menu rozwijanym wyboru szablonu. Aby uzyskać informacje na temat tworzenia szablonów lub zawartości szablonów, zapoznaj się z informacjami na [stronie GitHub](https://github.com/Microsoft/threat-modeling-templates)projektu .

![Opcja sprawdzania poprawności modelu](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)

## <a name="system-requirements"></a>Wymagania systemowe

- Obsługiwane systemy operacyjne
  - [Rocznicowa aktualizacja systemu Microsoft Windows 10 lub nowsza](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97)
- Wymagana wersja .NET
  - [.Net 4.7.1](https://go.microsoft.com/fwlink/?LinkId=863262) lub nowsze
- Wymagania dodatkowe
  - Do otrzymywania aktualizacji narzędzia oraz szablonów wymagane jest połączenie z Internetem.
