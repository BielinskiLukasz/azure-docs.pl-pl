# <a name="bing-search-sdk"></a>Wyszukiwanie Bing zestawu SDK
Przykłady interfejsu API wyszukiwania jednostek Bing obejmują scenariusze który:
1.  Wyszukuj jednostki, takie jak rejs Tom i uzyskuj szczegółowe informacje.
2.  Obsługuj uściślania warunków dla zapytań z potencjalnie wielu intencji.
3.  Wyszukaj lokalnej jednostki, takie jak restauracja i bogate informacje wokół niej.
4.  Wyszukaj lokalnych firm, np. restauracje i Uzyskaj szczegółowe informacje.
5.  Wyzwalanie złe żądanie oraz obsługi błędów.

> [!NOTE] 
> Niektóre zestawy SDK są teraz w wersji ogólnie dostępnej i oczekujących zmian w dokumentacji. 

Zestawy SDK wyszukiwania Bing należy funkcji wyszukiwania w sieci web łatwo dostępne w następujących językach programowania:
* Rozpoczynanie pracy z usługą [przykłady dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.EntitySearch/1.2.0)
    * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykłady dla platformy Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [biblioteki technologii Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykładów w języku Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [biblioteki Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykłady w języku Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-entitysearch) dla definicji i zależności.

Przykłady zestawu SDK dla każdego z języków zawierają pliku ReadMe ze szczegółowymi informacjami o wymaganiach wstępnych oraz instalowanie/uruchamiania przykładów.
