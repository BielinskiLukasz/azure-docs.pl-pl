# <a name="azure-cosmos-db-query-cheat-sheets"></a>Arkusze porad zapytania w usłudze Azure Cosmos DB

**Arkusze porad zapytań usługi Azure Cosmos DB** pomagają w szybkim zapytania zapisu danych przez wyświetlanie typowych zapytań do bazy danych, operacje, funkcje i operatory łatwe drukowanie dokumentów PDF. Arkusze porad obejmują informacje dotyczące programu SQL, MongoDB, Table i interfejsów API rozwiązania Gremlin. 

Wybierz z pobrania rozmiarze Letter lub o rozmiarze A3. 

## <a name="letter-sized-cheat-sheets"></a>Arkusze porad o rozmiarze Letter

Pobierz [arkusze porad o rozmiarze letter zapytań usługi Azure Cosmos DB](https://go.microsoft.com/fwlink/?LinkId=623215) Jeśli zamierzasz drukować na papierze o rozmiarze letter (8,5 "x 11").

![Azure Cosmos DB zapytania arkusze porad — wymiarach, za pomocą interfejsu API SQL, JavaScript, MongoDB, Gremlin i interfejsu API tabel zapytań i functions](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Arkusze porad zbyt duży
Pobierz [arkusze porad zapytania o rozmiarze usługi Azure Cosmos DB A3](https://go.microsoft.com/fwlink/?linkid=870413) Jeśli zamierzasz wydruku przy użyciu ploterów lub drukarki na dużą skalę na dokument o rozmiarze A3 (11,7 "x 16.5").

![Arkusze porad zapytań usługi Azure Cosmos DB — A3 wielkości, przy użyciu interfejsu API SQL, JavaScript, MongoDB, Gremlin i interfejsu API tabel zapytania i funkcje](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
