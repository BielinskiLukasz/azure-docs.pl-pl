# <a name="managing-owners-for-a-group"></a>Zarządzanie właścicielami grupy
Gdy właściciel zasobu ma przypisany dostęp do tego zasobu do grupy usługi Azure AD, członkostwo w grupie jest zarządzana przez właściciela grupy. Właściciel zasobu przekazuje skutecznie uprawnienia do przypisywania użytkowników do zasobów do właściciela grupy.

## <a name="add-an-owner-to-a-group"></a>Dodawanie właściciela do grupy

1. W [Centrum administracyjnego usługi Azure AD](https://aad.portal.azure.com), wybierz opcję **użytkowników i grup**.
2. Wybierz **wszystkich grup**, a następnie otwórz grupę, którą chcesz dodać właścicieli.
3. Wybierz **Dodawanie właścicieli**.
4. Na **Dodawanie właścicieli** wybierz użytkownika, który chcesz dodać jako właściciela tej grupy i upewnij się, że nazwa została dodana do **wybrane** okienka.

## <a name="remove-an-owner-from-a-group"></a>Usuwanie właściciela z grupy

1. W [Centrum administracyjnego usługi Azure AD](https://aad.portal.azure.com), wybierz opcję **użytkowników i grup**.
2. Wybierz **wszystkich grup**, a następnie otwórz grupę, z którego chcesz usunąć właścicieli.
3. Wybierz **właścicieli** kartę.
4. Wybierz właściciela, który chcesz usunąć z tej grupy, a następnie wybierz **Usuń**.
