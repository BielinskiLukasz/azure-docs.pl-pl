## <a name="setupdevenv"></a>Konfigurowanie środowiska projektowego
Aby rozpocząć, skonfiguruj środowisko projektowe za pomocą programu Visual Studio i zestawu Azure SDK.

* Visual Studio 2017 zawiera zestaw Azure SDK. Jeśli używasz programu VS2017, nie jest wymagana żadna dodatkowa konfiguracja środowiska projektowego.
* W przypadku programu Visual Studio 2015 kliknij następujący link, aby zainstalować [zestaw Azure SDK dla programu Visual Studio 2015](http://go.microsoft.com/fwlink/?linkid=518003).
* W przypadku programu Visual Studio 2013 kliknij następujący link, aby zainstalować [zestaw Azure SDK dla programu Visual Studio 2013](http://go.microsoft.com/fwlink/?LinkID=324322).

* Jeśli nie zainstalowano programu Visual Studio, użyj następującego linku, aby zainstalować program [Visual Studio 2017](https://www.visualstudio.com/) z zestawem Azure SDK.

> [!NOTE]
> Zależnie od liczby składników zależnych zestawu SDK, które znajdują się już na komputerze, instalowanie zestawu SDK może trwać od kilku minut do pół godziny lub dłużej.
