# <a name="cloud-partner-portal-management"></a>Zarządzanie portal Cloud Partner

W tej sekcji opisano typy operacji dostępne do tworzenia kont i zarządzania nimi wymaganych do korzystania z [Portal Cloud partner](https://cloudpartner.azure.com).  Te konta są wymagane w [witrynie Azure Marketplace](https://azuremarketplace.microsoft.com) i [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Informacje ogólne

Przed rozpoczęciem korzystania z portal Cloud Partner należy zapoznać się z następującymi artykułami:

- [Zasady i postanowienia w witrynie Azure Marketplace](../../policies-terms.md) wymieniają różne wymagania prawne i techniczne oraz zasady dotyczące użytkowników i wydawców w [portalu Azure Marketplace](https://azuremarketplace.microsoft.com) i [AppSource](https://appsource.microsoft.com).
- [Zostań wydawcą portalu Marketplace w chmurze](https://docs.microsoft.com/azure/marketplace/become-publisher) zawiera przegląd wszystkich kroków wymaganych do uzyskania oficjalnego wydawcy w chmurze firmy Microsoft.


## <a name="account-creation-and-management"></a>Tworzenie kont i zarządzanie nimi

Ta sekcja zawiera następujące artykuły objaśniające sposób zarządzania kontami wydawcy portalu Marketplace.

- [Wymagania wstępne dotyczące zarządzania kontami](./cpp-manage-prerequisites.md) wyjaśniają, jak stać się zarejestrowanym wydawcą w chmurze, wymaganym przy użyciu portalu Cloud partner.
- [Zarządzanie](./cpp-manage-publisher-profile.md) profilem wydawców w portalu Azure Marketplace i AppSource — opis sposobu używania Portal Cloud partner 
- [Zarządzanie użytkownikami na portal Cloud partner](./cpp-manage-users.md) Dodawanie użytkowników do konta publikacji w chmurze w organizacji oraz zarządzanie dostępem, aby umożliwić wielu osobom korzystanie z tych samych ofert.
