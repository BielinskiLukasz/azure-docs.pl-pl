# <a name="servicenow-and-azure-active-directory"></a>Usługi ServiceNow i Azure Active Directory

Przyspiesz wdrażanie usługi ServiceNow w całej organizacji i Zwiększ liczbę użytkowników, korzystając z naszej wbudowanej integracji. Usprawnij zarządzanie usługami IT, łącząc usługi ServiceNow z Azure Active Directory (Azure AD).

![Pokazuje logo aplikacji usługi ServiceNow](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Wypróbuj usługi ServiceNow z usługą Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Co możesz zrobić z usługi ServiceNow i usługą Azure AD?

- **Łatwe logowanie** jednokrotne — eliminowanie wielu nazw użytkowników i haseł oraz uproszczenie dostępu przez umożliwienie użytkownikom logowania się do usługi ServiceNow przy użyciu kont usługi Azure AD.
- **Automatyzowanie aprowizacji użytkowników** — Automatyzuj tworzenie, konserwację i usuwanie kont użytkowników w usłudze usługi ServiceNow, gdy użytkownicy dołączają, przenosiją lub opuszczają organizację.
- **Zabezpieczanie, zarządzanie dostępem i monitorowanie** dostępu do usługi ServiceNow przez stosowanie zasad dostępu warunkowego, zarządzanie dostępem za pomocą przeglądów dostępu i monitorowanie dostępu przy użyciu dzienników logowania.

## <a name="get-started"></a>Wprowadzenie

Skonfiguruj [Inicjowanie obsługi użytkowników](https://aka.ms/ServiceNowProvisioning) i [Logowanie](https://aka.ms/ServiceNowSSO) jednokrotne dla usługi ServiceNow przy użyciu naszych łatwych przewodników konfiguracji.
