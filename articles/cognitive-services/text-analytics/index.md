    <h1>Dokumentacja interfejsu API analizy tekstu</h1>
    <div class="intro" style="min-width: 200px">
        <p>Interfejs API analizy tekstu to oparta na chmurze usługa udostępniająca zaawansowane przetwarzanie języka naturalnego z zastosowaniem tekstu nieprzetworzonego. Obejmuje ona trzy główne funkcje: analizę tonacji, wyodrębnianie kluczowych fraz i wykrywanie języka. Dowiedz się, jak na różne sposoby analizować zawartość, korzystając z przewodników Szybki start, samouczków i przykładów.</p>
    </div>
