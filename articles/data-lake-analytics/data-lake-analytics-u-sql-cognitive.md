# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Wprowadzenie do funkcji poznawczych języka U-SQL

## <a name="overview"></a>Omówienie
Funkcje poznawcze dla języka U-SQL umożliwiają deweloperom korzystanie z analizy w swoich programach dużych zbiorów danych. 

Dostępne są następujące przykłady korzystające z funkcji poznawczych:
* Obrazowanie: [Wykrywanie twarzy](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Obrazowanie: [Wykrywanie emocji](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Obrazowanie: [wykrywanie obiektów (tagowanie)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Obrazowanie: [OCR (optyczne rozpoznawanie znaków)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: [Wyodrębnianie fraz kluczowych & analiza tonacji](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
