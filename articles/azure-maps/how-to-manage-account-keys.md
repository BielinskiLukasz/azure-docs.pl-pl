# <a name="manage-your-azure-maps-account"></a>Zarządzanie kontem usługi Azure Maps

Możesz zarządzać kontem usługi Azure Maps za pośrednictwem witryny Azure portal. Po dokonaniu konta można zaimplementować interfejsy API w witrynie sieci Web lub w aplikacji mobilnej.

Jeśli nie masz subskrypcji platformy Azure, utwórz [bezpłatne konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) przed rozpoczęciem.

## <a name="create-a-new-account"></a>Tworzenie nowego konta

1. Zaloguj się do [Portalu Azure](https://portal.azure.com).

2. W lewym górnym rogu witryny Azure portal wybierz pozycję **Utwórz zasób**.

3. Wyszukaj i **wybierz**mapy . Następnie wybierz pozycję **Utwórz**.

4. Wprowadź informacje dotyczące nowego konta.

[![Wprowadzanie informacji o koncie usługi Azure Maps w witrynie Azure portal](./media/how-to-manage-account-keys/new-account-portal.png)](./media/how-to-manage-account-keys/new-account-portal.png#lightbox)

## <a name="delete-an-account"></a>Usuwanie konta

Konto można usunąć z witryny Azure Portal. Przejdź do strony przeglądu konta i wybierz pozycję **Usuń**.

[![Usuwanie konta usługi Azure Maps w witrynie Azure portal](./media/how-to-manage-account-keys/account-delete-portal.png)](./media/how-to-manage-account-keys/account-delete-portal.png#lightbox)

Następnie zostanie wyświetlona strona potwierdzenia. Możesz potwierdzić usunięcie swojego konta, wpisując jego nazwę.
