# <a name="threat-modeling-tool-releases"></a>Wydania narzędzia do modelowania zagrożeń

Narzędzie Microsoft Threat Modeling Tool jest obecnie wydawane jako bezpłatna aplikacja [typu "kliknij, aby pobrać"](https://aka.ms/threatmodelingtool) dla systemu Windows. Ten mechanizm dostarczania pozwala nam wypychać najnowsze ulepszenia i poprawki błędów do klientów za każdym razem, gdy otwierają narzędzie.

## <a name="system-requirements"></a>Wymagania systemu

- Obsługiwane systemy operacyjne
  - Rocznicowa aktualizacja systemu Microsoft Windows 10 lub nowsza
- Wymagana wersja .NET
  - .NET 4.7.1 lub nowsze
- Wymagania dodatkowe
  - Do odbierania aktualizacji narzędzia i szablonów wymagane jest połączenie z Internetem.
