# <a name="service-bus-messaging-samples"></a>Przykłady obsługi wiadomości usługi Service Bus
Przykłady obsługi wiadomości usługi Service Bus pokazują kluczowe funkcje [w komunikatach usługi Service Bus](https://azure.microsoft.com/services/service-bus/). Obecnie można znaleźć próbki w następujących miejscach:

| Język programowania | SDK lub lokalizacja próbek |
| -------------------- | ----------------------- |
| .NET, Java i zarządzanie | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Eksplorator usługi Service Bus

Ponadto [Eksplorator usługi Service Bus](https://github.com/paolosalvatori/ServiceBusExplorer) jest przykładem hostowanym w usłudze GitHub, który umożliwia łączenie się z obszarem nazw usługi Service Bus i łatwe zarządzanie jednostkami obsługi wiadomości. Narzędzie zapewnia zaawansowane funkcje, takie jak funkcje importu/eksportu oraz możliwość testowania jednostek obsługi wiadomości i usług przekazywania. Pełne źródło i dokumentacja eksploratora usługi Service Bus explorer można znaleźć na [stronie GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
