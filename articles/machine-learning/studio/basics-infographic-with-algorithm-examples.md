# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>Grafika informacyjna do pobrania: podstawy uczenia maszynowego z przykładowymi algorytmami

[!INCLUDE [Notebook deprecation notice](../../../includes/aml-studio-notebook-notice.md)]

Pobierz ten łatwy do zrozumienia przegląd infografiki podstawowych uczenia maszynowego, aby dowiedzieć się więcej o popularnych algorytmach używanych do odpowiadania na typowe pytania dotyczące uczenia maszynowego. Przykłady algorytmów pomagają początkującym uczenia maszynowego zrozumieć, których algorytmów użyć i do czego są używane.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Popularne algorytmy w Machine Learning Studio (klasyczny)

[Usługa Azure Machine Learning Studio (klasyczna)](https://studio.azureml.net/) zawiera dużą bibliotekę algorytmów do analizy predykcyjnej. Ta infografika identyfikuje cztery popularne rodziny algorytmów - regresję, wykrywanie anomalii, klastrowanie i klasyfikację — i zawiera łącza do przykładów pracy w [Galerii sztucznej inteligencji platformy Azure.](https://gallery.azure.ai/) Galeria zawiera przykładowe eksperymenty i samouczki, które pokazują, jak te algorytmy mogą być stosowane w wielu rzeczywistych rozwiązaniach.

## <a name="more-help-with-algorithms-for-beginners-and-advanced-users"></a>Więcej pomocy z algorytmami dla początkujących i zaawansowanych użytkowników

* Aby uzyskać głębszą dyskusję na temat różnych typów algorytmów uczenia maszynowego, sposobu ich użycia i sposobu wyboru odpowiedniego dla rozwiązania, zobacz [Jak wybrać algorytmy dla programu Microsoft Azure Machine Learning Studio (klasyczny).](algorithm-choice.md)
* Aby uzyskać listę według kategorii wszystkich algorytmów uczenia maszynowego dostępnych w udiociepau machine learning (klasyczny), zobacz [Inicjowanie modelu][initialize-model] w studio uczenia maszynowego (klasyczny) algorytm i pomoc modułu.
* Aby uzyskać pełną alfabetyczną listę algorytmów i modułów w udiocieli Machine Learning Studio (klasycznym), zobacz [listę A-Z modułów Machine Learning Studio (klasyczny)][a-z-list] w udiociele machine learning studio (klasyczny) algorytm i moduł pomocy.
* Aby zapoznać się z omówieniem galerii sztucznej inteligencji platformy Azure i wielu dostępnych tam zasobów generowanych przez społeczność, zobacz [Udostępnianie i odnajdywanie zasobów w galerii sztucznej inteligencji platformy Azure.](gallery-how-to-use-contribute-publish.md)
