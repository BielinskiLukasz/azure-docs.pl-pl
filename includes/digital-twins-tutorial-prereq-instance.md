### <a name="prepare-an-azure-digital-twins-instance"></a>Przygotowanie wystąpienia usługi Azure Digital bliźniaczych reprezentacji

Do ukończenia tego samouczka potrzebne jest **wystąpienie usługi Azure Digital bliźniaczych reprezentacji** . 

Jeśli masz już skonfigurowane wystąpienie usługi Azure Digital bliźniaczych reprezentacji na podstawie poprzedniej pracy, możesz użyć tego wystąpienia, a następnie przejdź do następnej sekcji.

W przeciwnym razie wykonaj następujące kroki, aby skonfigurować wystąpienie:
1. **Utwórz wystąpienie** przy użyciu instrukcji przedstawionych w instrukcje [*: Tworzenie wystąpienia usługi Azure Digital bliźniaczych reprezentacji*](../articles/digital-twins/how-to-set-up-instance.md). 
2. **Skonfiguruj rejestrację aplikacji Azure Active Directory dla swojego wystąpienia** , korzystając z sekcji *Tworzenie rejestracji aplikacji* [*: uwierzytelnianie aplikacji klienckiej*](../articles/digital-twins/how-to-authenticate-client.md#create-an-app-registration).
