# <a name="toggle-live-chat"></a>Przełączanie czatu na żywo

W tym artykule opisano, jak przełączać czat na żywo w aplikacji IoT Central. Możesz użyć czatu na żywo, aby uzyskać dostęp do pomocy technicznej.

> [!NOTE]
> Opcja czatu jest dostępna tylko dla aplikacji utworzonych przy użyciu bezpłatnego planu cenowego.

## <a name="chat-with-us"></a>Porozmawiaj z nami

Aby uzyskać pomoc techniczną, otwórz aplikację IoT Central i wybierz **opcję Porozmawiaj z nami.**

![Porozmawiaj z nami](media/howto-show-hide-chat/chat-with-us.png)

Pytanie można wpisać na poniższym zrzucie ekranu:

![Formularz "Konfigurowanie szczegółów urządzenia" ze szczegółami ustawień i właściwości](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Ukryj czat

Aby ukryć czat, wybierz **pozycję Ukryj czat** w panelu **Pomocy:**

 ![Ukryj czat](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Włącz czat

Aby wyświetlić czat, wybierz pozycję **Pokaż czat** w panelu **Pomocy:**

 ![Pokaż czat](media/howto-show-hide-chat/show-chat.png)
