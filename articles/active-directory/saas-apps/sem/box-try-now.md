# <a name="box-and-azure-active-directory"></a>I Azure Active Directory

Przyspieszenie wdrożenia w całej organizacji i zwiększenie wdrożenia użytkowników przy użyciu naszej wbudowanej integracji. Umożliwienie użytkownikom szybkiego współdziałania przez połączenie z usługą Azure Active Directory (Azure AD).

![Wyświetla logo aplikacji Box](./media/box.png)

> [!div class="nextstepaction"]
> [Wypróbuj usługę Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Co możesz zrobić z funkcją Box i usługą Azure AD?

- **Łatwe logowanie** jednokrotne — eliminowanie wielu nazw użytkowników i haseł oraz uproszczenie dostępu przez umożliwienie użytkownikom logowania się w usłudze Box przy użyciu kont usługi Azure AD.
- **Automatyzowanie aprowizacji użytkowników** — automatyzuje tworzenie, konserwację i usuwanie kont użytkowników w usłudze Box, gdy użytkownicy dołączają, przenosiją lub opuszczają organizację.
- **Zabezpiecz, reguluj i monitoruj** dostęp bezpieczny dostęp do pola, stosując zasady dostępu warunkowego, Zarządzaj dostępem za pomocą przeglądów dostępu i monitoruj dostęp przy użyciu dzienników logowania.

## <a name="get-started"></a>Wprowadzenie

Skonfiguruj [Inicjowanie obsługi użytkowników](https://aka.ms/BoxSSODoc) i [Logowanie](https://aka.ms/BoxProvisioningDoc) jednokrotne w usłudze Box przy użyciu naszych przewodników związanych z prostymi konfiguracjami.
