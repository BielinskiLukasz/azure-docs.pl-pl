* Aktywne konto platformy Azure. (Jeśli nie masz konta, możesz utworzyć [bezpłatne konto](https://azure.microsoft.com/pricing/free-trial/) w ciągu zaledwie kilku minut).

* [Python 2.x lub 3.x](https://www.python.org/downloads/). Upewnij się, że używasz 32-bitowej lub 64-bitowej instalacji zgodnie z wymaganiami konfiguracji. Po wyświetleniu monitu podczas instalacji upewnij się, że język Python został dodany do zmiennej środowiskowej specyficznej dla platformy. Jeśli używasz środowiska Python 2.x, może być konieczne [zainstalowanie lub uaktualnienie systemu zarządzania pakietami języka Python — *pip*](https://pip.pypa.io/en/stable/installing/).

* W razie potrzeby zainstaluj pakiet [urządzenia azure-iot,](https://pypi.org/project/azure-iot-device/) używając polecenia`pip install azure-iot-device`

* W razie potrzeby zainstaluj pakiet [azure-iothub-service-client,](https://pypi.org/project/azure-iothub-service-client/) używając polecenia`pip install azure-iothub-service-client`
