# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Należy utworzyć dodatkową subskrypcję w witrynie Azure portal

Możesz utworzyć dodatkowych subskrypcji dla swojego konta na platformie Azure. Możesz chcieć dodatkową subskrypcję, aby uniknąć osiągnięcia limitów subskrypcji, aby utworzyć oddzielne środowiska dla zabezpieczeń i rozliczeniami lub do izolowania danych w celu zachowania zgodności.

Jeśli chcesz tworzyć subskrypcje platformy Azure w ramach umowy Enterprise Twoja organizacja musi mieć rolę właściciela konta, dla Twojej organizacji.

Jeśli nie masz jeszcze subskrypcji platformy Azure, Utwórz [bezpłatne konto](https://azure.microsoft.com/free).

## <a name="create-an-additional-azure-subscription"></a>Należy utworzyć dodatkową subskrypcję platformy Azure

1. Zaloguj się w witrynie [Azure Portal](https://portal.azure.com).
1. Wyszukaj **subskrypcje**.

   ![Zrzut ekranu pokazujący wyszukiwania w portalu dla subskrypcji](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. Wybierz pozycję **Dodaj**.

   ![Zrzut ekranu przedstawiający przycisk Dodaj w widoku subskrypcji](./media/billing-create-subscription/subscription-add.png)

Można również utworzyć dodatkowe subskrypcje programistycznie. Aby uzyskać więcej informacji, zobacz [programowe tworzenie subskrypcji Azure Enterprise](../azure-resource-manager/programmatically-create-subscription.md).
