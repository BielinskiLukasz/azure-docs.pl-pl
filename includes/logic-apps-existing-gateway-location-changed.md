**Q**: Dlaczego lokalizacji dla mojego istniejącego zmiany bramy? <br/>
**A**: zasoby bramy, utworzonych przed 3 maja 2017 zostały przeniesione do oryginalnej lokalizacji dzierżawy usługi Azure AD dla konta służbowego, które utworzone tych bram. Jednak te zmiany nie wpływa na aktualnie uruchomionej aplikacji logiki, które powinny nadal działać jak zwykle. Bramy ogólne udostępnienie w maju lokalizacje zasobów bramy może się różnić od lokalizacji aplikacji Logic Apps.
