# <a name="bing-search-sdk"></a>Wyszukiwanie Bing zestawu SDK
Przykłady interfejsu API wyszukiwania jednostek Bing obejmują scenariusze który:
1.  Wyszukuj jednostki, takie jak rejs Tom i uzyskuj szczegółowe informacje.
2.  Obsługuj uściślania warunków dla zapytań z potencjalnie wielu intencji.
3.  Wyszukaj lokalnej jednostki, takie jak restauracja i bogate informacje wokół niej.
4.  Wyszukaj lokalnych firm, np. restauracje i Uzyskaj szczegółowe informacje.
5.  Wyzwalanie złe żądanie oraz obsługi błędów.

> [!NOTE] 
> Niektóre zestawy SDK są teraz w wersji ogólnie dostępnej i oczekujących zmian w dokumentacji. 

Zestawy SDK wyszukiwania Bing należy funkcji wyszukiwania w sieci web łatwo dostępne w następujących językach programowania:
* Rozpocznij pracę, korzystając z [przykładów dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.EntitySearch/1.2.0)
    * Zobacz też [biblioteki platformy .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla środowiska Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [biblioteki środowiska Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla języka Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [biblioteki języka Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla języka Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-entitysearch) zawierające definicje i zależności.

Przykłady dotyczące zestawów SDK dla poszczególnych języków zawierają plik ReadMe ze szczegółowymi informacjami na temat wymagań wstępnych oraz instalowania i uruchamiania przykładów.
