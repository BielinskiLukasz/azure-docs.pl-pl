# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Przygotować dane za pomocą usługi Azure Machine Learning Prep zestawu SDK usługi Data dla języka Python
 
Przygotowywanie danych stanowi ważną część przepływu pracy machine learning. Modele będą bardziej dokładne i wydajne, jeśli mają dostęp do czyszczenia danych w formacie, który jest łatwiejsze korzystanie z. Możesz użyć [zestawu SDK usługi Azure Machine Learning danych Prep](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) do ładowania danych z różnych formatów, przekształcania, aby była bardziej użyteczne i zapisu danych do lokalizacji w celu swoje modele, aby uzyskać dostęp.

2. [Przekształcanie danych](how-to-transform-data.md )
3. [Zapisywanie danych](how-to-write-data.md )
