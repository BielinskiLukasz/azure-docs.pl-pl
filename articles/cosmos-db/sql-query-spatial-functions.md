# <a name="spatial-functions-azure-cosmos-db"></a>Funkcje przestrzenne (usługa Azure Cosmos DB)

Usługa Cosmos DB obsługuje następujące wbudowane funkcje OGC (Open Geospatial Consortium) w celu wykonywania zapytań dotyczących danych geoprzestrzennych. 

## <a name="functions"></a>Funkcje

Następujące funkcje skalarne wykonują operację na wartości wejściowej obiektu przestrzennego i zwracają wartość liczbową lub logiczną.
