> [!IMPORTANT]
> Podczas pracy z podsieciami bramy należy unikać kojarzenia sieciowej grupy zabezpieczeń (NSG, Network Security Group) z podsiecią bramy. Skojarzenie sieciowej grupy zabezpieczeń z tą podsiecią może spowodować, że brama Virtual Network (VPN, Express Route Gateway) przestanie działać zgodnie z oczekiwaniami. Aby uzyskać więcej informacji na temat sieciowych grup zabezpieczeń, zobacz [What is a Network Security Group?](../articles/virtual-network/security-overview.md) (Co to jest sieciowa grupa zabezpieczeń?)
