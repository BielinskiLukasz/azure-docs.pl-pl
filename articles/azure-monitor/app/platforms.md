## <a name="languages---community-supported"></a>Języki — obsługiwane przez społeczności

Istnieje szereg wspieranym przez społeczność zestawów SDK Application Insights, z których wiele zostały pierwotnie utworzone przez firmę Microsoft. Wspieranym przez społeczność zestawy SDK nie jest oficjalnie obsługiwane przez firmę Microsoft i nie możemy zapewnić obsługę dowolnego zestawu SDK, który nie znajduje się na liście oficjalnie obsługiwanych. Te zestawy SDK są traktowane jako eksperymentalne i nie są zalecane do użytku produkcyjnego.
