# <a name="overview-of-the-swift-csp-cscf-v2020-blueprint-sample"></a>Omówienie przykładowej strategii SWIFT CSP-CSCF v2020

Przykład strategii SWIFT CSP-CSCF v2020 udostępnia wytyczne dotyczące ładu w usłudze [Azure Policy](../../../policy/overview.md), które ułatwiają ocenę określonych mechanizmów kontroli SWIFT CSP. Ta strategia ułatwia klientom wdrażanie podstawowego zestawu zasad dla dowolnej architektury wdrożonej przez platformę Azure, która musi implementować mechanizmy kontroli SWIFT CSP.

## <a name="control-mapping"></a>Mapowanie kontrolek

Sekcja mapowania mechanizmów kontroli zawiera szczegółowe informacje dotyczące zasad w ramach tej strategii i o tym, jak te zasady pomagają spełnić wymagania różnych mechanizmów kontroli zgodnie z najnowszym standardem SWIFT CSP-CSCF. Po przypisaniu do architektury zasoby są oceniane przez usługę Azure Policy pod względem braku zgodności z przypisanymi zasadami. Aby uzyskać więcej informacji, zobacz [Azure Policy](../../../policy/overview.md).
