# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

Narzędzie Microsoft Security Response Center (MSRC) zostanie przeprowadzony przez niektóre z najbardziej doświadczonych ekspertów zabezpieczeń na świecie. Zidentyfikować, monitorują, reagować i rozwiązywać związane z bezpieczeństwem i w środowisku lokalnym i w chmurze luk w zabezpieczeniach wokół zegara, każdego dnia roku.

Oprócz ciągłej pracy zespół MSRC ma wiele zasobów, które są dostępne dla użytkownika, tak, aby zrozumieć, jak zabezpieczyć swoje zasoby platformy Azure i wdrożeń bardziej efektywnie.

## <a name="the-msrc-blog"></a>MSRC Blog

[MSRC blog](https://blogs.technet.microsoft.com/msrc/) jest miejscem, aby przejść do najnowsze wiadomości na MSRC działania do ochrony przed zagrożeniami w chmurze.

## <a name="white-papers"></a>Oficjalne dokumenty

Centrum MSRC został opublikowany szereg [oficjalne](https://technet.microsoft.com/library/bb969102.aspx) ułatwiające zrozumienie ich działania, oraz sposób ich funkcje. Niektóre zapewniają wgląd w jaki sposób zabezpieczamy firmy Microsoft w chmurze i zawierać przydatne informacje na temat sposobu można użyć tej samej konfiguracji zabezpieczeń.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Zabezpieczenia analitykowi zaangażowania i nagród programów

Centrum MSRC obsługuje współpracę i relacje z naukowo globalnie, aby awansować zabezpieczeń produktów firmy Microsoft.

Programy nagród firmy Microsoft płatność pracowników naukowo-badawczych dla:

- Wykorzystanie nowych technik
- Obrony pojęcia, które są eliminowanie nowej luki w zabezpieczeniach
- Identyfikacja krytycznych luk w zabezpieczeniach w programie Microsoft lokalnych i w chmurze oprogramowania.
