# <a name="connect-citrix-analytics-security"></a>Łączenie z rozwiązaniem Citrix Analytics (Security) 

Integracja citrix Analytics (Security) z usługą Azure Sentinel pomaga eksportować dane analizowane pod kątem ryzykownych zdarzeń z usługi Citrix Analytics (Security) do usługi Azure Sentinel. Możesz tworzyć niestandardowe pulpity nawigacyjne, analizować dane z innych źródeł wraz z danymi z citrix Analytics (Security) i tworzyć niestandardowe przepływy pracy przy użyciu aplikacji logiki w celu monitorowania i ograniczania zdarzeń zabezpieczeń. 



## <a name="prerequisites"></a>Wymagania wstępne

- Uprawnienia do odczytu i zapisu w obszarze roboczym usługi Azure Sentinel.

- Uprawnienie do citrix Security Analytics w citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Łączenie usługi Azure Sentinel z usługą Citrix Analytics (Security)

1. W witrynie Azure portal przejdź do łączników **usługi Azure Sentinel** > **Data,** a następnie wybierz łącznik **Citrix Analytics (Security).**

2. Wybierz **pozycję Otwórz stronę łącznika**.

3. Postępuj zgodnie z instrukcjami na stronie **Citrix Analytics (Security).**
