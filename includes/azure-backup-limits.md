| **Limit** | **Domyślne** |
| --- | --- |
| Serwerach lub maszynach, które mogą być rejestrowane w magazynie. | Windows Server/Windows klienta/System Center Data Protection Manager: 50. <br/><br/> Maszyny wirtualne IaaS: 1,000.  |
| Rozmiar źródła danych w magazynie w magazynie. |Maksymalna liczba 54,400 GB. Limit nie ma zastosowania do maszyn wirtualnych IaaS. |

| Zaplanuj codzienne wykonywanie kopii zapasowych. |Windows Server/kliencie: Trzy dziennie.<br/> System Center DPM: Dwa dziennie. <br/> Maszyny wirtualne IaaS: Raz dziennie.  |
