# <a name="azure-cdn-ddos-protection"></a>Usługa CDN Azure DDoS Protection

Usługa content delivery network zapewnia ochronę przed atakami DDoS, zgodnie z projektem. Oprócz globalnego pojemności w celu ochrony przed rozproszonymi atakami pomiarowej sieć CDN systemu Azure ma dodatkową ochronę przed atakami DDoS, co opisano poniżej, aby nie dodatkowych kosztów.

## <a name="azure-cdn-from-microsoft"></a>Usługa Azure CDN firmy Microsoft

Usługa Azure CDN firmy Microsoft jest chroniony przez [Azure DDoS podstawowe](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Jest zintegrowana w usłudze Azure CDN z platformy Microsoft domyślnie i bez ponoszenia dodatkowych kosztów. Skalowanie pełną i wydajność usługi Azure CDN z globalnie wdrożonych sieci firmy Microsoft zawiera obrony przed typowymi atakami warstwy sieci poprzez ograniczenie monitorowania i w czasie rzeczywistym zawsze w ruchu. Podstawowa ochrona przed atakami DDoS również systemów względem najbardziej typowych często występujących powodzie zapytania DNS 7 warstwy i warstwy 3 i 4 pomiarowej atakami tego docelowymi punktami końcowymi usługi CDN. Ta usługa również ma sprawdzonych poświadczać ochronę firmy Microsoft dla przedsiębiorstw i usług przed atakami na dużą skalę.

## <a name="azure-cdn-from-verizon"></a>Usługa Azure CDN from Verizon

Usługa Azure CDN from Verizon jest chroniony przez platformę ograniczania ryzyka ataków DDoS własności Verzion firmy. Jest zintegrowana w usłudze Azure CDN from Verizon domyślnie i bez ponoszenia dodatkowych kosztów. Zapewnia podstawową ochronę przed najbardziej typowych często pojawiają się warstwy 7 DNS kwerendy powodzie i warstwy 3 i 4 pomiarowej ataki tego docelowymi punktami końcowymi usługi CDN.
