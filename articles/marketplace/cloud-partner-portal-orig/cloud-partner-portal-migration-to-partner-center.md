# <a name="cloud-partner-portal-migration-to-partner-center"></a>Migracja z witryny Cloud Partner Portal do Centrum partnerskiego

Aby usprawnić Twoje konto i oferować środowisko zarządzania, firma Microsoft migruje oferty z portal Cloud Partner do Centrum partnerskiego. Jeśli obecnie zarządzasz materiałami do współsprzedaży w jednym z partnerów komercyjnych, przejdź do rynku (OCP GTM), będziesz mieć możliwość natychmiastowego zaktualizowania zabezpieczeń współsprzedaży w centrum partnerskim.

| **Typ oferty** | **Zarządzanie ofertą migrowane z portal Cloud Partner do Centrum partnerskiego** | **Możliwość aktualizacji dodatkowych zabezpieczeń w centrum partnerskim** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Oprogramowanie jako usługa (SaaS) |     Dostępny teraz       |   Ogólna dostępność kwietnia 2020      |
| Dynamics 365 dla zaangażowania klienta & PowerApps      |     Dostępny teraz        |   Dostępny teraz      |
| Dynamics 365 for Operations     |     Dostępny teraz       |   Dostępny teraz     |
| Dynamics 365 Business Central      |     Dostępny teraz        |   Ogólna dostępność kwietnia 2020     |
| Azure Application     |     Dostępny teraz       |   Ogólna dostępność kwietnia 2020     |
| Maszyna wirtualna platformy Azure      |     Ogólna dostępność kwietnia — maj 2020       |   Do ogłaszania    |
| Kontener platformy Azure     |     Ogólna dostępność kwietnia — maj 2020       |   Do ogłaszania     |
| Usługa konsultingowa      |     Ogólna dostępność kwietnia — maj 2020       |   Do ogłaszania     |
| Moduł IoT Edge      |     Ogólna dostępność kwietnia — maj 2020       |   Do ogłaszania     |
