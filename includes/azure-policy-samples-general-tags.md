|  |  |
|---------|---------|
| [Apply tag and its default value](../articles/azure-policy/scripts/apply-tag-def-val.md) (Zastosuj tag i jego wartość domyślną) | Dołącza określoną nazwę i wartość tagu, jeśli nie podano tagu. Należy określić nazwę i wartość tagu do zastosowania.  |
| [Billing tags policy initiative](../articles/azure-policy/scripts/billing-tags-policy-init.md) (Inicjatywa zasad tagów rozliczeń) | Wymaga określonych wartości tagów centrum kosztu i nazwy produktu.
