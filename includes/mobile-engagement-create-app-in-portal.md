1. Azure 管理ポータルにログオンし、をクリックして **+ 新規** 、画面の下部にあります。

2. をクリックして **App Services**, 、し **モバイル エンゲージメント**, 、し **作成**します。

    ![](./media/mobile-engagement-create-app-in-portal/create-mobile-engagement-app.png)

3. 表示されたポップアップに、次の情報を入力します。

    ![](./media/mobile-engagement-create-app-in-portal/create-azme-popup.png)

    - **アプリケーション名**: アプリケーションの名前。 
    - **プラットフォーム**: アプリ向けのターゲット プラットフォームです。 モバイル アプリケーションのターゲットとするプラットフォームごとに 1 つの Mobile Engagement アプリを作成する必要があります。 
    - **アプリケーション リソース名**: 名前をこのアプリケーションに Api と Url を通じてアクセスします。 
    - **場所**: このアプリとアプリ コレクションをホストする地域およびデータ センターです。
    - **コレクション**: 以前に作成したコレクションを選択するか [新しいコレクション] を選択します。
    - **コレクション名**: アプリケーションのグループを表します。 すべてのアプリを 1 つのグループに含めることで、それらをメトリック集計できます。 該当する場合は、ここで会社名や部門を使用する必要があります。

4. 作成したアプリを選択、 **アプリケーション** ] タブをクリックします。

5. をクリックして **接続情報]** にモバイル アプリに統合する SDK に組み込む接続設定を表示します。

6. コピー、 **接続文字列** -これは、アプリケーション コード内には、このアプリケーションを識別し、アプリからモバイル エンゲージメント サービスに接続する必要があります。

    ![](./media/mobile-engagement-create-app-in-portal/app-connection-info-page.png)
