# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool Update Release 7.1.60702.1 — 7/2/2019

Wersja 7.1.60702.1 Microsoft Threat Modeling Tool (TMT) została wydana w lipcu 2 2019 i zawiera następujące zmiany:

- Ulepszenia ułatwień dostępu
- Poprawki błędów
- Dołączanie zestawu wzorników typu "open source" w społeczności

## <a name="feature-changes"></a>Zmiany funkcji

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Dostępny jest nowy zestaw wzorników urządzeń medycznych udostępniany przez społeczność typu "open source"

Wzornik zestawu na potrzeby modelowania urządzeń medycznych został wniesiony przez społeczność typu open source. Po uaktualnieniu nowy zestaw wzornika pojawi się w menu rozwijanym wybór szablonu. Aby uzyskać informacje o współdziałaniu wzorników lub zawartości z szablonami, zapoznaj się z informacjami na [stronie witryny GitHub](https://github.com/Microsoft/threat-modeling-templates)projektu.

![Opcja walidacji modelu](./media/azure-security-threat-modeling-tool-releases-71607021/tmt-template-selection.png)

## <a name="system-requirements"></a>Wymagania systemowe

- Obsługiwane systemy operacyjne
  - [Rocznicowa Aktualizacja systemu Microsoft Windows 10](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97) lub nowsza wersja

  - [.NET 4.7.1](http://go.microsoft.com/fwlink/?LinkId=863262) lub nowszy
- Wymagania dodatkowe
  - Do otrzymywania aktualizacji narzędzia oraz szablonów są wymagane połączenia z Internetem.
