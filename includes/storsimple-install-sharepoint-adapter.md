#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Aby zainstalować Adapter usługi StorSimple dla programu SharePoint
1. Skopiuj Instalatora na serwerze sieci web serwer sieci Web (WFE), który również jest skonfigurowane do uruchamiania aplikacji sieci web Administracja centralna programu SharePoint. 
2. Użyj konta z uprawnieniami administratora do logowania się na serwerze WFE.
3. Kliknij dwukrotnie Instalatora. Rozpoczyna się adaptera StorSimple dla Kreatora instalacji programu SharePoint. Kliknij przycisk **dalej** do rozpoczęcia instalacji.
   
    ![Strony początkowej instalacji adaptera StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. W adaptera StorSimple dla strony konfiguracji instalacji programu SharePoint, wybierz miejsce instalacji, wpisz adres IP dla interfejs sieciowy DATA 0 na urządzeniu StorSimple, a następnie kliknij **dalej**. 
   
    ![Strona konfiguracji instalacji adaptera StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Na stronie Potwierdzenie instalacji, kliknij przycisk **zainstalować**.
   
    ![Strona potwierdzenia instalacji adaptera StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kliknij przycisk **Zakończ** aby zamknąć kreatora instalacji.
   
    ![Zakończono strony konfiguracji adaptera StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Otwórz stronę administracji centralnej programu SharePoint. Powinieneś zobaczyć grupy konfiguracji StorSimple, która zawiera adaptera StorSimple dla programu SharePoint, łączy.
8. Przejdź do następnego kroku: [skonfigurować SPZ](#configure-rbs).
