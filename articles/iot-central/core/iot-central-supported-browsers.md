# <a name="supported-browsers-for-azure-iot-central"></a>Obsługiwane przeglądarki dla usługi Azure IoT Central

*Ten artykuł dotyczy operatorów, konstruktorów i administratorów.*

Usługa Azure IoT Central jest dostępna na większości nowoczesnych komputerów stacjonarnych, tabletów i przeglądarek. W poniższym artykule opisano listę obsługiwanych przeglądarek i wymaganą łączność.  

## <a name="supported-browsers"></a>Obsługiwane przeglądarki
Zalecamy używanie najnowszej przeglądarki zgodnej z systemem operacyjnym. Obsługiwane są następujące przeglądarki:
- Microsoft Edge (najnowsza wersja)
- Safari (najnowsza wersja, tylko Mac)
- Chrome (najnowsza wersja)
- Firefox (najnowsza wersja)

## <a name="required-protocols"></a>Wymagane protokoły
Usługa Azure IoT Central wymaga, aby sieć obsługuje protokoły HTTPS i WebSocket dla łączności wychodzącej.
