# Azure サブスクリプションが無効になった場合、どうすればよいですか?

Azure サブスクリプションは、次の理由で無効になることがあります。

### 使用制限に達している

使用制限の詳細については、次を参照してください。 [使用制限](https://azure.microsoft.com/pricing/spending-limits/)します。 この記事では、プランに含まれる月間使用量に達した場合、その請求月のそれ以降の期間、サービスが無効化されることについて説明しています。 さらに使用制限を削除する方法についても説明しています。

### 請求の支払い期限を過ぎている

これを解決するには、記事を参照してください [理由が、通知を受信した、Azure サブスクリプションが過去である期日残高?](../billing-azure-subscription-past-due-balance.md)。

### Azure サブスクリプションが取り消されている

サブスクリプションを再アクティブ化するには、サポート チケットを作成する必要があります。 Azure サブスクリプションを再度有効にするチケットをサポートを作成するには、記事を参照して [Azure の課金とサブスクリプションの問題のサポート チケットを作成する方法](../billing-how-to-create-billing-support-ticket.md)します。
