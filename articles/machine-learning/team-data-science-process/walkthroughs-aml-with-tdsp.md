# <a name="team-data-science-structured-projects-in-azure-machine-learning-workbench"></a>Projekty do nauki o danych ze strukturą zespołu w usłudze Azure Machine Learning Workbench

Te samouczki i przykłady przedstawiają sposób tworzenia projektów zespołu do analizy danych ze strukturą za pomocą usługi Azure Machine Learning Workbench. Omówienie procesu do nauki o danych zespołu, zobacz [danych dla celów naukowych](overview.md). Aby zapoznać się z omówieniem Spark on HDInsight, zobacz [wprowadzenie do platformy Spark w HDInsight](../../hdinsight/spark/apache-spark-overview.md).

Przewodniki dotyczące przetwarzania dodatkowe dane, które są wykonywane w procesie nauki o danych zespołu są pogrupowane według **platformy** używane. Zobacz [przewodniki dotyczące wykonywania procesu do nauki o danych zespołu](walkthroughs.md) do podziału na pozycje z tych przykładów.
