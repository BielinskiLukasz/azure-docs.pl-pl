* Aktywne konto platformy Azure. (Jeśli nie masz konta, możesz utworzyć [bezpłatne konto](https://azure.microsoft.com/pricing/free-trial/) w ciągu zaledwie kilku minut).

* Zaleca się [stosowanie języka Python w wersji 3.7 lub nowszej.](https://www.python.org/downloads/) Upewnij się, że używasz 32-bitowej lub 64-bitowej instalacji zgodnie z wymaganiami konfiguracji. Po wyświetleniu monitu podczas instalacji upewnij się, że język Python został dodany do zmiennej środowiskowej specyficznej dla platformy. W przypadku innych obsługiwanych wersji języka Python zobacz [funkcje urządzenia Usługi Azure IoT](https://github.com/Azure/azure-iot-sdk-python/tree/master/azure-iot-device#azure-iot-device-features) w dokumentacji zestawu SDK.

    > [!IMPORTANT]
    > Ponieważ kod urządzenia w tym artykule używa asynchroniowego interfejsu API, nie można użyć języka Python 2.7.
