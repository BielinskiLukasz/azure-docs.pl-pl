Aby włączyć dzienniki przesyłania strumieniowego dla aplikacji funkcji na platformie Azure:

1. Wybierz klawisz F1, aby otworzyć paletę poleceń, a następnie wyszukaj i uruchom polecenie **Azure Functions: Start Streaming Logs**.

1. Wybierz aplikację funkcji na platformie Azure, a następnie wybierz **tak,** aby włączyć rejestrowanie aplikacji dla aplikacji funkcji.

1. Wyzwalanie funkcji na platformie Azure. Należy zauważyć, że dane dziennika są wyświetlane w oknie dane wyjściowe w programie Visual Studio Code.

1. Po zakończeniu pamiętaj, aby uruchomić polecenie **Usługi Azure Functions: Zatrzymaj dzienniki przesyłania strumieniowego,** aby wyłączyć rejestrowanie dla aplikacji funkcji.
