# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion-preview"></a>Zmień na widok pełnego ekranu dla sesji maszyny wirtualnej: Bastionu platformy Azure (wersja zapoznawcza)

Przed rozpoczęciem pracy z maszyną Wirtualną, upewnij się, że zostały wykonane kroki, aby [tworzenie hostem bastionu](bastion-create-host-portal.md). Następnie należy połączyć z maszyną wirtualną, którą chcesz pracować przy użyciu [RDP](bastion-connect-vm-rdp.md) lub [SSH](bastion-connect-vm-ssh.md).

## <a name="launch-the-clipboard-tool"></a>Uruchom narzędzie do Schowka

Podczas sesji zdalnej uruchomienie paletę Narzędzie dostępu do Schowka bastionu, wybierając dwie strzałki, które znajdują się w lewej na środku sesji.

![narzędzia](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>Wybierz pełny ekran

Wybierz **Fullscreen** przycisk, aby zmienić sesji środowiska pełnego ekranu. Po przełączeniu, sesja zostanie zainicjowana ponownie do pełnego ekranu.

![Pełny ekran](./media/bastion-vm-manage/full-screen.png)
