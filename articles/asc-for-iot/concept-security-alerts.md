# <a name="azure-security-center-for-iot-security-alerts"></a>Azure Security Center for IoT security alerts (Alerty zabezpieczeń usługi Azure Security Center dla IoT)

Usługa Azure Security Center for IoT stale analizuje rozwiązanie IoT przy użyciu zaawansowanej analizy i analizy zagrożeń, aby ostrzegać o złośliwej aktywności.
Ponadto można tworzyć niestandardowe alerty na podstawie wiedzy o oczekiwanym zachowaniu urządzenia.
Alert działa jako wskaźnik potencjalnego kompromisu i powinien zostać zbadany i naprawiony.

W tym artykule znajdziesz listę wbudowanych alertów, które mogą być wyzwalane na urządzeniach Usługi IoT Hub i/lub IoT.
Oprócz wbudowanych alertów usługa Azure Security Center for IoT umożliwia definiowanie niestandardowych alertów na podstawie oczekiwanego zachowania usługi IoT Hub i/lub urządzenia.
Aby uzyskać więcej informacji, zobacz [konfigurowalne alerty](concept-customizable-security-alerts.md).
