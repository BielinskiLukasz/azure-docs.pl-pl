# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Obsługiwane przeglądarki i urządzenia dla witryny Azure portal
Możesz uruchomić [witryny Azure portal](https://portal.azure.com) na wszystkie komputery stacjonarne nowoczesne, tablet urządzenia i przeglądarki.

### <a name="supported-devices"></a>Obsługiwane urządzenia
Azure portal działa doskonale na nowoczesnych PC, Mac i tabletów. Jeśli musisz zarządzać zasobami platformy Azure z urządzenia przenośnego, spróbuj aplikacja mobilna platformy Azure, która jest dostępna na urządzeniach iPhone i Android. Aby uzyskać więcej informacji, zobacz [aplikacja mobilna platformy Azure](https://azure.microsoft.com/features/azure-portal/mobile-app/).

### <a name="supported-browsers"></a>Obsługiwane przeglądarki
Zaleca się, że używasz najbardziej aktualne przeglądarki, która jest zgodna z systemem operacyjnym. Obsługiwane są następujące przeglądarki:

* Microsoft Edge (Najnowsza wersja)
* Internet Explorer 11
* Safari (Najnowsza wersja, tylko Mac)
* Chrome (Najnowsza wersja)
* Firefox (Najnowsza wersja)
