

## <a name="overview-of-azure-resource-manager-templates"></a>Omówienie szablonów usługi Azure Resource Manager
Szablony usługi Azure Resource Manager umożliwiają deklaratywne określenie infrastruktury IaaS platformy Azure w języku JSON przez zdefiniowanie zależności między źródłami.

Zobacz [Tworzenie szablonów rozszerzeń](../articles/virtual-machines/windows/template-description.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json), aby uzyskać więcej informacji na temat tworzenia szablonów do używania rozszerzeń.

W tym artykule omówiono rozwiązywanie problemów związanych z typowymi błędami rozszerzeń maszyn wirtualnych.
