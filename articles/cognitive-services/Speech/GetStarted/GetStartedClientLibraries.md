# <a name="get-started-with-speech-service-client-libraries"></a>Rozpoczynanie pracy z biblioteki klienta usługi mowy

Oprócz tworzenia bezpośrednie żądania HTTP za pośrednictwem interfejsu API REST, mowy usługi oferuje deweloperom biblioteki klienta mowy w różnych językach. Mowy biblioteki klienta:

- Obsługuje bardziej zaawansowanych funkcji Rozpoznawanie mowy, takich jak wyników pośrednich w czasie rzeczywistym, strumieniem audio czasu (do 10 minut) i ciągłe rozpoznawania.
- Zapewniają prosty i idiomatyczne interfejsu API w języku swoich preferencji.
- Ukryj szczegóły komunikacji niskiego poziomu.

Obecnie dostępne są następujące biblioteki klienta mowy:

- [Biblioteki pulpitu C#](GetStartedCSharpDesktop.md)
- [Biblioteka usługi C#](GetStartedCSharpServiceLibrary.md)
- [Biblioteka języka JavaScript](GetStartedJSWebsockets.md)
- [Biblioteka języka Java dla systemu Android](GetStartedJavaAndroid.md)
- [Biblioteka języka Objective C dla systemu iOS](Get-Started-ObjectiveC-iOS.md)

## <a name="additional-resources"></a>Zasoby dodatkowe

- [Przykłady](../samples.md) strona zawiera pełną próbek do użycia bibliotek klienckich mowy.
- Biblioteka klienta, który nie jest jeszcze dostępna, należy można utworzyć własny zestaw SDK. Implementowanie [protokół WebSocket mowy](../API-Reference-REST/websocketprotocol.md) platformy i stosowania wybranym języku.
