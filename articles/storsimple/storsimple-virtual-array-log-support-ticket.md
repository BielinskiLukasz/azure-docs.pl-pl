# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Rejestrowanie żądania pomocy technicznej dla rozwiązania StorSimple Virtual Array przy użyciu usługi Menedżer urządzeń StorSimple

## <a name="overview"></a>Omówienie

Menedżer urządzeń StorSimple zapewnia możliwość **rejestrować nowe żądanie pomocy technicznej** w bloku podsumowania usługi. W tym artykule wyjaśniono, jak rejestrować nowe żądanie pomocy technicznej i zarządzać jego cyklem życia z portalu.

## <a name="new-support-request"></a>Nowe żądanie pomocy technicznej

W zależności od Twojego [plan pomocy technicznej](https://azure.microsoft.com/support/plans/), tworzenia biletów pomocy technicznej dotyczącej problemu na macierz wirtualna StorSimple, bezpośrednio z poziomu bloku podsumowania usługi Menedżer urządzeń StorSimple.

#### <a name="to-log-a-new-request"></a>Aby rejestrować nowe żądanie

1. Przejdź do usługi Menedżer urządzeń StorSimple. W ustawieniach bloku podsumowania usługi, przejdź do **pomoc techniczna i rozwiązywanie problemów** sekcji, a następnie kliknij przycisk **nowe żądanie obsługi**.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. W **podstawy** blok, wykonaj następujące czynności:

    1. Z **typ problemu** listy rozwijanej wybierz **techniczne**. 
    
    2. Bieżący **subskrypcji**, **usługi** typu i **zasobów** (usługa Menedżer urządzeń StorSimple) są wybierane automatycznie. 

    3. Określ co najmniej jedno urządzenie zarejestrowane z usługą występują problemy.

    4. Wybierz odpowiednią **plan pomocy technicznej** Jeśli masz wiele planów skojarzonych z Twoją subskrypcją. Musisz mieć plan płatnej pomocy technicznej, umożliwiające pomocy technicznej.

3. W **kroku 2**, wybierz **ważność** i określ, czy problem jest związane tablicy lub usługi Menedżer urządzeń StorSimple. Ponadto należy wybrać **kategorii** tego wydania i zapewnić bardziej **szczegóły** o problemie.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. W **kroku 3**, podaj swoje informacje kontaktowe. Informacje te są wykorzystywane do skontaktować się z Tobą dodatkowe informacje, diagnozowania i rozwiązywania Microsoft Support.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Zarządzaj żądaniami obsługi

Po utworzeniu biletu pomocy technicznej możesz zarządzać jego cyklem życia z poziomu portalu.

#### <a name="to-manage-your-support-requests"></a>Aby zarządzać żądaniami pomocy technicznej

Aby uzyskać dostęp do strony pomocy i obsługi technicznej, przejdź do **Przeglądaj > Pomoc i obsługa techniczna**.

![Zarządzaj żądaniami obsługi](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
