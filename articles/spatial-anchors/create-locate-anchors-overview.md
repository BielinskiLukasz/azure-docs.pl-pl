# <a name="create-and-locate-anchors-using-azure-spatial-anchors"></a>Tworzenie i znajdowanie kotwic przy użyciu usługi Azure Spatial Anchors

Usługa Azure Spatial Anchors umożliwia udostępnianie kotwic na całym świecie między różnymi urządzeniami.

Została ona zaprojektowana do bezproblemowego współdziałania z dowolnym wybranym przez Ciebie środowiskiem deweloperskim.
