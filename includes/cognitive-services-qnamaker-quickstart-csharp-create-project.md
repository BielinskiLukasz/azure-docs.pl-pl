1. Otwórz program Visual Studio 2017 Community Edition.
1. Utwórz nowy projekt **Aplikacja konsoli (.NET Core)** i nadaj mu nazwę `QnaMakerQuickstart`. Zaakceptuj wartości domyślne pozostałych ustawień.
1. W Eksploratorze rozwiązań kliknij prawym przyciskiem myszy nazwę projektu **QnaMakerQuickstart**, a następnie wybierz pozycję **Zarządzaj pakietami NuGet...**.
1. W oknie NuGet wybierz pozycję **Przeglądarka**, a następnie wyszukaj pakiet **Newtonsoft.JSON** i zainstaluj go. Ten pakiet służy do analizowania danych JSON zwróconych z odpowiedzi HTTP interfejsu QnaMaker.
