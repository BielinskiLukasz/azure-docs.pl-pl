> Ten artykuł zawiera zawartość, która wymaga klasycznego wiersza polecenia platformy Azure. Bieżąca wersja interfejsu wiersza polecenia platformy Azure nie ma obsługi funkcji opisanych w tym artykule, więc klasyczny interfejs wiersza polecenia jest wymagane.

> Aby zainstalować klasyczny interfejs wiersza polecenia, zobacz [zainstalować Azure klasyczny interfejs wiersza polecenia](/cli/azure/install-classic-cli).
