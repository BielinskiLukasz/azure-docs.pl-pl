#Windows Phone Silverlight SDK リリース ノート

-   初期化直後にデバイス ID を要求すると発生するクラッシュを修正しました。
-   コンソールのログを改善しました。

-   既定の通知アイコンが更新されました。
-   通知がクリックされると、システム通知アクション フィードバックが返送されます。
-   システム通知の競合条件を修正しました。

-   Azure Mobile Engagement の最初のリリース。
-   appId/sdkKey 構成は、接続文字列の構成で置き換えられます。
-   セキュリティの強化。
