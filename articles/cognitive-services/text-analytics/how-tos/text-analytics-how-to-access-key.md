# <a name="how-to-find-endpoints-and-access-keys-for-the-text-analytics-cognitive-service"></a>Jak znaleźć punktów końcowych i dostępu do kluczy usługi Cognitive analizy tekstu

Po zarejestrowaniu się analizy tekstu, otrzymasz klucz dostępu do spersonalizowanych unikatowy do Twojej subskrypcji. Ten klucz jest wymagany przy każdym wywołaniu interfejsu API analizy tekstu. Jeśli jeszcze tego nie [zarejestrowali](text-analytics-how-to-signup.md), zrób to teraz pobierania klucza. 

Jeśli potrzebujesz pomocy, znajdowania klucza, lub aby określić, czy subskrypcja obejmuje już analizy tekstu, użyj poniższych instrukcji, aby uzyskać niezbędne informacje. 

## <a name="find-your-service-endpoint-and-access-key"></a>Znajdź swoje usługi punktu końcowego i klucza dostępu

1. Zaloguj się w [portalu Azure](https://portal.azure.com).

2. W okienku nawigacji po lewej stronie wybierz **wszystkich usług**.

3. W polu filtru wpisz *usług Cognitive Services*. Jeśli Twoja subskrypcja ma analizy tekstu, pojawia się na liście, korzystając z interfejsu API typu **interfejsu API analizy tekstu**.

4. Kliknij link, aby otworzyć blok usługi. Możesz teraz uzyskać klucz z **zasobów > klucze** lub przez kliknięcie przycisku **Pokaż klucze dostępu** w okienku podstawy. Punkt końcowy jest również widoczne w okienku podstawy.
