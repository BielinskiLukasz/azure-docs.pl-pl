Następujące limity mają zastosowanie tylko podczas wykonywania operacji zarządzania przy użyciu usługi Azure Resource Manager z usługą Azure Storage.

| Zasób | Limit |
| --- | --- |
| Operacje zarządzania kontem magazynu (odczyt) |800 na 5 minut |
| Operacje zarządzania kontem magazynu (zapis) |1200 na godzinę |
| Operacje zarządzania kontem magazynu (lista) |100 na 5 minut |
