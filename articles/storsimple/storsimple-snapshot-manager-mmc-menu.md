# StorSimple Snapshot Manager での MMC メニュー操作の使用

## 概要

StorSimple Snapshot Manager では、すべての操作メニューとのすべてのバリエーションに表示される次の操作が表示されます、 **アクション** ウィンドウです。 

- 表示
- ここから新しいウィンドウ 
- 更新 
- リストのエクスポート 
- ヘルプ 

これらの操作は Microsoft 管理コンソール (MMC) の機能であり、StorSimple Snapshot Manager に固有のものではありません。

このチュートリアルでは、これらの操作について説明し、StorSimple Snapshot Manager での各操作の使用方法を示します。

## 表示

使用することができます、 **ビュー** を変更するオプション、 **結果** ] ウィンドウの表示し、コンソール ウィンドウの表示を変更します。 

#### 結果ウィンドウの表示を変更するには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。

2.  **スコープ** ] ウィンドウで、任意のノードを右クリックし、またはノードを展開し、内の項目を右クリックし、 **結果** ] ウィンドウで、クリックして、 **ビュー** オプション。 

3. 表示される列を追加または削除、 **結果** ] ウィンドウで、をクリックして **列の追加/削除**します。  **列の追加/削除** ] ダイアログ ボックスが表示されます。

    ![結果ウィンドウの列の追加または削除](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Add_remove_columns.png) 

4. 次のようにフォームの操作を実行します。

    - 項目を選択、 **利用可能な** 列を一覧表示し、 **追加** に追加する、 **表示されている列** ] ボックスの一覧です。 

    - 項目をクリックして、 **表示されている列** 一覧を開き、をクリックして **削除** には、一覧から削除します。 

    - [項目を選択、 **表示** 列を一覧表示し、 **上へ移動** または **下へ移動** リストに項目を上または下に移動します。 

    - クリックして **既定値に戻す** を既定値を返す **結果** ウィンドウの設定です。 

5. クリックして、選択内容がしたら **OK**します。 

#### コンソール ウィンドウの表示を変更するには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。

2.  **スコープ** ] ウィンドウで任意のノードを右クリックし、をクリックして **ビュー**, 、クリックして **カスタマイズ**します。  **カスタマイズ** ] ダイアログ ボックスが表示されます。

    ![コンソール ウィンドウのカスタマイズ](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Customize.png) 

3. チェック ボックスをオンまたはオフにして、コンソール ウィンドウの項目の表示/非表示を切り替えます。 クリックして、選択内容がしたら **OK**します。

## ここから新しいウィンドウ

使用することができます、 **ここから新しいウィンドウ** 新しいコンソール ウィンドウを開くにはオプションです。

#### 新しいコンソール ウィンドウを開くには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。

2.  **スコープ** ] ウィンドウで、任意のノードを右クリックし、クリックして **ここから新しいウィンドウ**です。 

    新しいウィンドウが開き、選択したスコープだけが表示されます。 右クリックした場合など、 **バックアップ ポリシー** ノード、新しいウィンドウのみが表示されます、 **バックアップ ポリシー** 内のノード、 **スコープ** ウィンドウとの一覧はバックアップ ポリシーで定義されている、 **結果** ウィンドウです。 次の例を参照してください。

    ![ここから新しいウィンドウ](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_NewWindow.png) 
 
## 更新

使用することができます、 **更新** コンソール ウィンドウを更新する操作。

#### コンソール ウィンドウを更新するには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。

2.  **スコープ** ] ウィンドウで、任意のノードを右クリックし、またはノードを展開し、内の項目を右クリックし、 **結果** ] ウィンドウで、クリックして **更新**します。 

## リストのエクスポート

使用することができます、 **リストのエクスポート** 一覧をコンマ区切り値 (CSV) ファイルに保存するアクション。 たとえば、バックアップ ポリシーのリストやバックアップ カタログをエクスポートできます。 その後、CSV ファイルをスプレッドシート アプリケーションにインポートして分析することができます。

#### リストをコンマ区切り値 (CSV) ファイルに保存するには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。 

2.  **スコープ** ] ウィンドウで、任意のノードを右クリックし、またはノードを展開し、内の項目を右クリックし、 **結果** ] ウィンドウで、クリックして **リストのエクスポート**します。 

3.  **リストのエクスポート** ] ダイアログ ボックスが表示されます。 次のようにフォームの操作を実行します。 

    1.  **ファイル名** ボックス、CSV ファイルの名前を入力するか、ドロップ ダウン リストから選択する矢印をクリックします。

    2.  **ファイルの種類** ボックスで、矢印をクリックをドロップダウン リストからファイルの種類を選択します。

    3. 選択した項目だけを保存する行を選択し、をクリックして、 **選択した行のみ保存** チェック ボックスをオンします。 エクスポートされたすべてのリストを保存するには、オフ、 **選択した行のみ保存** チェック ボックスをオンします。

    4. クリックして **保存**します。

    ![リストをコンマ区切り値ファイルとしてエクスポート](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Export_List.png) 
 
## ヘルプ

使用することができます、 **ヘルプ** ] メニューの [StorSimple Snapshot Manager と MMC のオンライン ヘルプを表示します。

#### 利用可能なオンライン ヘルプを表示するには

1. デスクトップ アイコンをクリックして、StorSimple Snapshot Manager を起動します。

2.  **スコープ** ] ウィンドウで、任意のノードを右クリックし、またはノードを展開し、内の項目を右クリックし、 **結果** ] ウィンドウで、クリックして **ヘルプ**します。
