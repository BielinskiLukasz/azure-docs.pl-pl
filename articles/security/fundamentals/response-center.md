# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

Centrum Microsoft Security Response Center (MSRC) jest prowadzone przez niektórych najbardziej doświadczonych specjalistów ds. bezpieczeństwa. Identyfikują one, monitorują, reagują i rozwiązują zdarzenia związane z bezpieczeństwem oraz luki w zabezpieczeniach lokalnych i chmurowych dotyczące zegara, każdego dnia roku.

Oprócz jego ciągłego działania zespół MSRC ma wiele zasobów dostępnych dla Ciebie, dzięki czemu można zrozumieć, jak skutecznie zabezpieczać zasoby i wdrożenia platformy Azure.

## <a name="the-msrc-blog"></a>Blog MSRC

[Blog centrum MSRC](https://blogs.technet.microsoft.com/msrc/) jest miejscem, w którym można znaleźć najnowsze informacje na temat tego, co robi centrum MSRC, aby pomóc w ochronie przed zagrożeniami w chmurze.

## <a name="white-papers"></a>Oficjalne dokumenty

Centrum MSRC opublikowało wiele oficjalnych [dokumentów](https://technet.microsoft.com/library/bb969102.aspx) , które ułatwiają zrozumienie ich działania i sposobu ich działania. Niektóre zapewniają wgląd w sposób zabezpieczania chmury firmy Microsoft i zawierają przydatne informacje dotyczące korzystania z tych samych konfiguracji zabezpieczeń.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Zaangażowanie i Bounty programów do przeszukiwania zabezpieczeń

Centrum MSRC obsługuje współpracę i relacje z firmą Microsoft Security badaczs globalnie, aby uzyskać korzyści z używania zabezpieczeń produktów.

Programy Microsoft bountye zwracają naukowców dla:

- Nowe techniki wykorzystujące luki w zabezpieczeniach
- Pomysły obronne, które eliminują nowe luki w zabezpieczeniach
- Identyfikowanie krytycznych luk w zabezpieczeniach lokalnych i w chmurze firmy Microsoft.
