## <a name="prerequisites"></a>Wymagania wstępne

* Korzystanie z subskrypcji platformy Azure. Jeśli nie masz subskrypcji platformy Azure, przed rozpoczęciem utwórz [bezpłatne konto][free-account].
* Zainstaluj [wiersza polecenia platformy Azure][azure-cli] (az).
* [Aparat platformy docker][docker-engine] i Zweryfikuj, że interfejs wiersza polecenia platformy Docker działa w oknie konsoli.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
