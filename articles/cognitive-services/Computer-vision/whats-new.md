## <a name="january-2020"></a>Styczeń 2020 r.

### <a name="read-api-30-preview"></a>Odczyt wersji zapoznawczej interfejsu API 3.0

Teraz można użyć wersji 3.0 interfejsu API odczytu do wyodrębniania drukowanego lub odręcznego tekstu z obrazów. W porównaniu z wcześniejszymi wersjami, 3.0 zapewnia:
* zwiększona dokładność
* nowy format wyjściowy
* wynik zaufania dla każdego wyodrębnionego słowa
* obsługa zarówno języka hiszpańskiego, jak i angielskiego z dodatkowym parametrem językowym

Postępuj zgodnie [z wyodrębniaj tekst szybki start,](https://docs.microsoft.com/azure/cognitive-services/computer-vision/quickstarts/csharp-hand-text?tabs=version-3) aby rozpocząć korzystanie z interfejsu API 3.0.
