# <a name="explore-data-in-the-team-data-science-process"></a>Eksplorowanie danych w zespołowym przetwarzaniu danych dla celów naukowych
W tym dokumencie opisano sposób eksplorować dane w czterech środowiskach innego magazynu, które są zazwyczaj używane w procesie nauki danych:

* **Kontener obiektów blob platformy Azure** danych jest przedstawione za pomocą [Pandas](http://pandas.pydata.org/) pakiet języka Python.
* **SQL Server** danych jest przedstawione przy użyciu języka SQL i przy użyciu języka programowania, takich jak Python.
* **Hive tabeli** danych jest przedstawione za pomocą zapytań Hive.
* **Azure Machine Learning (AML) Studio** danych jest przedstawione za pomocą modułów AML.

Następujące **menu** linki do tematów opisujących sposób używania tych narzędzi, aby eksplorować dane w różnych środowiskach magazynu.
