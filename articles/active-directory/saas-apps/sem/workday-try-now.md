# <a name="workday-and-azure-active-directory"></a>Workday i Azure Active Directory

Połącz usługę Workday z usługą Azure Active Directory (Azure AD) przy użyciu wstępnie skompilowanej integracji i automatycznie zastrzegaj lub Anuluj obsługę kont użytkowników, gdy pracownicy przyłączą, przenieśą lub opuszczają.

![Pokazuje logo aplikacji Workday](./media/workday.png)

> [!div class="nextstepaction"]
> [Wypróbuj usługę Workday z usługą Azure AD >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Co możesz zrobić za pomocą produktu Workday i usługi Azure AD?

- **Automatyzowanie aprowizacji użytkowników** — automatyczne przypisywanie aplikacji do pracowników na podstawie ich grupy, roli lub innych atrybutów w dniu roboczym.
- **Synchronizuj profile pracowników** — Zapewnij aktualność danych pracownika między usługą Azure AD i dniem Workday.
- Uprość dołączanie **i** odłączanie — automatycznie udostępniaj konta użytkowników dla nowych pracowników, dzięki czemu mogą one pracować wydajnie w jednym dniu i zmniejszać zagrożenie bezpieczeństwa przez Automatyczne dezaktywowanie kont użytkowników, gdy pracownicy opuściją.
- **Łatwe logowanie** jednokrotne Eliminowanie nazw użytkowników i haseł oraz Zezwalanie użytkownikom na logowanie się przy użyciu konta organizacyjnego hostowanego w usłudze Azure AD.

## <a name="get-started"></a>Wprowadzenie

Skonfiguruj [Inicjowanie obsługi użytkowników](https://aka.ms/WorkdayProvisioning) i [Logowanie](https://aka.ms/WorkdaySSODoc) jednokrotne dla produktu Workday przy użyciu naszych przewodników związanych z prostymi konfiguracjami.
