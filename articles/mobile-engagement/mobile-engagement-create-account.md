# Azure Mobile Engagement アカウントを作成してアプリの管理を開始する

この記事は、使用する方法を示しています、 **簡易作成** を新規に作成メソッド **Azure モバイル エンゲージメント** アカウントです。 移動する方法も説明、 **モバイル エンゲージメント** ポータルの監視と、アプリの管理を開始するためにします。 

アプリのデータを収集してプッシュ通知を送信できるようにするには、最小限のセットである "基本的な統合" を追加する必要があります。 完全な統合のドキュメントについては記載されて、 [モバイル エンゲージメント統合](mobile-engagement-windows-store-integrate-engagement.md)します。

> [AZURE.IMPORTANT] 任意の Azure モバイル エンゲージメント チュートリアルを完了するには、アクティブな Azure アカウントが必要です。 アカウントがない場合は、無料試用版のアカウントを数分で作成することができます。 詳細については、「<a href="http://azure.microsoft.com/pricing/free-trial/?WT.mc_id=A0E0E5C02&amp;returnurl=http%3A%2F%2Fwww.windowsazure.com%2Fen-us%2Fdevelop%2Fmobile%2Ftutorials%2Fget-started%2F" target="_blank">Azure の無料試用版サイト</a>」を参照してください。

##Mobile Engagement ポータルに移動します

クリックして、モバイル エンゲージメント ポータルに移動しますの監視と管理、アプリケーションを開始する、 **の関心を引く** 下部にあるボタンをクリックします。

![](../../includes/media/mobile-engagement-connect-app-with-monitor/engage-button.png)

Analiyze をすることができますが表示されたら、モバイル エンゲージメント ポータルで、作成し、セグメントを管理、ユーザーなどに接続します。。
