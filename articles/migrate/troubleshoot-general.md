# <a name="troubleshoot-azure-migrate"></a>Rozwiązywanie problemów z usługą Azure Migrate

[Usługa Azure Migrate](migrate-services-overview.md) udostępnia centrum narzędzi do oceny i migracji, a także oferty niezależnych dostawców oprogramowania innych firm (ISV). Ten artykuł ułatwia rozwiązywanie problemów z migracją platformy Azure, oceną serwera migracji do usługi Azure i migracją serwera migracji do serwera migracji z programem Azure Migrate Server.

## <a name="how-do-i-create-or-find-a-project"></a>Jak utworzyć lub znaleźć projekt?

Zapoznaj się z [przewodnikiem rozwiązywania problemów z projektem migracji platformy Azure](troubleshoot-project.md).

## <a name="i-cant-get-the-appliance-working"></a>Nie mogę, aby urządzenie działało

Przejrzyj [odpowiedzi na typowe problemy](troubleshoot-appliance-discovery.md) z wdrażaniem urządzeń.

## <a name="machines-arent-discovered"></a>Maszyny nie są odnalezione

Przejrzyj typowe problemy z odnajdowaniem.

## <a name="app-discovery-isnt-working"></a>Odnajdowanie aplikacji nie działa

Odnajdowanie aplikacji, ról i funkcji działających na komputerach lokalnych jest obecnie obsługiwane tylko dla maszyn wirtualnych VMware. [Przejrzyj typowe błędy](troubleshoot-appliance-discovery.md#common-app-discovery-errors) dotyczące odnajdowania aplikacji.

## <a name="assessment-isnt-working"></a>Ocena nie działa

[Przejrzyj](troubleshoot-assessment.md) typowe problemy z oceną i błędy.
