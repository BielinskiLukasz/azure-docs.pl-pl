# <a name="sync-service-manager-metaverse-search"></a>Wyszukiwanie magazynu Metaverse Menedżera usługi synchronizacji

![Menedżera usługi synchronizacji](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Karta Wyszukiwanie magazynu metaverse jest przydatne podczas rozwiązywania problemów związanych z danymi. W górnej połowie można utworzyć kwerendy oparte na kombinacji atrybutów. Gdy jesteś zadowolony z zapytania, kliknij przycisk **wyszukiwania**. Wynik jest widocznych w siatce dolnej. Można wybrać kolumny, które powinny być widoczne z **ustawienia kolumny**.

W wynikach wyszukiwania, wybierz obiekt i **właściwości** się [właściwości obiektu metaverse](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
