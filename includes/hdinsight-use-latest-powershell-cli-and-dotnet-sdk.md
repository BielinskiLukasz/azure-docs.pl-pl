> [!IMPORTANT]
> Pomoc dla zarządzania zasobami HDInsight przy użyciu usługi Azure Service Manager (ASM) przy użyciu programu Azure PowerShell, interfejsu wiersza polecenia platformy Azure i zestawu .NET SDK HDInsight **przestarzałe**i została usunięta z dniem 1 stycznia 2017 r. Kroki opisane w tym dokumencie użycia nowych poleceń cmdlet programu Azure PowerShell, poleceń interfejsu wiersza polecenia platformy Azure i zestawu .NET SDK, które działają z usługą Azure Resource Manager.
> 
> Wykonaj poniższe kroki w [Instalowanie i Konfigurowanie interfejsu wiersza polecenia platformy Azure](/azure/cli/install-azure-cli) i [Instalowanie i konfigurowanie programu Azure PowerShell](/powershell/azureps-cmdlets-docs) zainstalować najnowszą wersję programu Azure PowerShell i wiersza polecenia platformy Azure. Aby zainstalować najnowszą wersję zestawu SDK dla platformy .NET usługi HDInsight, użyj informacji podanych w [zestawu .NET SDK usługi Microsoft Azure HDInsight](https://www.nuget.org/packages/Microsoft.WindowsAzure.Management.HDInsight/) stronę pakietu NuGet. Jeśli masz istniejące skrypty lub aplikacje, które muszą zostać zmodyfikowane w celu użycia nowych poleceń cmdlet, nowe polecenia interfejsu wiersza polecenia lub zestawu .NET SDK, zapoznaj się z [Migrowanie do narzędzi programistycznych opartych na usłudze Azure Resource Manager w celu obsługi klastrów HDInsight](../articles/hdinsight/hdinsight-hadoop-development-using-azure-resource-manager.md) Aby uzyskać więcej informacji.
