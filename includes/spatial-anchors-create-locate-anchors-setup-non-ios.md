## <a name="set-up-the-session"></a>Konfigurowanie sesji

Wywołaj Start() umożliwiające sesji do przetwarzania danych środowiska.

Aby obsługa zdarzeń wywołanych przez sesję, należy dołączyć program obsługi zdarzeń.
