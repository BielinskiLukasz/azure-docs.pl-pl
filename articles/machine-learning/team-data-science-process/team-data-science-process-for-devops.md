# <a name="team-data-science-process-for-developer-operations"></a>Proces nauki o danych zespołu dla operacji programistycznych

W tym artykule omówiono funkcje operacji deweloperskich (DevOps), które są specyficzne dla implementacji rozwiązania Zaawansowane analizy i usług Cognitive Services. Te materiały szkoleniowe implementują proces nauki o danych zespołu (TDSP) i microsoft oraz oprogramowanie i zestawy narzędzi typu open source, przydatne do przewidywania, wykonywania i dostarczania rozwiązań do nauki o danych. Odwołuje się do tematów, które obejmują DevOps Toolchain, który jest specyficzny dla nauki o danych i projektów i rozwiązań AI.
