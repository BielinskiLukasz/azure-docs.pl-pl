# <a name="additional-documentation"></a>Dodatkowa dokumentacja 

Niektóre zasoby spoza tej docs.microsoft.com witryny sieci Web mogą pomóc w zrozumieniu hybrydowej pamięci podręcznej Microsoft Azure FXT Edge Filer i pracy z nią.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT dla pamięci podręcznej platformy Azure dokumentacji

[Avere vFXT for Azure](https://docs.microsoft.com/azure/avere-vfxt/) to podobny produkt pamięci podręcznej klastra, który używa węzłów wirtualnych na platformie Microsoft Azure. Jest tworzony za pomocą witryny Azure portal, ale używa tego samego oprogramowania konfiguracji panelu sterowania, co pamięć podręczna sprzętu Azure FXT Edge Filer.

* [Przewodnik konfiguracji klastra](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) — zawiera informacje ogólne dotyczące typowych zadań konfiguracji klastra i zarządzania oraz szczegółowe objaśnienia stron ustawień klastra 

* [Przewodnik pulpitu nawigacyjnego](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) — wyjaśnia funkcje karty **Pulpitu nawigacyjnego** panelu sterowania

* [Przewodnik po tworzeniu klastrów FXT](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) — przewodnik po tworzeniu klastrów z poprzednich produktów
