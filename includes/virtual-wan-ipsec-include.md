>[!NOTE]
>Podczas pracy z domyślnymi zasadami platforma Azure może działać zarówno jako inicjator, jak i obiekt odpowiadający podczas konfiguracji tunelu IPsec. Nie ma obsługi platformy Azure tylko jako obiekt odpowiadający.
>

### <a name="initiator"></a>Inicjator

W poniższych sekcjach przedstawiono obsługiwane kombinacje zasad, gdy platforma Azure jest inicjatorem tunelu.

**Faza 1**

* AES_256, SHA1, DH_GROUP_2
* AES_256, SHA_256, DH_GROUP_2
* AES_128, SHA1, DH_GROUP_2
* AES_128, SHA_256, DH_GROUP_2

**Faza 2**

* GCM_AES_256, GCM_AES_256, PFS_NONE
* AES_256, SHA_1, PFS_NONE
* AES_256, SHA_256, PFS_NONE
* AES_128, SHA_1, PFS_NONE
