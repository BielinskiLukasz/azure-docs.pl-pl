# <a name="reset-password-overview"></a>Omówienie resetowania hasła

>[!Important]
>Ta zawartość jest przeznaczona dla użytkowników. Jeśli jesteś administratorem, więcej informacji na temat konfigurowania i zarządzania środowiskiem usługi Azure Active Directory (Azure AD) możesz znaleźć w [dokumentacji usługi Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

Jeśli nie pamiętasz hasła, dział pomocy technicznej firmy nie przekazała Ci go lub Twoje konto zostało zablokowane, możesz zresetować hasło przy użyciu informacji zabezpieczających i urządzenia przenośnego.

|------|------------|

|[Resetowanie hasła](user-help-reset-password.md)| Opisuje sposób resetowania własnego hasła.|
