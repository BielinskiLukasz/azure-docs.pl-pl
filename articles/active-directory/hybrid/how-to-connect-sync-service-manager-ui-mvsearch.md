# <a name="sync-service-manager-metaverse-search"></a>Synchronizacja Service Manager wyszukiwania Metaverse

![Service Manager synchronizacji](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Karta wyszukiwanie Metaverse jest przydatna do rozwiązywania problemów związanych z danymi. W górnej połowie można utworzyć zapytanie na podstawie kombinacji atrybutów. Gdy zapytanie jest zadowalające, kliknij przycisk **Wyszukaj**. Wynik jest widoczny w dolnej siatce. Można wybrać, które kolumny mają być widoczne dla **ustawień kolumny**.

W wynikach wyszukiwania wybierz obiekt i **Właściwości** , aby wyświetlić [właściwości obiektu metaverse](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
