Kroki i przykłady w tym artykule używają modułów Az programu Azure PowerShell. Aby zainstalować moduły Az lokalnie na komputerze, zobacz [Instalowanie programu Azure PowerShell](/powershell/azure/install-az-ps). Aby dowiedzieć się więcej o nowym module Az, zobacz [Przedstawianie nowego modułu Azure PowerShell Az](/powershell/azure/new-azureps-module-az). Polecenia cmdlet programu PowerShell są często aktualizowane. Jeśli nie używasz najnowszej wersji, wartości określone w instrukcjach mogą zakończyć się niepowodzeniem. Aby znaleźć zainstalowane wersje programu PowerShell w `Get-Module -ListAvailable Az` systemie, użyj polecenia cmdlet.
