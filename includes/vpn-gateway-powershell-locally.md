Można również zainstalować i uruchomić polecenia cmdlet programu Azure PowerShell lokalnie na komputerze. Polecenia cmdlet programu PowerShell są często aktualizowane. Jeśli nie zainstalowano najnowszej wersji, wartości określone w instrukcjach mogą zakończyć się niepowodzeniem. Aby znaleźć wersje programu Azure PowerShell zainstalowane `Get-Module -ListAvailable Az` na komputerze, użyj polecenia cmdlet. Aby zainstalować lub zaktualizować, zobacz [Instalowanie modułu programu Azure PowerShell](/powershell/azure/install-az-ps).
