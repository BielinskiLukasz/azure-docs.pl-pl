# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Otwieranie biletu pomocy technicznej dla pamięci podręcznej HPC usługi Azure

Użyj witryny Azure portal, aby otworzyć bilet pomocy technicznej. Przejdź do wystąpienia pamięci podręcznej, a następnie kliknij **łącze Nowe żądanie pomocy technicznej,** które pojawi się u dołu paska bocznego.

Aby otworzyć bilet, gdy nie masz aktywnej pamięci podręcznej, użyj głównej strony pomocy + pomocy technicznej z witryny Azure portal. Otwórz menu portalu z formantu w lewym górnym rogu ekranu, a następnie przewiń w dół i kliknij **przycisk Pomoc + obsługa**.

Wybierz **nowe żądanie pomocy technicznej** i wybierz pozycję **Techniczne,** aby uzyskać pomoc dotyczącą pamięci podręcznej HPC platformy Azure.

Wybierz subskrypcję z listy.

Aby znaleźć usługę Azure HPC Cache, kliknij przycisk **Wszystkie usługi** i wyszukaj hpc.

![Zrzut ekranu przedstawiający żądanie pomocy technicznej — karta Podstawy, częściowo wypełniona zgodnie z opisem](media/hpc-cache-support-request.png)

Wypełnij pozostałe pola swoimi informacjami i preferencjami, a następnie prześlij bilet, gdy będziesz gotowy.

Po przesłaniu wniosku otrzymasz wiadomość e-mail z potwierdzeniem z numerem biletu. Pracownik pomocy technicznej skontaktuje się z Tobą w sprawie wniosku.
