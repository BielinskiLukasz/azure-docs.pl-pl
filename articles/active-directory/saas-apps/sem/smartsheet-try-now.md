# <a name="smartsheet-and-azure-active-directory"></a>Narzędzia Smartsheet i Azure Active Directory

Przyspiesz wdrażanie narzędzia Smartsheet w całej organizacji i Zwiększ liczbę użytkowników, korzystając z naszej wbudowanej integracji. Umożliwienie użytkownikom szybkiego współdziałania, łącząc narzędzia Smartsheet z Azure Active Directory (Azure AD).

![Pokazuje logo aplikacji narzędzia Smartsheet](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Wypróbuj narzędzia Smartsheet z usługą Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Co możesz zrobić z narzędzia Smartsheet i usługą Azure AD?

- **Łatwe logowanie** jednokrotne — eliminowanie wielu nazw użytkowników i haseł oraz uproszczenie dostępu przez umożliwienie użytkownikom logowania się do narzędzia Smartsheet przy użyciu kont usługi Azure AD.
- **Automatyzowanie aprowizacji użytkowników** — Automatyzuj tworzenie, konserwację i usuwanie kont użytkowników w usłudze narzędzia Smartsheet, gdy użytkownicy dołączają, przenosiją lub opuszczają organizację.
- **Zabezpieczanie, zarządzanie dostępem i monitorowanie** dostępu do narzędzia Smartsheet przez stosowanie zasad dostępu warunkowego, zarządzanie dostępem za pomocą przeglądów dostępu i monitorowanie dostępu przy użyciu dzienników logowania.

## <a name="get-started"></a>Wprowadzenie

Skonfiguruj [Inicjowanie obsługi użytkowników](https://aka.ms/SmartsheetProvisioningDoc) i [Logowanie](https://aka.ms/SmartsheetSSODoc) jednokrotne dla narzędzia Smartsheet przy użyciu naszych łatwych przewodników konfiguracji.
