Host jest komputerem opartym na architekturze x64, na którym działa kontener platformy Docker. Może to być komputer lokalny lub usługa hostingu platformy Docker na platformie Azure, na przykład:

* [Usługa Azure Kubernetes](../articles/aks/index.yml).
* [Azure Container Instances](../articles/container-instances/index.yml).
* Klaster [Kubernetes](https://kubernetes.io/) wdrożony w [Azure Stack](/azure-stack/operator). Aby uzyskać więcej informacji, zobacz [wdrażanie Kubernetes do Azure Stack](/azure-stack/user/azure-stack-solution-template-kubernetes-deploy).
