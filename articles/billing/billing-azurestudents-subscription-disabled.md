# <a name="why-is-my-azure-for-students-subscription-disabled-and-how-do-i-reactivate-it"></a>Dlaczego jest Mój Azure dla uczniów lub studentów subskrypcji wyłączone i jak go uaktywnić?

Platformy Azure dla uczniów lub studentów subskrypcji może pobrać wyłączone, ponieważ użyto wszystkich środków, środków utracił ważność lub przypadkowo Anulowano subskrypcję. Zobacz, jaki problem dotyczy to Ciebie i Dowiedz się, jak można uzyskać subskrypcji ponownej aktywacji.

## <a name="you-have-used-all-of-your-credit"></a>Użyto wszystkich środków

Azure dla uczniów lub studentów konta zapewnia 100 w kredytowych i ograniczoną ilość bezpłatnych usług przez 12 miesięcy. Bez użycia poza bezpłatnych usług i ilości jest odejmowany od środków. Gdy skończy się środków Azure wyłącza usług i subskrypcji. Aby kontynuować korzystanie z usług Azure, należy uaktualnić subskrypcję z subskrypcji, kontaktując się z [pomocy technicznej platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) i udostępnia informacje o karcie kredytowej. Po zakończeniu uaktualnienia subskrypcji nadal ma dostęp do wolnego usług przez 12 miesięcy od daty rejestracji. Możesz tylko zostały naliczone opłaty za użycie poza bezpłatnych usług i ilości.

Można sprawdzić swojego pozostałego kredytu [portal reklamy sponsorowane pakietu Microsoft Azure](https://www.microsoftazuresponsorships.com/balance)

1. Zaloguj się przy użyciu platformy Azure dla uczniów lub studentów poświadczenia konta.
2. Na stronie saldo zawiera informacje dotyczące używanych i pozostałe środki. Środki daty wygaśnięcia można znaleźć pod wykresem kredytowej.  
   
    ![Zrzut ekranu, który zawiera informacje środki dla platformy Azure dla uczniów lub studentów subskrypcji](./media/billing-azurestudents-subscription-disabled/azurestudents-credit-balance.png)

3. Aby sprawdzić użycia usług, które wykorzystują środków platformy Azure, wybierz **użycia** z następnie nawigacji po lewej stronie wybierz platformy Azure dla uczniów lub studentów subskrypcji.

    ![Zrzut ekranu pokazujący informacje o użyciu platformy Azure dla uczniów lub studentów kredytowej](./media/billing-azurestudents-subscription-disabled/azurestudents-credit-usage.png)

Tabela zawiera następujące kolumny:

* **Nazwa usługi:** usługi, która zużywa środków platformy Azure.
* **Zasoby usługi:** jednostki miary są używane usługi.
* **Spędzają:** ilość środki w USD($) przeznaczonego na usługę.


## <a name="your-credit-has-expired"></a>Środków utracił ważność

Platformy Azure dla uczniów lub studentów kredytowej wygaśnie w ciągu 12 miesięcy. Po wygaśnięciu środków, Azure wyłącza subskrypcji. Aby kontynuować korzystanie z usług Azure, należy uaktualnić subskrypcję subskrypcji płatność za rzeczywiste użycie, kontaktując się z [pomocy technicznej platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) i udostępnia informacje o karcie kredytowej. Po uaktualnieniu, Azure opłat możesz płatności obejmujące szybkości dla wszystkich usług, którego używasz.


## <a name="you-have-accidentally-canceled-your-subscription"></a>Przypadkowo Anulowano subskrypcję

Jeśli przypadkowo Anulowano platformy Azure dla uczniów lub studentów subskrypcji, należy go ponownie uaktywnić kontaktując [pomocy technicznej platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). Po aktywowaniu, możesz nadal mieć dostęp do pozostałej liczbie kredytów i bezpłatnych usług przez 12 miesięcy od daty rejestracji.

## <a name="need-help-contact-support"></a>Potrzebujesz pomocy? Kontakt z pomocą techniczną

Jeśli nadal potrzebujesz pomocy, [się z pomocą techniczną](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade) uzyskać szybkie rozwiązanie problemu.
