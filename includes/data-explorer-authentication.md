## <a name="authentication"></a>Authentication

Aby uruchomić Poniższy przykład, potrzebna jest aplikacja Azure Active Directory (Azure AD) i nazwa główna usługi, która może uzyskiwać dostęp do zasobów. Aby utworzyć bezpłatną aplikację usługi Azure AD i dodać przypisanie roli na poziomie subskrypcji, zobacz [Tworzenie aplikacji usługi Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal). Wymagany jest również identyfikator katalogu (dzierżawy), identyfikator aplikacji i klucz tajny klienta.
