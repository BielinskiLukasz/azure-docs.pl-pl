# <a name="cloud-partner-portal-management"></a>Zarządzanie w portalu dla partnerów w chmurze

W tej sekcji opisano typy umożliwiające tworzenie i zarządzanie kontami, trzeba używać operacji [portalu Cloud Partner](https://cloudpartner.azure.com).  Te konta są wymagane dla [portalu Azure Marketplace](https://azuremarketplace.microsoft.com) i [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Informacje uzupełniające

Przed rozpoczęciem korzystania z portalu Cloud Partner, należy zapoznać się z następujących artykułów:

- [Azure Marketplace — zasady i warunki](../../policies-terms.md) Wyświetla listę różnych wymagań prawnych i technicznych i reguł, które mają zastosowanie do użytkowników i wydawców [portalu Azure Marketplace](https://azuremarketplace.microsoft.com) i [AppSource](https://appsource.microsoft.com).
- [Stają się wydawcy w witrynie Marketplace chmury](https://docs.microsoft.com/azure/marketplace/become-publisher) zawiera omówienie wszystkich kroków wymaganych do stają się oficjalny program Microsoft publisher chmury.


## <a name="account-creation-and-management"></a>Tworzenie konta i zarządzanie nimi

Ta sekcja zawiera następujące artykuły, które opisują sposób zarządzania kontami wydawcy witryny marketplace.

- [Wymagania wstępne dotyczące zarządzania kontami](./cpp-manage-prerequisites.md) wyjaśnia, jak zostać wydawcą zarejestrowanej chmury, wymagania dotyczące korzystania z portalu CLoud Partner.
- [Zarządzanie profilem wydawcy w portalu Azure Marketplace i AppSource](./cpp-manage-publisher-profile.md) wyjaśnia, jak używać portalu Cloud Partner do przeglądania i zarządzania swojego profilu wydawcy witryny marketplace firmy Microsoft. 
- [Zarządzanie użytkownikami w portalu Cloud Partner](./cpp-manage-users.md) Dodawanie użytkowników do chmury w organizacji publikowania konta oraz jak zarządzać ich dostępu, aby umożliwić wielu użytkownikom pracę na tej samej oferty.
