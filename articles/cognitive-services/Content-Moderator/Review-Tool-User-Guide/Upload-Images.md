# <a name="upload-images"></a>Przekazywanie obrazów

Aby zapoznać się z interfejsu użytkownika, należy przekazywać obrazy do pięciu jednocześnie, każdy obraz nie może przekraczać 2 MB rozmiaru pliku. Pamiętaj sprawdzić swoje rozmiary plików przed przekazaniem.

![Przekazywanie obrazów](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Prześlij do moderacji
Jeśli znajdą się przesyłania więcej niż pięciu obrazów w czasie, umożliwiają usuwanie niepożądanych obrazów ikona Kosza na śmieci. Następnie prześlij obrazy dla zalet zautomatyzowanego moderowania.

![Prześlij obrazy](images/1-try-image-2.png)
