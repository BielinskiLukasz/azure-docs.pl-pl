## <a name="create-an-azure-resource"></a>Tworzenie zasobu platformy Azure

Rozpocznij korzystanie z interfejsu API sprawdzanie pisowni Bing, tworząc jeden z następujących zasobów platformy Azure:

[Zasób wersji próbnej](https://azure.microsoft.com/try/cognitive-services/?api=spellcheck-api-v7)
   * Nie jest wymagana żadna subskrypcja platformy Azure.
   * Ważne przez siedem dni bezpłatnie. Po zarejestrowaniu się w [witrynie sieci Web platformy Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/)będą dostępne klucze próbne i punkt końcowy.

[Zasób sprawdzanie pisowni Bing](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSpellCheck-v7)
   * Dostępne w Azure Portal do momentu usunięcia zasobu.
   * Skorzystaj z warstwy cenowej bezpłatna do wypróbowania usługi i przeprowadź uaktualnienie później do warstwy płatnej dla środowiska produkcyjnego.
   * Interfejs API sprawdzanie pisowni Bing jest również oferowany w niektórych warstwach [zasobu wyszukiwanie Bing wersji 7](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7).
    
[Zasób obejmujący wiele usług](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * Dostępne w Azure Portal do momentu usunięcia zasobu.  
   * Użyj tego samego klucza i punktu końcowego dla aplikacji w wielu Cognitive Services.
