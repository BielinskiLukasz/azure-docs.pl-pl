# <a name="team-data-science-process-project-planning"></a>Planowanie projektu procesu nauki o danych zespołowych

Proces nauki o danych zespołu (TDSP) zapewnia cykl życia do struktury rozwoju projektów nauki o danych. Ten artykuł zawiera łącza do szablonów programu Microsoft Project i Excel, które ułatwiają planowanie tych etapów projektu i zarządzanie nimi.

Cykl życia przedstawia główne etapy, które projekty zazwyczaj wykonują, często iteracyjne:

- Zrozumienie biznesowe
- Pozyskiwanie i rozumienie danych
- Modelowanie
- wdrażania
- Akceptacja klienta

Opisy każdego z tych etapów można znaleźć w [cyklu życia procesu nauki o danych zespołu.](https://docs.microsoft.com/azure/machine-learning/team-data-science-process/lifecycle)

 
## <a name="microsoft-project-template"></a>Szablon programu Microsoft Project

Szablon programu Microsoft Project dla procesu nauki o danych zespołu jest dostępny tutaj: [Szablon programu Microsoft Project](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.mpp) 

Po otwarciu planu kliknij łącze po lewej stronie dla TDSP. Zmień nazwę i opis, a następnie dodaj inne zasoby zespołu, których potrzebujesz. Oszacuj daty wymagane na podstawie doświadczenia.

![1](./media/team-data-science-process-project-templates/ms-project-templates.png)

Każde zadanie ma notatkę. Otwórz te zadania, aby zobaczyć, jakie zasoby zostały już utworzone dla Ciebie.
