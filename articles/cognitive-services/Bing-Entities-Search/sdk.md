# <a name="bing-search-sdk"></a>Wyszukiwania usługi Bing zestawu SDK
Scenariusze zawierają przykłady interfejsu API Bing encji wyszukiwania który:
1.  Wyszukaj jednostek, takich jak rejs Tomasz i rozbudowane informacje.
2.  Obsługa Uściślanie warunki dla zapytań z prawdopodobnie wiele lokalizacji docelowych.
3.  Wyszukaj jednostki lokalnej, takich jak restauracji i rozbudowane informacje wokół niego.
4.  Wyszukaj lokalnych firm, takich jak restauracji i rozbudowane informacje.
5.  Wyzwalanie nieprawidłowe żądanie oraz obsługi błędów.

> [!NOTE] 
> Niektóre zestawy SDK znajdują się teraz na GA i oczekujących zmian w dokumentacji. 

Zestawy SDK wyszukiwania usługi Bing należy funkcji wyszukiwania w sieci web łatwo dostępne w następujących językach programowania:
* Rozpoczynanie pracy z [próbek .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.EntitySearch/1.2.0)
    * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch) definicje i zależności.
* Rozpoczynanie pracy z [Node.js — przykłady](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [bibliotek Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [bibliotek Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [bibliotek języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-entitysearch) definicje i zależności.

Przykłady zestawu SDK dla każdego języka obejmują plik ReadMe o szczegółowe informacje o wymaganiach wstępnych oraz instalowania/uruchomiony próbek.
