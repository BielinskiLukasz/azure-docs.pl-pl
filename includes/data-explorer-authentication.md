## <a name="authentication"></a>Uwierzytelnianie

Aby uruchomić Poniższy przykład, potrzebna jest aplikacja usługi Azure AD i nazwa główna usługi, która może uzyskiwać dostęp do zasobów. Zobacz [Tworzenie aplikacji usługi Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal) , aby utworzyć bezpłatną aplikację usługi Azure AD, a następnie Dodaj przypisanie roli na poziomie subskrypcji. Wymagany jest również `directory (tenant) ID`, `application ID` i `client secret`.
