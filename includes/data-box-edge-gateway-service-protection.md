- Aby uzyskać dostęp do usługi Data Box Edge, twoja organizacja musi mieć subskrypcję umowy Enterprise Agreement (EA) lub Dostawcy rozwiązań w chmurze (CSP). Aby uzyskać więcej informacji, zobacz [Rejestrowanie się w celu subskrypcji platformy Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Ponieważ ta usługa zarządzania jest hostowana na platformie Azure, jest chroniona przez funkcje zabezpieczeń platformy Azure. Aby uzyskać więcej informacji na temat funkcji zabezpieczeń oferowanych przez platformę Azure, przejdź do [Centrum zaufania platformy Microsoft Azure](https://azure.microsoft.com/support/trust-center/security/).
- W przypadku operacji zarządzania pakietem SDK można uzyskać klucz szyfrowania zasobu we **właściwościach urządzenia**. Klucz szyfrowania można wyświetlić tylko wtedy, gdy masz uprawnienia do interfejsu API wykresu zasobów.
