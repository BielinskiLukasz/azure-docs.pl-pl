# <a name="cloud-partner-portal-migration-to-partner-center"></a>Migracja z witryny Cloud Partner Portal do Centrum partnerskiego

Aby usprawnić zarządzanie kontem i oferować, firma Microsoft przeprowadza migrację ofert z portalu Cloud Partner Do Centrum partnerów. Jeśli obecnie zarządzasz zabezpieczeniem co-sell w katalogu One Commercial Partner Go-to-Market (OCP GTM), wkrótce będziesz mógł zaktualizować swoje zabezpieczenie co-sell w Centrum partnerów.

| **Typ oferty** | **Zarządzanie ofertami przeniesione z portalu cloud partner do centrum partnerów** | **Możliwość aktualizacji zabezpieczenia co-sell w Centrum partnerów** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Oprogramowanie jako usługa (SaaS) |     Dostępny teraz       |   Ogólna dostępność kwiecień 2020      |
| Dynamics 365 for Customer Engagement & PowerApps      |     Dostępny teraz        |   Dostępny teraz      |
| Dynamics 365 for Operations     |     Dostępny teraz       |   Dostępny teraz     |
| Dynamics 365 Business Central      |     Dostępny teraz        |   Ogólna dostępność kwiecień 2020     |
| Azure Application     |     Dostępny teraz       |   Ogólna dostępność kwiecień 2020     |
| Maszyna wirtualna platformy Azure      |     Ogólna dostępność kwiecień–maj 2020       |   Do ogłoszenia    |
| Kontener platformy Azure     |     Ogólna dostępność kwiecień–maj 2020       |   Do ogłoszenia     |
| Usługa konsultingowa      |     Ogólna dostępność kwiecień–maj 2020       |   Do ogłoszenia     |
| Moduł brzegowy IoT      |     Ogólna dostępność kwiecień–maj 2020       |   Do ogłoszenia     |
| Aplikacja usługi Power BI     |     Ogólna dostępność kwiecień–maj 2020       |   Do ogłoszenia     |
