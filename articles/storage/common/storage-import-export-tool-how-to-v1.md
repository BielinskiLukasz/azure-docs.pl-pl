# <a name="using-the-azure-importexport-tool-classic-deployment-model"></a>Korzystanie z narzędzia Azure Import/Export (klasyczny model wdrażania)

Narzędzie Azure Import/Export (WAImportExport. exe) służy do tworzenia zadań i zarządzania nimi w usłudze Azure Import/Export, co umożliwia Transferowanie dużych ilości danych do i z usługi Azure Blob Storage.

Ta dokumentacja dotyczy klasycznego modelu wdrażania narzędzia Azure Import/Export. Aby uzyskać informacje na temat korzystania z najnowszej wersji narzędzia, zobacz [Korzystanie z narzędzia Azure Import/Export](../storage-import-export-tool-how-to.md).

W poniższych artykułach pokazano, jak:

- Zainstaluj i skonfiguruj narzędzie Import/Export.
- Przygotuj dyski twarde w celu zaimportowania danych z dysków do usługi Azure Blob Storage.
- Sprawdź stan zadania za pomocą plików dziennika kopiowania.
- Naprawianie zadania importu.
- Naprawianie zadania eksportu.
- Rozwiązywanie problemów z narzędziem Azure Import/Export, jeśli wystąpił problem podczas procesu.
