## <a name="prerequisites"></a>Wymagania wstępne 
Aby ukończyć ten samouczek, musisz mieć plik VHD przekazany do konta magazynu skojarzonego z laboratorium, w którym tworzysz obraz niestandardowy. Wybierz dowolne z poniższych linków, aby przekazać plik VHD:

- [Przekazywanie przy użyciu narzędzia wiersza polecenia AzCopy](../articles/lab-services/devtest-lab-upload-vhd-using-azcopy.md)
- [Przekaż przy użyciu Eksplorator usługi Microsoft Azure Storage](../articles/lab-services/devtest-lab-upload-vhd-using-storage-explorer.md)
- [Przekazywanie przy użyciu konsoli programu PowerShell](../articles/lab-services/devtest-lab-upload-vhd-using-powershell.md)
