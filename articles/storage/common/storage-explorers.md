|-------------------------------------------|---------------------|------------|-----------|-------------|--------|--------|-------|
| [Azure Portal](https://portal.azure.com)                    | sieć Web                 | Tak          | Yes         | Yes           | Yes      | Yes      | Tak     |
| [Azure Storage Explorer](https://azure.microsoft.com/features/storage-explorer/)          | Windows, OSX        | Tak          | Yes         | Yes           | Yes      | Yes      | Tak     |
| [Microsoft Visual Studio w programie Cloud Explorer](/visualstudio/azure/vs-azure-tools-resources-managing-with-cloud-explorer) | Windows             | Tak          | Yes         | Yes           | Yes      | Yes      | Nie      |

Dostępne są również różne narzędzia innych firm umożliwiające pracę z danymi usługi Azure Storage.
