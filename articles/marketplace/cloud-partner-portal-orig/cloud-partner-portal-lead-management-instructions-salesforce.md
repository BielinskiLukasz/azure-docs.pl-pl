# <a name="configure-lead-management-for-salesforce"></a>Konfigurowanie zarządzania potencjalnych klientów usługi Salesforce

W tym artykule opisano sposób konfigurowania systemu Salesforce do obsługi potencjalnych klientów.

1. Zaloguj się do usługi Salesforce.
2. Wybierz **Instalatora**.
    ![Konfiguracja usługi SalesForce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

3. Rozwiń **kompilacji** menu **Dostosuj/potencjalnych klientów/Web do-potencjalny klient**.
    ![Moduł Web kierownik usługi SalesForce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

4. Na **Instalator sieci Web do realizacji**, wybierz opcję **Utwórz formularz sieci Web do realizacji**.
    ![Formularz tworzenia usługi SalesForce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

5. Na **Tworzenie formularza sieci Web do realizacji**, wybierz opcję **Generuj** ![Generowanie Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

6. Skopiuj identyfikator OID w przykładzie i zapisz go. Będziesz Wklej identyfikator OID w **identyfikatora obiektu** w portalu Cloud Partner.
![Pobierz SalesForce identyfikatora OID](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
