# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>Lokalne monitorowanie aplikacji bezkobowych w języku Java — usługa Azure Monitor Application Insights — publiczna wersja zapoznawcza

Monitorowanie aplikacji bezkodowych Java polega na prostocie - nie ma żadnych zmian w kodzie, agent Java można włączyć za pomocą zaledwie kilku zmian konfiguracji.

## <a name="overview"></a>Omówienie

Po włączeniu agent Java automatycznie zbiera wiele żądań, zależności, dzienników i metryk z najczęściej używanych bibliotek i struktur.

Należy postępować zgodnie [ze szczegółowymi instrukcjami](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent) dotyczącymi wszystkich środowisk, w tym lokalnych.
