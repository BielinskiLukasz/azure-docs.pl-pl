# <a name="azure-cosmos-db-query-cheat-sheets"></a>Arkusze cheatów zapytań usługi Azure Cosmos DB

**Arkusze cheat zapytań usługi Azure Cosmos DB** ułatwiają szybkie zapisywanie zapytań dotyczących danych przez wyświetlanie typowych zapytań, operacji, funkcji i operatorów w łatwych do wydrukowania arkuszach referencyjnych PDF. Arkusze kodów zawierają informacje referencyjne dla interfejsów API SQL, MongoDB, Table i Gremlin. 

Do wyboru jest pobieranie w rozmiarze letter lub A3. 

## <a name="letter-sized-cheat-sheets"></a>Arkusze o rozmiarze listu

Pobierz [arkusze zapytań o rozmiarze listu usługi Azure Cosmos DB,](https://go.microsoft.com/fwlink/?LinkId=623215) jeśli zamierzasz drukować na papierze o rozmiarze letter (8,5" x 11).

![Arkusze cheatów zapytań usługi Azure Cosmos DB — rozmiar liter z zapytaniami i funkcjami interfejsu API SQL, JavaScript, MongoDB, Gremlin i Table API](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Ponadwymiarowe arkusze oszustów
Pobierz [arkusze zapytań usługi Azure Cosmos DB A3,](https://go.microsoft.com/fwlink/?linkid=870413) jeśli zamierzasz drukować za pomocą plotera lub drukarki wielkoskalowej na papierze formatu A3 (11,7" x 16,5").

![Arkusze cheatów zapytań usługi Azure Cosmos DB — rozmiar A3 z zapytaniami i funkcjami interfejsu API SQL, JavaScript, MongoDB, Gremlin i Table API](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
