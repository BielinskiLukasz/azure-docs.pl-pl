# <a name="add-bank-and-tax-information"></a>Dodawanie informacji bankowych i podatkowych  

Informacje bankowe i podatkowe są wymagane dla wszystkich ofert komercyjnych przy użyciu typu aukcji transakcji.  

## <a name="partner-center"></a>Centrum partnerskie

Informacje bankowe i podatkowe są teraz zarządzane w Centrum partnerów. Instrukcje konfigurowania tych ustawień można znaleźć na stronie "Przegląd portalu Komercyjnego Marketplace" w sekcji [Zarządzanie kontem Centrum partnerów](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#payout-account).
