# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Wyświetlanie zasobu usługi Azure Arc Data Controller w Azure Portal

Po wykonaniu pierwszych [metryk lub załadowania do platformy Azure](upload-metrics-and-logs-to-azure-monitor.md) lub [przekazaniu użycia](view-billing-data-in-azure.md)będzie można zobaczyć, że zasób kontrolera danych usługi Azure Arc utworzony na platformie Azure jest w Azure Portal.

Otwórz Azure Portal przy użyciu specjalnego adresu URL:  [https://portal.azure.com](https://portal.azure.com) .

Wyszukaj swój kontroler danych według nazwy na pasku wyszukiwania i kliknij go.
