# Azure サブスクリプションに支払期限を過ぎた未払い額があるという通知を受信する理由
Azure サブスクリプションのアカウント管理者、時間どおりに支払いを行っていない場合は、電子メールが届きますが、過去の分散、またはする期日に関する通知 http://accounts.windowsazure.com または http://portal.azure.com でアラートが表示されます。  

### 支払い忘れた場合
サービスが取り消され、リソースは使用できなくなります。 サービスが停止されてから 90 日後にすべてのデータが削除されます。

### この問題を解決する方法

未払いの残高を全額支払います。

**シナリオ 1**: 支払、請求書モードの場合は、お客様のお支払いを請求書の下部に表示されている場所に送信します。 ヘルプを必要がある場合にお問い合わせください。 [Azure サポート](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)します。

**シナリオ 2**: 部品上、クレジット カードのために支払いが行われない場合ファイルが拒否されました有効期限が切れた、別のクレジット カードまたは支払方法を使用して、サブスクリプションの場合、または問題を解決するのには、金融機関にお問い合わせください。 Azure の支払方法を変更する方法の手順については、次を参照してください。 [Azure サブスクリプションの支払いに使用するクレジット_カードを変更する方法](./billing-how-to-change-credit-card.md)します。 クレジット カードを変更するには、アカウント管理者としてログオンする必要があります。


**シナリオ 3**: 会社または変更されたロール アカウント管理者が残っているため請求の通知が受信しなかった場合にお問い合わせください。 [Azure サポート](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) 、問題を解決します。
