# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Usługa w chmurze oparta na sztucznej inteligencji z usługą Azure Cognitive Search
[Usługa Azure Cognitive Search](https://azure.microsoft.com/services/search/) to rozwiązanie w chmurze typu "wyszukiwanie jako usługa", które zapewnia deweloperom interfejsy API i narzędzia do dodawania bogatego środowiska wyszukiwania za pośrednictwem prywatnej, heterogeniii zawartości w aplikacjach sieci Web, urządzeń mobilnych i korporacyjnych. Kod niestandardowy wywołuje pozyskiwania danych (indeksowanie) do tworzenia i ładowania indeksu. Z drugiej strony kod aplikacji wystawia żądania zapytań i obsługuje odpowiedzi. Środowisko wyszukiwania jest zdefiniowany w kliencie przy użyciu funkcji z usługi Azure Cognitive Search, z wykonywaniem zapytań za pośrednictwem utrwalonego indeksu, który można utworzyć, własne i przechowywać w usłudze.

## <a name="azure-cognitive-search-features"></a>Funkcje usługi Azure Cognitive Search
- **Wyszukiwanie tekstu w postaci swobodnej:** umożliwia wyszukiwanie pełnotekstowe, składnię kwerendy prostej i składnię zapytania Lucene.
- **Proste profile oceniania**: Modelowanie trafności jako funkcji wartości w samych dokumentach.
- **Geosearch:** Eksploruje dane na podstawie bliskości wyniku wyszukiwania do fizycznej lokalizacji.
- **Funkcje środowiska użytkownika**: Podaj autouzupełnianie, sugestie wyszukiwania, sortowanie i stronicowanie.
- **Wyszukiwanie poznawcze:** Może być stosowane do potoku indeksowania, aby wyodrębnić informacje tekstowe z zawartości nieprzetworzonej do analizy obrazu i tekstu.
- **Magazyn wiedzy**: Zapisuje wzbogacenia utworzone podczas indeksowania.
- **Źródła danych:** wyodrębnianie zawartości z przeszukiwalną w podstawowych magazynach danych, gdy masz połączenie z usługą Azure SQL Database, usługa Azure Cosmos DB i magazynem obiektów blob platformy Azure.
- **Kreator importu danych**: Konfiguruje indeksatory. 
- **Projektant indeksu**: Wystaje indeks. 
- **Eksplorator wyszukiwania:** testuje zapytania i udoskonala profile oceniania.

Znajdź szczegółowy opis [funkcji](/azure/search/search-what-is-azure-search#feature-descriptions)usługi Azure Cognitive Search .
