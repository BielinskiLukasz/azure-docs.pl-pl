# <a name="cloud-solution-providers"></a>Dostawców rozwiązań w chmurze

Oferty dotyczące oprogramowania można teraz uruchamiać na milionach kwalifikowaną klientów firmy Microsoft przez partnerów Cloud Solution Provider (CSP), przy minimalnym inwestycji sprzedaży i marketingu.  Wydawcy mogą teraz korzystać kanałów partnera CSP, oprócz publiczną dostępność oferty za pośrednictwem [sklepów sieci web firmy Microsoft](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Wydawcy można wybrać ofertę dla dostępności w kanale partnera programu CSP dla nowej oferty lub istniejąca uczestnictwo. Ta opcja umożliwia partnerzy programu CSP sprzedawać swoje produkty i tworzyć powiązane rozwiązania dla klientów.

Wydawcy są odpowiedzialne za świadczenie pomocy technicznej naprawa klientów końcowych oraz zapewniające mechanizm partnerzy programu CSP i/lub klienci mogą skontaktować się z pomocy technicznej. Jest to najlepsze rozwiązania, aby zapewnić partnerzy programu CSP z dokumentacji użytkownika, szkoleń i usługę powiadomienia o kondycji/przestojach (o ile dotyczy), partnerzy programu CSP są wyposażone do obsługi warstwy obsługi 1 żądań od klientów.

W poniższej tabeli przedstawiono typy oferty i modele cen, które zgodzić się na kanale partnerskim programu CSP:

| **Typ oferty**    | **BYOL**  |  **Bezpłatna** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Maszyny wirtualne  | Yes | Yes | Yes |
| Containers | ND | Yes | ND |
| Szablony rozwiązań | ND | Yes | ND |
| Aplikacje zarządzane | ND | Yes | Yes |
| IT rozwiązanie SaaS (dla poszczególnych lokacji ceny) | ND | Yes | Yes |
|   |   |   |
