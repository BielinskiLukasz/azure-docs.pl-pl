# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Jak skonfigurować serwer Proxy aplikacji aplikacji na używanie PingAccess

Nasza współpraca z PingAccess teraz umożliwia czerpać korzyści z serwera Proxy aplikacji w aplikacji przy użyciu uwierzytelniania opartego na nagłówka. Jeśli aplikacje nie należy używać nagłówków, zobacz nasze [rejestracji jednokrotnej dokumentacji](manage-apps/application-proxy-configure-single-sign-on-with-kcd.md) szczegółowe informacje na temat innych.

## <a name="overview-of-steps-and-recommended-documents"></a>Omówienie kroków i zalecane dokumentów

Aby skonfigurować aplikację z PingAccess, istnieją cztery kroki:

1.  Skonfiguruj Konektory serwera Proxy aplikacji

2.  Tworzenie aplikacji serwera Proxy aplikacji usługi Azure AD

3.  & Pobierz skonfigurować PingAccess

4.  Konfigurowanie aplikacji w PingAccess

Aby uzyskać szczegółowe informacje na każdym z tych kroków, zobacz nasze [rejestracji jednokrotnej z dokumentacją nagłówki](manage-apps/application-proxy-configure-single-sign-on-with-ping-access.md).
