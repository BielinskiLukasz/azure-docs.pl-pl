### <a name="azure-active-directory-authentication"></a>Uwierzytelnianie usługi Azure Active Directory

Zakotwiczenia przestrzenne platformy Azure umożliwiają również aplikacjom uwierzytelnienie za pomocą tokenów usługi Azure AD (Active Directory). Na przykład można użyć tokenów usługi Azure AD do integracji z zakotwiczeniami przestrzennymi platformy Azure. Jeśli przedsiębiorstwo utrzymuje użytkowników w usłudze Azure AD, można dostarczyć token usługi Azure AD użytkownika w usłudze Azure Spatial Anchors SDK. W ten sposób można uwierzytelnić bezpośrednio do usługi Azure Spatial Anchors dla konta, które jest częścią tej samej dzierżawy usługi Azure AD.
