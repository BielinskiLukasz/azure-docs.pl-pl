> [!NOTE]
>
> W tym artykule została zaktualizowana w celu użycia nowego [modułu Azure PowerShell Az](/powershell/azure/new-azureps-module-az). Moduł AzureRM będzie w dalszym ciągu otrzymywać aktualizacje z poprawkami krytycznych usterek, lecz nowe funkcje będą udostępniane tylko modułowi Az.
> * Moduł Az jest już dostępna w usłudze Cloud Shell ze zgodnością AzureRM włączone.
> * Aby zainstalować moduł Az lokalnie, zobacz [Instalowanie programu Azure PowerShell](/powershell/azure/install-az-ps).
> * Po zainstalowaniu modułu Az lokalnie, aby umożliwić zgodności AzureRM, uruchamiając `Enable-AzureRmAlias`.
