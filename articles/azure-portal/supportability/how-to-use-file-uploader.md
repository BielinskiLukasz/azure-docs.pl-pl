# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Wskazówki dotyczące narzędzia do przekazywania plików żądania obsługi technicznej platformy Microsoft Azure

Podczas [tworzenia żądania pomocy technicznej](https://portal.azure.com/#create/Microsoft.Support)opcja przekazywania plików może pomóc w łatwym przekazywaniu plików diagnostycznych lub innych plików, które uważasz za istotne dla incydentu.  

> [!NOTE]
> Aby chronić swoją prywatność, nie umieszczaj żadnych danych osobowych w przesłaniu.
>
>

Podczas korzystania z opcji przekazywania plików postępuj zgodnie z tymi wskazówkami:

* Wszystkie pliki muszą mieć rozszerzenie nazwy pliku, takie jak *.docx* lub *xlsx*. Zobacz tabelę dla dozwolonych rozszerzeń nazw plików.
* Nazwa pliku nie może być dłuższa niż 110 znaków.
* Nie można przesłać więcej niż jednego pliku.
* Pliki nie mogą być większe niż 4 MB.
