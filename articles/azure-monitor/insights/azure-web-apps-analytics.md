# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Wyświetl dane analityczne metryk we wszystkich zasobach aplikacji sieci Web platformy Azure

![Symbol aplikacji sieci Web](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Rozwiązanie analiza aplikacji sieci Web platformy Azure została przeniesiona do pomoc techniczna w społeczności.  
>- Rozwiązanie nie jest już dostępna w witrynie Azure Marketplace, ale mogą być instalowane z [szablony szybkiego startu platformy Azure](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) gdzie nie jest obsługiwany przez społeczność.
>- Klienci, którzy już zainstalowano rozwiązania można nadal z niego korzystać ze zmianami.  
>- Firma Microsoft zaleca się, że używasz [usługi Application Insights](../../application-insights/app-insights-overview.md) do monitorowania aplikacji sieci web.
