# <a name="power-bi-apps-prerequisites"></a>Wymagania wstępne dotyczące usługi Power BI aplikacje

W tym artykule wymieniono techniczne i wymagań biznesowych, które muszą spełnić, przed można publikować oferty aplikacji Power BI do usługi Microsoft AppSource Marketplace.

## <a name="technical-requirements"></a>Wymagania techniczne

Podstawowy zasobów technicznych, należy dla tego typu oferty jest [aplikacja Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Aby uzyskać więcej informacji, zobacz [zasoby techniczne tworzenia usługi Power BI](./cpp-create-technical-assets.md).

Po utworzeniu aplikacji usługi i przetestowane w usłudze Power BI, należy zapisać adres URL instalacji aplikacji, która generuje programie Power BI. Będzie potrzebny do [Tworzenie nowej oferty aplikacji Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Wymagania biznesowe

Wymagania biznesowe obejmują procedurach umowne stosowane w UE oraz prawne zobowiązań:

-   Musi być zarejestrowany wydawcy chmury w witrynie Marketplace. Jeśli nie zostały jeszcze zarejestrowane, postępuj zgodnie z instrukcjami w artykule [stają się wydawcy w witrynie Marketplace chmury](../../become-publisher.md).

    > [!NOTE] 
    > Należy używać tego samego konta Microsoft Developer Center rejestrowania się na [portalu Cloud Partner](https://cloudpartner.azure.com). Obsługi ofert w usłudze AppSource, powinien mieć tylko jedno konto Microsoft. Nie powinien być specyficzne dla poszczególnych usług lub oferty.
 
-   Odpowiedzialność za udostępnianie pomocy technicznej dla klientów w sposób rozsądny z komercyjnego punktu widzenia.
-   Należy podać zawartość, która spełnia kryteria za ofertę użytkownika był wyświetlany w usłudze AppSource. Aby uzyskać więcej informacji, zobacz [mieć aplikację do listy w usłudze AppSource? Oto jak](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how).
-   Muszą być zgodne z [zasady zachowania poufności informacji firmy Microsoft](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Kolejne kroki

Gdy spełniono wszystkie wymagania, jesteś gotowy do [Utwórz ofertę usługi Power BI](./cpp-create-offer.md) w portalu Cloud Partner.
