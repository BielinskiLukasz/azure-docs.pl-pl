# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Wyświetlanie ostatniej aktywności logowania w portalu Mój profil (wersja zapoznawcza)
Możesz wyświetlać wszystkie ostatnie działania logowania przy użyciu konta służbowego z **ostatnią aktywność** strony **Mój profil** portalu.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Wyświetl swoją ostatnią aktywność logowania
1. Zaloguj się do swojego konta firmowego lub szkolnego, a następnie przejdź do swojej https://myprofile.microsoft.com/ strony.

2. Wybierz **ostatnią aktywność** z lewego okienka nawigacji lub wybierz **działanie recenzowania** link z **ostatnią aktywność** bloku.

    ![Strona Moje profilu, przedstawiający wyróżniony ostatnie działania łącza](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Rozwiń węzeł, a następnie przejrzyj poszczególne elementy logowania, upewniając się, że znasz każdej z nich. Jeśli okaże się logowania elementu, który nie wygląda znajomo, zdecydowanie zalecamy zmienisz hasło, aby chronić swoje konto, jeśli zostało naruszone bezpieczeństwo.

    ![Strony ostatnich działań z rozwiniętej szczegóły logowania](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Jeśli uważasz, że Twoje konto zostało naruszone, zobacz [sposób dostępu o naruszeniu bezpieczeństwa konta Microsoft](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) Aby uzyskać więcej pomocy przy użyciu resetowania hasła lub odblokowywania konta.

## <a name="next-steps"></a>Kolejne kroki

- Po obejrzeniu swoją ostatnią aktywność logowania, można wybrać wyświetlanie i zarządzanie swoje [zabezpieczające](user-help-security-info-overview.md), połączone [urządzeń](myprofile-portal-devices-page.md) lub [organizacje](myprofile-portal-organizations-page.md), lub aby wyświetlić jak usługi Organizacja [używa danych](myprofile-portal-privacy-page.md).
