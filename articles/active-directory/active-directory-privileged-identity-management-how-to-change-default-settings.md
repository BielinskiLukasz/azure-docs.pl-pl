# Azure Privileged Identity Management: ロールの既定のアクティブ化設定を変更または表示する方法

## ロールの既定のアクティブ化の変更と表示
1. ダッシュボードのロールのテーブルで、構成するロールをクリックします。
2. クリックして **設定**します。
3. アクティブ化の既定の期間 (時間単位) を、スライダーを調整するか、テキスト フィールドに時間数を入力して、設定します。
4. クリックして **を有効にする** または **を無効にする** か、管理者に送信されるライセンス認証に関する通知が希望される場合。
5. クリックして **を有効にする** または **を無効にする** か、ライセンス認証要求に tickeing 情報を入力する管理者を許可するようにします。
6. クリックして **を有効にする** または **を無効にする** か、ライセンス認証要求の多要素認証を要求するようにします。  PIM での MFA の使用の詳細については、次を参照してください。 [MFA を必要とする方法](active-directory-privileged-identity-management-how-to-require-mfa.md)します。
7. クリックして **を有効にする** または **を無効にする** 一時的なグローバル管理者を許可するようにします。 この設定の詳細については以下を参照してください。
8. クリックして **保存**します。
