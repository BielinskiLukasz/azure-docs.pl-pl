# <a name="register-your-azure-digital-twins-app-with-azure-active-directory-legacy"></a>Zarejestruj swoją aplikację Twins cyfrowych platformy Azure z usługą Azure Active Directory starszej wersji

Ten artykuł pokazuje, że używasz starych lub starszej wersji sposób rejestrowania przykładowej aplikacji do usługi Azure Active Directory (Azure AD), dzięki czemu będzie miał dostęp do wystąpienia usługi cyfrowego bliźniaczych reprezentacji. Warto wypróbować tę metodę w przypadku nowej Azure rejestrację aplikacji w usłudze Active Directory nie działa w przypadku konfiguracji.
