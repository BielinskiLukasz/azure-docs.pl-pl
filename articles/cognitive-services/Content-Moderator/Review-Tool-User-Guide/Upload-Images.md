# <a name="upload-images"></a>Przekazywanie obrazów #

Aby zapoznać się z interfejsu użytkownika, należy przekazać do pięciu obrazy w czasie, obraz nie może przekraczać 2 MB, rozmiar pliku. Należy sprawdzić Twoje rozmiary plików przed przesłaniem.

![Przekazywanie obrazów](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Wprowadzane do łagodzenia ##
Jeśli na końcu przesyłania więcej niż pięć obrazów w czasie, należy użyć ikony pojemnik na śmieci, aby usunąć niechciane obrazów. Następnie należy przesłać obrazy zautomatyzowanych łagodzenia.

![Przesyłania obrazów](images/1-try-image-2.png)
