# <a name="azure-and-dynamics-365-consulting-service-offer"></a>Oferty platformy Azure i Dynamics 365, usługa DORADCZA

W tym artykule wyjaśniono, jak opublikować ofertę usługi konsultingowe firmy Microsoft <a href="https://azuremarketplace.microsoft.com">portalu Azure Marketplace</a> lub <a href="https://appsource.microsoft.com">AppSource</a>. Rozwiązania oparte na programie Microsoft <a href="https://dynamics.microsoft.com">Dynamics 365</a>, <a href="https://powerbi.microsoft.com">usługi Power BI</a>, i <a href="https://powerapps.microsoft.com">PowerApps</a> mogą być wyświetlane w usłudze AppSource. Inne oferty oparte na innym Microsoft <a href="https://azure.microsoft.com/services">usług platformy Azure</a> mogą być wyświetlane w portalu Azure Marketplace.

## <a name="publishing-benefits"></a>Publikowanie korzyści

Publikowanie do jednej z platformach handlowych firmy Microsoft ma następujące zalety:

- Podwyższanie poziomu firmy dzięki wykorzystaniu marki firmy Microsoft.
- Potencjalnie dotarcia do usługi Office 365 i Dynamics 365 ponad 100 milionów użytkowników w usłudze AppSource i dotrzeć do ponad 200 000 organizacji za pośrednictwem portalu Azure Marketplace.
- Pobierz wysokiej jakości potencjalnych klientów z tych rynków.
- Uzyskaj usługi wspierane przez zespoły pola i telesales firmy Microsoft.

## <a name="publish-a-consulting-service-offer"></a>Opublikuj ofertę usługi doradcze

W tej sekcji przedstawiono elementy publikowania doradcze oferty usługi. Publikowanie jest podzielona na następujące główne elementy:

- [Wymagania wstępne dotyczące usług doradczych](./cpp-consulting-service-prerequisites.md) wymagania aby usługa doradcza publikowania oferuje dla listy:
 
    - Dynamics 365 for Customer Engagement 
    - Dynamics 365 for Finanse i operacje 
    - Dynamics 365 Business Central 
    - Power BI 
    - PowerApps
- [Utwórz nową ofertę](./cpp-consulting-service-create-offer.md) zawiera czynności wymagane do utworzenia doradcze wpisu oferty usługi za pomocą portalu Cloud Partner:
    - [Zdefiniuj ustawienia oferty](./cpp-consulting-service-define-offer-settings.md).
    - [Wprowadź szczegóły sklepu i czy publikować w witrynie Azure Marketplace lub w usłudze AppSource.](./cpp-consulting-service-storefront-details.md)
- [Publikowanie oferty](./cpp-consulting-service-publish-offer.md) wyjaśnia, jak przesyłać oferty do opublikowania.
