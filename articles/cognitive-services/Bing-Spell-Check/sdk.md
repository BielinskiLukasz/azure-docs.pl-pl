# <a name="bing-spell-check-sdk-samples"></a>Przykłady dotyczące zestawów SDK sprawdzania pisowni Bing
Przykłady dotyczące sprawdzania pisowni Bing obejmują następujące scenariusze:
1. Wyszukiwanie zapytań z błędami pisowni i analizowanie odpowiedni.
2. Wyzwalanie odpowiedzi z informacją o błędzie z poziomu interfejsu API.

Zestawy SDK sprawdzania pisowni Bing ułatwiają korzystanie z funkcji kontekstowego sprawdzania pisowni i gramatyki przy użyciu następujących języków programowania:
* Rozpocznij pracę, korzystając z [przykładów dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.SpellCheck/1.2.0)
    * Zobacz też [biblioteki platformy .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Language/SpellCheck) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla środowiska Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [biblioteki środowiska Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/spellCheck) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla języka Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [biblioteki języka Java](https://github.com/Azure/azure-sdk-for-java/tree/master/cognitiveservices/azure-spellcheck) zawierające definicje i zależności.
* Rozpocznij pracę, korzystając z [przykładów dla języka Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-language-spellcheck) zawierające definicje i zależności.

Przykłady dotyczące zestawów SDK dla poszczególnych języków zawierają plik ReadMe ze szczegółowymi informacjami na temat wymagań wstępnych oraz instalowania i uruchamiania przykładów.
