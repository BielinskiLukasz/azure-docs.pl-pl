# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>Program dostawcy rozwiązań w chmurze StorSimple: często zadawane pytania


## <a name="overview"></a>Omówienie
Poniżej przedstawiono pytania i odpowiedzi, które mogą mieć jako partner StorSimple podczas wdrażania lub zarządzania StorSimple Virtual Array w witrynie Azure portal.

Pytania i odpowiedzi są podzielone na następujące kategorie:

 - [Model programu Dostawcy rozwiązań storsimple i chmurowych (CSP)](#storsimple-and-csp-model)
 - [Wdrażanie macierzy wirtualnej i zarządzanie nią jako partner](#deploy-and-manage-storsimple-as-a-partner)
 - Wsparcie dla partnerów StorSimple dla dostawcy usług kryptograficznych

## <a name="storsimple-and-csp-model"></a>Model StorSimple i CSP

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Co to znaczy mieć StorSimple jako część programu Dostawcy rozwiązań w chmurze (CSP)?
 StorSimple for CSP umożliwia naszym partnerom CSP odsprzedać StorSimple Virtual Array swoich klientów. Ten model umożliwia partnerom dostawcy usług kryptograficznych posiadanie kompleksowego cyklu życia klienta z bezpośrednim inicjowaniu obsługi administracyjnej, fakturowania i obsługi usług firmy Microsoft w chmurze. 

Aby uzyskać więcej informacji, przejdź do [StorSimple w programie CSP](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Czy StorSimple jest oparte na użyciu lub na podstawie licencji na miejsca, takich jak Office 365 i Enterprise Mobility Suite (EMS)?
StorSimple jest usługą opartą na użyciu. Klienci, którzy zaopatrują się w tablicę wirtualną StorSimple za pośrednictwem trasy dostawcy usług kryptograficznych, płacą za użycie macierzy wirtualnej i platformy Azure, rozliczanych jako oddzielne elementy zamówienia na rachunku dostawcy usług kryptograficznych. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Czy model biznesowy StorSimple jest podobny do modelu biznesowego innych usług, takich jak Office 365 i Pakiet Mobilności w przedsiębiorstwie? 
Tak. Model jest hurtowym modelem rabatu, podobnie jak inne usługi oparte na użyciu na platformie Azure. Rabat hurtowy dla StorSimple jest podobny do innych usług platformy Azure. Ten model zapewnia naszym partnerom dostawcy usług kryptograficznych możliwość sprzedaży większej ilości i zwiększenia zużycia platformy Azure za pośrednictwem portalu StorSimple Virtual Array.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Które jednostki SKU StorSimple są dostępne w dniu premiery?
Te same jednostki SKU, które są dostępne z autonomicznych usług StorSimple. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Czy istnieje jakaś różnica między StorSimple bezpośrednio zakupione od firmy Microsoft w ramach EA i StorSimple jako część dostawcy usług kryptograficznych?
Nie. W ramach dostawcy usług kryptograficznych klienci mogą również zakupić inne usługi oferowane przez partnera dostawcy usług kryptograficznych wraz z programem StorSimple Virtual Array (SVA) na jednej fakturze. Należy pamiętać, że w przypadku zakupu urządzenia fizycznego z serii StorSimple 8000 klienci nadal muszą przejść przez trasę EA. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>Na jakich rynkach, storsimple jest dostępny dla CSP w dniu premiery?
StorSimple Virtual Array for CSP jest dostępny na wszystkich rynkach, na których StorSimple jest już dostępny. Aby uzyskać więcej informacji, przejdź do [listy regionów, w których dostępny jest program StorSimple](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Jakie rozwiązania może dostarczać partner z StorSimple i CSP? 
Istnieje wiele rozwiązań:

- Partnerzy mogą odsprzedawać StorSimple Virtual Array. 
- Partnerzy mogą dostarczać rozwiązania oparte na storsimple i innych usługach platformy Azure lub oprogramowaniu innych firm. Aby uzyskać więcej informacji, zapoznaj się z [możliwymi rozwiązaniami, które istnieją wokół StorSimple](https://aka.ms/storsimple-build-solutions).
- Mogą również odkrywać nowe modele biznesowe, dostarczając usługi zarządzane.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Jakie są zachęty dostępne dla partnerów StorSimple CSP? Czy kwalifikuję się do jakichkolwiek zachęt programu CSP?
Aby uzyskać informacje na temat zachęt, przejdź do [zachęt programu CSP](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Wdrażanie storSimple i zarządzanie nim jako partnerem

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Jak administrować tablicą wirtualną StorSimple w dostawcy usług kryptograficznych? 
Subskrypcje StorSimple Virtual Array można dodać do konta klientów za pośrednictwem Centrum partnerów. Ponadto można użyć witryny Azure portal, aby dodać użytkowników do subskrypcji.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Czy podejście witryny Azure portal jest takie samo jak inne usługi dla usługi CSP? 
Tak. Z StorSimple Virtual Array dla dostawcy usług kryptograficznych najlepszym sposobem dostępu partnerów do witryny Azure portal powinien być za pośrednictwem [Centrum partnerskiego,](https://partnercenter.microsoft.com/) gdzie mogą zarządzać klientami i subskrypcjami. Aby zarządzać subskrypcjami StorSimple, partner lub klient (w zależności od przyznanych uprawnień) powinien zalogować się do witryny Azure portal. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Czy firma Microsoft wysyła nowy portal dla StorSimple dla dostawcy usług kryptograficznych? 
Nie. Będzie można administrować StorSimple Virtual Array w dostawcy usług kryptograficznych za pośrednictwem witryny Azure portal.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Mam aprowizowanie autonomicznej subskrypcji StorSimple dla mojego klienta. Czy muszę korzystać z witryny Azure portal do administrowania urządzeniem? 
Tak. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Czy zespół konta Microsoft jest kompensowany w wyniku sprzedaży StorSimple za pośrednictwem dostawcy usług kryptograficznych?
Tak. Partner sprzedaży wykonawczej i zespół konta klienta dla odbiorcy końcowego zostaną zrekompensowane sprzedaży CSP. Oczekuje się, że partnerzy przejmą wiodącą rolę w sprzedaży ich zróżnicowanej wartości i oferty.

## <a name="support-for-storsimple-and-csp"></a>Obsługa StorSimple i CSP

### <a name="are-there-any-forums-to-get-additional-support-for-csp-partners-to-deploy-and-manage-storsimple"></a>Czy są jakieś fora, aby uzyskać dodatkową pomoc dla partnerów dostawcy usług kryptograficznych do wdrażania i zarządzania StorSimple?
Możesz odwiedzić forum StorSimple w CSP, aby uzyskać odpowiedzi na niektóre często zadawane pytania. Należy dołączyć do [grupy Usługi Azure Advisors Yammer](https://www.yammer.com/azureadvisors) pierwszy. Następnie wyszukaj grupę i dołącz do niej - **StorSimple Partner Advisors**.

### <a name="how-does-the-support-work-for-storsimple-for-csp"></a>Jak działa pomoc techniczna dla StorSimple dla dostawcy usług kryptograficznych? 
Model pomocy technicznej dla StorSimple w dostawcy usług kryptograficznych jest taki sam jak w przypadku innych usług platformy Azure w dostawcy usług kryptograficznych. Aby uzyskać więcej informacji, przejdź do [działu obsługi klienta w zakresie CSP](/partner-center/csp-overview).

Aby uzyskać więcej informacji na temat StorSimple dla dostawcy usług kryptograficznych, przejdź do:

- [Program dostawcy rozwiązań w chmurze firmy Microsoft](https://partner.microsoft.com/Membership)
- [Centrum partnerskie](https://partnercenter.microsoft.com/)
