# <a name="sign-up-for-content-moderator"></a>Utwórz konto moderatora zawartości #
Przejdź do [zawartości moderatora Zarejestruj](http://contentmoderator.cognitive.microsoft.com/Account/Signup) strony. Możesz zalogować przy użyciu konta Microsoft lub utworzyć konto w witrynie sieci web moderatora zawartości.

![Rejestrowanie](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Nazwa Twojego zespołu ##
Podaj nazwę dla swojego zespołu przeglądu. Opcjonalnie możesz zaprosić współpracowników do przyłączenia zespołu.

![Tworzenie zespołu](images/1-Team-Name.PNG)
