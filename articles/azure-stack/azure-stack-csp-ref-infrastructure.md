## <a name="usage-reporting-infrastructure-for-cloud-service-providers"></a>Użycie raportowanie infrastruktury dla dostawcy usług w chmurze

Azure Stack zapewnia infrastrukturę potrzebną do śledzenia użycia, ponieważ występuje i przekazuje je do platformy Azure. Na platformie Azure Azure Commerce przetwarza dane użycia i opłat użycia do odpowiednich subskrypcji platformy Azure, w taki sam sposób jak użycia, która ma miejsce w chmurze Azure globalnej.

Należy pamiętać, że niektóre pojęcia są spójne Azure globalnych i stosu Azure. Stos Azure ma subskrypcje lokalne, które spełniają podobną rolę do subskrypcji platformy Azure. Subskrypcje lokalne są prawidłowe tylko lokalnie. Subskrypcje lokalne są mapowane na subskrypcji platformy Azure, w przypadku użycia jest przekazywany do platformy Azure.

Azure stos nie zawiera liczników użycia lokalnego. Użycie lokalnego jest mapowana na liczników używane w handlu platformy Azure. Jednak identyfikatory licznika są różne. Brak dostępnych liczników więcej lokalnie niż wykorzystywane przez firmę Microsoft do rozliczeń.

Istnieją pewne różnice między jak cenowo usług Azure stosu i Azure. Na przykład w stosie Azure, opłat dla maszyn wirtualnych jest tylko na podstawie vcore/godziny, o tym samym częstotliwość dla wszystkich serii maszyn wirtualnych, w przeciwieństwie do usługi Azure. Dzieje się tak na platformie Azure globalne różne ceny odzwierciedlają sprzętowe. W stosie Azure klienta zawiera sprzęt, dlatego nie ma powodu do obciążania różne szybkości dla różnych klas maszyny Wirtualnej.

Można dowiedzieć się o liczniki stosu Azure używana w handlu i ich cen w Centrum partnerskiego w taki sam sposób jak w przypadku usług Azure:

1. W Centrum partnerskiego, przejdź do **menu nawigacyjnego** > **cennik i oferty**.
2. W obszarze **usługi opartej na użyciu**, wybierz pozycję **bieżącego**.
3. Otwórz **Azure w globalnej dostawcy usług Kryptograficznych cennik** arkusza kalkulacyjnego.
4. Filtrowanie według **regionu Azure stosu =**.

## <a name="terms-used-for-billing-and-usage"></a>Pojęcia dotyczące rozliczeń i użycia

Poniższe terminy i pojęcia są używane do użycia i rozliczeń w stosie Azure:

| Termin | Definicja |
| --- | --- |
| Bezpośrednie partnera dostawcy usług Kryptograficznych | Bezpośrednie partnera Cloud Solution Provider (CSP) bezpośrednio odbiera faktury bezpośrednio z programu Microsoft Azure i stosu Azure użycia i rachunków klientów. |
| Pośrednie dostawcy usług Kryptograficznych | Pośrednie odsprzedawców współpracować z pośrednich dostawcy (znanej także jako dystrybutor). Odsprzedawców rekrutacji klientów końcowych; pośrednie dostawcy zawiera rozliczeń relację z firmą Microsoft, zarządza klienta rozliczeń i zapewnia dodatkowe usługi, takie jak Centrum pomocy technicznej. |
| Odbiorcy końcowego | Klientów końcowych są firm i agencji rządowych, do których należą aplikacje oraz innych obciążeń uruchamianych na stosie Azure. |
