# <a name="bing-search-sdk"></a>Wyszukiwania usługi Bing zestawu SDK
Przykładowe interfejsu API Bing niestandardowe wyszukiwania wyszukuje termin pojedynczego zapytania i drukowania `name` i `URL` do pierwszego wyniku sieci web.

Zestawy SDK wyszukiwania usługi Bing należy funkcji wyszukiwania w sieci web łatwo dostępne w następujących językach programowania:
* Rozpoczynanie pracy z [próbek .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.CustomSearch/1.2.0)
    * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingCustomSearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek NodeJS](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [biblioteki NodeJS](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/customSearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [bibliotek Java](https://github.com/Azure/azure-sdk-for-java/tree/master/cognitiveservices/azure-customsearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [bibliotek języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-customsearch) definicje i zależności.

Przykłady zestawu SDK dla każdego języka obejmują plik ReadMe o szczegółowe informacje o wymaganiach wstępnych oraz instalowania/uruchomiony próbek.
