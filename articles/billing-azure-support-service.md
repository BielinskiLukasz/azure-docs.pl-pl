# Azure サポートの購入方法は?

Azure では、クラウドでアプリケーションの開発を開始する開発者からビジネス クリティカルなアプリケーションをデプロイする企業まで、あらゆるお客様向けの柔軟なサポート オプションをご用意しています。 課金とサブスクリプションの管理に関する問題に対するサポートは無償で提供されますが、Azure のテクニカル サポートについては Azure サポートのサブスクリプションを購入する必要があります。

購入可能な Azure サブスクリプションの詳細については、クリックして [ここ](https://azure.microsoft.com/support/legal/offer-details/)します。 テクニカル サポートをご注文には、次を参照してください。、 [Azure サポート プラン ページ](https://azure.microsoft.com/support/plans/)します。

セルフヘルプ リソースも利用できます。 Azure のサポートを表示する [FAQ](https://azure.microsoft.com/support/faq/) ここでは、およびアクセスの任意の掲載している Azure.com のページで、[ドキュメント] タブからの記事をサポートします。
