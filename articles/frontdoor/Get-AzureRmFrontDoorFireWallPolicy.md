# Get-AzureRmFrontDoorFireWallPolicy

Pobieranie zasad zapory aplikacji sieci Web

## SKŁADNIA

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## OPIS
**Get AzureRmFrontDoorFireWallPolicy** cmdletGet pobiera zasady zapory aplikacji sieci Web w grupie zasobów w ramach bieżącej subskrypcji

## PRZYKŁADY

### Przykład 1: Pobierz zasady zapory aplikacji sieci Web o nazwie $Name w $resourceGroup
