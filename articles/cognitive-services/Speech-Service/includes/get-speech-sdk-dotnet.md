#### <a name="c-nuget-package"></a>Pakiet NuGet języka C#

Zestaw SDK mowy dla platformy .NET można zainstalować z **interfejs wiersza polecenia platformy .NET Core** przy użyciu następującego `dotnet add` polecenia.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

Zestaw SDK mowy dla platformy .NET można zainstalować z **Menedżera pakietów** przy użyciu następującego `Install-Package` polecenia.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
