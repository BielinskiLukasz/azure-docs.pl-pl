Na platformie Unity, otwórz projekt w `Unity` folderu. Unity może spowodować wyświetlenie monitu o różnych wersji aparatu Unity między projektem a ten, który został zainstalowany na tym komputerze. To ostrzeżenie nie szkodzi, tak długo, jak używana wersja programu Unity Editor jest nowszy niż ten, którego projekt został utworzony za pomocą. W takim przypadku należy po prostu kliknij **Kontynuuj**. Jeśli Twoja wersja Unity Editor jest starsza niż jeden projekt wymaga, kliknij przycisk **Quit**i uaktualnianie programu Unity Editor.
