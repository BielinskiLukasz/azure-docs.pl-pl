# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Oprogramowanie innych firm zgodne z usługą Azure AD Domain Services

## <a name="one-identity-authentication-services"></a>Jedna z usług uwierzytelniania tożsamości

![Jedna tożsamość](./media/compatible-software-icons/OneIdentity.png)

Jedna z usług uwierzytelniania tożsamości umożliwia systemom UNIX, Linux i Mac OS X korzystanie z dostępu, uwierzytelniania i autoryzacji istniejącej infrastruktury Active Directory (AD) w organizacji. Usługi uwierzytelniania obsługują teraz Azure Active Directory Domain Services włączania przez zasoby spoza systemu Windows tej samej platformy nowej generacji, która już używa istniejących rozwiązań SaaS.

[Jedna z usług uwierzytelniania tożsamości](https://www.quest.com/products/authentication-services/)

[Artykuł z bazy wiedzy — używanie jednej usługi uwierzytelniania tożsamości z usługą Azure AD Domain Services](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux to wiodąca na świecie platforma korporacyjna systemu Linux, która spełnia wymagania współczesnego przedsiębiorstwa. Red Hat Enterprise Linux jest preferowanym wyborem dla obciążeń maszyn wirtualnych w przedsiębiorstwie z systemem Linux na Microsoft Azure. Red Hat Enterprise Linux to otwarta, niezawodna i bezpieczna platforma przeznaczona dla klientów, którzy chcą uzyskać elastyczność wdrażania dla swoich obciążeń o krytycznym znaczeniu dla firmy — od centrum danych do chmury platformy Azure, dzięki ścisłej integracji z usługą klasy korporacyjnej Red Hat i Microsoft.

[Dołącz maszynę wirtualną do Red Hat Enterprise Linux 7,2 do Azure AD Domain Services](active-directory-ds-admin-guide-join-rhel-linux-vm.md)
