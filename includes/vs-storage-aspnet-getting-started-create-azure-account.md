## <a name="create-an-azure-storage-account"></a>Tworzenie konta usługi Azure Storage
Najprostszym sposobem utworzenia pierwszego konta usługi Azure Storage jest użycie witryny [Azure Portal](https://portal.azure.com). Więcej informacji można znaleźć w temacie [Tworzenie konta magazynu](https://docs.microsoft.com/azure/storage/common/storage-quickstart-create-account).

Można również utworzyć konto usługi Azure Storage przy użyciu programu [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md), [interfejsu wiersza polecenia Azure](../articles/storage/common/storage-azure-cli.md) lub [biblioteki klienta dostawcy zasobów magazynowych dla platformy .NET](https://msdn.microsoft.com/library/azure/mt131037.aspx).
