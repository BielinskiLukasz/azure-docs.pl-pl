# <a name="what-is-the-linguistic-analysis-api-preview"></a>Czym jest interfejs API analizy językowej? (Wersja zapoznawcza)

> [!IMPORTANT]
> Wersja zapoznawcza analizy językowej została wycofana 9 sierpnia 2018 r. W celu przetwarzania i analizy tekstu zalecamy korzystanie z [modułów analizy tekstu w usłudze Azure Machine Learning](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics).

Interfejs API analizy językowej zapewnia dostęp do narzędzi przetwarzania języka naturalnego, które identyfikują strukturę tekstu. Bieżąca wersja udostępnia trzy typy analizy:

- [Oddzielanie wyrazów i tokenów](Sentences-and-Tokens.md)
- [Tagowanie części mowy](POS-tagging.md)
- [Analizowanie fraz](Constituency-Parsing.md)

## <a name="getting-started"></a>Wprowadzenie

Gdy zapoznasz się z głównymi typami analizy wymienionymi powyżej, możesz przyjrzeć się punktowi końcowemu do [uzyskiwania listy dostępnych analizatorów](AnalyzersMethod.md) i ich [nazw](Analyzer-Names.md), a następnie punktowi końcowemu do [analizowania nowych danych wejściowych](AnalyzeMethod.md).
