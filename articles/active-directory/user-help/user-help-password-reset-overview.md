# <a name="reset-your-own-work-or-school-password-overview"></a>Resetuj własnej pracy lub szkołą Przegląd hasła

Nie pamiętasz hasła firmowego lub szkolnego, nigdy nie masz hasła z Twojej organizacji, czy został zablokowany ze swojego konta, informacjom zabezpieczającym i urządzenia przenośnego można użyć do zresetowania hasła firmowego lub szkolnego.

Administrator musi włączyć tę funkcję dla Ciebie można było zarejestrować informacje i zresetować własne hasło. Jeśli nie widzisz **nie pamiętam hasła** opcji, oznacza to, administrator nie włączenia tej funkcji dla Twojej organizacji. Jeśli uważasz, że to jest niepoprawna, skontaktuj się z pomocy technicznej.

>[!Important]
>Ta zawartość jest przeznaczona dla użytkowników. Jeśli jesteś administratorem, więcej informacji na temat konfigurowania i zarządzania środowiskiem usługi Azure Active Directory (Azure AD) możesz znaleźć w [dokumentacji usługi Azure Active Directory](https://docs.microsoft.com/azure/active-directory).
