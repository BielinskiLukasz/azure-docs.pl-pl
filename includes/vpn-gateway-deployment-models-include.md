Obecnie platforma Azure obsługuje dwa modele wdrażania: model wdrażania przy użyciu usługi Azure Resource Manager i model klasyczny. Przed rozpoczęciem konfiguracji należy sprawdzić, czy stosowane są instrukcje dotyczące odpowiedniego modelu wdrażania. Aby uzyskać więcej informacji, zobacz [Understanding deployment models](../articles/resource-manager-deployment-model.md) (Omówienie modeli wdrażania).
