# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Jak zarządzać warstwy cenowej konta usługi Azure Maps

Możesz zarządzać warstwy cenowej konta usługi Azure Maps w witrynie Azure portal. Po utworzeniu [konta](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), można wyświetlić lub zmienić warstwę cenową konta.

Aby uzyskać więcej informacji o wybieraniu warstwy cenowej, po prawej stronie znajduje się pozycja [Wybieranie odpowiedniej warstwy cenowej](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)

## <a name="view-pricing-tier"></a>Wyświetl warstwy cenowej

Aby wyświetlić wybranego warstwę cenową, przejdź do **warstwy cenowej** opcji w menu Ustawienia.

![Widok wybrana warstwa cenowa](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Zmień warstwę cenową

Po utworzeniu konta usługi Azure Maps, można uaktualnić lub obniżyć warstwę cenową dla konta usługi Azure Maps. Aby uaktualnić lub obniżyć wersję, przejdź do **warstwy cenowej** opcji w menu Ustawienia, a następnie wybierz warstwę cenową, która nie jest wyróżniona. Kliknij przycisk Zapisz, aby zapisać wybranego opcji warstwy cenowej.

![Zmień warstwę cenową](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Nie masz do generowania nowych kluczy subskrypcji, uaktualnić lub obniżyć warstwę cenową dla konta usługi Azure Maps.
