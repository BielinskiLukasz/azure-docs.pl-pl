Usługi Azure Cloud Shell można używać do uruchamiania większości poleceń cmdlet i poleceń interfejsu wiersza polecenia programu PowerShell, zamiast instalować lokalnie usługi Azure PowerShell lub CLI. Usługa Azure Cloud Shell to bezpłatna powłoka interaktywna, która ma preinstalowane wspólne narzędzia platformy Azure i jest skonfigurowana do używania z twoim kontem.
Aby uruchomić dowolny kod zawarty w tym artykule w usłudze Azure Cloud Shell, otwórz sesję usługi Cloud Shell, użyj przycisku **Kopiuj** na bloku kodu, aby skopiować kod i wklej go do sesji Cloud Shell za pomocą __klawiszy Ctrl+Shift+V__ w systemie Windows i Linux lub __Cmd+Shift+V__ w systemie macOS. Wklejony tekst nie jest wykonywany automatycznie, naciśnij klawisz **Enter,** aby uruchomić kod.

Istnieje kilka sposobów uruchomienia usługi Cloud Shell:

|  |   |
|-----------------------------------------------|---|
| Kliknij przycisk **Wypróbuj** w prawym górnym rogu bloku kodu. | ![Usługa Cloud Shell w tym artykule](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-powershell-try-it.png) |
| Otwórz usługę Cloud Shell w swojej przeglądarce. | [![https://shell.azure.com/powershell](./media/vpn-gateway-cloud-shell-powershell/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Kliknij przycisk **Powłoki chmury** w menu w prawym górnym rogu witryny Azure portal. | [![Usługa Cloud Shell w portalu](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-menu.png)](https://portal.azure.com) |
