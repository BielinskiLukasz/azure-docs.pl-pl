# <a name="what-is-appsource"></a>Co to jest usługa AppSource
Usługa AppSource umożliwia partnerom firmy Microsoft stał się ich technologii i usług wykrywalny dla klientów za pośrednictwem sklepu online obsługiwane przez firmę Microsoft. Jako wydawca AppSource mogą angażować użytkowników biznesowych w usłudze AppSource, aby znaleźć, spróbuj i uzyskać aplikacje SaaS line-of-business, a także implementacji usługi kierować wynikami biznesowymi i zmniejszyć wartość godziny: 

| Potrzeba klienta       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Szukasz rozwiązania biznesowe, które działają z produktami firmy Microsoft już używają**    | Umożliwia partnerom umożliwia rozszerzanie aplikacji w chmurze firmy Microsoft i technologii usług i aplikacji innych firm. |
| **Możliwość, aby łatwo znaleźć odpowiednią usługę rozwiązania lub wdrożenia po prawej stronie.**     | Udostępnia kompleksowym do uzyskania wersji próbnej i aplikacjami i usługami, dodatków i wiele innych. |
| **Rozwiązanie line-of-business branżowe na swoje wyzwania biznesowe specyficzne**   | Oferuje Zakończono branży end-to-end rozwiązania pomagające w wielu branżach adres określonych wymagań. |
| **Aplikacje, aby zwiększyć produktywność, wydajności i analiz biznesowych**       | Udostępnia aplikacje biznesowe, łącznie z działem obsługi klienta, HR, operacji i wiele innych. |
| **Doświadczeni pomocy partnera ułatwia dostosowanie aplikacji do ich unikatowych sytuacji**     | Podaj katalog konsultacji oferty usług dla rozwiązań opartych na Dynamics 365, Power BI, PowerApps i aplikacji innych firm 3, które są dostępne w usłudze AppSource, aby ułatwić użytkownikom firm Znajdź usługi doradcze przeznaczonych do dostarczenia przewidywalne wyniki. |

## <a name="appsource-publishing"></a>Publikowanie w usłudze AppSource
Za pomocą usługi AppSource możesz wyświetlić listę aplikacji lub konsultacji oferta, która pomaga w spełniających potrzeby klientów podczas korzystania z produktów firmy Microsoft, takich jak usługi Office 365, Dynamics 365, Power BI i Power Apps. Następujące oferty są dostępne w witrynie Marketplace usługi AppSource: 
*   **Dynamics 365 for Finance and Operations**: (ERP), rozwiązanie do planowania zasobów przedsiębiorstwa. Zróżnicowane rozwiązania są średnie do dużych firm w wersji Enterprise i Business Edition dla małych i średnich firm.
*   **Dynamics 365 for Customer Engagement**: rozwiązanie zarządzania (CRM) relacji klienta, które zawiera sprzedaży, Customer Service, usługi terenowe i automatyzacja obsługi projektowej aplikacji.
*   **Zarządzana usługa w programie Dynamics NAV**: Microsoft **Dynamics NAV** jest pakiet oprogramowania (ERP) w przypadku średnich organizacji do planowania zasobów przedsiębiorstwa. Ta usługa zapewnia funkcje specjalne produkcji, dystrybucji, dla instytucji rządowych, handlu detalicznego i inne branże.
*   **Usługa Power BI**: rozwiązaniu wizualizacji do analizy biznesowej.
*   **Konsultacji oferty**: usługi dostarczone przez firmę Microsoft partnerami, aby pomóc klientom zrozumieć, wypróbuj i wykonywania wyspecjalizowanych rozwiązań technologicznych.
*   **Pakiet Cortana Intelligence**: pakiet Cortana Intelligence Suite jest zbiorem niezależne, ale w pełni zintegrowane danych i narzędzi analitycznych platformy oferowanych na platformie Azure.
*   **Usługi Office 365**: nowoczesnych miejscach pracy w chmurze. Współpraca za darmo z programu Microsoft Word, PowerPoint, Excel i programu OneNote.

## <a name="azure-active-directory-integration"></a>Integracja z usługą Azure Active Directory
Niektóre usługi AppSource StoreFront — aplikacje są wymagane, aby zintegrować z usługą Azure Active Directory (Azure AD) do opublikowania. Integracja aplikacji z usługą Azure AD jest dobrze udokumentowana; Firma Microsoft udostępnia wiele zestawów SDK i dodatkowe zasoby, zgodnie z wymaganiami. 

| Nazwa oferty | Wymaga integracji z usługą Azure AD | Uwagi |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Nie | Ofertę na podstawie licencji nie wymaga Integracja z usługą Azure AD |
| Dynamics 365 for Finance | Nie | Ofertę na podstawie licencji nie wymaga Integracja z usługą Azure AD |
| Dynamics 365 for Customer Engagement | Nie | Ofertę na podstawie licencji nie wymaga Integracja z usługą Azure AD |
| Usługa zarządzana systemu Dynamics NAV | Nie | Ofertę na podstawie licencji nie wymaga Integracja z usługą Azure AD |
| Power BI | Nie | Ofertę na podstawie licencji nie wymaga Integracja z usługą Azure AD |
| Konsultacji oferty | Nie | Usługi są koordynowany przez partnera, a nie przy użyciu środowiska sieci web |
| Cortana Intelligence | Yes | Usługa Azure AD jest najlepszym rozwiązaniem, aby zapewnić bezproblemową obsługę klientów, doświadczenia i dyski, które prowadzi najwyższej jakości |
| Aplikacje SaaS (wcześniej aplikacji sieci Web) | Preferowane | Usługa Azure AD jest najlepszym rozwiązaniem, aby zapewnić bezproblemową obsługę klientów, doświadczenia i dyski, które prowadzi najwyższej jakości |
