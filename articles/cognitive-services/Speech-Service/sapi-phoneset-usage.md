# <a name="sapi-phone-set-usage"></a>Użycie zestawu telefonu SAPI

Firma Microsoft definiuje zestaw telefonów SAPI dla siedmiu języków (EN-US, fr-FR, de-DE, es-ES, ja-JP, zh-CN, zh-TW). Zestaw telefonu SAPI jest stosunkowo łatwy w użyciu i należy pamiętać, szczególnie w przypadku głośnika natywnego. Na przykład ja-JP Użyj kana jako numeru telefonu, zh-CN Użyj PinYin jako numeru telefonu. Czasami nie jest to jednak wystarczająco wydajne, ale przykładowo zestaw SAPI z firmy fr-FR nie może reprezentować łącznika francuskiego.
