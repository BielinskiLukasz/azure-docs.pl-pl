# <a name="bing-spell-check-sdk-samples"></a>Przykłady SDK sprawdzania pisowni usługi Bing
Scenariusze zawierają przykłady sprawdzania pisowni usługi Bing który:
1. Wyszukaj błędnie zapytania i przeanalizować odpowiedzi.
2. Wyzwalanie odpowiedzi na błąd z interfejsu API.

SDK sprawdzania pisowni usługi Bing upewnij kontekstowe gramatyki i sprawdzanie pisowni łatwo dostępne w następujących językach programowania:
* Rozpoczynanie pracy z [próbek .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.SpellCheck/1.2.0)
    * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Language/SpellCheck) definicje i zależności.
* Rozpoczynanie pracy z [Node.js — przykłady](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [bibliotek Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/spellCheck) definicje i zależności.
* Rozpoczynanie pracy z [próbek Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [bibliotek Java](https://github.com/Azure/azure-sdk-for-java/tree/master/cognitiveservices/azure-spellcheck) definicje i zależności.
* Rozpoczynanie pracy z [próbek Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [bibliotek języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-language-spellcheck) definicje i zależności.

Przykłady zestawu SDK dla każdego języka obejmują plik ReadMe o szczegółowe informacje o wymaganiach wstępnych oraz instalowania/uruchomiony próbek.
