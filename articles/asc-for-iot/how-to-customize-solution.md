# <a name="customize-your-azure-security-center-for-iot-solution"></a>Dostosowywanie Azure Security Center na potrzeby rozwiązania IoT

W tym przewodniku dowiesz się, jak dostosować różne ustawienia w Azure Security Center na potrzeby IoT.

> [!div class="checklist"]
> * Skonfiguruj zalecenia dotyczące rozwiązania
> * Zmień ustawienia

## <a name="change-settings"></a>Zmień ustawienia

Zarządzaj ustawieniami usługi Azure Security Center dla IoT:

Na IoT Hub przejdź do bloku Przegląd zabezpieczeń w lewym górnym rogu, zobacz "Ustawienia", aby skonfigurować Azure Security Center ustawień usługi IoT, wykonaj następujące czynności:

1. Otwórz **IoT Hub** w Azure Portal.
1. W menu po lewej stronie **zabezpieczenia**, wybierz i Otwórz **Przegląd**.
1. W obszarze Ustawienia wybierz ustawienie rozwiązania, które chcesz zmienić.
1. Pamiętaj, aby zawsze klikać pozycję **Zapisz** w górnej części ekranu ustawienia, aby zapisać zmiany ustawień.

## <a name="configure-solution-recommendations"></a>Skonfiguruj zalecenia dotyczące rozwiązania

Aby skonfigurować Azure Security Center na potrzeby rozwiązań IoT, wykonaj następujące czynności:

1. Otwórz **IoT Hub** w Azure Portal.
1. Wybierz i Otwórz pozycję **Przegląd** w obszarze **zabezpieczenia** w menu po lewej stronie.
1. W obszarze Ustawienia wybierz pozycję **zalecana konfiguracja** w menu po lewej stronie.
1. Wyłącz/Włącz zalecenia dotyczące rozwiązania odpowiednie dla Twojej organizacji i przepływów pracy.
1. Wybierz pozycję **Zapisz** w górnej części ekranu, aby zapisać wybrane opcje.
