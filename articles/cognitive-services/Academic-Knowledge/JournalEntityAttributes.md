# <a name="journal-entity"></a>Jednostki dziennika

<sub> * Następujące atrybuty są specyficzne dla jednostki dziennika. (Ty = "2") </sub>

Name (Nazwa)    |Opis                            |Typ       | Operacje
------- | ------------------------------------- | --------- | ----------------------------
Identyfikator      |Identyfikator jednostki                              |Int64      |Równa się
DJN     |Znormalizowaną nazwę dziennika                |Ciąg     |brak
JN      |Nazwa wyświetlana arkusza                   |Ciąg     |Równa się
DW      |Liczba całkowita cytatu dziennika           |Int32      |brak  
ECC     |Liczba całkowita cytatu szacowany dziennika |Int32      |brak
