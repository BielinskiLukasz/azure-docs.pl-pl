# <a name="azure-security-overview"></a>Przegląd zabezpieczeń platformy Azure

Witryna informacji o zabezpieczeniach Microsoft Azure w systemie Azure.com jest miejscem, w którym znajdują się odpowiedzi na pytania zabezpieczające dotyczące platformy Azure. Jeśli interesuje Cię zgodność z platformą Azure i sposób, w jaki platforma Azure zabezpiecza własną infrastrukturę, odwiedź [Centrum zaufania firmy Microsoft](https://microsoft.com/en-us/trustcenter/default.aspx).

Wiemy, że zabezpieczenia to zadanie jedno w chmurze i jak ważne jest, aby znaleźć dokładne i aktualne informacje o zabezpieczeniach platformy Azure. Jednym z najlepszych powodów używania platformy Azure dla aplikacji i usług jest skorzystanie z zalet szerokiej gamy dostępnych narzędzi i funkcji zabezpieczeń. Te narzędzia i funkcje ułatwiają tworzenie bezpiecznych rozwiązań na bezpiecznej platformie Azure.

Ta strona i ta witryna zostaną rozłączone jako nowe usługi platformy Azure w trybie online, a istniejące usługi zostaną zaktualizowane. Nasze oferty zabezpieczeń są stale ulepszane i będziemy udostępniać nam szczegółowe informacje techniczne dotyczące tych ulepszeń. Zalecamy, aby regularnie wrócić do tej witryny i zobaczyć, co nowego!

## <a name="using-the-azure-security-information-site"></a>Korzystanie z witryny Azure Security Information

Po lewej stronie strony zostanie wyświetlona kolekcja kategorii tematów dotyczących zabezpieczeń. Kliknij lub naciśnij temat, aby zobaczyć artykuły, które mamy dla tej kategorii.

Każda kategoria zawiera przegląd artykułu. Celem tego artykułu jest przedstawienie podsumowania platformy Azure w zakresie zabezpieczeń dla tej kategorii. Artykuły z omówieniem pomagają zrozumieć produkty, usługi i funkcje dostępne na platformie Azure, których można użyć w celu zwiększenia bezpieczeństwa wdrożeń. Te artykuły z omówieniem zawierają liczne linki do zawartości technicznej, dzięki czemu można dowiedzieć się więcej o tematach i sposobach ich implementacji.

Witryna informacji o zabezpieczeniach platformy Azure obecnie rozwiązuje następujące kategorie:

* Zasoby
* Bezpieczeństwo sieci
* Zabezpieczenia magazynu
* Zabezpieczenia maszyn wirtualnych
* Ochrona punktu końcowego
* Zarządzanie zabezpieczeniami i monitorowanie ich
* Zarządzanie tożsamościami
* Usługi PaaS
* Specyficzne dla branży
* Architektura

> [!NOTE]
> Wszystkie artykuły w witrynie informacji o zabezpieczeniach platformy Azure są wyświetlane w obszarze nawigacji po lewej stronie. Jeśli chcesz zobaczyć wszystkie artykuły, kliknij lub naciśnij pozycję **Zobacz więcej** u dołu listy kategorii.
