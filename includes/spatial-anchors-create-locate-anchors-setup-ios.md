## <a name="setting-up-the-library"></a>Konfigurowanie biblioteki

Invoke Start (), aby umożliwić sesji przetwarzania danych środowiska.

Aby obsłużyć zdarzenia zgłoszone przez sesję, `delegate` należy ustawić właściwość sesji na obiekt, taki jak widok. Ten obiekt musi implementować protokół SSCCloudSpatialAnchorSessionDelegate.
