#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Aby zainstalować kartę StorSimple dla programu SharePoint
1. Skopiuj instalatora na serwer frontu sieci Web (WFE), który jest również skonfigurowany do uruchamiania aplikacji sieci Web Administracji Centralnej programu SharePoint. 
2. Zaloguj się na serwerze WFE za pomocą konta z uprawnieniami administratora.
3. Kliknij dwukrotnie instalatora. Zostanie uruchomiony Kreator konfiguracji storaproszka dla programu SharePoint. Kliknij **przycisk Dalej,** aby rozpocząć instalację.
   
    ![Strona startowa ustawienia karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. Na stronie konfiguracji konfiguracji StorSimple adapter for SharePoint wybierz lokalizację instalacji, wpisz adres IP interfejsu sieciowego DATA 0 na urządzeniu StorSimple, a następnie kliknij przycisk **Dalej**. 
   
    ![Strona konfiguracji karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Na stronie potwierdzenia konfiguracji kliknij pozycję **Zainstaluj**.
   
    ![Strona potwierdzenia konfiguracji karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kliknij **przycisk Zakończ,** aby zamknąć Kreatora instalacji.
   
    ![StorSimple konfiguracja karty gotowej strony](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Otwórz stronę Administracja centralna programu SharePoint. Powinna zostać wyświetlona grupa konfiguracji StorSimple zawierająca łącza StorSimple Adapter for SharePoint.
8. Przejdź do następnego kroku: [Konfigurowanie SPZ](#configure-rbs).
