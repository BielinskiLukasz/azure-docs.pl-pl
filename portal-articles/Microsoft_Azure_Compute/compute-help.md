# <a name="virtual-machines-documentation"></a>Dokumentacja dotycząca maszyn wirtualnych
Usługa Azure Virtual Machines zapewnia bezpieczną infrastrukturę zwirtualizowaną na żądanie o dużej skali. Można utworzyć maszyny wirtualne z systemem Windows lub maszyny wirtualne z systemem Red Hat, Ubuntu albo wybraną dystrybucją systemu Linux. Aby dowiedzieć się, jak to zrobić, zobacz:

- [Tworzenie maszyny wirtualnej z systemem Windows](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot)
- [Tworzenie maszyny wirtualnej z systemem Linux](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot)
