> [!NOTE]
> W grudniu 2021 wersje programu SSMS przed 18,6 nie będą już uwierzytelniane za pomocą Azure Active Directory za pomocą usługi MFA.
>
> Aby nadal korzystać z uwierzytelniania Azure Active Directory za pomocą usługi MFA, wymagany jest program [SSMS 18,6 lub nowszy](/sql/ssms/download-sql-server-management-studio-ssms).
