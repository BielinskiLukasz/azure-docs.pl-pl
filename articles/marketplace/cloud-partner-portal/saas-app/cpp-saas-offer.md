# <a name="azure-saas-application-offer"></a>Oferta Azure aplikacji SaaS

[Portalu Cloud Partner](https://cloudpartner.azure.com/) już nie obsługuje tworzenia i zarządzania ofertami aplikacji SaaS.  Zamiast tego, ta funkcja została przeniesiona do firmy Microsoft [portalu Centrum partnerskiego](https://partner.microsoft.com/pcv/).  Aby uzyskać więcej informacji, zobacz [Tworzenie nowej oferty SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Kolejne kroki

Przed utworzeniem nowej oferty aplikacji SaaS, powinien przeczytać [aplikacji SaaS oferują Podręcznik publikowania](../../marketplace-saas-applications-technical-publishing-guide.md) i przejrzyj kroki portalu Centrum partnerskiego na liście [Lista kontrolna tworzenia oferty](../../partner-center-portal/offer-creation-checklist.md).
