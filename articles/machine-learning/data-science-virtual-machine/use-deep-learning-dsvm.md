# <a name="using-the-deep-learning-virtual-machine"></a>Za pomocą maszyny wirtualnej do uczenia głębokiego

Po aprowizowaniu usługi głębokiego Virtual Machine Learning (DLVM), możesz rozpocząć tworzenie modeli sieci neuronowej do kompilowania aplikacji sztucznej Inteligencji w domenach, takich jak przetwarzanie obrazów i language understanding. 

Istnieje szereg narzędzi dostępnych na maszynie Wirtualnej z głębokiego uczenia dla sztucznej Inteligencji. [Strony platform uczenia głębokiego i sztucznej Inteligencji](dsvm-deep-learning-ai-frameworks.md) zawiera szczegółowe informacje dotyczące sposobu używania tych narzędzi. 

## <a name="deep-learning-tutorials-and-walkthroughs"></a>Głębokiego uczenia, samouczki i przewodniki

Oprócz przykładów oparte na strukturze zbiór — wszechstronne wskazówki udostępniane są również że zweryfikowany w DLVM. Przewodniki te pomagają szybko Rozpocznij korzystanie z usługi opracowywania aplikacji uczenia głębokiego w domenach, takich jak obraz i tekst/language understanding. Więcej samouczków end-to-end w różnych domenach i technologii, będą w dalszym ciągu można dodać.   


- [Uruchamianie sieci neuronowych dla różnych platform](https://github.com/ilkarman/DeepLearningFrameworks): Kompleksowy przewodnik, który pokazuje, jak przeprowadzić migrację kodu z jednej struktury. Ilustruje też sposób porównywania modelu i uruchamianie wydajności w czasie dla platform. 

- [Przewodnik rozwiązania end-to-end do wykrywania produktów w obrębie obrazów](https://github.com/Azure/cortana-intelligence-product-detection-from-images): Wykrywanie obrazu to technika, który można zlokalizować i sklasyfikować obiekty w obrębie obrazów. Ta technologia może potencjalnie doprowadzić ogromne korzyści w wielu dziedzinach biznesowych realnym. Na przykład sprzedawcy detaliczni, można użyć tej techniki ustalenie produktu, który klientów została pobrana z półki. Informacje te pomagają z kolei magazyny Zarządzanie spis produktów. 

- [Uczenie głębokie dla audio](https://blogs.technet.microsoft.com/machinelearning/2018/01/30/hearing-ai-getting-started-with-deep-learning-for-audio-on-azure/) ten samouczek pokazuje sposób trenowania modelu uczenia głębokiego, wykrywania zdarzeń audio na [dataset miejskich dźwięki](https://serv.cusp.nyu.edu/projects/urbansounddataset/urbansound8k.html) i omówiono sposób pracy z danymi audio.

- [Klasyfikacja dokumentów tekstowych](https://github.com/anargyri/lstm_han): W tym przewodniku pokazano, jak tworzyć i szkolenie z dwóch różnych sieci neuronowych architektury: Hierarchiczna sieć sieci uwagi i długi krótki okres pamięci (LSTM). Te sieci neuronowych na użytek interfejsu API biblioteki Keras głębokie uczenie do klasyfikowania dokumentów tekstowych. Biblioteki Keras jest frontonu do trzech spośród najbardziej popularnych struktur do uczenia głębokiego: Microsoft Cognitive Toolkit, TensorFlow, a Theano.
