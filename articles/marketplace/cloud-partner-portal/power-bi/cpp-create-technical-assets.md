# <a name="create-power-bi-app-offer-technical-assets"></a>Tworzenie oferty aplikacji Power BI, zasoby techniczne

Podstawowy zasobów technicznych, potrzebne dla tego typu oferty jest aplikacja Power BI. Aplikacja Power BI to kolekcja podstawowe zestawy danych, raportów lub pulpitów nawigacyjnych i opcjonalnie podłączonych usług i osadzone zestawów danych. Powstała się od typu zasobu wcześniej znana jako [pakiet zawartości](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction). Aby uzyskać więcej informacji na temat tworzenia tego typu aplikacji, zobacz [co to są aplikacje szablonu usługi Power BI?](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtain-an-installation-url"></a>Uzyskiwanie adresu URL instalacji

Można tworzyć tylko w aplikacji Power BI [usługi Power BI](https://powerbi.microsoft.com/). Aby rozpocząć, możesz zarejestrować się za pomocą [licencji usługi Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Po utworzeniu i przetestuj swoją aplikację z usługi w usłudze Power BI należy zapisać adres URL instalacji aplikacji, która generuje programie Power BI. Na koniec Podaj adres URL na [informacje techniczne](./cpp-technical-info-tab.md) kartę formularz zgłoszeniowy portalu Cloud Partner.
 

## <a name="next-steps"></a>Kolejne kroki

Po po utworzeniu i przetestowaniu aplikacji usługi Power BI i utworzeniu skojarzonych oferty, możesz [publikowanie oferty aplikacji Power BI](./cpp-publish-offer.md).
