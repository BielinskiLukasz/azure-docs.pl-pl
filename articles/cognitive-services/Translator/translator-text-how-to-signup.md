# <a name="how-to-sign-up-for-the-translator-text-api"></a>Jak zarejestrować się dla interfejsu API Translator tekstu

## <a name="sign-in-to-the-azure-portal"></a>Logowanie się do witryny Azure Portal

- Nie masz jeszcze konta? Można utworzyć [bezpłatne konto](https://azure.microsoft.com/free/) wypróbować bez dodatkowych opłat.
- Masz już konto? [Rejestrowanie](https://ms.portal.azure.com/)

## <a name="create-a-subscription-to-the-translator-text-api"></a>Utwórz subskrypcję do interfejsu API Translator tekstu

Po zalogowaniu się do portalu w przypadku tworzenia subskrypcji do interfejsu API Translator tekstu w następujący sposób:

1. Wybierz pozycję **+ Utwórz zasób**.
1. W **wyszukiwania portalu Marketplace** polu wyszukiwania, wprowadź **tekst Translator** i wybierz go z wyników.
1. Wybierz **Utwórz** można zdefiniować szczegóły dla subskrypcji.
1. Z **warstwa cenowa** wybierz warstwę cenową, która najlepiej odpowiada Twoim potrzebom.
    1. Każda subskrypcja ma warstwę bezpłatna. Warstwa bezpłatna ma te same funkcje i funkcje, ponieważ płatną planów i nie wygasa.
    1. Może mieć tylko jedną bezpłatną subskrypcję dla Twojego konta.
1. Wybierz **Utwórz** na zakończenie tworzenia subskrypcji.

## <a name="authentication-key"></a>Klucz uwierzytelniania

Gdy zarejestrujesz się tekstu Translator otrzymasz klucz dostępu do spersonalizowanych unikatowy do subskrypcji. Ten klucz jest wymagany dla każdego wywołania interfejsu API tekst translatora.

1. Pobrać klucz uwierzytelniania, wybierając najpierw odpowiednią subskrypcję.
1. Wybierz **klucze** w **zarządzanie zasobami** sekcji Szczegóły Twojej subskrypcji.
1. Skopiuj albo klawisze dla Twojej subskrypcji.
