
1. 参照してください、 [Azure Portal]します。 クリックして **すべてを参照** > **Mobile Apps** > 作成したバックエンドです。 モバイル アプリの設定 [ **クイック スタート** > **Android)**します。  **構成、クライアント アプリケーションことはできます**, 、クリックして **ダウンロード**します。 これにより、バックエンドに接続するように事前に構成されたアプリ用の完成した Android プロジェクトがダウンロードされます。 

2. 使用してプロジェクトを開く **Android Studio**, を使用して、 **(Eclipse ADT, Gradle,) プロジェクトのインポート**します。 JDK エラーを回避するために、このインポート オプションを必ず選択してください。

3. キーを押して、 **'app' を実行** 、プロジェクトをビルドし、Android のシミュレーターでアプリケーションを開始] ボタンをクリックします。

4. アプリケーションで、意味のあるテキストをなど入力 _に関するチュートリアルを完了_ [追加] ボタンをクリックします。 これにより、事前にデプロイした Azure バックエンドに POST 要求が送信されます。 バックエンドは要求から取得したデータを TodoItem SQL テーブルに挿入し、新しく格納されたアイテムに関する情報をモバイル アプリに返します。 モバイル アプリでは、このデータが一覧に表示されます。
