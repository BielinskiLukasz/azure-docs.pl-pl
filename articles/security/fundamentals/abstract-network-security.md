# <a name="azure-network-security"></a>Zabezpieczenia sieci na platformie Azure
## <a name="abstract"></a>Abstrakcyjny
Usługa Azure Network Services maksymalizuje elastyczność, dostępność, odporność, bezpieczeństwo i integralność przez projektowanie. Ten oficjalny dokument zawiera szczegółowe informacje o funkcjach sieciowych platformy Azure. Opisano w nim również, w jaki sposób klienci mogą korzystać z natywnych funkcji zabezpieczeń na platformie Azure w celu ochrony zasobów informacji.
Zaznaczeni odbiorcy tego oficjalnego dokumentu obejmują:
- Kierownicy techniczne, Administratorzy sieci i deweloperzy szukający rozwiązań zabezpieczeń, które są dostępne i obsługiwane na platformie Azure.
- MŚP lub dyrektorzy procesów roboczych, którzy chcą uzyskać ogólne omówienie technologii i usług platformy Azure, które odnoszą się do zabezpieczeń sieci w chmurze publicznej platformy Azure.
