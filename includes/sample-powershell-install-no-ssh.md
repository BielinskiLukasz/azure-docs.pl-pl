Ten przykład wymaga programu Azure PowerShell Az 1.0 lub nowszego. Uruchom, `Get-Module -ListAvailable Az` aby zobaczyć, które wersje są zainstalowane. Jeśli chcesz zainstalować, zobacz [Instalowanie modułu programu Azure PowerShell](/powershell/azure/install-az-ps). 

Uruchom [connect-AzAccount,](/powershell/module/az.accounts/connect-azaccount) aby zalogować się na platformie Azure.
