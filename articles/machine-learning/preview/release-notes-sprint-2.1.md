# <a name="sprint-2-qfe---december-2017"></a>Przebieg 2 (QFE) — grudzień 2017 

#### <a name="version-number-01171115323"></a>Numer wersji: 0.1.1711.15323

>Poniżej przedstawiono sposób [Znajdź numer wersji](known-issues-and-troubleshooting-guide.md).

Witamy w wersji QFE (Quick Fix Engineering) trzeci aktualizacji z usługi Azure Machine Learning Workbench. Ta wersja jest wersji pomocniczej. On rozwiązuje kilka problemów telemetrii i pomaga zespołowi produktu w celu lepszego zrozumienia sposobu używania produktu. Wiedza może przejść do przyszłych wysiłki mające na celu poprawę środowiska produktu. 

Ponadto istnieją dwie ważne aktualizacje:

- Rozwiązane usterki w przygotowanie bazy danych, który uniemożliwił inspektora serii czas wyświetlania w pakietach przygotowywania danych.
- W narzędziu wiersza polecenia nie trzeba być właścicielem subskrypcji platformy Azure do udostępnienia klastrów usługi Machine Learning obliczeniowe ACS.
