# <a name="connect-one-identity-safeguard"></a>Połącz jedną ochronę tożsamości
Łącznik danych One Identity Safeguard rozszerza standardowe łączniki wspólnego formatu zdarzeń (CEF) za pomocą pulpitów nawigacyjnych specyficznych dla sesji uprzywilejowanych. Użyj tego łącznika, aby łatwo rozpocząć korzystanie ze zdarzeń generowanych przez urządzenie do wizualizacji, alertów, badań i innych.


## <a name="prerequisites"></a>Wymagania wstępne

- Uprawnienia do odczytu i zapisu w obszarze roboczym usługi Azure Sentinel.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Łączenie usługi Azure Sentinel z zabezpieczeniem jednej tożsamości

1. W witrynie Azure portal przejdź do łączników **usługi Azure Sentinel** > **Data,** a następnie wybierz łącznik One Identity **Safeguard.**

2. Wybierz **pozycję Otwórz stronę łącznika**.

3. Postępuj zgodnie z instrukcjami na stronie **Ochrona jednej tożsamości.**
