## <a name="clone-the-repository-to-your-machine"></a>Klonowanie repozytorium na komputerze

Za pomocą repozytorium utworzonego na koncie usługi GitHub Sklonuj projekt na komputerze lokalnym przy użyciu poniższego polecenia.

```bash
git clone https://github.com/<YOUR_GITHUB_ACCOUNT_NAME>/my-first-static-web-app.git
```

Pamiętaj o zamianie na `<YOUR_GITHUB_ACCOUNT_NAME>` nazwę użytkownika usługi GitHub.
