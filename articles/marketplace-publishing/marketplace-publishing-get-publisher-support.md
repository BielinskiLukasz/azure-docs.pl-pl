# <a name="accessing-publisher-support-for-the-azure-marketplace"></a>Obsługa wydawcy podczas uzyskiwania dostępu do portalu Azure Marketplace
Firma Microsoft udostępnia obsługę szerokiej gamy produktów i usług. Zapewnienie, że kwerenda znajduje zespół pomocy technicznej prawym ważne jest, aby zapewnić odpowiednią i czas odpowiedzi. Rozważ następujące scenariusze, które powinny pomóc w kierować kwerendy do odpowiednich zespołu:

* Są wydawcy w portalu Azure Marketplace, nadal tworzenia maszyny Wirtualnej należy współpracować z Twojej dołączaniu rozwiązania Microsoft lub obliczeniowe zespołów.
* Jeśli jesteś wydawcy w witrynie Marketplace i masz pytanie klienta, poinformuj klientów, aby zażądać pomocy technicznej za pośrednictwem łącza pomocy technicznej na https://portal.azure.com/.
* Jeśli jesteś wydawcy w witrynie Marketplace i masz pytanie dotyczące Twojej aplikacji lub usługi, nadal odczytu poniżej.

W tym artykule opisano, jak do:

* Zaloguj się do swojego konta Microsoft.
* Przejdź do strony pomocy technicznej wydawcy.
* Wybierz typ problemu i kategoria.
* Żądanie pomocy technicznej.
* Podaj informacje kontaktowe.
* Opisz problem.

## <a name="1-sign-in-to-your-microsoft-account"></a>1. Zaloguj się do swojego konta Microsoft
Przejdź do https://publish.windowsazure.com/ i zaloguj się przy użyciu poświadczeń konta Microsoft.

  ![Ekran logowania][1]

## <a name="2-navigate-to-the-publisher-support-pages"></a>2. Przejdź do strony pomocy technicznej wydawcy
Wybierz z menu górnym prawym **Get Support**.

  ![Uzyskiwanie pomocy technicznej][2]

## <a name="3-select-the-problem-type-and-category"></a>3. Wybierz typ problemu i kategoria
![Typ problemu i kategoria][3]

Typy problemów dostępne są następujące:

* **Zarządzanie kontami**
* **Raporty i szczegółowych informacji**
* **Certyfikacja obrazu maszyny Wirtualnej**
* **Zarządzanie klientami**
* **Zarządzanie oferty**
* **Payouts**
* **Centrum deweloperów**
* **Inne**

## <a name="4-request-support"></a>4. Żądanie obsługi
Wybierz typ problemu i kategoria, kliknij **żądanie uruchomienia** przycisku.

![Uruchomić obsługi][4]

## <a name="5-provide-contact-information-and-describe-the-problem"></a>5. Podaj informacje kontaktowe i opisz problem
Opisz problem pełni jako użytkownik może, a następnie podaj wszystkie pliki (zrzuty ekranu, śladów sieciowych itd.), które mogą być pomocne w rozwiązaniu problemu.

![Opisz problem][5]

## <a name="6-submission-confirmation"></a>6. Potwierdzenie przesyłania
Po przesłaniu problemu firma Microsoft będzie otrzymanie w ciągu 24 godzin i przypisz problemu, priorytetu i wagi. Dedykowany zespół pracowników działu pomocy technicznej będzie pomagać w jego rozwiązanie zgodnie z naszym czasy odpowiedzi umową dotyczącą poziomu usług.

![Potwierdzenie][6]

* Jeśli potrzebujesz Porozmawiaj z nami o problem, należy użyć numer potwierdzenia wszystkich zgodność.
* Postęp można wyświetlać na problem w dowolnym momencie.

## <a name="next-steps"></a>Następne kroki
Aby dowiedzieć się, jak rozwiązywanie określonych problemów, zobacz [Rozwiązywanie typowych problemów w witrynie Azure Marketplace](marketplace-publishing-support-common-issues.md).
