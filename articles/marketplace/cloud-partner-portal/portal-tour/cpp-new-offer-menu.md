# <a name="new-offer-menu"></a>Menu nowej oferty

**Nowe menu oferta** umożliwia partnerom Tworzenie wystąpień typów ofert obsługiwanych przez [Portal Cloud partner](https://cloudpartner.azure.com).  Po wybraniu elementu menu **Nowa oferta** na pasku nawigacyjnym po lewej stronie zostanie wyświetlona następująca lista typów ofert.  Wybranie jednego z tych typów rozpoczyna proces tworzenia oferty i publikowania.

### <a name="next-steps"></a>Następne kroki

Jeśli tworzysz nową ofertę, możesz użyć poprzedniej tabeli, aby przejść do sekcji oferta w celu uzyskania wskazówek dotyczących tworzenia i publikowania oferty.  W przeciwnym razie samouczek portalu jest kontynuowany z badaniem [strony wszystkie oferty](./cpp-all-offers-page.md).
