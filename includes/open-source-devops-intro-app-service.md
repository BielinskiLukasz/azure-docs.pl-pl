[Usługa Azure App Service](/azure/app-service/overview) umożliwia tworzenie i hostowania kodu. Ten kod może mieć postać aplikacji sieci web, zaplecza mobilnego i interfejsów API RESTful. Korzystając z usługi App Service, można tworzyć kod przy użyciu wybranego języka programowania bez zarządzania infrastrukturą. Usługa App Service obsługuje zarówno systemy Windows, jak i Linux. Obsługiwane są automatyczne wdrożenia z dowolnego repozytorium Git, w tym GitHub i Azure DevOps.
