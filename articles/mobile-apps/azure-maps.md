# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Interfejsy API geoprzestrzenne umożliwiające dodawanie map, analiz przestrzennych i rozwiązań do mobilności za pomocą Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) zapewnia natywną platformę lokalizacyjną w chmurze platformy Azure, oferuje portfolio usług geoprzestrzennych, które używają danych do odwzorowania w celu zapewnienia dokładnego kontekstu geograficznego dla aplikacji sieci Web i mobilnych. Umożliwia ona deweloperom, przedsiębiorstwom i niezależnym dostawcom oprogramowania Tworzenie aplikacji obsługujących lokalizacje oraz rozwiązań IoT, mobilności, logistycznych i śledzenia zasobów.

## <a name="azure-maps-features"></a>Funkcje Azure Maps
- Funkcja **Search** zawiera szczegółowe informacje o pobliżu punktów zainteresowania (punkt POI), miejscach renderowania na mapie oraz geokodowych (lub odwróconych geokodowych) adresach w celu uzyskania odpowiednich współrzędnych szerokości geograficznej.
- Zwiększ czas podróży i unikaj korków przy użyciu **ruchu**.
- Integruj czytelne i łatwe w odczytaniu **mapy** w rozwiązaniu.
- **Routing** do prezentowania najkrótszych lub najszybszych dostępnych tras — do wielu miejsc docelowych w danym momencie lub zapewniają wyspecjalizowane trasy i fakty dla podejść, rowerzystów i pojazdów komercyjnych.
- **Strefy czasowe** ułatwiają użytkownikom wyświetlanie informacji o czasie, w którym znajduje się w dowolnym miejscu na świecie.
- **Operacje przestrzenne** usprawniające analizę lokalizacji.
- **Geolokalizacja** do wyszukiwania kraju w adresie IP.
- **Usługa danych** do przekazywania i przechowywania danych geoprzestrzennych do użycia z operacjami przestrzennymi i kompozycjami obrazów.
- **Mobilność** zapewnia analizę lokalizacji w czasie rzeczywistym na publicznych usługach tranzytowych, w tym informacje o zatrzymaniu, trasie i szacunkach czasu podróży.
