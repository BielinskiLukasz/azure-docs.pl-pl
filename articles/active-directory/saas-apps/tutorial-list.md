# <a name="saas-application-integration-with-azure-active-directory"></a>Integracja aplikacji SaaS w usłudze Azure Active Directory

Ułatwia integrowanie z obsługą chmury [(SaaS)](https://azure.microsoft.com/overview/what-is-saas/) aplikacji za pomocą usługi Azure Active Directory, opracowaliśmy kolekcji samouczków, które umożliwia przeprowadzenie konfiguracji.

Gdzie mamy wskazówki inicjowania obsługi administracyjnej użytkowników, wprowadzono samouczek aplikacji do aprowizowania użytkowników w prawej kolumnie obok samouczek aplikacji.

Aby uzyskać pełną listę aplikacji SaaS, które zostały wstępnie zintegrowane w usłudze Azure AD, zobacz [Active Directory Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
