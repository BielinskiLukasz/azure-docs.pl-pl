# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Rozpoczynanie pracy z funkcjami kognitywnych U-SQL

## <a name="overview"></a>Przegląd
Kognitywnych możliwości U-SQL umożliwiają deweloperom używanie analizy put w swoich programach danych big data. 

Dostępne są następujące funkcje kognitywnych:
* Obrazowania: Wykrywanie kroje [próbki](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Obrazowania: Wykrywanie emocji [próbki](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Tworzenie obrazów: Wykrycia obiektów (znakowanie) [próbki](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Tworzenie obrazów: Rozpoznawania (OCR) [próbki](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: Klucz wyodrębniania frazy & analizę wskaźniki nastrojów klientów [próbki](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
