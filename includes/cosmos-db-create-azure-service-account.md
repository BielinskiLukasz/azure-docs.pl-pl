Możesz korzystać z tabel przy użyciu usługi Azure Table Storage lub Azure Cosmos DB. Aby dowiedzieć się więcej o różnicach między ofertami tabel w tych dwóch usługach, zobacz artykuł [oferty tabeli](../articles/cosmos-db/table-introduction.md#table-offerings) . Aby móc korzystać z wybranej usługi, musisz w niej utworzyć konto. W poniższych sekcjach pokazano, jak utworzyć zarówno usługę Azure Table Storage, jak i konto Azure Cosmos DB, ale możesz użyć jednego z nich.
