# <a name="secure-hybrid-access-secure-legacy-apps-with-app-delivery-controllers-and-networks"></a>Bezpieczny dostęp hybrydowy: bezpieczne starsze aplikacje za pomocą kontrolerów dostarczania aplikacji i sieci

Teraz można chronić lokalne i starsze aplikacje uwierzytelniania w chmurze, łącząc je z usługą Azure AD z istniejącym kontrolerem dostarczania aplikacji lub siecią. W ten sposób można wypełnić lukę i wzmocnić swoją postawę zabezpieczeń we wszystkich aplikacjach za pomocą funkcji usługi Azure AD, takich jak dostęp warunkowy usługi Azure AD i usługa Azure AD Identity Protection.

Korzystając z istniejącego kontrolera sieci i dostarczania, można łatwo chronić starsze aplikacje, które są nadal krytyczne dla procesów biznesowych, ale nie można było wcześniej chronić za pomocą usługi Azure AD. Prawdopodobnie masz już wszystko, czego potrzebujesz, aby rozpocząć ochronę tych aplikacji.
