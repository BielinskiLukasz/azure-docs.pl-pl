Aby uzyskać dostęp do usługi Azure Storage, potrzebujesz subskrypcji platformy Azure. Jeśli nie masz jeszcze subskrypcji, przed rozpoczęciem utwórz [bezpłatne konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F).

Cały dostęp do usługi Azure Storage odbywa się za pośrednictwem konta magazynu. Jeśli nie masz jeszcze konta magazynu, możesz łatwo je utworzyć w witrynie [Azure Portal](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Możesz utworzyć konto magazynu przy użyciu programu Azure PowerShell lub interfejsu wiersza polecenia platformy Azure. Aby uzyskać pomoc przy tworzeniu konta, zobacz [Tworzenie konta magazynu](../articles/storage/common/storage-quickstart-create-account.md).
