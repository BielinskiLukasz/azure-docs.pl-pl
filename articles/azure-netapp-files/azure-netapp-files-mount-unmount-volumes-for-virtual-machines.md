# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Zainstalowanie lub odinstalowanie woluminu dla maszyn wirtualnych Windows lub Linux 

Można zainstalować lub odinstalować woluminu dla maszyn wirtualnych Windows lub Linux zgodnie z potrzebami.  Instrukcje instalacji dla maszyn wirtualnych systemu Linux są dostępne w usłudze Azure Files NetApp.  

1. Kliknij przycisk **woluminów** bloku, a następnie wybierz polecenie eksportuje woluminu, dla którego chcesz zainstalować systemu plików NFS. 
2. Kliknij przycisk **instrukcje dotyczące instalacji** wybranego woluminu, a następnie postępuj zgodnie z instrukcjami.
