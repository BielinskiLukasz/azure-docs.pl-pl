# <a name="version-release-history"></a>Historia wersji
Zespół Microsoft Genomics regularnie aktualizuje klienta Microsoft Genomics Python poprawki i nowe funkcje. 

## <a name="latest-release"></a>Najnowsza wersja
Bieżące klienta Python jest wersja 0.7.4. Została wydana marca 2018 5, a używa GATK 3.3. 


## <a name="release-history"></a>Historia wersji 
Nowe wersje klienta Microsoft Genomics Python są wydawane o dwa razy w roku. Jak są wydawane nowe wersje klienta Microsoft Genomics Python, listę poprawki i funkcje jest aktualizowana w tym miejscu. Po udostępnieniu nowej wersji, wcześniejsze wersje powinno być kontynuowane do obsługi co najmniej 90 dni. Jeśli wcześniejsze wersje nie są już obsługiwane, będą wyświetlane na tej stronie. 

### <a name="version-074"></a>Wersja 0.7.4
Wersja 0.7.4 obsługuje akceptować tokeny sygnatury dostępu Współdzielonego zamiast klucze konta w `config.txt` wejściowego. Aby uzyskać więcej informacji, zobacz [dane wejściowe SAS tokeny szybkiego startu](quickstart-input-sas.md). 

### <a name="version-073"></a>Wersja 0.7.3
Wersja 0.7.3 obejmuje niewielkie poprawki błędów

### <a name="version-072"></a>Wersja 0.7.2
0.7.2 jest początkowa wersja. Została wydana 2017 1 listopada, a używa GATK 3.3.
