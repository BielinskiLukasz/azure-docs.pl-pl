# <a name="supported-postgresql-database-versions"></a>Bazą danych PostgreSQL obsługiwane wersje
Microsoft ma na celu obsługuje n-2 wersji aparatu PostgreSQL w bazie danych Azure dla usługi PostgreSQL, co oznacza aktualnie wydanej wersji głównej (n) i dwa poprzednie wersje główne-(2).

Bazy danych platformy Azure dla PostgreSQL obecnie obsługuje następujące wersje:

## <a name="postgresql-version-962"></a>Wersja PostgreSQL 9.6.2
Zapoznaj się [dokumentacji PostgreSQL](https://www.postgresql.org/docs/9.6/static/release-9-6-2.html) Aby dowiedzieć się więcej na temat ulepszeń i poprawek w PostgreSQL 9.6.2.

## <a name="postgresql-version-957"></a>Wersja PostgreSQL 9.5.7
Zapoznaj się [dokumentacji PostgreSQL](https://www.postgresql.org/docs/9.5/static/release-9-5-7.html) Aby dowiedzieć się więcej na temat ulepszeń i poprawek w PostgreSQL 9.5.7.

## <a name="managing-updates-and-upgrades"></a>Zarządzanie aktualizacjami i uaktualnieniami
Bazy danych platformy Azure dla PostgreSQL automatycznie zarządza poprawki aktualizacji wersji pomocniczej. Obecnie w wersji zapoznawczej, wersja główna uaktualnienie nie jest obsługiwane. Na przykład uaktualnienie z PostgreSQL 9.5 do PostgreSQL 9.6 nie jest obsługiwane.
