# Azure Search: チュートリアル、ビデオ デモ、サンプル

次のビデオ、デモ、チュートリアル、サンプル アプリを使って作業をスピードアップできます。

## デモ サイト

[* * ニューヨーク市 (ニューヨーク) ジョブ ポータル デモの Azure Search *](http://aka.ms/azjobsdemo) NYC オープン データ構想からデータを使用して Azure Search の機能を示すライブの ASP.NET web サイトです。

[* * App Service の試用 + Azure Search で ASP.NET MVC コンボ app * *](search-tryappservice.md) 、ASP.NET アプリケーション内のさまざまな検索機能は、無料の 1 時間セッション データ、関数型の web アプリケーション、およびオンライン テスト ドライブに変更できるソース コードを含む Azure Search をします。

[* * Azure Search デモ サイト * *](https://searchsamples.azurewebsites.net/#/) Azure Search のデモ アプリのコレクションです。 Azure Search を利用し、メディア検索、チリ検索、製品在庫のアプリを試します。

## コード サンプル

[* * * * https://github.com/liamca/AzureSearchDemos](https://github.com/liamca/AzureSearchDemos) Liam cavanagh 作成したサンプル コードを含む GitHub リポジトリです。 このリポジトリには次のサンプルがあります。

- [AdventureWorksWP8]() は Windows phone アプリです。
- [AzSearchAutoScaleWeb](https://github.com/liamca/AzureSearchDemos/tree/master/AzSearchAutoScaleWeb)
- [BabyNameJS](https://github.com/liamca/AzureSearchDemos/tree/master/BabyNamesJS) index.html ファイルで、チュートリアル内のデータを提供します。
- [DotNetSample](https://github.com/liamca/AzureSearchDemos/tree/master/DotNetSample) は Azure Search を ASP.NET MVC アプリケーションです。
- [NYCJobs.NET](https://github.com/liamca/AzureSearchDemos/tree/master/NYCJobs.NET) NYCJobs ライブ デモ サイトのソース コードを提供します。
- [の推奨事項](https://github.com/liamca/AzureSearchDemos/tree/master/Recommendations) Active Data Factory、SQL データベース、Blob ストレージと HDInsight を使用して、製品がともに購入されてどのくらいの頻度に基づいて製品推奨を実装します。

[* * * * https://github.com/AzureSearch](https://github.com/AzureSearch) 別の GitHub リポジトリは、Azure Search、Azure Search 開発チームが作成したコード サンプルが含まれているのです。 リボジトリを参照してその内容を確認できます。または、次のリンクをクリックして、特定のサンプルに直接移動できます。

- [* * JavaIndexerDemo * *](https://github.com/AzureSearch/AzureSearchJavaIndexerDemo)
- [* * NodeJSIndexerDemo * *](https://github.com/AzureSearch/AzureSearchNodeJSIndexerDemo)
- [**.NET-2015年-02-28-* * サンプル](https://github.com/AzureSearch/.NET-2015-02-28-Sample) は検索操作の .NET クライアント ライブラリを使用する ASP.NET MVC 5 アプリケーションです。

別の GitHub リポジトリが使用される [* * AzureSearch Microsoft Virtual Academy トレーニング資料 * *](https://github.com/MicrosoftLearning/AzureSearch-MVA)します。 これには、7 部構成のコースで使用されるいくつかのサンプルが含まれます。

CodePlex で、最新バージョンと関連のあるパブリック プレビュー期間中に公開されたサンプルがいくつか記載されています。

- [* * 管理 REST API demo](https://azuresearchmgmtapi.codeplex.com/) 管理 REST API を使用する Windows コンソール アプリケーションを示します。
- [* * AdventureWorks demo](https://azuresearchadventureworksdemo.codeplex.com/) は、ASP.NET MVC 4 アプリケーションです。

- [* * SQL Server を Azure Search Sync サンプル * *](http://sqlserver2azuresearch.codeplex.com/) 埋め込みローカル データベースから SQL Server 行セットを読み込む Windows コンソール アプリケーションです。
- [* * スコアリング プロファイルのデモ *](https://azuresearchscoringprofiles.codeplex.com/) Windows コンソール アプリケーションを定義しが実行され、検索結果をチューニングするためのスコアリング プロファイルの更新です。
