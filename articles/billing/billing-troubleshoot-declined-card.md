# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>Rozwiązywanie problemów z odrzuconym kartą na platformie Azure — Rejestracja

Ten artykuł pomaga w rozwiązywaniu problemów, w których karta kredytowa została odrzucona w ramach rejestracji na platformie Azure w witrynie Azure Portal lub centrum konta platformy Azure. Przed rozpoczęciem rozwiązywania problemu należy sprawdzić następujące kwestie:

- Upewnij się, że informacje podane dla profilu konta platformy Azure, takie jak kontaktowe adresy e-mail, adres i numer telefonu, są poprawne.
- Upewnij się, że informacje o karcie kredytowej są poprawne.
- Upewnij się, że nie masz jeszcze konta Microsoft z tymi samymi informacjami.
- Karty debetowe nie są akceptowane

## <a name="issues"></a>Problemy

Poniżej wymieniono typowe problemy, które mogą spowodować odrzucenie karty kredytowej na platformie Azure.

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>Dostawca karty kredytowej nie został zaakceptowany w Twoim kraju

Po wybraniu karty na platformie Azure zostaną wyświetlone opcje karty, które są prawidłowe w wybranym kraju. Skontaktuj się z wystawcą banku lub karty, aby upewnić się, że karta kredytowa jest włączona dla transakcji międzynarodowych. Aby uzyskać więcej informacji o obsługiwanych krajach i walutach, zobacz [często zadawane pytania dotyczące zakupu platformy Azure](https://azure.microsoft.com/pricing/faq/).

>[!Note]
>Karty kredytowe American Express nie są obecnie obsługiwane jako instrument płatniczy w Indiach. Nie ma przedziału czasowego, ponieważ może to być zaakceptowana forma płatności.

### <a name="youre-using-a-virtual-or-prepaid-card"></a>Korzystasz z karty wirtualnej lub przedpłaty 

Karty kredytowe lub debetowe nie są akceptowane jako płatność za subskrypcje platformy Azure.

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>Informacje kredytowe są niedokładne lub niekompletne 

Wprowadzona nazwa, adres i kod CVV muszą dokładnie pasować do tego, co jest drukowane na karcie.

### <a name="the-card-is-inactive-or-blocked"></a>Karta jest nieaktywna lub zablokowana 

Skontaktuj się z bankiem, aby upewnić się, że Twoja karta jest aktywna.

Mogą występować inne problemy z rejestrowaniem 

Aby uzyskać więcej informacji na temat rozwiązywania problemów z rejestracją na platformie Azure, zobacz następujący artykuł bazy wiedzy: 

[Nie możesz zarejestrować się na platformie Azure w centrum konta platformy Azure Azure Portal lub](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>Reprezentujesz firmę, która nie chce płacenia według karty 

Jeśli reprezentujesz firmę, możesz skorzystać z metod płatności faktury, takich jak czeki, testy noclegowe lub transfery w sieci, aby zapłacić za subskrypcję platformy Azure. Po skonfigurowaniu konta do płatności za pomocą faktury nie można zmienić innej opcji płatności, chyba że masz umowę klienta firmy Microsoft i zarejestrowano się na platformie Azure za pomocą witryny sieci Web systemu Azure.

Aby uzyskać więcej informacji o sposobach płacenia według faktury, zobacz Przesyłanie żądania zaliczania [subskrypcji platformy Azure według faktury](billing-how-to-pay-by-invoice.md).

### <a name="your-credit-card-information-is-outdated"></a>Informacje o karcie kredytowej są nieaktualne 

Aby uzyskać informacje o sposobach zarządzania informacjami o karcie, w tym o zmianie lub usunięciu karty, zobacz [Dodawanie, aktualizowanie lub usuwanie środków na platformie Azure](billing-how-to-change-credit-card.md).
