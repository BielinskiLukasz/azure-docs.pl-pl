# <a name="create-azure-marketplace-and-appsource-offers"></a>Tworzenie ofert portalu Azure Marketplace i appsource

Jednym z podstawowych celów portalu cloud partner jest umożliwienie wydawcom tworzenia (a następnie publikowania) ofert w witrynach Microsoft Azure i AppSource Marketplaces.  Ta operacja zawsze rozpoczyna się od wybrania żądanego typu oferty z [menu Nowa oferta](../portal-tour/cpp-new-offer-menu.md).  W odpowiedzi dla tego typu oferty zostanie wyświetlona odpowiednia strona **Nowa oferta.**  Na przykład na poniższej ilustracji przedstawiono domyślną stronę **Nowa oferta** dla typu aplikacji platformy Azure.

![Nowa oferta domyślna strona](./media/new-offer-page.png)

W poziomym pasku menu wyświetlanym w górnej części tej strony dostępne są dwa opcje kart: 
- **Karta Edytor** — umożliwia wprowadzanie informacji i przekazywanie zasobów dla nowego wystąpienia oferty.  Ta karta jest wyświetlana domyślnie.
- **Karta Stan** — określa stan publikowania i wyświetla wszelkie problemy z sprawdzaniem poprawności i przeglądem. 

Podczas tworzenia oferty należy użyć karty **Edytor,** aby wprowadzić informacje o tej ofercie. 

## <a name="editing-operations"></a>Operacje edycji

Poziomy pasek narzędzi znajdujący się nad obszarem wprowadzania danych wyświetla następujące przyciski:

|   Button    |   Przeznaczenie                                                          |
|   ------    |  --------                                                          |
| **Zapisz**    | Zapisuje wszystkie ostatnie zmiany wprowadzania danych.  Przed odejmą się od strony lub utratą zmian należy ręcznie zapisać zmiany. | 
| **Odrzuć** | Odrzuca ostatnie zmiany wprowadzania danych (od ostatniego zapisu)             |
| **Porównanie** | Porównuje stan bieżącej oferty z opublikowaną ofertą.  Włączono tylko po pomyślnym opublikowaniu oferty.  |
| **Publikowanie** | Rozpoczyna proces publikowania dla tej oferty                       |
| **Usuwanie**  | Usuwa tę ofertę po jej utworzeniu, ale przed jej opublikowaniem. |
|   |   |


## <a name="editing-tabs"></a>Edytowanie kart

Podczas tworzenia oferty należy podać wymagane i opcjonalne dane na każdej karcie znajdującej się w lewej pionowej kolumnie strony **Nowa oferta.**  Standardowe formanty interfejsu użytkownika — takie jak pola tekstowe, menu rozwijane i pola wyboru — są wyświetlane do zbierania danych.  Chociaż określona kolekcja kart edycji zależy od typu oferty, w poniższej tabeli wymieniono niektóre typowe karty.

|      Nazwa karty       |   Przeznaczenie                                                            |
|      --------       |   -------                                                            |
| **Ustawienia oferty**  | Zbiera informacje o ofercie i tożsamości wydawcy.                    |
| **Jednostki SKU**            | Określa charakterystykę techniczną i biznesową każdej wersji oferty dla każdej jednostki magazynowej (SKU) |
| **Wersja testowa**      | Dla tych typów, które obsługują tę opcjonalną funkcję, definiuje demonstrację dla oferty.  Aby uzyskać więcej informacji, zobacz [Co to jest dysk testowy?](../test-drive/what-is-test-drive.md)  |
| **Marketplace** lub **Storefront** | Zbieranie ciągów tekstowych, dokumentów i obrazów używanych do wystawiania oferty na rynku |
| **Pomoc techniczna**         | Zbiera informacje kontaktowe dla klienta, inżynierii i pomocy technicznej online  |
|  |  |

Zawartość kart o podobnej nazwie może się różnić w zależności od typu oferty.  Szczegółowe informacje dotyczące oferty dotyczące tych kart są dostępne w sekcji "Utwórz ofertę" dla każdego typu oferty.
