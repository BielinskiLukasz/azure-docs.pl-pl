# <a name="supported-browsers-for-azure-iot-central"></a>Przeglądarki obsługiwane przez usługi Azure IoT Central

*Ten artykuł dotyczy operatorów, konstruktorów i administratorów.*

Azure IoT Central jest dostępna na nowoczesnych komputery stacjonarne, tabletów i przeglądarki. Następujący artykuł przedstawia listę obsługiwanych przeglądarek. 

Zaleca się, że używasz najbardziej aktualne przeglądarki, która jest zgodna z systemem operacyjnym. Obsługiwane są następujące przeglądarki:
- Microsoft Edge (Najnowsza wersja)
- Safari (Najnowsza wersja, tylko Mac)
- Chrome (Najnowsza wersja)
- Firefox (Najnowsza wersja)
