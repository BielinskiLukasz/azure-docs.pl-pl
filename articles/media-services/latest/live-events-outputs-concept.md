# <a name="live-events-and-live-outputs"></a>Wydarzenia na żywo i na żywo dane wyjściowe

Usługa Azure Media Services umożliwia dostarczanie wydarzeń na żywo dla klientów w chmurze Azure. Aby skonfigurować usługi przesyłania strumieniowego wydarzeń na żywo usługi Media Services v3, musisz Dowiedz się więcej o [zdarzenia na żywo](https://docs.microsoft.com/rest/api/media/liveevents) i [na żywo dane wyjściowe](https://docs.microsoft.com/rest/api/media/liveoutputs).
