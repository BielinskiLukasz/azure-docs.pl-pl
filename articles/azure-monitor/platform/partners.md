## <a name="alertlogic-log-manager"></a>Menedżer dzienników AlertLogic
Alert Logic Log Manager umożliwia zbieranie informacji o maszynie Wirtualnej, aplikacji i dzienników platformy Azure do przechowywania i analizy zabezpieczeń. Gromadzone są również dziennik aktywności platformy Azure za pośrednictwem interfejsu API usługi Azure Monitor. Informacja ta jest używany do wykrywania malfeasance i spełnianie wymagań dotyczących zgodności.

[Przejdź do dokumentacji.][alertlogic-doc]

## <a name="appdynamics"></a>AppDynamics
AppDynamics zarządzania wydajnością aplikacji (APM) umożliwia właścicielom aplikacji szybkie rozwiązywanie problemów z wąskich gardeł wydajności i zoptymalizować wydajność ich aplikacji, które działają w środowisku platformy Azure. Azure Cloud Services (PaaS) może monitorować ról sieć web i proces roboczy, Virtual Machines (IaaS), zdalnej usługi wykrywania (Microsoft Azure Service Bus), Microsoft Azure Queue, zdalnej usługi Microsoft Azure (Azure Blob), kolejki platformy Azure (Microsoft Service Bus), dane Magazyn i usługi Microsoft Azure Blob Storage. AppDynamics APM jest dostępna w witrynie Azure Marketplace.

[Przejdź do dokumentacji.][appdynamics-doc]

## <a name="circonus"></a>Circonus
Circonus to mikrousług, monitorowania i w obsłudze platforma analityczna myślą lokalne lub wdrożenie SaaS. Jest w pełni możliwą opartej na interfejsach API platformy jest bardziej skalowalna i monitoruje niezawodną metodą od systemów. Opracowane dla wymagań metodyki DevOps, Circonus dostarcza alerty na podstawie percentyla, wykresy, pulpity nawigacyjne i analizy na uczeniu maszynowym, pozwalających na korzystanie z optymalizacji biznesowej. Circonus monitoruje swoje zasoby w chmurze Microsoft Azure i ich aplikacji w czasie rzeczywistym. Circonus służy do zbierania i śledzenia metryk zmiennych, które chcesz zmierzyć dla zasobów i aplikacji. Dzięki Circonus możesz uzyskać systemowe wgląd w wykorzystanie zasobów platformy Azure, wydajność aplikacji i kondycji operacyjnej.

[Przejdź do dokumentacji.][circonus-doc]

## <a name="datadog"></a>Pomocą usługi Datadog
Pomocą usługi Datadog jest wiodących na świecie monitorowania usługi dla aplikacji w skali chmury. Jego gromadzi dane z serwerów, baz danych, narzędzia i usługi do przedstawienia ujednoliconego podglądu całego stosu. Te możliwości zostały podane na platformy do analizy danych oparta na rozwiązaniu SaaS. Ta usługa umożliwia deweloperów i Współpracuj, aby uniknąć przestoju, zespoły Ops rozwiązać problemy z wydajnością i upewnij się, że rozwoju i cykli wdrażania oferujemy teraz zdążyć na czas. Po zintegrowaniu pomocą usługi Datadog i na platformie Azure, można zbierać i wyświetlać metryki w infrastrukturze. Korelowanie metryki maszyny Wirtualnej z metrykami na poziomie aplikacji. Kątami metryk przy użyciu kombinacji właściwości i tagów niestandardowych.

[Przejdź do dokumentacji.][datadog-doc]

## <a name="dynatrace"></a>Rozwiązania Dynatrace
Dynatrace OneAgent integruje się z usługą Azure Virtual Machines i usług aplikacji za pośrednictwem mechanizmu rozszerzenie platformy Azure. W ten sposób Dynatrace OneAgent można zbierać metryki wydajności dotyczące hostów, sieci i usług. Oprócz po prostu wyświetlanie metryk, Dynatrace wizualizuje środowisk end-to-end. Pokazuje transakcji po stronie klienta, aby warstwa bazy danych. Dynatrace zapewnia oparte na sztucznej Inteligencji korelacji problemów i w pełni zintegrowane — analiza głównej przyczyny — aby zapewnić metody poziomu szczegółowych informacji o kodzie i bazie danych. Te szczegółowe informacje sprawia, że optymalizacje Rozwiązywanie problemów i wydajności jest znacznie łatwiejsze.

[Przejdź do dokumentacji.][dynatrace-doc]

Chmura LM kompleksowego monitorowania platformy Azure umożliwia użytkownikom skorelować wydajność platformy Azure w chmurze, lokalnych i zasobów w chmurze hybrydowej — wszystko za pomocą pojedynczej platformy. Odnajdywanie automatyczne zasobów, wbudowane monitorowanie szablonów, wstępnie skonfigurowane progi alertów i dostosowywane pulpity nawigacyjne są łączone w celu zapewniają IT szybkości, elastyczności i widoczność wymagane zakończyło się sukcesem.

## <a name="signalfx"></a>SignalFx
SignalFx jest liderem w aktualne operacyjne dane analityczne dla opartych na danych metodyki DevOps. Ta usługa umożliwia odnalezienie i zbiera metryki na każdy składnik w chmurze. Zastępuje punkt tradycyjnych narzędzi i zapewnia wgląd w czasie rzeczywistym w współczesnych dynamicznych środowiskach. Korzystanie z wysoce skalowalnej platformy SignalFx platformy SaaS jest zoptymalizowana pod kątem kontenerów i mikrousług na podstawie architektury i zapewnia zaawansowane wizualizacje, aktywne alerty i funkcji Klasyfikacja współpracy między organizacjami z wszystkie rozmiary. SignalFx integruje się bezpośrednio z usługą Azure Monitor, a także za pomocą łączników typu open source, takich jak Telegraf, statsD i zebrane, aby zapewnić najlepsze w klasy pulpitów nawigacyjnych, analiz i alertów dla platformy Azure.

[Przejdź do dokumentacji.][signalfx-doc]

## <a name="signl4"></a>APLIKACJI SIGNL4
Wywołuje SIGNL4 — alertów aplikacji mobilnej dla zespołów operacyjnych — jest to najszybszy sposób trasy krytyczne alerty z usługi Azure Monitor do odpowiednich osób we właściwym czasie — w dowolnym miejscu, wypychania, tekstu i głosu. SIGNL4 zarządza dyżurów obowiązków i przesunięcia zespołu, ścieżki dostarczania i własności alerty i przekazuje problem w razie potrzeby. Podano pełną przezroczystość całemu zespołowi. Za pomocą bardzo łatwe REST-webhook o SIGNL4 dowolnej usługi platformy Azure może być połączona z Brak nakładu pracy. Za pomocą aplikacji SIGNL4 zostanie wyświetlony maksymalnie 10 x szybciej uzyskać odpowiedź, za pośrednictwem powiadomienia e-mail i ręczne alerty.

[Przejdź do dokumentacji.][signl4-doc]

## <a name="sumo-logic"></a>Logika sumo
Logika sumo jest maszyny bezpieczny i natywnych dla chmury, Usługa do analizowania danych, dostarczania analizy w czasie rzeczywistym, zapewniająca ciągłą dane ze strukturą, częściową strukturą i bez struktury w cyklu życia całej aplikacji i stosu. Więcej niż 1000 klientów na całym świecie, zależą od Sumo logikę analiz i szczegółowych informacji do tworzenia, uruchamiania i zabezpieczanie ich aplikacji i infrastruktury chmury. Wraz z logiką Sumo klienci uzyskują wielodostępne, model usług zaletą zwiększyć przewagę konkurencyjną, wartości biznesowej i wzrost.

## <a name="turbonomic"></a>Turbonomic
Usługa Turbonomic automatyzuje obciążenia dla chmur hybrydowych, jednocześnie optymalizując wydajność, koszty i zgodność ze standardami w czasie rzeczywistym. Turbonomic pomaga organizacjom znajdować się elastycznych w ich platformy Azure ze standardami, stale optymalizując nieruchomości, aby upewnić się, że aplikacje stale pobierają zasoby wymagają one dostarczać ich umowy SLA i nic w obliczeń, magazynu i sieci dla rozwiązań IaaS i PaaS warstwy. Organizacje mogą symulować migracje, prawidłowo obciążenia SAP Hana i wycofywać zasoby centrum danych odpowiedzialnego migrację do platformy Azure na czas, w ramach budżetu przy jednoczesnym zapewnieniu zarówno wydajności, jak i zgodności. Usługa Turbonomic pracuje w oparciu o interfejs API i działa jako bezagentowa maszyna wirtualna na platformie Azure oraz lokalnie.

[Dowiedz się więcej.][turbonomic-doc]
