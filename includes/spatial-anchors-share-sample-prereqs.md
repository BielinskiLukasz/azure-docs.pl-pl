## <a name="prerequisites"></a>Wymagania wstępne

Aby ukończyć kroki tego samouczka, upewnij się, że dysponujesz następującymi elementami:

* Przeczytaj [omówienie zakotwiczenia przestrzennego platformy Azure](../articles/spatial-anchors/overview.md).
* Ukończono jeden z [5-minutowych programów Szybki start.](../articles/spatial-anchors/index.yml)
* Podstawowa wiedza na temat języka C# i Unity.
* Podstawowa wiedza na temat <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore,</a> jeśli chcesz używać systemu Android lub <a href="https://developer.apple.com/arkit/" target="_blank">ARKit,</a> jeśli chcesz używać systemu iOS.
* Komputer z systemem Windows, na którym jest <a href="https://www.visualstudio.com/downloads/" target="_blank">zainstalowany program Visual Studio 2017</a> lub nowszy z **obciążeniem ASP.NET i tworzenia sieci Web.**
* [Zestaw .NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* Co najmniej jedno urządzenie (iOS lub Android), na którym ma być wdrażana i uruchamiana aplikacja.
  * Jeśli korzystasz z Androida, musisz mieć:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> lub nowszy, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 lub 2019.2</a>i <a href="https://git-scm.com/download/win" target="_blank">Git dla systemu Windows</a> zainstalowane na komputerze z systemem Windows.
    * Urządzenie z systemem Android <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">z obsługą dewelopera</a> i <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">obsługą ARCore.</a>
  * Jeśli korzystasz z systemu iOS, musisz mieć:
    * Komputer z systemem macOS, na którym jest zainstalowany <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">kod Xcode 10</a> lub <a href="https://cocoapods.org" target="_blank">nowszy, CocoaPods</a>i <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 lub 2019.2.</a>
    * Urządzenie z systemem iOS zgodne z funkcją <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit z</a> obsługą dewelopera.
    * Git zainstalowany przez Homebrew. Wprowadź następujące polecenie w jednym wierszu `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` w `brew install git`terminalu: Następnie uruchom polecenie .
