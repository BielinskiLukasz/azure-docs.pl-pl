# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Jak utworzyć konto Microsoft Translator Speech API

Wykonaj następujące kroki w celu uzyskania interfejsu API usługi Microsoft Translator mowy.

## <a name="1-sign-into-azure"></a>1: Zaloguj się do platformy Azure   
- Nie masz jeszcze konta? [Załóż konto Microsoft Azure](http://azure.com/)
- Masz już konto? [Rejestrowanie](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: subskrybować Translator firmy Microsoft     
- Po zalogowaniu się do platformy Azure, przejdź do [kognitywnych usług](https://portal.azure.com/#create/Microsoft.CognitiveServices) sekcji
- W obszarze "Typ interfejsu API" Wybierz Translator Speech API
- W sekcji warstwy cenowej wybierz warstwę cenową, która najlepiej odpowiada Twoim potrzebom
    - Każda subskrypcja ma warstwę bezpłatna. Warstwa bezpłatna ma te same funkcje i funkcje, ponieważ płatną planów i nie ma datę wygaśnięcia.
    - Może mieć tylko jedną bezpłatną subskrypcję na Twoim koncie
- Wypełnianie pozostałej części formularza i kliknij przycisk Utwórz
- Subskrypcja jest gotowa do użycia

## <a name="3-authentication-key"></a>3: klucz uwierzytelniania    
- Pobierz klucz uwierzytelniania, przechodząc do menu wszystkie zasoby > kliknij subskrypcję
- Wartość klucza jest używany do uwierzytelniania, należy zachować wartość klucza poufne. Wartość klucza potrzebne podczas opracowywania aplikacji
