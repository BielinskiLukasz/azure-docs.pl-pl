# <a name="avere-vfxt-demo-projects"></a>Projekty demo Avere vFXT

Przykładowe samouczki są dostępne na [GitHub](https://github.com/Azure/Avere). Te małe projekty zademonstrować kluczowe funkcje i przypadki użycia dla Avere vFXT dla platformy Azure.

## <a name="video-rendering"></a>Renderowanie wideo

* [Renderowanie przy użyciu usługi Azure Batch i Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/maya_azure_batch_avere_vfxt_demo.md) — 60-minutowy projekt, który pokazuje, jak używać programu Autodesk Maya z usługą Azure Batch i klastra Avere vFXT do generowania filmu animowanego

* [Dlaczego warto używać Avere vFXT do renderowania?](https://github.com/Azure/Avere/blob/master/docs/why_avere_for_rendering.md) - Demonstracja, która porównuje czas renderowania z sieciowej pamięci masowej z klastrem VFXT i bez niego

## <a name="high-performance-computing"></a>Obliczenia o wysokiej wydajności
