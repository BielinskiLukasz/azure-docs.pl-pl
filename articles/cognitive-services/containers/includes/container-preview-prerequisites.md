## <a name="prerequisites"></a>Wymagania wstępne

* Użyj subskrypcji platformy Azure. Jeśli nie masz subskrypcji platformy Azure, utwórz [bezpłatne konto][free-account] przed rozpoczęciem.
* Zainstaluj [interfejsu wiersza polecenia platformy Azure][azure-cli] (az).
* [Docker aparatu][docker-engine] i sprawdź, czy docker CLI działa w oknie konsoli.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
