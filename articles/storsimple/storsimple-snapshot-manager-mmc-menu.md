# <a name="use-the-mmc-menu-actions-in-storsimple-snapshot-manager"></a>Używanie akcji menu programu MMC programu StorSimple Snapshot Manager

## <a name="overview"></a>Omówienie
Programie StorSimple Snapshot Manager zobaczysz następujące akcje wymienione na wszystkich menu akcji i wszystkie odmiany funkcji **akcje** okienka.

* Widok
* Nowe okno, w tym miejscu 
* Odśwież 
* Eksportuj listę 
* Help 

Te akcje są dostępne w ramach programu Microsoft Management Console (MMC) i nie są specyficzne do programu StorSimple Snapshot Manager. W tym samouczku opisano te akcje oraz wyjaśniono sposób używania każdego z nich w programie StorSimple Snapshot Manager.

## <a name="view"></a>Widok
Możesz użyć **widoku** opcję, aby zmienić **wyniki** widoku okienka i zmienić widok okna konsoli. 

#### <a name="to-change-the-results-pane-view"></a>Aby zmienić widok okienko wyników
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager.
2. W **zakres** okienku kliknij prawym przyciskiem myszy dowolny węzeł lub rozwiń węzeł i kliknij prawym przyciskiem myszy element w **wyniki** okienku, a następnie kliknij przycisk **widoku** opcji. 
3. Aby dodać lub usunąć kolumny, które pojawiają się w **wyniki** okienku kliknij **Dodaj/Usuń kolumny**. **Dodaj/Usuń kolumny** pojawi się okno dialogowe.
   
    ![Dodawanie lub usuwanie kolumn w okienku wyników](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Add_remove_columns.png) 
4. Wypełnij formularz w następujący sposób:
   
   * Wybierz elementy z **dostępne** kolumny i kliknij przycisk **Dodaj** Aby dodać je do **wyświetlane kolumny** listy. 
   * Kliknij elementy na **wyświetlane kolumny** listy, a następnie kliknij przycisk **Usuń** je usunąć z listy. 
   * Wybierz element w **wyświetlane** kolumny i kliknij przycisk **Przenieś w górę** lub **Przenieś w dół** do Przenieś element w górę lub w dół na liście. 
   * Kliknij przycisk **Przywróć domyślne** aby powrócić do domyślnego **wyniki** Konfiguracja okienka. 
5. Po zakończeniu opcje kliknij **OK**. 

#### <a name="to-change-the-console-window-view"></a>Aby zmienić widok okna konsoli
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager.
2. W **zakres** okienku kliknij prawym przyciskiem myszy dowolny węzeł, kliknij przycisk **widoku**, a następnie kliknij przycisk **Dostosuj**. **Dostosuj** pojawi się okno dialogowe.
   
    ![Dostosowywanie okna konsoli](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Customize.png) 
3. Zaznacz lub wyczyść pola wyboru, aby pokazać lub ukryć elementy w oknie konsoli. Po zakończeniu opcje kliknij **OK**.

## <a name="new-window-from-here"></a>Nowe okno, w tym miejscu
Możesz użyć **nowe okno, w tym miejscu** opcję, aby otworzyć nowe okno konsoli.

#### <a name="to-open-a-new-console-window"></a>Aby otworzyć nowe okno konsoli
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager.
2. W **zakres** , kliknij prawym przyciskiem myszy dowolny węzeł, a potem kliknij **nowe okno, w tym miejscu**. 
   
    Nowe okno zostanie wyświetlony tylko zakres czy wybrane. Przykładowo po kliknięciu prawym przyciskiem myszy **zasady tworzenia kopii zapasowych** węzła, nowe okno zostanie wyświetlone tylko **zasady tworzenia kopii zapasowych** w węźle **zakres** okienka i listę zdefiniowanych kopii zapasowej zasady w **wyniki** okienka. Zobacz poniższy przykład.
   
    ![Nowe okno, w tym miejscu](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_NewWindow.png) 

## <a name="refresh"></a>Odśwież
Możesz użyć **Odśwież** akcji w celu zaktualizowania okna konsoli.

#### <a name="to-update-the-console-window"></a>Aby zaktualizować okna konsoli
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager.
2. W **zakres** okienku kliknij prawym przyciskiem myszy dowolny węzeł lub rozwiń węzeł i kliknij prawym przyciskiem myszy element w **wyniki** okienku, a następnie kliknij przycisk **Odśwież**. 

## <a name="export-list"></a>Eksportuj listę
Możesz użyć **Eksportuj listę** akcję, aby zapisać listę w pliku wartości rozdzielanych przecinkami (CSV). Na przykład możesz wyeksportować listę zasad tworzenia kopii zapasowych i wykazem kopii zapasowych. Następnie można zaimportować pliku CSV do arkusza kalkulacyjnego aplikacji do analizy.

#### <a name="to-save-a-list-in-a-comma-separated-value-csv-file"></a>Aby zapisać listę w pliku wartości rozdzielanych przecinkami (CSV)
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager. 
2. W **zakres** okienku kliknij prawym przyciskiem myszy dowolny węzeł lub rozwiń węzeł i kliknij prawym przyciskiem myszy element w **wyniki** okienku, a następnie kliknij przycisk **Eksportuj listę**. 
3. **Eksportuj listę** pojawi się okno dialogowe. Wypełnij formularz w następujący sposób: 
   
   1. W **nazwy pliku** wpisz nazwę pliku CSV lub kliknij strzałkę, aby wybrać z listy rozwijanej.
   2. W **Zapisz jako typ** polu, kliknij strzałkę i wybierz typ pliku z listy rozwijanej.
   3. Aby zapisać tylko wybrane elementy, wybierz wiersze, a następnie kliknij przycisk **Zapisz tylko wybrane wiersze** pole wyboru. Aby zapisać wszystkie eksportowane listy, wyczyść **Zapisz tylko wybrane wiersze** pole wyboru.
   4. Kliknij pozycję **Zapisz**.
      
      ![Eksportuj listę w formacie wartości rozdzielanych przecinkami](./media/storsimple-snapshot-manager-mmc-menu/HCS_SSM_Export_List.png) 

## <a name="help"></a>Help

#### <a name="to-view-available-online-help"></a>Aby wyświetlić dostępne Pomoc online
1. Kliknij ikony pulpitu, które można uruchomić programu StorSimple Snapshot Manager.
2. W **zakres** okienku kliknij prawym przyciskiem myszy dowolny węzeł lub rozwiń węzeł i kliknij prawym przyciskiem myszy element w **wyniki** okienku, a następnie kliknij przycisk **pomocy**.
