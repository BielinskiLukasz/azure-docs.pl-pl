### <a name="azure-active-directory-authentication"></a>Uwierzytelnianie usługi Azure Active Directory

Kotwice przestrzenne platformy Azure umożliwia również aplikacjom uwierzytelnianie za pomocą tokenów usługi Azure AD (Active Directory) użytkownika. Na przykład można użyć tokenów usługi Azure AD Aby zintegrować z usługą Azure przestrzenne kotwic. Jeśli przedsiębiorstwo zachowuje użytkowników w usłudze Azure AD, możesz podać token użytkownika usługi Azure AD w przestrzenne kotwic zestawu SDK usługi Azure. Dzięki temu można uwierzytelnić bezpośrednio z usługą Azure przestrzenne kotwic dla konta które jest częścią tej samej dzierżawie usługi Azure AD.
