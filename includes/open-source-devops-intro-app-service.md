[Usługa Azure App Service](/azure/app-service/overview) umożliwia tworzenie i hostowanie kodu. Ten kod może być w formie aplikacji internetowych, zapleczy aplikacji mobilnych i interfejsów API RESTful. Za pomocą usługi App Service, możesz tworzyć kod przy użyciu preferowanego języka programowania bez zarządzania infrastrukturą. Usługa App Service obsługuje systemów Windows i Linux. Używanie wdrożeń zautomatyzowanych z dowolnym repozytorium Git są obsługiwane, takich jak GitHub i DevOps platformy Azure.
