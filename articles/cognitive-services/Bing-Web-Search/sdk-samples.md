# <a name="bing-web-search-sdk-samples"></a>Przykłady zestawu SDK wyszukiwania sieci Web Bing

Zestaw SDK wyszukiwania w sieci Web Bing jest dostępna w języku Python, Node.js, C# i Java. Przykłady kodu, wstępnie wymagane składniki i kompilacji instrukcje znajdują się w witrynie GitHub. Omówione są następujące scenariusze:

* Zapytania pojedynczego wyrazu i Drukuj nazwy i adresu URL pierwszego wyniku dla stron sieci web, obrazów, artykuły i filmy wideo.
* Zapytania frazę, sprawdź liczbę wyników, a następnie wydrukuj nazwy i adresu URL pierwszego wyniku.
* Zapytania wyszukiwanego terminu z filtrami odpowiedzi równa `news` i wydrukować szczegóły wyniki wiadomości.
* Zapytanie wyszukiwany termin `answerCount` i `promote` parametrów, a następnie wydrukować szczegóły wyników.
