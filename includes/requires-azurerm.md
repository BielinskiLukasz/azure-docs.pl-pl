> [!IMPORTANT]
>
> Korzystanie z tej funkcji platformy Azure za pomocą programu PowerShell wymaga `AzureRM` zainstalowany moduł. Jest to moduł starsze dostępne tylko dla Windows PowerShell 5.1, która nie będzie już otrzymywać nowych funkcji.
> `Az` i `AzureRM` moduły są __nie__ zgodnego zainstalowany dla tej samej wersji programu PowerShell.
> Jeśli potrzebujesz obie wersje:
>
> 1. [Odinstalowanie modułu Az](/powershell/azure/uninstall-az-ps) w sesji programu PowerShell 5.1.
> 2. [Zainstaluj moduł AzureRM](/powershell/azure/azurerm/install-azurerm-ps) w sesji programu PowerShell 5.1.
> 3. [Pobierz i zainstaluj program PowerShell Core w wersji 6.x lub nowszej](/powershell/scripting/install/installing-powershell-core-on-windows).
> 4. [Zainstaluj moduł Az](/powershell/azure/install-az-ps) w sesji programu PowerShell Core.
