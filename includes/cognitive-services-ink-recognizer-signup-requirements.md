Musisz mieć [konto interfejsu API Cognitive Services](../articles/cognitive-services/cognitive-services-apis-create-account.md) z dostępem do interfejsu API rozpoznawania pisma odręcznego. Jeśli nie masz subskrypcji platformy Azure, możesz [utworzyć konto](https://azure.microsoft.com/try/cognitive-services/) bezpłatnie. Klucz subskrypcji możesz uzyskać z [Azure Portal](../articles/cognitive-services/cognitive-services-apis-create-account.md#get-the-keys-for-your-resource) po utworzeniu konta lub [witrynie sieci Web systemu Azure](https://azure.microsoft.com/try/cognitive-services/my-apis) po aktywowaniu bezpłatnej wersji próbnej.
