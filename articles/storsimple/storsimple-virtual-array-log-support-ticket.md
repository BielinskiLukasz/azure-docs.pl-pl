# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Użyj usługi Menedżer urządzeń StorSimple, aby zarejestrować żądanie pomocy technicznej dla tablicy wirtualnej StorSimple

## <a name="overview"></a>Omówienie

Menedżer urządzeń StorSimple udostępnia możliwość **rejestrowania nowego żądania pomocy technicznej** w bloku podsumowania usługi. W tym artykule wyjaśniono, jak można rejestrować nowe żądanie pomocy technicznej i zarządzać jego cyklem życia z poziomu portalu.

## <a name="new-support-request"></a>Nowe żądanie pomocy technicznej

W zależności od [planu pomocy technicznej](https://azure.microsoft.com/support/plans/)można utworzyć bilety pomocy technicznej dla problemu w tablicy StorSimple Virtual bezpośrednio z bloku podsumowania usługi StorSimple Device Manager.

#### <a name="to-log-a-new-request"></a>Aby zarejestrować nowe żądanie

1. Przejdź do usługi Menedżer urządzeń StorSimple. W ustawieniach bloku podsumowania usługi przejdź do sekcji **POMOC + ROZWIĄZYWANIE PROBLEMÓW,** a następnie kliknij pozycję **Nowe żądanie pomocy technicznej**.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. W bloku **Podstawy** wykonaj następujące czynności:

    1. Z listy rozwijanej **Typ problemu** wybierz pozycję **Techniczne**. 
    
    2. Bieżąca **subskrypcja,** typ **usługi** i **zasób** (usługa StorSimple Device Manager) są wybierane automatycznie. 

    3. Określ co najmniej jedno urządzenie zarejestrowane w usłudze, na których występują problemy.

    4. Wybierz odpowiedni **plan pomocy technicznej,** jeśli masz wiele planów skojarzonych z subskrypcją. Aby włączyć pomoc techniczną, potrzebny jest płatny plan pomocy technicznej.

3. W **kroku 2**wybierz **ważność** i określ, czy problem jest związany z tablicą lub usługą StorSimple Device Manager. Ponadto wybierz **kategorię** dla tego problemu i podaj więcej **szczegółów** dotyczących problemu.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. W **kroku 3**podaj swoje dane kontaktowe. Pomoc techniczna firmy Microsoft będzie korzystać z tych informacji, aby uzyskać dalsze informacje, diagnozę i rozwiązanie problemu.
   
    ![Nowe żądanie pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Zarządzanie żądaniem pomocy technicznej

Po utworzeniu biletu pomocy technicznej możesz zarządzać jego cyklem życia z poziomu portalu.

#### <a name="to-manage-your-support-requests"></a>Aby zarządzać żądaniami pomocy technicznej

Aby przejść do strony pomocy i pomocy technicznej, przejdź do **sekcji Przeglądaj > Pomoc + pomoc + pomoc .**

![Zarządzanie żądaniami pomocy technicznej](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
