# <a name="view-or-search-for-your-user-groups-in-azure-active-directory"></a>Wyświetlanie lub wyszukiwanie grup użytkowników w usłudze Azure Active Directory
W tym artykule wyjaśniono, jak wyświetlać grupy w usłudze Azure Active Directory. (Azure AD). Jedną z funkcji zarządzania użytkownikami w usłudze Azure AD jest możliwość używania grup do wykonywania zadań administracyjnych, takich jak przypisywanie licencji lub uprawnień do wielu użytkowników jednocześnie.

## <a name="how-do-i-see-all-the-groups"></a>Jak mogę wyświetlić wszystkie grupy?
1. Zaloguj się do witryny [Azure Portal](https://portal.azure.com) przy użyciu konta, które jest administratorem globalnym katalogu.
2. Wybierz pozycję **Wszystkie usługi**, w polu tekstowym wprowadź filtr **Użytkownicy i grupy**, a następnie naciśnij klawisz **Enter**.

   ![Otwieranie użytkowników i grup](./media/active-directory-groups-view-azure-portal/search-user-management.png)
3. W bloku **Użytkownicy i grupy** wybierz pozycję **Wszystkie grupy**.

   ![Otwieranie bloku grup](./media/active-directory-groups-view-azure-portal/view-groups-blade.png)
4. W bloku **Użytkownicy i grupy — Wszystkie grupy** możesz dodawać lub usuwać wyświetlane kolumny, filtrować listę, aby wyszukać grupę, lub wprowadzać zmiany w grupach, do których masz wystarczające uprawnienia.
