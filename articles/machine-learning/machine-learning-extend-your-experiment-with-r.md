# R を使用した実験の拡張

使用して、R 言語の ML Studio の機能を拡張することができます、 [R スクリプトの実行 ][execute-r-script] モジュールです。

このモジュールは、複数の入力データセットを受け取り、1 つのデータセットを出力として生成します。 R スクリプトを入力することができます、 **R スクリプトの** のパラメーター、 [R スクリプトの実行 ][execute-r-script] モジュールです。

モジュールの各入力ポートにアクセスするには、次のようなコードを使用します。

    dataset1 <- maml.mapInputPort(1)

## 現在インストールされているすべてのパッケージを一覧表示する

インストールされているパッケージの一覧を変更できます。 完全な一覧を取得するには、次の行を含む、 [R スクリプトの実行 ][execute-r-script] モジュールの出力データセットに一覧が送信します。

    out <- data.frame(installed.packages())
    maml.mapOutputPort("out")

パッケージの一覧を表示するには、変換モジュールをなど接続 [[convert csv] の CSV に変換][convert-to-csv] の出力に、 [R スクリプトの実行 ][execute-r-script] 、実験を実行しているモジュールをクリックし、変換モジュールの出力 **ダウンロード**します。

## パッケージをインポートする

インポートすることもできますが既にインストールされていないステージングされた ML Studio リポジトリからでは、次のコマンドを使用して、パッケージ、 [R スクリプトの実行 ][execute-r-script] モジュールと zip 圧縮したパッケージのアーカイブします。

    install.packages("src/my_favorite_package.zip", lib = ".", repos = NULL, verbose = TRUE)
    success <- library("my_favorite_package", lib.loc = ".", logical.return = TRUE, verbose = TRUE)

ここで、 `my_favorite_package.zip` 、パッケージの zip ファイルが含まれています。
