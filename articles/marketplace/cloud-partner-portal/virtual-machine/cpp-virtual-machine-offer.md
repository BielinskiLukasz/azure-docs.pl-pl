# <a name="azure-virtual-machine-offer"></a>Oferta maszyny wirtualnej platformy Azure

[Portal Cloud partner](https://cloudpartner.azure.com/) nie obsługuje już tworzenia ofert maszyn wirtualnych platformy Azure i zarządzania nimi. Ta funkcja została przeniesiona do [Centrum partnerskiego](https://partner.microsoft.com/pcv/)firmy Microsoft. Aby uzyskać więcej informacji, zobacz [Tworzenie oferty maszyny wirtualnej platformy Azure](../../partner-center-portal/azure-vm-create-offer.md).
