
* [アプリ ID をアプリの登録](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991)します。 明示的なアプリ ID (いないワイルドカード アプリ ID) を作成および **バンドル ID**, 、正確なを使用して **バンドル ID** Xcode クイック スタート プロジェクト内にあります。 確認することが重要です、 **プッシュ通知** オプション。 

* 次に、 [プッシュ通知を構成する](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6)です。 "開発用" または "配布用" の SSL 証明書を作成できます (後で必ず Azure ポータル内の該当するオプションを選択してください)。
