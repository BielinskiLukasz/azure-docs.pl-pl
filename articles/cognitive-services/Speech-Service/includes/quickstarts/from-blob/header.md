W tym przewodniku szybki start użyjesz interfejsu API REST do rozpoznawania mowy z plików w procesie wsadowym. Proces wsadowy wykonuje transkrypcję mowy bez żadnych interakcji użytkownika. Oferuje prosty model programowania bez konieczności zarządzania współbieżnością, niestandardowymi modelami mowy ani innymi szczegółami. Wiąże się to z zaawansowanymi opcjami kontrolek, a jednocześnie wydajnym użyciem zasobów usługi Azure Speech.

[Przegląd transkrypcji usługi Batch](../../../batch-transcription.md) zawiera szczegółowe informacje dotyczące korzystania z tej funkcji. Szczegółowy interfejs API jest dostępny jako [dokument struktury Swagger](https://westus.cris.ai/swagger/ui/index#/Custom%20Speech%20transcriptions%3A), pod nagłówkiem `Custom Speech transcriptins`. 

Poniższy przewodnik Szybki Start przeprowadzi Cię przez przykład użycia.
