# Pierwsze zadania usługi Azure Stack POC

Podstawowe obiekty w usłudze Azure Stack — [plany, oferty i subskrypcje](azure-stack-key-features.md#services-plans-offers-and-subscriptions) — zapewniają dzierżawcom usługi IaaS, takie jak maszyny wirtualne i witryny sieci Web. Te usługi zapewniają wartość dla dzierżawców.

Oto następne kroki, które należy wykonać po [wdrożeniu](azure-stack-deploy.md) usługi Azure Stack POC i [zalogowaniu się](azure-stack-connect-azure-stack.md) w niej.

Najpierw [zaloguj się jako administrator usługi](azure-stack-connect-azure-stack.md#log-in-as-a-service-administrator), a następnie wykonaj poniższe czynności.

1.  [Utwórz plan](azure-stack-create-plan.md), aby zapewnić usługi obliczeniowe, sieciowe i magazynowe.

2.  [Utwórz ofertę](azure-stack-create-offer.md), która będzie obejmować ten plan.

Następnie [zaloguj się jako dzierżawca](azure-stack-connect-azure-stack.md#log-in-as-a-tenant) i wykonaj poniższe czynności:

1.  [Subskrybuj ofertę](azure-stack-subscribe-plan-provision-vm.md) i uzyskaj dostęp do planów oferty.

2.  [Użyj usług](azure-stack-provision-vm.md) zapewnianych przez plan oferty do utworzenia maszyny wirtualnej.
