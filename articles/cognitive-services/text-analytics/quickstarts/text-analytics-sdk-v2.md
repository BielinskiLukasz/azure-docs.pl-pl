# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>Szybki Start: korzystanie z biblioteki klienta analiza tekstu w wersji 2

Rozpocznij pracę z biblioteką klienta analiza tekstu v2. Wykonaj następujące kroki, aby zainstalować pakiet i wypróbować przykładowy kod dla podstawowych zadań.

Za pomocą analiza tekstu Client Library v2 wykonaj następujące:

* Analiza opinii
* Wykrywanie języka
* Rozpoznawanie jednostek
* Wyodrębnianie kluczowych fraz
