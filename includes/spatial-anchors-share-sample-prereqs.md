## <a name="prerequisites"></a>Wymagania wstępne

Aby ukończyć kroki tego samouczka, upewnij się, że dysponujesz następującymi elementami:

* Komputer Windows, na którym <a href="https://www.visualstudio.com/downloads/" target="_blank">programu Visual Studio 2017</a> lub nowszy jest zainstalowany za pomocą **ASP.NET i tworzenie aplikacji internetowych** obciążenia.
* [Zestaw .NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* Co najmniej jedno urządzenie (z systemem iOS lub Android) na którym chcesz wdrożyć i uruchomić aplikację.
  * Jeśli używasz systemu Android, musisz mieć:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> lub nowszym, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> lub nowszym, oraz <a href="https://git-scm.com/download/win" target="_blank">Git dla Windows</a> zainstalowane na komputerze Windows.
    * A <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">włączone dla deweloperów</a> i <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">obsługą ARCore</a> urządzenia z systemem Android.
  * Jeśli używasz systemu iOS, musisz mieć:
    * Komputer z systemem macOS, na którym <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> lub nowszym, <a href="https://cocoapods.org" target="_blank">CocoaPods</a>, i <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> lub nowszy.
    * Włączone dla deweloperów <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">zgodnego z ARKit</a> urządzenia z systemem iOS.
    * Zainstalowane za pomocą Homebrew narzędzie Git. Wprowadź następujące polecenie w jednym wierszu w terminalu: `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` Następnie uruchom `brew install git`.
