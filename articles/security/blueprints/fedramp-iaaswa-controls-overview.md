# <a name="fedramp-overview"></a>Omówienie FedRAMP

[Federalne ryzyka i Program do zarządzania autoryzacji (FedRAMP)](https://www.fedramp.gov), to program całej dla instytucji rządowych USA, który zawiera standardowe podejście do oceny zabezpieczeń, autoryzacji i ciągłego monitorowania produktów chmury i usługi. FedRAMP definiuje proces oceny zabezpieczeń oparte na standardowych zestaw wymagań pochodzące z [National Institute of Standards i technologii (NIST) specjalne SP publikacji 800-53](https://csrc.nist.gov/publications/detail/sp/800-53/rev-4/final) środki zabezpieczające.
