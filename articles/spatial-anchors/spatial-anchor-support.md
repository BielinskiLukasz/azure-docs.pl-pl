# <a name="azure-spatial-anchors-support-options"></a>Opcje pomocy technicznej Azure kotwic przestrzenne

## <a name="open-a-tech-support-ticket"></a>Otwórz bilet pomocy technicznej tech

Aby otworzyć bilet pomocy technicznej w witrynie Azure Portal do kotwic przestrzenne platformy Azure:

1. Za pomocą [witryny Azure portal](https://azure.microsoft.com/account/) Otwórz, wybierz ikonę Pomoc w pasku menu u góry, a następnie wybierz **Pomoc i obsługa techniczna** przycisku. 

![Azure portal Pomoc i obsługa techniczna](./media/spatial-anchor-support.png)

1. Pomoc i obsługa techniczna otwarty, wybierz stronę **+ nowe żądanie pomocy technicznej**.

![Azure portal nowe żądanie obsługi](./media/spatial-anchor-support2.png)

1. Po wypełnieniu pól bilet pomocy technicznej: 

- Typ problemu: Techniczna
- Usługa: Spatial Anchors

![Pola bilet pomocy technicznej platformy Azure portal](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Pomoc techniczna społeczności

Uzyskać pomoc od społeczności użytkowników, zobacz [Azure przestrzenne kotwice w witrynie Stack Overflow](https://stackoverflow.com/questions/tagged/azure-spatial-anchors). 

## <a name="provide-content-feedback"></a>Opinii w zawartości

W dolnej części każdego artykułu zawartości istnieje możliwość Otwórz problem w usłudze GitHub i wyraź swoją opinię zawartość dokumentacji Azure przestrzenne zakotwiczenia.
