Ułatwia integrowanie z obsługą chmury [oprogramowanie jako usługa (SaaS)](https://azure.microsoft.com/overview/what-is-saas/) aplikacji za pomocą usługi Azure Active Directory, opracowaliśmy kolekcji samouczków, które umożliwia przeprowadzenie konfiguracji.

Aby uzyskać listę wszystkich aplikacji SaaS, które zostały wstępnie zintegrowane w usłudze Azure AD, zobacz [Active Directory Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
