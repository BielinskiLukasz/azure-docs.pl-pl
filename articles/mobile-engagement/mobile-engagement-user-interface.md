# Azure Mobile Engagement ユーザー インターフェイス (UI) の使用方法

## 概要:

Azure モバイル エンゲージメントを統合した後 [SDK ][link 5] アプリケーションに統合し、Azure Mobile Engagement の基本を理解する [概念 ][link 6], 、アプリの Azure モバイル エンゲージメントと対話する 2 つの方法があります: 標準ユーザーが使用できる、 [ユーザー インターフェイス ][link 1] し、開発者ユーザーは HTTP REST ベース [Api ][link 4]します。 Azure モバイル エンゲージメントを参照してください。 問題が発生する必要がある場合は [トラブルシューティング ガイド ][link 2]します。
