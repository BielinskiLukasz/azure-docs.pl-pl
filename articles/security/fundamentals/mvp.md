# <a name="azure-security-mvp-program"></a>Program MVP dotyczący zabezpieczeń platformy Azure

Firma Microsoft najbardziej cennych specjalistów (MVP) to liderzy społeczności, którzy wykazały, że RETORSYJNE zaangażowanie w inne osoby. MVP umożliwianie innym użytkownikom korzystania z technologii firmy Microsoft. Współdzielą oni swoją pasjąą wiedzę i wiedzą techniczną ze społecznością oraz z firmą Microsoft.

Microsoft Azure teraz rozpoznaje ekspertów społeczności ze szczególnym doświadczeniem w zakresie zabezpieczeń platformy Azure. Program Microsoft Mvpe może udzielić SPECJALISTy w Microsoft Azure w obszarze udział zabezpieczeń platformy Azure.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

Nie ma żadnego wzorca dla MVP. Jest to częściej, ponieważ różni się ona od technologii i cyklu życia. Niektóre z tych kryteriów obejmują:

- Wpływ wkładu kandydat na fora online, takie jak odpowiedzi firmy Microsoft, TechNet i MSDN
- Wiki i zawartość online
- Konferencje i grupy użytkowników
- Podcast, witryny sieci Web, blogi i Media społecznościowe
- Artykuły i książki.
