# <a name="view-support-information-lab-user-in-azure-lab-services"></a>Wyświetlanie informacji o pomocy technicznej (użytkownik laboratorium w usługach Azure Lab Services)
W tym artykule wyjaśniono, jak (jako użytkownik laboratorium) można wyświetlić następujące informacje pomocy technicznej:

- Adres URL
- Adres e-mail
- Telefon
- Dodatkowe instrukcje

Można użyć tych informacji, aby uzyskać pomoc, gdy napotkasz wszelkie problemy techniczne podczas korzystania z laboratorium na koncie laboratorium.

 
## <a name="view-support-information"></a>Wyświetlanie informacji o pomocy technicznej
1. Zaloguj się do [portalu usług Azure Lab Services](https://labs.azure.com).
2. Wybierz **laboratorium lub maszynę wirtualną,** dla których potrzebujesz pomocy, i **wybierz?** w prawym górnym rogu strony. 
3. Upewnij się, że widzisz linki do **witryny pomocy technicznej widoku,** **obsługi poczty e-mail**i **numeru telefonu pomocy technicznej.**

    ![Wyświetlanie informacji o pomocy technicznej](../media/lab-user-support-information/support-information.png)
4. Można wyświetlić informacje kontaktowe pomocy technicznej dla innego laboratorium, przełączając się do tego laboratorium na liście rozwijanej. 

    ![Przełączanie do innego laboratorium](../media/lab-user-support-information/switch-another-lab.png)
5. Teraz zobaczysz informacje kontaktowe pomocy technicznej dla innego laboratorium. 

    ![Informacje o pomocy technicznej innych laboratoriów](../media/lab-user-support-information/second-lab-support-information.png)
