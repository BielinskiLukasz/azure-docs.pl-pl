# <a name="view-requests-for-an-access-package-in-azure-ad-entitlement-management"></a>Wyświetlanie żądań pakietu dostępu w zarządzaniu uprawnieniami usługi Azure AD

W zarządzania uprawnieniami usługi Azure AD można zobaczyć, kto zażądał pakietów dostępu, ich zasad i stanu. W tym artykule opisano sposób wyświetlania żądań dla pakietów dostępu.

## <a name="view-requests"></a>Wyświetlanie żądań

**Rola wstępna:** Administrator globalny, administrator użytkownika, właściciel katalogu lub menedżer pakietów programu Access

1. W witrynie Azure portal kliknij pozycję **Usługa Azure Active Directory,** a następnie kliknij pozycję **Zarządzanie tożsamościami**.

1. W menu po lewej stronie kliknij pozycję **Pakiety programu Access,** a następnie otwórz pakiet dostępu.

1. Kliknij **pozycję Żądania**.

1. Kliknij konkretne żądanie, aby wyświetlić dodatkowe szczegóły.

    ![Lista żądań dotyczących pakietu dostępu](./media/entitlement-management-access-package-requests/requests-list.png)
