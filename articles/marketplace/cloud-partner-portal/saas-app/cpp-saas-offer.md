# <a name="azure-saas-application-offer"></a>Oferta aplikacji Azure SaaS

[Portal cloud partnerów](https://cloudpartner.azure.com/) nie obsługuje już tworzenia ofert aplikacji SaaS i zarządzania nimi.  Zamiast tego ta funkcja została przeniesiona do [portalu Centrum partnerów](https://partner.microsoft.com/pcv/)firmy Microsoft .  Aby uzyskać więcej informacji, zobacz [Tworzenie nowej oferty SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Następne kroki

Przed utworzeniem nowej oferty aplikacji SaaS należy zapoznać się z [przewodnikiem publikowania ofert aplikacji SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) i przejrzeć kroki portalu Centrum partnerów wymienione na [liście kontrolnej Tworzenie oferty](../../partner-center-portal/offer-creation-checklist.md).
