# <a name="azure-services-by-fedramp-and-dod-cc-srg-audit-scope"></a>Usługi platformy Azure, FedRAMP i DoD DW SRG inspekcji zakresu

Usług w chmurze firmy Microsoft dla instytucji rządowych spełnia wymagania wymagających nam Federal Risk & Authorization Management Program (FedRAMP) oraz Departamentu Obrony USA, z informacji na poziomie wpływu 2 do 5. Wdrażając chronione usług w tym Azure dla instytucji rządowych, Office 365 w Stanach Zjednoczonych Dla instytucji rządowych i Dynamics 365 dla instytucji rządowych, federalnych i agencjach obrony mogą korzystać z szeroką gamę zgodnych usług.

Ten artykuł zawiera szczegółową listę usług w chmurze w zakresie platformy Azure i Azure dla instytucji rządowych, FedRAMP i DoD DW SRG ofert zgodności.

#### <a name="terminologysymbols-used"></a>Terminologia/symbole używane

* DoD CC SRG = Departament Obrony Cloud Computing Security Requirements Guide
* Zagadnienie = poziom wpływu informacji
* FedRAMP = Federal Risk and Authorization Management Program  
* : heavy_check_mark: = wskazuje usługę zdobyła ten zakres inspekcji.
