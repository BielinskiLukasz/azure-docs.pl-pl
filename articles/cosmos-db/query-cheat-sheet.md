# <a name="azure-cosmos-db-query-cheat-sheets"></a>Azure DB rozwiązania Cosmos zapytania cheat arkuszy

**Arkusze cheat kwerendy bazy danych Azure rozwiązania Cosmos** pomagają w szybkim zapytania zapisu danych przez wyświetlanie typowych kwerend bazy danych, operacje, funkcje i operatory łatwe drukowanie dokumentów PDF. Arkusze cheat zawierają informacje dotyczące interfejsu API SQL, interfejsu API bazy danych MongoDB, tabela interfejsu API i interfejsu API programu Gremlin/Graph. 

Wybierz spośród pobierania wymiarach lub rozmiaru A3. 

## <a name="letter-sized-cheat-sheets"></a>Arkusze wymiarach wskazówkami

Pobierz [arkusze cheat wymiarach kwerendy bazy danych Azure rozwiązania Cosmos](http://go.microsoft.com/fwlink/?LinkId=623215) Jeśli zamierzasz drukowania wymiarach papieru (8,5 "x 11").
