## Azure IoT Suite

Microsoft Azure IoT Suite は、*リモート モニタリング*や*接続済みデバイス*など、一般的な IoT シナリオに対処する、拡張可能な一連の事前構成済みソリューションを介してすばやく開始できるエンタープライズ グレードのソリューションです。

構成済みのソリューションは次のとおりです。

- 包括的な機能、エンド ツー エンド ソリューションを始めるために、シミュレートされたデバイスを含む構成済みの Azure サービスをなどが [Azure IoT Hub][], 、[Azure Event Hubs][], 、[Azure Stream Analytics][], 、[Azure Machine Learning][], 、および [Azure ストレージの][], 、およびソリューションの特定の管理コンソールです。
- 独自の IoT シナリオを実装するためにカスタマイズおよび拡張できる、実績のある、実稼働可能なコードが含まれています。

興味場合もあります。、 [Azure IoT Hub []][] 構成済みのソリューションの多くを使用するサービスです。 [Azure IoT Hub:operator[]][] データの処理と分析機能、サンプルのソリューション アーキテクチャで定義されている多数用意されています。
