# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Wskazówki dotyczące usługi Azure Data Lake Storage Gen1 klientów

Usługa Azure Data Lake magazyn Gen2 — wersja zapoznawcza oferuje natywne obiektu Blob magazynu funkcje, w tym warstw danych i zarządzanie cyklem życia i większe bezpieczeństwo w sieci, oprócz dostępnych w Data Lake Storage Gen1 system plików Hadoop zgodny.

Jeśli jesteś istniejącym klientem Data Lake Storage Gen1, firma Microsoft zachęca do oceny wszystkich nowych i ulepszonych funkcji Data Lake Storage Gen2. Ponadto Data Lake Storage Gen1 będą w dalszym ciągu być obsługiwane. Data Lake Storage Gen1 i Gen2 mogą współistnieć, co z istniejących zasobów i aplikacji korzystających z programu Data Lake Storage Gen1 nadal działać bez aktualizacji. Można opracować plan uaktualnić swoje rozwiązanie do Data Lake Storage Gen2 wygodne. Aby uzyskać instrukcje dotyczące uaktualniania, zobacz [uaktualniania rozwiązań do analizy danych big data do usługi Azure Data Lake Gen2 — wersja zapoznawcza](data-lake-storage-upgrade.md).

Jeśli oceniasz obecnie Data Lake Storage Gen1, firma Microsoft zachęca do uczestnictwa w wersji zapoznawczej Gen2 i ocenić nowych i ulepszonych funkcji oferowanych przez Data Lake Storage Gen2. Podobnie, jeśli jesteś istniejącym klientem magazynu obiektów Blob, firma Microsoft zachęca do oceny, czy system plików Hadoop zgodny jest odpowiednia dla potrzeb analitycznych.
