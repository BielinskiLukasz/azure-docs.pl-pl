## <a name="install-the-azure-functions-extension"></a>Instalowanie rozszerzenia usługi Azure Functions

Rozszerzenie usługi Azure Functions służy do tworzenia i testowania funkcji i wdrażania ich na platformie Azure.

1. W programie Visual Studio Code otwórz **rozszerzenia** i wyszukaj **funkcje platformy Azure**lub wybierz to [łącze w programie Visual Studio Code](vscode:extension/ms-azuretools.vscode-azurefunctions).

1. Wybierz **pozycję Zainstaluj,** aby zainstalować rozszerzenie dla programu Visual Studio Code:

    ![Instalowanie rozszerzenia usługi Azure Functions](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. Po instalacji wybierz ikonę platformy Azure na pasku aktywności. Powinieneś widzieć obszar usługi Azure Functions na pasku bocznym.

    ![Obszar usługi Azure Functions na pasku bocznym](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)
