# <a name="azure-data-factory-documentation"></a>Dokumentacja usługi Azure Data Factory

Azure Data Factory to usługa integracji danych oparta na chmurze, która automatyzuje operacje przenoszenia i przekształcania danych. Usługa Data Factory tworzy rozwiązania do integracji danych, które umożliwiają ich pozyskiwanie z różnych magazynów, przekształcanie i przetwarzanie danych oraz publikowanie wyników w magazynach danych.
