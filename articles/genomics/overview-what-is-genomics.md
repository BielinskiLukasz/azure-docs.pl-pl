# <a name="what-is-microsoft-genomics"></a>Co to jest Microsoft Genomics?
Microsoft Genomics oferuje implementację chmury Burrows-Wheeler Aligner (BWA) i Zestawu narzędzi do analizy genomu (GATK) do analizy wtórnej. Usługa jest certyfikowana przez ISO i zgodna z przepisami HIPAA i oferuje przewidywalność cen dla twoich potrzeb w zakresie sekwencjonowania genomu. Dowiedz się, jak korzystać z usługi Microsoft Genomics i integrować się z naszym interfejsem API, czytając nasze przewodniki Szybki start, samouczki i dokumentację.

## <a name="support-your-most-demanding-sequencing-needs"></a>Wesprzyj najbardziej wymagające potrzeby w zakresie sekwencjonowania
Zamiast zarządzać własnymi centrami danych, skorzystaj ze skali i środowiska firmy Microsoft w uruchamianiu obciążeń w skali eksabajtów. Nasza implementacja BWA-GATK w chmurze jest bardzo zgowalna z rurociągiem najlepszych praktyk Broad Institute.


## <a name="keep-your-business-running"></a>Utrzymuj działalność firmy
Firma Microsoft Genomics oferuje umowę dotyczącą poziomu usług dostępności (SLA) o wartości 99,99% do odbierania żądań przepływu pracy. Dowiedz się więcej o [umowy SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/)microsoft genomics .


## <a name="secure-your-data"></a>Zabezpieczanie danych
Usługa Microsoft Genomics posiada certyfikaty ISO 27001, ISO 27018 i ISO 9001 oraz zgodność z przepisami USTAWY HIPAA. Odwiedź [Centrum zaufania Azure](https://www.microsoft.com/trustcenter/security), aby uzyskać informacje o zabezpieczeniach platformy Azure.
