# <a name="glossary"></a>Słownik

[Niestandardowe w usłudze Translator](https://portal.customtranslator.azure.ai) glosariusz opisano terminy, które można napotkać podczas pracy z niestandardowych w usłudze Translator.

| **Wyraz lub frazę**       | **Definicja**                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Język źródłowy          | Źródło jest język rozpoczynasz korzystanie z i do przekonwertowania na inny język ("target").                                                                                                                                                                                                                                                                                                                                                         |
| Język docelowy          | Język docelowy jest język, który ma tłumaczenie maszynowe, aby podać po odebraniu języka źródłowego.                                                                                                                                                                                                                                                                                                                                               |
| Plik jednojęzyczne         | Jednojęzyczne plik ma jeden język, który nie jest sparowana z innego pliku innego języka.                                                                                                                                                                                                                                                                                                                                                                 |
| Pliki równoległe           | Równoległe pliku jest kombinacją dwóch plików o odpowiedni tekst. Język źródłowy ma jeden plik. Na drugim języka docelowego.                                                                                                                                                                                                                                                                                                                                         |
| Wyrównanie zdania       | Zestaw równoległy danych mają wyrównane do zdań zdań, które reprezentują ten sam tekst w obu językach. Na przykład w pliku źródłowym równoległe zdanie pierwsze powinny być teoretycznie mapowane do zdanie równoległe pliku docelowego.                                                                                                                                                                                                                               |
| Tekst wyrównany             | Jedną z najważniejszych kroków weryfikacji pliku jest dopasowanie zdań w dokumentach równoległych. Elementy są wyrażane inaczej w różnych językach. Różne języki mają również różnych zleceniach programu word. Ten krok jest zadaniem wyrównywanie zdania z tej samej zawartości, dzięki czemu może służyć do trenowania. Wyrównanie zdania niski wskazuje, że może występować problem z jedną lub obie te pliki. |
| Word istotne / Unbreaking | Dzielenie wyrazów jest funkcja oznaczania granice między wyrazami. Wiele systemów pisania użyj spacji do określenia granic między wyrazami. Word unbreaking odnosi się do usunięcia widoczny znacznik, który może być wstawiany między wyrazami w poprzednim kroku.                                                                                                                                                                                                  |
| ograniczniki               | Ograniczniki sposobów, które zdanie jest podzielony na segmenty lub ograniczone czasowo margines między zdaniami. Na przykład w angielskim spacji rozgraniczania wyrazów w dwukropki, i średnikami rozgraniczania klauzule okresy rozgraniczania zdań.                                                                                                                                                                                                                                         |
| Pliki szkolenia           | Plik szkolenia służy do nauki system tłumaczenia maszynowego sposób mapowania z jednego języka (źródło) na język docelowy (docelowy). Im więcej danych, które można podać lepiej system będzie wykonywać na tłumaczenia.                                                                                                                                                                                                               |
| Dostosowywanie plików             | Te pliki są często losowo uzyskiwane z szkolenia ustawienie (o ile nie zaznaczaj żadnego zestawu dostrajania). Autoselected zdania są używane do dostosowywania systemu i upewnij się, że działa prawidłowo. Należy podjąć decyzję o utworzenia plików dostrajania, upewnij się, że są one losowy zestaw zdań między domenami Jeśli chcesz utworzyć model tłumaczenia ogólnego przeznaczenia.                                                                                 |
| Testowanie plików            | Są to często pochodnej pliki, wybierane losowo spośród zestawu szkoleniowego (Jeśli nie zaznaczono żadnego zestawu testów). Celem tych zdań jest do oceny dokładności modelu tłumaczenia. Są to zdania, przeznaczonych do upewnij się, że dokładnie tłumaczy systemu. Dlatego możesz utworzyć zestaw testów i przekaż go do usługi translator, aby upewnić się, że te zdania są używane w wersji ewaluacyjnej systemu (Generowanie wyników BELEU).   |
| Pole kombi pliku               | Typ pliku, w którym źródło i zdań przetłumaczonych znajdują się w tym samym pliku. Obsługiwane formaty plików (".tmx", ".xliff", ".xlf", ".lcl", "xlsx").                                                                                                                                                                                                                                                                                                                       |
| Plik archiwum             | Plik, który zawiera inne pliki. Obsługiwane formaty plików (pliku zip, gz, tgz).                                                                                                                                                                                                                                                                                                                                                                                                |
| Wynik BLEU               | [BELEU](what-is-bleu-score.md) jest standardowa metoda oceny "dokładności" lub dokładności modelu tłumaczenia. Jednak istnieją inne metody oceny, Microsoft Translator korzysta z metody BELEU dokładności raportów do projektu właścicieli.
