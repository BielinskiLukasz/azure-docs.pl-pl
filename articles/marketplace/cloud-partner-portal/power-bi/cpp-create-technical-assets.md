# <a name="create-power-bi-app-technical-assets"></a>Tworzenie aplikacji Power BI, zasoby techniczne

Podstawowy zasobów technicznych należy dla tego typu oferty jest aplikacja Power BI, który jest kolekcją podstawowego zestawu danych, raport i (lub) pulpit nawigacyjny i opcjonalnie podłączonych usług i osadzone zestawów danych. (Usługa power BI aplikacje są unowocześnienia typów zasobów, które wcześniej znana jako [pakiety zawartości](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction).)  Aby uzyskać pełne informacje na temat tworzenia tych aplikacji, zobacz [aplikacje usługi Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Uzyskiwanie adresu URL instalacji

Tworzenie aplikacji Power BI w tylko dostępne w ramach [usługi Power BI](https://powerbi.microsoft.com/) i wymaga zalogować się przy użyciu [licencji usługi Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Po utworzeniu aplikacji usługi i przetestowane w usłudze Power BI, Zapisz adres URL instalacji aplikacji, która generuje programie Power BI.  Należy podać ten adres URL w [karta informacje techniczne](./cpp-technical-info-tab.md) portalu Cloud Partner przesyłania formularza.
 

## <a name="next-steps"></a>Kolejne kroki

Po utworzeniu i przetestowane aplikacji usługi Power BI i utworzyć skojarzone oferty można [opublikować ofertę usługi Power BI](./cpp-publish-offer.md).
