> [!NOTE]
>
> Ten artykuł został ostatnio zaktualizowany o polecenia nowego modułu Azure PowerShell Az. Jest to nowy moduł programu PowerShell dla platformy Azure, zgodny z wersjami programu PowerShell 5.x i 6. Moduł AzureRM będzie w dalszym ciągu otrzymywać aktualizacje z poprawkami krytycznych usterek, lecz nowe funkcje będą udostępniane tylko modułowi Az.
>
> * Aby dowiedzieć się więcej na temat modułu Az i zapewnić dalszą obsługę modułu AzureRM, zobacz [Wprowadzenie do nowego modułu Azure PowerShell Az](/powershell/azure/new-azureps-module-az).
> * Moduł Az jest już dostępny w usłudze Cloud Shell. Zgodność z modułem AzureRM jest zapewniona automatycznie.
>   Aby uzyskać instrukcje dotyczące instalowania modułu Az, zobacz [Instalowanie programu Azure PowerShell](/powershell/azure/install-az-ps).
> * Aby włączyć polecenia zgodności z modułem AzureRM, uruchom polecenie `Enable-AzureRmAlias` po zakończeniu instalacji.
>   Aby uzyskać więcej informacji o zgodności, patrz [Migracja z modułu Azure PowerShell AzureRM do modułu Az](/powershell/azure/migrate-from-azurerm-to-az).
