### <a name="open-the-mfa-page"></a><a name="mfa"></a>Otwieranie strony usługi MFA

1. Zaloguj się do Portalu Azure.
2. Przejdź do **usługi Azure Active Directory -> Wszyscy użytkownicy**.
3. Wybierz **opcję Uwierzytelnianie wieloskładnikowe,** aby otworzyć stronę uwierzytelniania wieloskładnikowego.

   ![Logowanie](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Wybierz użytkowników

1. Na stronie **uwierzytelniania wieloskładnikowego** wybierz użytkowników, dla których chcesz włączyć uwierzytelnianie wieloskładnikowe.
2. Wybierz pozycję **Włącz**.

   ![Wybierz pozycję](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
