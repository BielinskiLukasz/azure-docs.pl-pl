* [アプリ ID をアプリの登録](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/MaintainingProfiles/MaintainingProfiles.html#//apple_ref/doc/uid/TP40012582-CH30-SW991)します。 ワイルドカード アプリ ID ではなく明確なアプリ ID を作成し、**バンドル ID** では、Xcode クイックスタート プロジェクト内に含まれる **バンドル ID** を使用します。 **[プッシュ通知]** チェック ボックスをオンにすることも重要です。

* 次に、 [プッシュ通知を構成する](https://developer.apple.com/library/ios/documentation/IDEs/Conceptual/AppDistributionGuide/AddingCapabilities/AddingCapabilities.html#//apple_ref/doc/uid/TP40012582-CH26-SW6)します。 "開発用" または "配布用" の SSL 証明書を作成できます (後で必ず Azure ポータル内の該当するオプションを選択してください)。
