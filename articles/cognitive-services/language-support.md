# <a name="natural-language-support-for-azure-cognitive-services"></a>Obsługa języka naturalnego dla usług Azure Cognitive Services

Usługi Azure Cognitive Services umożliwia tworzenie aplikacji, które Zobacz, słyszeć, porozmawiać z przedstawicielem i zrozumieć użytkowników. Między tymi usługami więcej niż trzech tuzina języki są obsługiwane, dzięki czemu użytkownicy mogą komunikować się ze swoją aplikacją w sposób naturalny.

W tym artykule jest dzielony na dwie sekcje. Pierwszy zawiera listę języków podstawowych, które często są obsługiwane w usługach Azure Cognitive Services. Druga sekcja koncentruje się na obsługę dodatkowych języków dla poszczególnych usług. Dostępne wersje językowe mogą się znacząco różnić rynku i kraju/regionu.

## <a name="core-languages"></a>Języki Core

Te języki core są obsługiwane w usługach Azure Cognitive Services:

* Chiński
* Angielski
* Francuski
* Niemiecki
* Włoski
* Japoński
* Koreański
* Portugalski
* Hiszpański

> [!NOTE]
> ¹ LUIS i Video Indexer: Zamiana mowy na tekst nie jest obsługiwana.

## <a name="additional-language-availability-by-service"></a>Dostępność dodatkowych języków przez usługę
