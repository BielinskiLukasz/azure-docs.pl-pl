カーネギーメロン大学 StatLib 研究所が提供したデータセットを多少変更したものです。 このデータセットは 1983 年に米国統計学会で使用された実績があります。<p> </p>さまざまな車体の燃料消費量 (ガロンあたりのマイル数) と、シリンダー数、エンジン排気量、馬力、総重量、加速度などの情報が記載されています。<p> </p><b>用途:</b> 3 つの多値離散属性と 5 つの連続属性に基づいて燃費効率を予測します。 <p> </p><b>関連の研究:</b> カーネギーメロン大学 StatLib (1993 年)。 UCI Machine Learning Repository <a href="http://archive.ics.uci.edu/ml">http://archive.ics.uci.edu/ml</a>. カリフォルニア州アーバイン: カリフォルニア大学、情報・コンピューター サイエンス学部
