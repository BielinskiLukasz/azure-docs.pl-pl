> [AZURE.NOTE] .NET 用ストレージ クライアント ライブラリの ODataLib 依存は、ODataLib で解決されます (バージョン 5.0.2 以上) と WCF Data Services ではなく、NuGet から利用可能なパッケージです。 ODataLib ライブラリは、直接ダウンロードすることも、NuGet を使用してコード プロジェクトで参照することもできます。 ストレージ クライアント ライブラリによって使用される ODataLib パッケージは [OData](http://nuget.org/packages/Microsoft.Data.OData/5.0.2), 、[Edm](http://nuget.org/packages/Microsoft.Data.Edm/5.0.2), 、および [空間](http://nuget.org/packages/System.Spatial/5.0.2)します。
> 
> これらのライブラリが Azure テーブル ストレージ クラスで使用されるときは、ストレージ クライアント ライブラリを使用したプログラミングの必須の依存関係です。
