## <a name="setupdevenv"></a>Konfigurowanie środowiska projektowego
Aby rozpocząć, skonfiguruj środowisko projektowe za pomocą programu Visual Studio i zestawu Azure SDK.

* Visual Studio 2017 i nowszym zawiera zestaw SDK platformy Azure. Jeśli używasz programu Visual Studio 2017 lub później, żadna dodatkowa konfiguracja jest wymagany dla środowiska programistycznego.
* W przypadku programu Visual Studio 2015 kliknij następujący link, aby zainstalować [zestaw Azure SDK dla programu Visual Studio 2015](https://go.microsoft.com/fwlink/?linkid=518003).
* W przypadku programu Visual Studio 2013 kliknij następujący link, aby zainstalować [zestaw Azure SDK dla programu Visual Studio 2013](https://go.microsoft.com/fwlink/?LinkID=324322).

* Jeśli nie masz zainstalowanego programu Visual Studio, zainstaluj [Visual Studio 2019](https://www.visualstudio.com/) przy użyciu zestawu SDK platformy Azure.

> [!NOTE]
> Zależnie od liczby składników zależnych zestawu SDK, które znajdują się już na komputerze, instalowanie zestawu SDK może trwać od kilku minut do pół godziny lub dłużej.
