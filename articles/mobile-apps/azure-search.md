# <a name="ai-powered-cloud-service-with-azure-search"></a>Usługa w chmurze oparta na AI z Azure Search
[Azure Search](https://azure.microsoft.com/services/search/) to rozwiązanie w chmurze typu "wyszukiwanie jako usługa", które udostępnia deweloperom interfejsy API i narzędzia umożliwiające dodawanie bogatego środowiska wyszukiwania do aplikacji sieci Web, mobilnych i firmowych. Niestandardowy kod wywołuje pozyskiwanie danych (indeksowanie) w celu utworzenia i załadowania indeksu. Po drugiej stronie kod aplikacji wystawia żądania zapytań i obsługuje odpowiedzi. Środowisko wyszukiwania jest zdefiniowane w kliencie przy użyciu funkcji z Azure Search, z wykonywaniem zapytań w odniesieniu do utrwalonego indeksu tworzonego, własnego i przechowywanego w usłudze.

## <a name="azure-search-features"></a>Funkcje Azure Search
- **Wyszukiwanie tekstu swobodnego** za pomocą wyszukiwania pełnotekstowego, prostej składni zapytań i składni zapytań Lucene.
- **Proste profile oceniania** do modelowania znaczenia jako funkcja wartości w samych dokumentach.
- **Wyszukiwanie geograficzne** umożliwia użytkownikom Eksplorowanie danych na podstawie bliskości wyniku wyszukiwania do lokalizacji fizycznej.
- **Funkcje środowiska użytkownika** z autouzupełnianiem, sugestiami wyszukiwania, sortowaniem i stronicowaniem.
- **Wyszukiwanie poznawcze** dla analizy obrazów i tekstu można zastosować do potoku indeksowania, aby wyodrębnić informacje tekstowe z nieprzetworzonej zawartości.
- **Magazyn wiedzy** do zapisywania wzbogacań utworzonych podczas indeksowania.
- Łączenie się z usługą Azure SQL Database, Azure Cosmos DB lub Azure Blob Storage jako **źródłami danych** w celu wyodrębnienia zawartości z możliwością wyszukiwania w podstawowych magazynach danych.
- **Kreator importu danych** służy do konfigurowania indeksatorów, **projektanta indeksów** w celu rozdzielenia indeksu i **Eksploratora wyszukiwania** do testowania zapytań i udoskonalania profilów oceniania.

Znajdź szczegółowy opis [funkcji](/azure/search/search-what-is-azure-search#feature-descriptions)Azure Search.
