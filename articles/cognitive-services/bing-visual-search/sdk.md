# <a name="bing-visual-search-sdk"></a>Wyszukiwanie wizualne Bing zestawu SDK

Wyszukiwanie wizualne Bing udostępnia następujące klienta zestawów SDK, których można użyć do korzystania z interfejsu API wyszukiwania wizualnego.

* [Przykłady dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Biblioteki .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) dla definicji i zależności
* [Przykłady dla platformy Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Biblioteki środowiska node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) dla definicji i zależności
* [Przykłady języka Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Biblioteki Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) dla definicji i zależności
* [Przykłady w języku Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) dla definicji i zależności

Przykłady zestawu SDK dla każdego z języków zawierają plik ReadMe z wymagań wstępnych dotyczących używania zestawu SDK oraz kroki dotyczące instalowania i uruchamiania przykładów.

### <a name="sdk-tools"></a>Narzędzia zestawu SDK
W tym repozytorium Git można znaleźć narzędzi zestawu SDK, aby poznać możliwości wyszukiwania wizualnego Bing: [narzędzia wyszukiwania wizualnego Bing](https://github.com/Azure-Samples/bing-visualsearch-tools)
