# <a name="grant-permissions-to-users-from-partner-organizations-in-your-azure-active-directory-tenant"></a>Udziel uprawnień użytkownikom z organizacji partnerskich w dzierżawie usługi Azure Active Directory

Użytkownicy współpracy B2B usługi Azure Active Directory (Azure AD) są dodawane jako użytkownicy-goście do katalogu, a uprawnienia gościa w katalogu są domyślnie ograniczone. Twoja firma może potrzebować niektórych użytkowników-gości do wypełnienia ról o wyższych uprawnieniach w organizacji. Aby obsługiwać definiowanie ról o wyższych uprawnieniach, użytkownicy-goście mogą być dodani do dowolnych ról, które chcesz, w zależności od potrzeb organizacji.
