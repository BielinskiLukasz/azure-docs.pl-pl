Musisz mieć [konto interfejsu API usług Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) z dostępem do interfejsu API sprawdzania pisowni Bing. Jeśli nie masz subskrypcji platformy Azure, możesz bezpłatnie utworzyć [konto](https://azure.microsoft.com/try/cognitive-services/?api=text-analytics). Zanim będzie można kontynuować, potrzebny będzie klucz dostępu [podawany po aktywacji bezpłatnej wersji próbnej](https://azure.microsoft.com/try/cognitive-services/my-apis/) lub klucz płatnej subskrypcji z pulpitu nawigacyjnego platformy Azure.
