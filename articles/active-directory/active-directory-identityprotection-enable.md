# <a name="enabling-azure-active-directory-identity-protection"></a>Włączenie ochrony tożsamości usługi Azure Active Directory

Azure Active Directory Identity Protection jest możliwość usługi Azure Active Directory (Azure AD). Usługa Azure AD Identity Protection umożliwia:

- uzyskanie skonsolidowanego widoku oflagowanych użytkowników i zdarzeń o podwyższonym ryzyku wykrytych za pomocą algorytmów usługi Machine Learning
- ustawienie zasad dostępu warunkowego opartych na ryzykach w celu automatycznego chronienia użytkowników
- polepszenie stanu bezpieczeństwa przez wykonywanie akcji na podstawie luk w zabezpieczeniach 

Aby uzyskać więcej informacji, zobacz [Azure Active Directory Identity Protection](active-directory-identityprotection.md).

W tym artykule przedstawiono sposób włączania usługi Azure Active Directory Identity Protection.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Kroki, aby włączyć usługi Azure Active Directory Identity Protection
1. [Zaloguj się na](https://ms.portal.azure.com/) do portalu Azure jako administrator globalny. 
2. W portalu Azure kliknij **Marketplace**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/01.png "tworzenie")
3. Na liście aplikacji kliknij **bezpieczeństwo i Obsługa tożsamości**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/02.png "tworzenie")
4. Kliknij przycisk **Azure AD Identity Protection**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/03.png "tworzenie")
5. Na **Azure AD Identity Protection** bloku, kliknij przycisk **Utwórz**.
   
    ![Utwórz](./media/active-directory-identityprotection-enable/04.png "tworzenie")
