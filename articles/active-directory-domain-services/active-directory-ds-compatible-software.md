# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Oprogramowanie innych firm, zgodny z usług domenowych Azure AD

## <a name="one-identity-authentication-services"></a>Jedna tożsamość usługi uwierzytelniania

![Jedna tożsamość](./media/compatible-software-icons/OneIdentity.png)

Jeden usług uwierzytelniania tożsamości umożliwia systemów Unix, Linux i Mac OS X, dostępu, uwierzytelniania i autoryzacji istniejącej infrastruktury usługi Active Directory (AD) w organizacji. Usługi uwierzytelniania obsługuje teraz usługi Azure Active Directory Domain Services Włączanie zasobów innych niż Windows korzystanie z tego samego Platforma następnej generacji, którego już używać istniejących rozwiązań SaaS.

[Jedna tożsamość usługi uwierzytelniania](https://www.quest.com/products/authentication-services/)

[Artykuł bazy wiedzy — usługi uwierzytelniania tożsamości użyj jednej z usług domenowych Azure AD](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux to na świecie wiodących platforma dla przedsiębiorstw Linux opracowany pod kątem potrzeb współczesnych nowoczesnych przedsiębiorstwach. Red Hat Enterprise Linux jest preferowany wyborem dla obciążeń maszyny wirtualnej (VM) systemu Linux enterprise w systemie Microsoft Azure. Red Hat Enterprise Linux jest otwarte, niezawodną i bezpieczną platformę, przeznaczona dla klientów chcących elastyczność wdrażania dla ich działalności obciążeń — od centrum danych w chmurze platformy Azure — wspierany przez ściśle zintegrowane, pomoc techniczną klasy przedsiębiorstwa Red Hat i Microsoft.

[Dołącz do programu Red Hat Enterprise Linux 7.2 maszyny Wirtualnej do usług domenowych Azure AD](active-directory-ds-admin-guide-join-rhel-linux-vm.md)
