# <a name="my-profile-preview-portal-overview"></a>Omówienie portalu Mój profil (wersja zapoznawcza)

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

Portal **Mój profil** (wersja zapoznawcza) umożliwia zarządzanie kontem służbowym w celu:

- Konfigurowania informacji zabezpieczających i zarządzania nimi na potrzeby weryfikacji dwuetapowej i resetowania hasła

- Wyświetlania aktywności logowania

- Zarządzania połączonymi organizacjami i urządzeniami

- Przeglądania sposobów korzystania z danych w organizacji

>[!Important]
>Ta zawartość jest przeznaczona dla użytkowników. Jeśli jesteś administratorem, więcej informacji na temat konfigurowania i zarządzania środowiskiem usługi Azure Active Directory (Azure AD) możesz znaleźć w [dokumentacji usługi Azure Active Directory](https://docs.microsoft.com/azure/active-directory).

## <a name="access-the-my-profile-portal"></a>Dostęp do portalu Mój profil
Do portalu **Mój profil** możesz przejść, otwierając adres https://myprofile.microsoft.com w bieżącej wersji dowolnej z następujących przeglądarek:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11
