# <a name="resource-types-that-extend-capabilities-of-other-resources"></a>Typy zasobów rozszerzające możliwości innych zasobów

Zasób rozszerzenia jest zasobem, który dodaje do możliwości innego zasobu. Na przykład blokada zasobów jest zasobem rozszerzenia. Blokadę zasobu należy zastosować do innego zasobu, aby zapobiec jego usunięciu lub zmodyfikowaniu. Nie ma sensu tworzyć blokady zasobów samodzielnie. Zasób rozszerzenia jest zawsze stosowany do innego zasobu.
