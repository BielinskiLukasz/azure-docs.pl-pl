# <a name="pricing"></a>Cennik

## <a name="compute-cost"></a>Obliczanie kosztu
Powłoka chmury Azure działa na maszynie bezpłatnie dostarczany przez platformę Azure, ale wymaga udziału plików na platformę Azure do użycia. 

## <a name="storage-cost"></a>Koszt magazynowania
Chmura powłoki wymaga nowego lub istniejącego udziału plików platformy Azure ma zostać zainstalowany, aby zachować pliki między sesjami. Koszty zwykłe wiąże się z magazynu.

Sprawdź [tutaj szczegółowe informacje na temat plików Azure koszty](https://azure.microsoft.com/pricing/details/storage/files/).
