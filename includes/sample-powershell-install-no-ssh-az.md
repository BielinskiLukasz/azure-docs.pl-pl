Ten przykładowy skrypt wymaga programu Azure PowerShell. Uruchom polecenie `Get-Module -ListAvailable Az`, aby dowiedzieć się, jaka wersja jest używana. Jeśli konieczna będzie instalacja lub uaktualnienie, zobacz [Instalowanie modułu Azure PowerShell](/powershell/azure/install-az-ps). 

Uruchom polecenie `Connect-AzAccount`, aby utworzyć połączenia z platformą Azure.
