# <a name="create-a-power-bi-application-offer"></a>Utwórz ofertę aplikacja Power BI

W tej sekcji przedstawiono kroki wymagane do utworzenia nowej oferty aplikacji Power BI dla [AppSource](https://appsource.microsoft.com). Każdy oferty pojawia się jako swoje własne jednostki w usłudze AppSource.  Po utworzeniu nowej oferty w [portalu Cloud Partner](https://cloudpartner.azure.com/), należy podać cztery grupy zasobów dla oferty.

|   Grupy zasobów      | Opis                                                                         |
| ----------------   | ----------------                                                                    |
| Ustawienia oferty     | Podstawowy identyfikatory i nazwy dla oferty                                      |
| Informacje techniczne     | Instalator adres URL używany do instalowania aplikacji w obszarze roboczym usługi Power BI klienta. Aby uzyskać więcej informacji na temat generowania tego adresu URL, zapoznaj się [dokumentacji aplikacji Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).   |
| Szczegóły sklepu | Zawiera zasoby zarządzania marketingu, prawne i potencjalnych klientów. Marketing zasoby zawierają opis oferty i logo użytkownika.  Zasoby prawne zawierają zasady zachowania poufności informacji, warunki użytkowania i inne dokumenty prawne.  Włącza zasady zarządzania prowadzić można określić sposób obsługi potencjalnych klientów z poziomu portalu użytkownika końcowego usługi AppSource. |
| Kontakty           | Zawiera informacje dotyczące kontaktu i zasady pomocy technicznej                                     |
|    |     |


## <a name="new-offer-form"></a>Nowy formularz oferty

Gdy Twoja Zaloguj się do portalu Cloud Partner, kliknij przycisk **+ nowa oferta** element na pasku menu po lewej stronie.  Wynikowy menu, kliknij pozycję **aplikacje usługi Power BI** do wyświetlenia **nowa oferta** formularza i rozpocząć proces Definiowanie zasobów dla nowej oferty aplikacji.

![Z menu usługi Power BI oferty](./media/new-offer-menu.png)

> [!WARNING] 
> Jeśli **aplikacje usługi Power BI** opcja nie jest wyświetlany lub nie jest włączona, a następnie Twoje konto nie ma uprawnień do utworzenia tego typu oferty. Sprawdź, czy zostały spełnione wszystkie [wymagania wstępne](./cpp-prerequisites.md) dla tego typu oferty, w tym rejestrowanie dla konta dewelopera.
