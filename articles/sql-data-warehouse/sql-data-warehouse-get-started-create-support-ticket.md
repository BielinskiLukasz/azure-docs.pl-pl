# SQL Data Warehouse のサポート チケットを作成する
 この記事では、SQL Data Warehouse のサポート チケットを作成する方法を示します。


## サポート チケットを作成する方法

1. 開いている、 [Azure クラシック ポータル][]します。 
2. [ホーム画面をクリックして、 **ヘルプとサポート** を並べて表示します。

    ![ヘルプとサポート](./media/sql-data-warehouse-get-started-create-support-ticket/help-support.png)

3. [ヘルプとサポート] ブレードで、をクリックして **作成サポート要求**します。

    ![新しいサポート要求](./media/sql-data-warehouse-get-started-create-support-ticket/create-support-request.png)

4. 選択、 **要求の種類**します。

    ![要求の種類](./media/sql-data-warehouse-get-started-create-support-ticket/request-type.png)

5. 選択、 **サブスクリプション** 、報告された問題を使用してデータベースをホストします。 

    ![[サブスクリプション]](./media/sql-data-warehouse-get-started-create-support-ticket/subscription.png)

6. 選択 **SQL Data Warehouse** リソースとします。

    ![リソース](./media/sql-data-warehouse-get-started-create-support-ticket/resource.png)

7. 選択、 **サポート プラン**します。

    - 課金とサブスクリプション管理に関連する問題のサポートは、すべてのサポート レベルでご利用いただけます。
    - 障害対応の問題に関するサポートは、Developer、Standard、Pro-Direct、Premier サポートで提供いたします。 障害対応のサポートとは、Azure のご利用に際して発生した、マイクロソフトが原因と想定される問題への対応です。
    - 開発者支援およびアドバイザリ サービスは、Professional Direct サポートと Premier サポートでご利用いただけます。
    - スコープ、応答時間、料金などを含む Azure のサポートの詳細を参照 [Azure サポート プラン][]します。

    ![サポート プラン](./media/sql-data-warehouse-get-started-create-support-ticket/support-plan.png)

    プレミア サポート プランがある場合は、SQL Data Warehouse も報告できますに関連する問題、 [マイクロソフト プレミア サポート オンライン ポータル][]します。

8. 選択、 **問題の種類** と **カテゴリ**します。

    ![問題の種類のカテゴリ](./media/sql-data-warehouse-get-started-create-support-ticket/problem-type-category.png)

9. 問題を記述し、業務への影響のレベルを選択します。

    ![問題の説明](./media/sql-data-warehouse-get-started-create-support-ticket/problem-description.png)

10.  **連絡先情報** このサポート チケットが事前に入力されます。 必要な場合は更新します。

    ![連絡先情報](./media/sql-data-warehouse-get-started-create-support-ticket/contact-info.png)

11. クリックして **作成** サポート要求を送信します。


## 次のステップ
サポート要求を送信すると、Azure サポート チームから連絡があります。 要求の状態と詳細を確認する] をクリックして **サポートの要求の管理** ダッシュ ボード。

![状態の確認](./media/sql-data-warehouse-get-started-create-support-ticket/check-status.png)

接続できるさらに、SQL Data Warehouse と azure sqldw を使用したコミュニティのタグを [Stack Overflow][] または [Azure SQL データ ウェアハウスの MSDN フォーラム][]します。
