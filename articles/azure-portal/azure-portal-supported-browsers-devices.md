# <a name="supported-devices"></a>Obsługiwane urządzenia

[Witryny Azure portal](https://portal.azure.com) jest konsola sieci web i działa w przeglądarce wszystkich nowoczesnych komputerów stacjonarnych i urządzeń typu tablet. Jeśli musisz zarządzać zasobami platformy Azure z urządzenia przenośnego, spróbuj [aplikacja mobilna platformy Azure](https://azure.microsoft.com/features/azure-portal/mobile-app/). Jest ona dostępna dla systemów iOS i Android.

## <a name="recommended-browsers"></a>Zalecane przeglądarki

Zaleca się, że używasz najbardziej aktualne przeglądarki, która jest zgodna z systemem operacyjnym. Obsługiwane są następujące przeglądarki:

* Microsoft Edge (Najnowsza wersja)
* Internet Explorer 11
* Safari (Najnowsza wersja, tylko Mac)
* Chrome (Najnowsza wersja)
* Firefox (Najnowsza wersja)
