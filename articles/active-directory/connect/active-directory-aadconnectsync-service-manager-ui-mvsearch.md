# <a name="sync-service-manager-metaverse-search"></a>Wyszukiwanie Metaverse Menedżera usługi synchronizacji

![Menedżera usługi synchronizacji](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

Karta wyszukiwanie metaverse jest przydatne podczas rozwiązywania problemów związanych z danymi. W górnej połowie można utworzyć kwerendy, oparty na kombinacji atrybutów. Po zakończeniu kwerendy kliknij **wyszukiwania**. Wynik jest widoczny w siatce dolnej. Można wybrać kolumny, które powinny być widoczne z **ustawienia kolumny**.

W wynikach wyszukiwania wybierz obiekt i **właściwości** wyświetlić [właściwości obiektu metaverse](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties).
