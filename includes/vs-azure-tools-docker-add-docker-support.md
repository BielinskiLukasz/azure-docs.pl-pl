1. W programie Visual Studio **Eksploratora rozwiązań**, kliknij prawym przyciskiem myszy projekt i wybierz **Dodaj > obsługę platformy Docker** z menu kontekstowego.
   
    ![Dodaj obsługę platformy Docker menu kontekstowe](media/vs-azure-tools-docker-add-docker-support/docker-support-context-menu.png)
2. Dodawanie obsługi platformy Docker do platformy ASP.NET Core projektu sieci web powoduje dodanie kilku plików związanych z platformy Docker, dodawane do projektu, w tym pliku Dockerfile, skrypty programu Windows PowerShell wdrażania i pliki właściwości platformy Docker. 
   
    ![Pliki docker dodane do projektu](media/vs-azure-tools-docker-add-docker-support/docker-files-added.png)
