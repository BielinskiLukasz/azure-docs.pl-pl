# <a name="add-bank-and-tax-information"></a>Dodawanie informacji o banku i podatku  

Informacje o banku i podatku są wymagane dla wszystkich ofert komercyjnych przy użyciu typu listy języka Transact.  

## <a name="partner-center"></a>Centrum partnerskie

Informacje dotyczące banku i podatku są teraz zarządzane w centrum partnerskim. Instrukcje dotyczące konfigurowania tych ustawień można znaleźć na stronie "przegląd komercyjnej witryny Marketplace" w temacie [Zarządzanie kontem Centrum partnerskiego](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#payout-account).
