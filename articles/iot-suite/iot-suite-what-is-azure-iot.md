## Azure IoT Suite

Microsoft Azure の IoT Suite は、エンタープライズ レベル ソリューションなど、一般的な IoT シナリオに対応する拡張可能な構成済みのソリューションのセットをすばやく開始することができます *リモート監視* と *接続されているデバイス*します。

構成済みのソリューションは次のとおりです。

- 包括的な機能、エンド ツー エンド ソリューションを始めるために、シミュレートされたデバイスを含む構成済みの Azure サービスをなど、 [Azure IoT Hub][], 、[Azure Event Hubs][], 、[Azure Stream Analytics][], 、[Azure Machine Learning][], 、および [Azure ストレージ][], 、およびソリューションの特定の管理コンソールです。
- 独自の IoT シナリオを実装するためにカスタマイズおよび拡張できる、実績のある、実稼働可能なコードが含まれています。

興味場合もあります。、 [Azure IoT Hub][] 構成済みのソリューションの多くを使用するサービスです。 [Azure の IoT Hub][] データの処理と分析機能、サンプルのソリューション アーキテクチャで定義されている多数用意されています。
