# <a name="explore-data-in-the-team-data-science-process"></a>Eksplorowanie danych w zespołowym przetwarzaniu danych dla celów naukowych

Eksplorowanie danych jest krokiem w [procesie nauki o danych zespołu.](overview.md)

W poniższych artykułach opisano sposób eksplorowania danych w trzech różnych środowiskach magazynu, które są zwykle używane w procesie nauki o danych:

* Eksploruj dane [kontenera obiektów blob platformy Azure](explore-data-blob.md) przy użyciu pakietu [Pandas](https://pandas.pydata.org/) Python.
* Eksploruj dane [programu SQL Server](explore-data-sql-server.md) przy użyciu języka SQL i przy użyciu języka programowania, takiego jak Python.
* Eksploruj dane [tabeli gałęzi](explore-data-hive-tables.md) przy użyciu zapytań hive.

Ponadto w klipie wideo, [Przetwarzanie wstępne danych w usłudze Azure Machine Learning Studio,](https://azure.microsoft.com/documentation/videos/preprocessing-data-in-azure-ml-studio/)opisano często używane moduły do czyszczenia i przekształcania danych.
