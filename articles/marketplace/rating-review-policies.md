# <a name="azure-appsource-and-marketplace-review-policies"></a>Azure AppSource i Portal Marketplace Przejrzyj zasady

Ocena i przejrzyj zasady dla platformy Azure w usłudze AppSource i Portal Marketplace są wyświetlane w tym miejscu.

* W przeglądach usługi AppSource i portalu Azure Marketplace Microsoft nie dopuszczają następujące zachowania lub zawartości:
    * Bot/cyberatakami
    * Naruszenia zgodności ochrony prywatności
    * Zawartość wiadomości-śmieci
    * Zawartość oszustwo
    * Obraźliwą zawartość: obsceniczne, obsceniczne lub obraźliwy język lub gestów
    * Nielegalnych treści
    * Niewłaściwą, hateful lub niebezpiecznym charakterze danej zawartości.
    * Oszczerczych zawartości
    * Powtórzony przesłanych treści takie same lub podobne
    * Reklamy, łącznie z promocji z innych aplikacji i usług

* Firma Microsoft zastrzega sobie prawo do usunięcia związanych z przeglądu zawartości przesłane przez użytkownika do jakiegokolwiek powodu.
* Firma Microsoft zastrzega sobie prawo do uniemożliwić użytkownikowi przesyłanie zawartość związana z przeglądu jakiegokolwiek powodu.
* Wydawca otrzyma bez powiadomienia przy usuwaniu związane z przeglądu zawartości, powiązane ze swoimi aplikacjami.
* Użytkownicy będą otrzymywać nie powiadomień przy usuwaniu związane z przeglądu zawartości przesłane przez nich.
* Microsoft nie ma obowiązku podanie wyjaśnienia dotyczące usunięcia związanych z przeglądu zawartości.
* Microsoft nie będzie honorować żądania do przywrócenia usuniętych związane z przeglądu zawartości.

## <a name="review-guidelines-for-customers"></a>Wytyczne dla klientów

* Nie przesyłać zawartość, która może być narusza [usługi AppSource i portalu Azure Marketplace, przejrzyj zasady](./rating-review-policies.md).
* Nie dołączaj identyfikowalne dane osobowe (PII) takich jak adres e-mail i adres fizyczny, numerów telefonów itd.
* Upewnij się, że zapoznania się z nimi jest jasna, czytelny i zawierającego wiele użytecznych informacji.
* Upewnij się, że punkt widzenia wyrażone w recenzji zrównoważonych oraz neutralne.
* Należy pamiętać, księgowania forum publiczne i podejmowanie odpowiednich działań.
