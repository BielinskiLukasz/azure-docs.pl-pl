Aby wyświetlić zdarzenie urządzenia, należy wykonać następujące kroki w witrynie Azure Portal.

1. W witrynie Azure portal przejdź do zasobu Data Box Edge/Data Box Gateway, a następnie przejdź do **monitorowania zdarzeń > urządzenia.**
2. Wybierz zdarzenie i wyświetl szczegóły alertu. Podejmij odpowiednie działania, aby rozwiązać warunek alertu.

    ![Wybieranie szczegółów wydarzenia i wyświetlania](media/data-box-edge-gateway-view-device-events/view-device-events.png)
