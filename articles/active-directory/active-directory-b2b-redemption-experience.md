# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Realizacji zaproszenia współpraca w usłudze Azure Active Directory B2B

## <a name="azure-ad-and-microsoft-account-users"></a>Usługi Azure AD i użytkowników kont Microsoft
Dla użytkowników przy użyciu istniejącego konta usługi Azure AD lub konta Microsoft środowisko realizacji jest równie proste jak logowanie.

## <a name="social-id-user-first-time-redemption"></a>Społecznościowych identyfikator użytkownika po raz pierwszy realizacji
Współpraca AD B2B Azure ułatwia dowolnego adresu e-mail używanego do realizacji. Spójrz na środowisko realizacji, jeśli adres e-mail innych niż Microsoft jest używany do współpracy B2B. Ten przepływ realizacji jest bardziej skomplikowane, ponieważ może być konieczne utworzenie konta w czasie realizacji. Go wyewidencjonować w poniższym klipie wideo:

> [!VIDEO https://channel9.msdn.com/Blogs/Azure/b2b-collaboration-redemption/Player]
