# <a name="azure-remote-rendering-asset-tool-arrt"></a>Narzędzie zasobów renderowania zdalnego na platformie Azure (ARRT)

Narzędzie zasobów renderowania zdalnego na platformie Azure (ARRT) to aplikacja klasyczna "open source" opracowana w języku C++/QT.

![ARRT](./media/azure-remote-rendering-asset-tool.png "Zrzut ekranu ARRT")

## <a name="features"></a>Funkcje

Aplikacja może służyć do:

* Przekaż model 3W
* Sterowanie konwersją modelu
* Tworzenie zdalnej sesji renderowania i zarządzanie nią
* Załaduj model 3W
* Podgląd modelu 3W
* Modyfikowanie jego materiałów

Możesz również użyć go jako przykładu typu open source, aby dowiedzieć się, jak zaimplementować fronton dla zestawu SDK języka C++ dla usługi Azure Remote Rendering przy użyciu biblioteki klienta usługi Azure Storage do zarządzania konwersją modelu 3W.

## <a name="source-repository"></a>Repozytorium źródła

Możesz znaleźć kod źródłowy i dokumentację w [repozytorium GITHUB ARRT](https://github.com/Azure/azure-remote-rendering-asset-tool).
