# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Szybki Start: przygotowywanie sieci do wdrożenia usługi Azure Defender for IoT

## <a name="choose-your-hardware"></a>Wybierz sprzęt
Usługa Azure Defender dla IoT obsługuje różne certyfikowane urządzenia, w tym:
- Certyfikowane wstępnie skonfigurowane urządzenia, na których jest już zainstalowane oprogramowanie
- Nieskonfigurowane urządzenia certyfikowane, na których można pobrać i zainstalować oprogramowanie czujnika
- Specyfikacje urządzenia lokalnej konsoli zarządzania. Lokalna Konsola zarządzania nie jest dostępna jako urządzenie wstępnie skonfigurowane. 

Aby uzyskać więcej informacji, zobacz [Przewodnik po specyfikacjach sprzętu](https://aka.ms/AzureDefenderforIoTBareMetalAppliance)


## <a name="install-components"></a>Zainstaluj składniki
Po uzyskaniu składników sprzętowych i pobraniu pakietu oprogramowania z portalu usługi Azure Defender dla IoT można zainstalować oprogramowanie składników. Aby uzyskać więcej informacji, zobacz [Przewodnik instalacji](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Dołączanie czujników do portalu usługi Azure Defender dla IoT 
 Ten artykuł zawiera omówienie procesów wdrażania, dołączania i aktualizacji dla usługi Azure Defender na potrzeby czujników IoT i lokalnej konsoli zarządzania. Ten proces jest realizowany przez następujące instrukcje na stronie Witamy w usłudze Azure Defender for IoT i kolejnych stronach.

![Zrzut ekranu przedstawiający widok strony powitalnej platformy Azure](media/updates/image4.png)

Aby otworzyć stronę powitalną:

1. Zaloguj się do swojego konta platformy Azure i Wyszukaj usługę Azure Defender for IoT.

1. **Wdrożenie czujnika** wymaga: zakup wstępnie skonfigurowanego sprzętu z poziomu strzałki lub zakupienie własnego urządzenia i zainstalowanie na nim oprogramowania czujnika.
1. **Wdrożenie lokalnej konsoli zarządzania** wymaga zakupienia własnego urządzenia i zainstalowania na nim lokalnego oprogramowania konsoli zarządzania. Więcej informacji.




## <a name="network-setup"></a>Konfiguracja sieci
Po zainstalowaniu składnika usługi Defender for IoT można skonfigurować sprzęt sieciowy pod kątem optymalnego monitorowania sieci. Aby uzyskać więcej informacji, zobacz [Przewodnik konfigurowania sieci](https://aka.ms/AzureDefenderForIoTNetworkSetup).
