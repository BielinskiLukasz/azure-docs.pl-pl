# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Zainstalowanie lub odinstalowanie woluminu dla maszyn wirtualnych Windows lub Linux 

Można zainstalować lub odinstalować woluminu dla maszyn wirtualnych Windows lub Linux zgodnie z potrzebami.  Instrukcje instalacji dla maszyn wirtualnych systemu Linux są dostępne w usłudze Azure Files NetApp.  

1. Kliknij przycisk **woluminów** bloku, a następnie wybierz wolumin, dla którego chcesz zainstalować. 
2. Kliknij przycisk **instrukcje dotyczące instalacji** wybranego woluminu, a następnie postępuj zgodnie z instrukcjami, aby zainstalować wolumin. 

    ![Instrukcje dotyczące instalacji systemu plików NFS](../media/azure-netapp-files/azure-netapp-files-mount-instructions-nfs.png)

    ![Instrukcje instalacji SMB](../media/azure-netapp-files/azure-netapp-files-mount-instructions-smb.png)
