## <a name="citrix"></a>Citrix

![Logo Citrix](./media/partners/citrix.png)

Citrix jest zatwierdzonym dostawcą, który oferuje przedsiębiorstwom scentralizowane hybrydowe zarządzanie wirtualnymi aplikacjami i obciążeniami pulpitu na platformie Azure, obok wdrożeń lokalnych. Citrix Workspace z usługą Aplikacje wirtualne i pulpity umożliwia użytkownikom dostęp do aplikacji i pulpitów z dowolnego urządzenia, wykorzystując zaawansowany protokół Citrix HDX, aby zapewnić wysokiej rozdzielczości środowisko z dowolnego miejsca.

Citrix rozszerza wartość pulpitu wirtualnego systemu Windows o niezawodne narzędzia dla przedsiębiorstw, aby poprawić gęstość i wydajność użytkowników, aprowizować obciążenia na żądanie oraz uprościć zarządzanie obrazami i aplikacjami. It może optymalizować koszty za pomocą inteligentnych narzędzi skalowania, zapewniając jednocześnie niesamowite środowisko użytkownika, które jest testowane w terenie pod kątem najtrudniejszych zastosowań w różnych branżach. Ponadto Citrix Managed Desktops to program pulpitu wirtualnego z obsługą pulpitu windowsowego jako usługa, który zapewnia proste, oparte na chmurze rozwiązanie do zarządzania do dostarczania wirtualnych aplikacji i pulpitów na dowolnym urządzeniu.

- [Przejdź do witryny partnera](https://more.citrix.com/wvd).

## <a name="vmware"></a>VMware

![VMware Logo](./media/partners/vmware.png)

Usługa VMware Horizon Cloud na platformie Microsoft Azure umożliwia organizacjom połączenie własnego wystąpienia platformy Microsoft Azure z prostą, intuicyjną płaszczyzną sterowania horizon cloud, tworząc bezpieczne, kompleksowe rozwiązanie hostowane w chmurze do dostarczania zwirtualizowanych aplikacje i komputery stacjonarne systemu Windows. Wraz z wydaniem pulpitu wirtualnego systemu Windows firma VMware nawiązała współpracę z firmą Microsoft w celu rozszerzenia funkcji pulpitu wirtualnego systemu Windows na klientów korzystających z usługi Horizon Cloud na platformie Microsoft Azure. Ta funkcja będzie dostępna w wersji Tech Preview do końca 2019 r.
 
Jako zatwierdzony dostawca, korzyści z pulpitu wirtualnego systemu Windows zostaną rozszerzone na VMware Horizon Cloud na klientach platformy Microsoft Azure, w tym windows 10 Enterprise dla wielu sesji, Windows 7 z bezpłatnymi rozszerzonymi aktualizacjami zabezpieczeń przez okres do trzech lat oraz FSLogix. Ponadto klienci otrzymają funkcje klasy korporacyjnej programu Horizon Cloud, takie jak obsługa szerokiego punktu końcowego; globalne, zoptymalizowane pod kątem chmury brokering; elastyczne opcje i konfiguracje pulpitu; zarządzanie środowiskiem użytkownika; i wsparcie dla środowisk hybrydowych.

- [Zarejestruj się w wersji Tech Preview programu Horizon Cloud na platformie Microsoft Azure](https://www.vmware.com/learn/323526_TECH_PREVIEW_REG.html).
- [Przejdź do witryny partnera](https://www.vmware.com/products/horizon-cloud-virtual-desktops.html).
