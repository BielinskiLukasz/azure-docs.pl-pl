# <a name="share-and-discover-resources-in-the-azure-ai-gallery"></a>Udostępniaj i odnajduj zasoby w galerii Azure AI

**[Galeria sztucznej Inteligencji platformy Azure](http://gallery.cortanaintelligence.com)**  jest witryną tworzonych metodą społecznościową do odnajdywania i udostępnianie rozwiązań utworzonych za pomocą sztucznej Inteligencji platformy Azure.
Galeria ma wiele zasobów, które można użyć do tworzenia własnych rozwiązań do analizy.

[!INCLUDE [machine-learning-free-trial](../../../includes/machine-learning-free-trial.md)]

## <a name="what-can-i-find-in-the-gallery"></a>Co można znaleźć w galerii?
Galeria sztucznej Inteligencji platformy Azure zawiera wiele zasobów, które można użyć do tworzenia własnych rozwiązań do analizy.

* **[Eksperymenty](gallery-experiments.md)**  — Galeria zawiera szeroką gamę eksperymentów, które zostały opracowane w usłudze Azure Machine Learning Studio. Rozwiązania z zakresu od szybkiego eksperymentów weryfikacji koncepcji, które przedstawiają określonej usługi machine learning technikę do w pełni rozwinięte rozwiązań uczenia maszynowego złożonych problemów.
* **[Program Jupyter Notebooks](gallery-jupyter-notebooks.md)**  — notesów programu Jupyter dołączyć kod, wizualizacje danych i dokumentacji w pojedynczym interaktywny obszar roboczy.
  Notesy w galerii zapewniają samouczków i uzyskać bardziej szczegółowe objaśnienia zaawansowanych technik i rozwiązań uczenia maszynowego.

* **[Rozwiązania](gallery-solutions.md)**  — szybkie tworzenie rozwiązania Cortana Intelligence z wstępnie skonfigurowanych rozwiązań, architektur referencyjnych i wzorców projektowych. Nadaj im z dołączonymi instrukcjami lub z pomocy promowanego partnera.
* **[Samouczki](gallery-tutorials.md)**  — dostępnych jest kilka samouczków, przeprowadzi Cię przez machine learning technologie i koncepcje lub do opisania zaawansowane metody rozwiązywania różnych problemów learning maszyny.

Tych podstawowych zasobów galerii można grupować logicznie na kilka różnych sposobów:

* **[Kolekcje](gallery-collections.md)**  — kolekcji pozwala na grupowanie eksperymentów, interfejsów API i innych galerii elementów tego adresu konkretne rozwiązanie lub koncepcji.
* **[Branże](gallery-industries.md)**  -branżach sekcji galerii umożliwia połączenie ze sobą różnych zasobów, które są specyficzne dla branż, takich jako sprzedaży detalicznej, produkcji, bankowości i opiece zdrowotnej.

Na koniec **[konkursy](gallery-competitions.md)** udostępniają ekscytujące możliwość konkurują ze społecznością scientsts danych w celu rozwiązywania złożonych problemów przy użyciu pakietu Cortana Intelligence.

![Usługa Azure stronę główną galerii sztucznej Inteligencji](./media/gallery-how-to-use-contribute-publish/gallery-home-page.png)

## <a name="discover-and-learn"></a>Znajdowanie i poznawanie
Każda osoba można przeglądać i wyszukiwać różnych typów zasobów w galerii, które zostały udostępnione przez firmę Microsoft i społeczność zaawansowanych analiz.
Umożliwiają one Dowiedz się więcej i od razu na rozwiązywaniu własnych problemy związane z analizą danych.
Możesz również pobrać eksperymentów i notesy Jupyter do własnego obszaru roboczego usługi Machine Learning Studio.

Ostatnio opublikowane i popularnych zasoby można łatwo znaleźć w galerii, lub możesz wyszukiwać według nazwy, tagów, algorytmów i innych atrybutów.
Kliknij przycisk **Przeglądaj wszystko** w nagłówku galerii, a następnie wybierz opcję wyszukiwanie udoskonalenia w lewej części strony i wprowadź terminy wyszukiwania w górnej.

Przeglądaj wkłady od konkretnego autora, klikając imię i nazwisko autora z żadnej z kafelków:

![Wyświetl wkładów autora](./media/gallery-how-to-use-contribute-publish/view-by-author.png)

Możesz przekształcić w komentarz, przesłania opinii lub zadawać pytania za pomocą sekcji komentarzy na każdej stronie zasobów.
Możesz nawet udostępniać zasobu zainteresowania znajomym lub współpracowników, korzystając z możliwości udziału usługi LinkedIn i Twitter.
Może również wysłać pocztą e-mail łączy do tych zasobów, aby zaprosić innych użytkowników, aby wyświetlić strony.

![Udostępnij ten element ze znajomymi](./media/gallery-how-to-use-contribute-publish/comment-and-share.png)

## <a name="contribute-to-the-gallery"></a>Współtworzenie galerii
Podczas logowania użytkownik staje się członkiem społeczności galerii. Dzięki temu można współtworzyć elementów galerii, tak aby inne osoby mogą korzystać z rozwiązania, które już znasz.

Jak inne spotkać się Twój wkład w galerii, możesz wykonać wiele widoków i pliki do pobrania wkładu:

![Udostępnij ten element ze znajomymi](./media/gallery-how-to-use-contribute-publish/view-and-download-counts.png)

Użytkownicy mogą również dodawać komentarze i udostępnianie Twojego wkładu z innymi członkami społeczności do nauki o danych.
Możesz zalogować się przy użyciu narzędzia dyskusji, takiego jak usługi Disqus i otrzymywać powiadomienia dotyczące komentarzy na Twój wkład.

![Postępuj zgodnie z komentarzami użytkowników](./media/gallery-how-to-use-contribute-publish/follow-comments.png)

Poniższe elementy może przyczynić się do galerii — skorzystaj z poniższych linków, aby uzyskać więcej informacji:

* **[Kolekcje](gallery-collections.md#contribute)**
* **[Eksperymenty](gallery-experiments.md#contribute)**
* **[Tutorials](gallery-tutorials.md#contribute)**
