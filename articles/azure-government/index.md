#<a name="azure-government-documentation"></a>Dokumentacja platformy Azure dla instytucji rządowych

Platforma Microsoft Azure dla instytucji rządowych jest platformą w chmurze zbudowaną na fundamentalnych zasadach bezpieczeństwa, zachowania prywatności i kontroli, zgodności oraz przejrzystości. Jednostki sektora publicznego otrzymują fizycznie izolowane wystąpienie platformy Microsoft Azure, które obejmuje światowej klasy usługi związane z bezpieczeństwem i zgodnością, mające krytyczne znaczenie dla rządu USA, w zakresie wszystkich systemów i aplikacji tworzonych na podstawie tej architektury.
