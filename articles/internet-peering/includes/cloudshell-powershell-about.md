Aby uruchomić polecenia cmdlet, można użyć usługi Azure Cloud Shell, bezpłatna powłoka interaktywna. Udostępnia ona wstępnie zainstalowane i najczęściej używane narzędzia platformy Azure, które są skonfigurowane do użycia na koncie. Wybierz **pozycję Kopiuj,** aby skopiować kod, i wklej go do powłoki cloud shell. Następnie wybierz pozycję **Enter,** aby go uruchomić. Istnieje kilka sposobów uruchamiania usługi Cloud Shell:


|  |   |
|-----------------------------------------------|---|
| Otwórz usługę Cloud Shell w swojej przeglądarce. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Wybierz przycisk **Powłoki chmury** na pasku narzędzi w prawym górnym rogu witryny Azure portal. | [![Usługa Cloud Shell w portalu](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Jeśli nie chcesz używać usługi Azure Cloud Shell, możesz zainstalować program PowerShell lokalnie. Jeśli zdecydujesz się zainstalować i używać programu PowerShell lokalnie, należy zainstalować najnowszą wersję poleceń cmdlet programu Azure Resource Manager programu PowerShell. Polecenia cmdlet programu PowerShell są często aktualizowane. Aby uzyskać najnowsze funkcje funkcji, należy zaktualizować polecenia cmdlet programu PowerShell. Jeśli tego nie zrobisz, mogą wystąpić problemy.

Aby znaleźć wersję programu PowerShell, która jest uruchomiona lokalnie, należy użyć polecenia cmdlet **"Get-Module -ListAvailable Az".** Aby zaktualizować, zobacz [Instalowanie modułu programu Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps). Aby uzyskać więcej informacji, zobacz [jak zainstalować i skonfigurować program Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview).

Jeśli używasz programu PowerShell w systemie macOS, wykonaj kroki opisane w [instalowaniu programu PowerShell w systemie macOS.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
