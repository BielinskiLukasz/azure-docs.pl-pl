Aby włączyć dzienniki przesyłania strumieniowego dla aplikacji funkcji na platformie Azure:

1. Naciśnij klawisz F1, aby otworzyć paletę poleceń, a następnie wyszukaj i uruchom polecenie **usługi Azure Functions: Rozpocząć przesyłanie strumieniowe dzienników**.

1. Wybierz aplikację funkcji na platformie Azure, a następnie wybierz **tak** Aby włączyć rejestrowanie aplikacji dla aplikacji funkcji.

1. Wyzwalanie funkcji na platformie Azure i zwróć uwagę, że dane dziennika są wyświetlane w oknie danych wyjściowych programu Visual Studio Code.

1. Gdy skończysz, pamiętaj, aby uruchomić polecenie **usługi Azure Functions: Zatrzymać przesyłanie strumieniowe dzienników** Aby wyłączyć rejestrowanie dla aplikacji funkcji.
