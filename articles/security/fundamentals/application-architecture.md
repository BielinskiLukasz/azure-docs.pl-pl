# <a name="application-architecture-on-azure"></a>Architektura aplikacji na platformie Azure
Aby pomóc w zabezpieczeniu rozwiązań opartych na chmurze na Microsoft Azure, silna platforma architektury jest niezwykle ważna. Architekty, projektanci i realizatorzy mogą korzystać z mocnej wiedzy o architekturze aplikacji i usług. Ta podstawowa wiedza pomaga zrozumieć wszystkie składniki rozwiązań chmurowych i ułatwić integrację zabezpieczeń ze wszystkimi aspektami projektu i implementacji.

Firma Microsoft udostępnia następujące elementy, które ułatwiają badanie architektury i projektowanie:

* Plany architektoniczne
* Zestaw symboli w chmurze i przedsiębiorstwie
* Szablon programu Visio planu 3W

## <a name="architectural-blueprints"></a>Plany architektoniczne
Firma Microsoft publikuje zestaw [planów architektury](https://aka.ms/azblueprints) wysokiego poziomu przedstawiających sposób tworzenia określonych typów systemów przy użyciu produktów firmy Microsoft.
Każdy plan obejmuje:

* Prosty 2D plik z programem Visio 2003, który można pobrać i zmodyfikować
* Kolorowy plik PDF z perspektywą 3W, aby wprowadzić plan do mniej technicznych odbiorców
* Film wideo, który przeprowadzi Cię przez wersję 3D

## <a name="cloud-and-enterprise-symbol-set"></a>Zestaw symboli w chmurze i przedsiębiorstwie
[Obejrzyj wideo szkoleniowe dotyczące programu Visio i symboli](https://aka.ms/CnESymbolsVideo) , a następnie [Pobierz zestaw symboli w chmurze i przedsiębiorstwie](https://aka.ms/CnESymbols) , aby ułatwić tworzenie materiałów technicznych, które opisują platformę Azure, system Windows Server, SQL Server i wiele innych. Symboli można używać w diagramach architektury, materiałach szkoleniowych, prezentacjach, arkuszach danych, grafiki informacyjne, oficjalnych, oficjalnych książkach, a także w przypadku, gdy książka pociąga za siebie osoby do korzystania z produktów firmy Microsoft. Nie są one jednak przeznaczone do użycia w interfejsach użytkownika.

## <a name="3d-blueprint-visio-template"></a>Szablon programu Visio planu 3W
Wersje trójwymiarowe [planów architektury firmy Microsoft](https://aka.ms/azblueprints) zostały początkowo utworzone w narzędziu innym niż firmy Microsoft. Nowy szablon programu Visio 2013 (i nowszego) wydany 5 sierpnia 2015 w ramach [kursu certyfikacji architektury firmy Microsoft dystrybuowanego na EDX.org](https://docs.microsoft.com/azure/architecture/).

Szablon jest również dostępny poza kursem.

* [Obejrzyj wideo szkoleniowe](https://aka.ms/3dBlueprintTemplateVideo) , aby dowiedzieć się, co można zrobić
* Pobierz [szablon programu Visio dla planu Microsoft 3D](https://aka.ms/3DBlueprintTemplate)
* Pobierz [symbole chmury i przedsiębiorstwa](https://docs.microsoft.com/azure/architecture/) do użycia z szablonem 3W
