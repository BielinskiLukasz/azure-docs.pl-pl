# <a name="lcs-subscription"></a>Subskrypcja LCS

W tym samouczku dowiesz się jak utworzyć subskrypcję. Ta subskrypcja zapewni to dzierżawa testowa Online firmy Microsoft i projektu programu Microsoft Dynamics Lifecycle Services gdzie możesz wdrożyć środowiska. Zostanie również konfigurowanie dodatkowych użytkowników w dzierżawie usługi Online firmy Microsoft i zdobycia doświadczenia z następującymi funkcjami administracji usługi. Poniżej przedstawiono umiejętności, które dowiesz się:

- Subskrypcja i tworzenie nowej witryny Microsoft Online testu dzierżawy.
- Przejdź do projektów usługi cyklu życia.
- Korzystanie z funkcji usług cyklu życia.
- Dodawanie dodatkowych użytkowników do usługi Microsoft Azure Active Directory i klienta.
- Wyświetlanie zasobów w subskrypcji wiadomości e-mail.

## <a name="key-terms"></a>Kluczowe terminy

**Dzierżawy Microsoft Online Services** — dzierżawa to grupy wszystkie subskrypcje i użytkowników w Twojej organizacji. Dzierżawy jest tworzony w tym samym czasie co w przypadku pierwszego subskrypcji w Microsoft Online Services.

**Subskrypcja** — subskrypcja zapewnia środowisko online. Środowisko to umożliwia środowiska Dynamics 365 dla operacji w chmurze i umożliwia także zobaczyć, jak można wdrożyć dostosowania, które tworzysz aplikacje w chmurze.

**Microsoft Azure Active Directory** — w środowisku chmury obejmuje usługi Azure Active Directory (AD), która ułatwia zarządzanie użytkownikami, grupami, role zabezpieczeń i licencji na aplikacje online, podobnie jak sposób zarządzania nimi dla środowiska lokalnego środowiska.

**Użytkownicy** — użytkowników, usług, które zostały zasubskrybowane przez organizację odbywa się w usłudze Azure AD. Wszyscy użytkownicy w Twojej dzierżawie mogą być dodawane lub przypisane do ról zabezpieczeń.

**Deweloperzy i Administratorzy** -deweloperom i administratorom są użytkownicy, którzy mają także dostęp do usług cyklu życia, które pozwala im zarządzać projektami i środowisk. Ci użytkownicy są także użytkownikom końcowym.

Zobacz [subskrypcji LCS](https://docs.microsoft.com/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription) Aby uzyskać więcej informacji.
