# Azure AD ドメイン サービス *(プレビュー)*

## 特徴
Azure AD ドメイン サービス プレビュー リリースでは、次の機能を使用できます。

- **単純な展開の操作:** 、たった数回のクリックを使用して、Azure AD テナントの Azure AD ドメイン サービスを有効にすることができます。 Azure AD テナントがクラウド テナントかオンプレミスのディレクトリと同期しているかにかかわらず、管理対象ドメインを短時間でプロビジョニングできます。

- **ドメイン参加のサポート:** Azure AD ドメイン サービスで使用できる Azure の仮想ネットワークでドメイン参加コンピューターが簡単にできます。 Windows クライアントおよび Windows Server オペレーティング システムで行うドメイン参加操作は、Azure AD ドメイン サービスが提供するドメインに対してシームレスに機能します。 また、このようなドメインに対して自動化されたドメイン参加ツールを使用することもできます。

- **Azure AD ディレクトリごとに 1 つのドメイン インスタンス:** Azure AD ディレクトリごとに 1 つの Active Directory ドメインを作成することができます。

- **カスタムの名前を持つドメインを作成します。** ドメインを作成するにはカスタム名 (例: です。 contoso.local) Azure AD ドメイン サービスを使用します。 確認済みのドメイン名だけでなく、未確認のドメイン名を指定できます。 必要に応じて、Azure AD ディレクトリが提供する組み込みのドメイン サフィックス (*.onmicrosoft.com など) を指定してドメインを作成することができます。

- **Azure AD と統合:** 構成または Azure AD ドメイン サービスへのレプリケーションを管理する必要はありません。 Azure AD ディレクトリのユーザー アカウント、グループ メンバーシップ、およびユーザー資格情報 (パスワード) が、Azure AD ドメイン サービスで自動的に利用可能になります。 Azure AD テナントやオンプレミスのディレクトリで新規作成されたユーザーやグループおよび変更が加えられた属性は、Azure AD ドメイン サービスに自動的に同期されます。

- **NTLM および Kerberos 認証:** の NTLM と Kerberos 認証のサポートにより、Windows 統合認証に依存するアプリケーションを展開することができます。

- **企業の資格情報やパスワードを使用して:** Azure ad ユーザーのパスワードをテナントの Azure AD ドメイン サービスを使用します。 つまり、組織のユーザーはドメインで会社の資格情報を使用でき、ドメイン参加しているコンピューターに対話的に、またはリモート デスクトップ経由でログインして、DC などに対して認証できます。

- **LDAP バインドと LDAP 読み取りのサポート:** Azure AD ドメイン サービスによって処理されたドメイン内のユーザーを認証するために LDAP バインドに依存するアプリケーションを使用することができます。 さらに、LDAP 読み取り操作を使用してディレクトリのユーザーまたはコンピューターの属性をクエリするアプリケーションも Azure AD ドメイン サービスに対して機能します。

- **グループ ポリシー:** 組み込み GPO を 1 つそれぞれのユーザーとコンピューターを利用するユーザー アカウントとドメインの必要なセキュリティ ポリシーの対応を強制するためにコンテナーには、コンピューターが参加しています。

- **複数の Azure リージョンで使用できる:** を参照してください、 [Azure リージョン](active-directory-ds-regions.md) Azure AD ドメイン サービスは利用可能な Azure リージョンの一覧のページです。

- **高可用性:** Azure AD ドメイン サービスが、ドメイン用の高可用性を提供します。 これにより、サービスの稼働時間とエラーに対する復元性が確実に向上します。 また、組み込みの正常性監視では、インスタンスに障害が発生したときに新しいインスタンスを作成することで障害を自動的に修正し、ドメインに継続的なサービスを提供します。

- **使い慣れた管理ツールを使用します。** Azure AD ドメイン サービスによって提供されるドメインを管理するには、Active Directory 管理センターまたは Active Directory PowerShell などの使い慣れたの Windows Server Active Directory 管理ツールを使用することができます。
