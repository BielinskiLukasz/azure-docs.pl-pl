Poniższe limity mają zastosowanie do usługi Azure Backup.

| **Limit** | **Domyślne** |
| --- | --- |
| Serwerów/maszyn, które mogą być rejestrowane w magazynie | Windows Server/Windows klienta/System Center DPM: 50 <br/><br/> Maszyny wirtualne IaaS: 1000  |
| Rozmiar źródła danych w magazynie magazynu |Maksymalna liczba 54400 GB. Limit nie ma zastosowania do maszyn wirtualnych IaaS |
| Magazyny kopii zapasowych w subskrypcji platformy Azure |500 magazynów na region |
| Zaplanuj codzienne wykonywanie kopii zapasowych |Windows Server/kliencie: 3 dziennie<br/> System Center DPM: 2 dziennie <br/> Maszyny wirtualne IaaS: Raz dziennie  |
| Dysków danych podłączonych do maszyny Wirtualnej platformy Azure do utworzenia kopii zapasowej | 16 |
| Dysk danych dołączony do maszyny Wirtualnej platformy Azure do utworzenia kopii zapasowej| 4095 GB|
