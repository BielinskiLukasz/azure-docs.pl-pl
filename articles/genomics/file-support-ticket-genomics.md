# <a name="how-to-contact-microsoft-genomics-for-support"></a>Jak skontaktować się z Microsoft Genomics w celu uzyskania pomocy technicznej
W tym omówieniu opisano sposób tworzenia żądania pomocy technicznej w celu skontaktowania się z Microsoft Genomics. Może to być przydatne, jeśli nie można rozwiązać problemu przy użyciu [przewodnika rozwiązywania problemów](troubleshooting-guide-genomics.md) lub [często zadawanych pytań](frequently-asked-questions-genomics.md). 


## <a name="file-a-support-ticket-through-the-azure-portal"></a>Zaplikowanie biletu pomocy technicznej za pomocą Azure Portal
Najpierw Zaloguj się do konta platformy Azure w Azure Portal. Kliknij znak zapytania w prawym górnym rogu i wybierz pozycję `Help + support`.

![Skontaktuj się z pomocą techniczną], aby uzyskać pomoc(./media/file-support-ticket/genomics-contact-support.png "techniczną Azure Portal Azure Portal") 



Następnie wybierz pozycję `New support request`. 

![Nowe](./media/file-support-ticket/new-support-request.png "żądanie obsługi") nowe żądanie pomocy technicznej 

Wypełnij `Basics`, w tym `Issue type` i `Subscription`. Wybierz najbardziej odpowiednie `Issue type` do problemu; Przykładowo na poniższym obrazie zaznaczono `Technical`. Pamiętaj o wybraniu `Microsoft Genomics` jako odpowiedniej usługi.  `Microsoft Genomics` będzie wyświetlana w kategorii `Intelligence and Analytics`.   Po wybraniu `Microsoft Genomics` zostanie również wyświetlony monit o podanie typu `Support plan`.

Podstawowe(./media/file-support-ticket/support-request-basics.png "informacje Support request") ![support Request]


Następnie Dodaj szczegóły dotyczące problemu. Narzędzie żądania również łączy się z istniejącą dokumentacją, która może być pomocna. W tym przykładzie problem dotyczy `Genomics keys`, ale dostępne są również różne opcje.

Problem(./media/file-support-ticket/support-request-problem.png "Support request") ![support Request problemu]

Na koniec Dodaj swoje informacje kontaktowe i wybierz `Create` u dołu ekranu w celu przesłania żądania pomocy technicznej.

Skontaktuj się z(./media/file-support-ticket/support-request-contact.png "kontaktem Support request") ![support Request]

## <a name="next-steps"></a>Następne kroki
W tym artykule przedstawiono sposób przesyłania żądania pomocy technicznej. Możesz również rozwiązywać typowe problemy, korzystając z naszych [często zadawanych pytań](frequently-asked-questions-genomics.md) i naszego [przewodnika rozwiązywania problemów](troubleshooting-guide-genomics.md).
