# <a name="virtual-machine-prerequisites"></a>Wymagania wstępne maszyny wirtualnej

W tym artykule wymieniono zarówno wymagania techniczne, jak i biznesowe, które należy spełnić przed opublikowaniem oferty maszyny Wirtualnej w [portalu Azure Marketplace.](https://azuremarketplace.microsoft.com/marketplace/)  Jeśli jeszcze tego nie zrobiono, zapoznaj się z [przewodnikiem publikowania ofert maszyn wirtualnych](../../marketplace-virtual-machines.md).


## <a name="technical-requirements"></a>Wymagania techniczne

Techniczne wymagania wstępne dotyczące publikowania rozwiązania maszyny wirtualnej są proste:

- Musisz mieć aktywne konto platformy Azure. Jeśli go nie masz, możesz zarejestrować się w [witrynie microsoft azure](https://azure.microsoft.com).  
- Musisz mieć środowisko skonfigurowane do obsługi tworzenia maszyn wirtualnych z systemem Windows lub Linux.  Aby uzyskać więcej informacji, zobacz skojarzoną witrynę dokumentacji maszyny Wirtualnej:
    - [Dokumentacja maszyny wirtualnej z systemem Linux](https://docs.microsoft.com/azure/virtual-machines/linux/)
    - [Dokumentacja maszyny wirtualnej z systemem Windows](https://docs.microsoft.com/azure/virtual-machines/windows/)
