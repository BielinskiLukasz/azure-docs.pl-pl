# <a name="python-packages-for-azure-machine-learning"></a>Pakiety języka Python dla usługi Azure Machine Learning

Dowiedz się więcej o zastrzeżonych pakietów języka Python firmy Microsoft dla usługi Azure Machine Learning. Te biblioteki i funkcji można użyć w połączeniu z innymi pakietami open source lub innych firm, ale do używania zastrzeżonych pakiety programu Python kod należy uruchomić przed usługi lub komputera, który udostępnia tłumaczy.

Pakiety usługi Azure Machine Learning **instalowalnych pip rozszerzenia języka Python dla usługi Azure Machine Learning** umożliwiających danych i analityków AI do szybkiego tworzenia i wdrażania, uczenie maszynowe bardzo dokładne i uczenie się bezpośrednich modele dla różnych domen.

<a name="amlpcv"></a>
## <a name="azure-ml-package-for-computer-vision"></a>Pakiet Azure ML do widzenia komputera

Z pakietem uczenia Maszynowego Azure wizji komputera, można tworzyć, dostosowywanie, wdrażanie modeli uczenia głębokie klasyfikacji obrazu obiekt wykrywania i podobieństwa obrazu.

Spróbuj wykonać te czynności dla tego pakietu:
1. [Pobierz](https://aka.ms/aml-packages/vision/download) pakietu.
1. Odczyt [zainstalować docs](https://aka.ms/aml-packages/vision).
1. [Skompiluj i Wdróż model wizji komputera](how-to-build-deploy-image-classification-models.md) z notesem Jupyter.
1. Eksploruj pakietu [odwołania dokumentacji](https://aka.ms/aml-packages/vision).
