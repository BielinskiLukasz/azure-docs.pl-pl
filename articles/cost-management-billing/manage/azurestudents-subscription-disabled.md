# <a name="why-is-my-azure-for-students-subscription-disabled-and-how-do-i-reactivate-it"></a>Dlaczego moja subskrypcja platformy Azure for Students została wyłączona i jak można ją ponownie uaktywnić?

Subskrypcja platformy Azure for Students może zostać wyłączona z powodu wykorzystania wszystkich środków, wygaśnięcia środków lub przypadkowego anulowania subskrypcji. Zobacz, która z wymienionych przyczyn ma zastosowanie w Twojej sytuacji, i dowiedz się, jak można ponownie uaktywnić subskrypcję.

## <a name="youve-used-all-of-your-credit"></a>Wykorzystano wszystkie środki

Konto Azure for Students zapewnia środki w wysokości 100 USD i ograniczoną liczbę bezpłatnych usług przez 12 miesięcy. Wszelkie użycie wykraczające poza bezpłatne usługi i ilości jest potrącane ze środków. Po wyczerpaniu środków platforma Azure wyłącza Twoje usługi i subskrypcję. Aby móc nadal korzystać z usług platformy Azure, musisz podwyższyć poziom swojej subskrypcji do subskrypcji w modelu płatności zgodnie z rzeczywistym użyciem, kontaktując się z [pomocą techniczną platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). Po podwyższeniu poziomu Twoja subskrypcja będzie nadal miała dostęp do bezpłatnych usług przez 12 miesięcy od daty rejestracji. Opłata jest naliczana tylko za użycie wykraczające poza bezpłatne usługi i ilości.

Pozostałe środki możesz sprawdzić w portalu [dostępu sponsorowanego Microsoft Azure](https://www.microsoftazuresponsorships.com/balance)

1. Zaloguj się przy użyciu poświadczeń konta platformy Azure for Students.
2. Strona salda zawiera informacje o zużytych i pozostałych środkach. Datę wygaśnięcia środków możesz sprawdzić poniżej wykresu dotyczącego środków.  

    ![Zrzut ekranu pokazujący informacje o środkach dla subskrypcji platformy Azure for Students](./media/azurestudents-subscription-disabled/azurestudents-credit-balance.png)

3. Aby sprawdzić użycie usług korzystających ze środków platformy Azure, wybierz pozycję **Użycie** w lewym obszarze nawigacji, a następnie wybierz subskrypcję platformy Azure for Students.

    ![Zrzut ekranu pokazujący informacje o użyciu środków subskrypcji platformy Azure for Students](./media/azurestudents-subscription-disabled/azurestudents-credit-usage.png)

Tabela zawiera następujące kolumny:

* **Nazwa usługi:** Usługa, która zużywa środki na korzystanie z platformy Azure.
* **Zasób usługi:** Jednostka miary dla używanej usługi.
* **Wydano:** Kwota środków w USD wydana na usługę.

## <a name="your-credit-has-expired"></a>Twoje środki wygasły

Środki na korzystanie z platformy Azure for Students wygasają po upływie 12 miesięcy. Po wygaśnięciu środków platforma Azure wyłączy twoją subskrypcję. Aby móc nadal korzystać z usług platformy Azure, musisz podwyższyć poziom swojej subskrypcji do subskrypcji w modelu płatności zgodnie z rzeczywistym użyciem, kontaktując się z [pomocą techniczną platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). Po podwyższeniu poziomu na platformie Azure są naliczane opłaty zgodnie ze stawkami płatności zgodnie z rzeczywistym użyciem za wszystkie używane usługi.

## <a name="youve-accidentally-canceled-your-subscription"></a>Twoja subskrypcja została przypadkowo anulowana

Jeśli przypadkowo anulowano subskrypcję platformy Azure for Students, można ją ponownie uaktywnić, kontaktując się z [pomocą techniczną platformy Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). Po ponownym uaktywnieniu nadal masz dostęp do pozostałych środków i bezpłatnych usług przez 12 miesięcy od daty rejestracji.
