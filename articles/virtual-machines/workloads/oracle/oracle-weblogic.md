# <a name="oracle-weblogic-server-azure-applications"></a>Oracle webWebLogicc Server — aplikacje platformy Azure

## <a name="oracle-weblogic-server-is-a-scalable-enterprise-ready-java-ee-application-server"></a>Oracle webWebLogicc Server to skalowalny, gotowy do zastosowania w przedsiębiorstwie serwer aplikacji Java EE.

Oracle webWebLogicc Server to pierwszy, natywny chmurowy serwer aplikacji platformy Java dla przedsiębiorstw na potrzeby tworzenia i wdrażania aplikacji rozproszonego przedsiębiorstwa wielowarstwowego. Usługa Azure WebLogic Server oferuje możliwość tworzenia rozwiązań w chmurze, co zapewnia większą możliwość wyboru i elastyczność migracji WebLogic, w tym podnoszenia i przesunięć aplikacji Java EE do chmury platformy Azure z mniejszym nakładem pracy i największym wpływem. Oferty mogą szybko rozpocząć pracę z aplikacjami biznesowymi, automatycznie udostępniając zasoby sieci wirtualnej, magazynu i systemu Linux, instalując serwer WebLogic, konfigurując zabezpieczenia z sieciową grupą zabezpieczeń, równoważenia obciążenia za pomocą usługi Azure App Gateway, uwierzytelniania przy użyciu Azure Active Directory i zwiększania łączności z bazą danych.

Dostępne są cztery oferty do realizacji różnych scenariuszy: pojedynczy węzeł bez serwera administracyjnego, pojedynczy węzeł z serwerem administracyjnym, klastrem i klastrem dynamicznym.  Możesz skorzystać z bezpłatnej oferty, aby umożliwić im wypróbowanie ofert.

### <a name="oracle-weblogic-server-single-node"></a>Pojedynczy węzeł serwera Oracle WebLogic Server

Ta oferta udostępnia pojedynczą maszynę wirtualną i instaluje na niej serwer Oracle webWebLogicc. Nie tworzy domeny ani nie uruchamia serwera administracyjnego. Jest to przydatne w scenariuszach z wysoce dostosowaną konfiguracją domeny.

### <a name="oracle-weblogic-server-with-admin-server"></a>Serwer Oracle webWebLogicc Server z serwerem administracyjnym

Ta oferta udostępnia pojedynczą maszynę wirtualną i instaluje na niej serwer Oracle webWebLogicc. Tworzy domenę i uruchamia serwer administracyjny, który umożliwia zarządzanie domeną.

### <a name="oracle-weblogic-server-cluster"></a>Klaster serwera Oracle WebLogic

Ta oferta służy do tworzenia klastra o wysokiej dostępności z maszynami wirtualnymi programu Oracle WebLogic Server. Serwer administracyjny i wszystkie zarządzane serwery są uruchamiane domyślnie, co umożliwia zarządzanie domeną.

### <a name="oracle-weblogic-server-dynamic-cluster"></a>Dynamiczny klaster programu Oracle WebLogic Server

Ta oferta tworzy wysoce dostępny i skalowalny dynamiczny klaster maszyn wirtualnych Oracle webWebLogicc Server. Serwer administracyjny i wszystkie zarządzane serwery są uruchamiane domyślnie, co umożliwia zarządzanie domeną.
