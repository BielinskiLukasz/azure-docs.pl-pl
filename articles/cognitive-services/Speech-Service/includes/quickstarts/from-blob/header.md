W tym przewodniku Szybki start użyjesz interfejsu API REST do rozpoznawania mowy z plików w procesie wsadowym. Proces wsadowy wykonuje transkrypcję mowy bez żadnych interakcji użytkownika. Zapewnia prosty model programowania, bez konieczności zarządzania współbieżności, niestandardowych modeli mowy lub innych szczegółów. Pociąga za sobą zaawansowane opcje sterowania, jednocześnie efektywnie korzystając z zasobów usługi mowy platformy Azure.

Aby uzyskać więcej informacji na temat dostępnych opcji i szczegółów konfiguracji, zobacz [transkrypcja partii](../../../batch-transcription.md).

Poniższy przewodnik Szybki start spowoduje przejście przez przykład użycia.
