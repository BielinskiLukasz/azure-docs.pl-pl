Najprostszym sposobem utworzenia konta magazynu platformy Azure jest użycie [witryny Azure portal](https://portal.azure.com). Więcej informacji można znaleźć w temacie [Tworzenie konta magazynu](../articles/storage/common/storage-account-create.md).

Możesz utworzyć konto usługi Azure Storage przy użyciu programu [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md) lub [interfejsu wiersza polecenia platformy Azure](../articles/storage/common/storage-azure-cli.md).

Jeśli nie chcesz teraz tworzyć konta magazynu, możesz także użyć emulatora usługi Azure Storage do uruchomienia i testowania kodu w środowisku lokalnym. Aby uzyskać więcej informacji, zobacz [Używanie emulatora magazynu platformy Azure do tworzenia i testowania.](../articles/storage/common/storage-use-emulator.md)
