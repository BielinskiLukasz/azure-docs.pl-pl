# <a name="pricing-and-billing-for-azure-machine-learning-services"></a>Cennik i rozliczenia Azure Machine Learning usług

Aby uzyskać szczegółowe informacje, lub aby wyświetlić zestawienie próbki, odwiedź [Azure cennikiem](https://azure.microsoft.com/pricing/details/machine-learning-services/).  

Poniżej przedstawiono niektóre często zadawane pytania dotyczące rozliczeń i cenach.

## <a name="can-i-use-azure-machine-learning-for-free-during-preview"></a>Można używać usługi Azure Machine Learning bezpłatnej wersji zapoznawczej?    

Aplikacja Azure Machine Learning Workbench jest dostępny bezpłatnie do subskrybentów Azure.

Model zarządzania i usługi eksperymenty oferują warstwy bezpłatna, oprócz warstw płatnych będzie dostępna w obniżonej cenie publicznej wersji zapoznawczej.

## <a name="am-i-charged-based-on-how-many-experiments-i-run"></a>Jestem daną opłatę oparte na ile eksperymenty uruchomić?

Nie. Usługa Eksperymentowanie umożliwia wykonanie dowolnej liczby eksperymentów w zależności od potrzeb, a opłaty są naliczane wyłącznie na podstawie liczby użytkowników. Opłaty za zasoby obliczeniowe usługi Eksperymentowanie są naliczane oddzielnie.  Firma Microsoft zachęca do wykonywania wielu eksperymenty w modelu najlepsze rozwiązania. 

## <a name="am-i-charged-based-on-how-many-times-my-web-services-is-called"></a>Jestem daną opłatę na podstawie na ile razy nosi nazwę moich usług sieci web?

Nie. Usługi sieci Web można wywołać tak często, zgodnie z wymaganiami, bez konieczność zarządzania modelu rozliczeń. Masz pełną kontrolę nad skalowaniem swoich wdrożeń, aby spełniały wymagania aplikacji.

## <a name="how-can-i-scale-the-units-i-purchase-in-the-azure-machine-learning-model-management"></a>Jak można skalować jednostki, które można kupić w pakiecie administracyjnym modelu Azure Machine Learning?

Można zmienić liczbę jednostek, w górę lub w dół, za pomocą portalu Azure lub interfejsu wiersza polecenia. 

## <a name="what-does-a-bill-look-like"></a>Zestawienie jak wygląda?

Opłaty są produkowane codziennie. Dla celów rozliczeniowych dzień rozpoczyna się o północy (czas UTC). Rachunki są generowane raz w miesiącu. Oddzielne jest obciążany dla wszystkich usług platformy Azure, używane w połączeniu z usługi Azure Machine Learning. Opłaty mogą obejmować, ale nie są ograniczone do: 
- Opłaty za obliczenia bazy danych
- HDInsight
- Azure Container Service
- Azure Container Registry 
- Azure Blob Storage
- Application Insights
- W usłudze Azure Key Vault
- Visual Studio Team Services
- Azure Event Hub
- Azure analizy strumienia dla bardziej szczegółowe informacje, lub aby wyświetlić zestawienie przykładowe, odwiedź [Azure cennikiem](https://azure.microsoft.com/pricing/details/machine-learning-services/).
