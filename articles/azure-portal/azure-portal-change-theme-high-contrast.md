# <a name="turn-on-high-contrast-or-change-the-theme-in-the-azure-portal"></a>Włączanie wysokiego kontrastu lub zmienianie motywu w witrynie Azure portal

Ustawienia wysokiego kontrastu ułatwiają czytanie witryny Azure Portal. Można również wybrać motyw, który zmienia kolory tła portalu.

Jeśli potrzebujesz więcej kontrastu lub chcesz zmienić schemat kolorów w witrynie Azure portal, przejdź do ustawień portalu, aby wprowadzić zmiany.

## <a name="turn-on-high-contrast"></a>Włączanie wysokiego kontrastu

1. W nagłówku [portalu Azure](https://portal.azure.com)wybierz pozycję **Ustawienia**.

    ![Zrzut ekranu przedstawiający ikonę koła zębatego ustawień portalu w witrynie Azure portal](./media/azure-portal-change-theme-high-contrast/azure-portal-settings-icon.png)
1. Wybierz **biały** lub **czarny**.

    ![Zrzut ekranu przedstawiający opcje wysokiego kontrastu w ustawieniach witryny Azure portal](./media/azure-portal-change-theme-high-contrast/azure-portal-highcontrast-options.png)
1. Wybierz przycisk **Zastosuj**.

> [!NOTE]
> Jeśli wybierzesz motyw o wysokim kontraście, zastąpi on wszystkie inne wybrane motywy.
>
>

## <a name="change-theme"></a>Kompozycja Zmienianie

1. W nagłówku [portalu Azure](https://portal.azure.com)wybierz pozycję **Ustawienia**.

    ![Zrzut ekranu przedstawiający ikonę koła zębatego ustawień portalu w witrynie Azure portal](./media/azure-portal-change-theme-high-contrast/azure-portal-settings-icon.png)

1. Wybierz motyw.

    ![Zrzut ekranu przedstawiający opcje motywu w ustawieniach witryny Azure portal](./media/azure-portal-change-theme-high-contrast/azure-portal-theme-options.png)

1. Wybierz przycisk **Zastosuj**.
