# SQL Data Warehouse ソリューション パートナー

エンド ツー エンドのデータ ウェアハウス ソリューションを作成するには、さまざまな業界をリードするツールから選択します。 この記事では、SQL Data Warehouse 向けのソリューションを提供するパートナー企業の一覧を示します。


| パートナー | 説明 | 詳細 |
| ------- | ----------- |------|
| ![Attunity][1] |**Attunity**<br>Attunity CloudBeam は、SQL Data Warehouse にデータを読み込むため、自動化および最適化されたソリューションを提供します。 "クリックして読み込み" 設計を使用して、SQL Server、Oracle、DB2、Sybase、mySQL など、多種多様なデータベースからのバッチ読み込みと増分レプリケーションを簡素化し、迅速化します。 | [詳細][attunity_more]|
| ![Birst][2] |**Birst**<br>Birst は世界的なクラウド BI および分析します。 主要なビジネス ユーザーごとの状況に応じて、信頼できるデータ ソースを作成できます。 次に、組織全体のビジネス ユーザーが、世界最高クラスの BI ツールを使用して情報をレポートおよび分析できます。| [詳細][birst_more] |
| ![Bryte Systems][3] |**Bryte Systems**<br>Bryte Systems は、さまざまなデータベースやソースから SQL Data Warehouse へのデータ転送を自動化します。 Bryte Systems は、データの移動および統合テクノロジの専門企業です。 スケーラブルなクラウド ベースのデータ ウェアハウスと Data Lake を構築することで、知見を見いだすことができます。 | [詳細][bryte_systems_more] |
| ![ClearStory Data][4] |**ClearStory Data**<br>ClearStory Data は、SQL Data Warehouse に格納されているさまざまなデータわたる高速サイクル分析します。 ClearStory の Spark ベース プラットフォームと分析アプリケーションにより、多種多様なデータ セットへのアクセスと協調が高速化されます。 また、高速で調和的な探索が実現されるため、ビジネス ユーザーは他のユーザーに頼らずに知見を得ることができます。| [詳細][clearstory_data_more] |
| ![Coffing Data Warehousing][5] |**Coffing Data Warehousing**<br>Coffing Data Warehousing、設計がクエリ実行システムに専用の 10 年ある、ツール、Nexus Chameleon を提供します。 Nexus は SQL Data Warehouse のクエリ ツールとして使用できます。 Nexus を使用して、社内でクエリを実行し、さまざまなプラットフォームでクラウド コンピューターとデータを結合します。 ポイントしてクリックするだけでレポートが生成されます。 | [詳細][coffing_data_warehousing_more] |
| ![Dell Statistica][6] |**Dell Statistica**<br>Dell Statistica は、高度な分析プラットフォーム将来に関する予測を行う、プロセス最適化、コスト削減と収益を上げることが可能です。  Dell Statistica は、ビッグ データ分析を簡略化し、実装とデプロイを効率化します。また、次の行動につながる洞察をテキストから抽出し、ビジネスを支援します。 | [詳細][dell_statistica_more] |
| ![Dundas][7] |**Dundas Software**<br>Dundas Data Visualization はビジネス インテリジェンスおよびデータ可視化ソフトウェアのグローバルの的な一流プロバイダーです。  Dundas のダッシュボード、レポート作成、およびビジュアル データ分析により、ビジネス アプリケーションがシームレスに統合されることで、適切な決断とすばやい分析が可能になります。 | [詳細][dundas_software_more] |
| ![Inbrein][8] |**Inbrein**<br>Inbrein MicroERD は、正確なデータ モデルを作成、データの冗長性の削減、生産性の向上、および標準を観察するために必要なツールを提供します。 広範なユーザー エクスペリエンスに基づいて開発されたこのツールの UI を使用することで、モデル作成者は容易かつ使い勝手よく DB モデルに取り組むことができます。 プロンプト機能の向上と更新により、MicroERD の強化された新機能を途切れることなく利用できます。  | [詳細][inbrein_more] |
| ![Infolibrarian][9] |**Infolibrarian**<br>InfoLibrarian カタログ化、格納、およびデータの管理キーの問題点を解決するためのメタデータを管理します。 また、メタデータ管理、データ ガバナンス、および資産管理のソフトウェア ソリューションを提供することで、多種多様なツールとテクノロジの組み合わせにおいてメタデータを管理したり公開したりできるようにします。 | [詳細][infolibrarian_more] |
| ![Informatica][10] |**Informatica**<br>Informatica は世界的に有名な独立したデータ統合ソフトウェアのプロバイダーの 1 つです。 業界初の組み込み可能な仮想データ マシンの 1 つである Informatica Vibe は、 どこにでもデプロイします。" Informatica プラットフォームの機能です。 全世界で 5,500 以上の企業が Informatica を使用することで、情報資産を十二分に活用しています。| [詳細][informatica_more] |
| ![Jinfonet][11] |**Jinfonet**<br>JReport は、企業の組み込み可能な BI ソリューションです。 このソリューションでは、クラウド、ビッグ データ、およびトランザクション データ ソースに関するレポート、ダッシュボード、データ分析でユーザーを支援します。 ユーザーはデータを視覚化することにより、独自のレポート作成とデータ探索を実行して、機敏で継続的な意思決定を実現できます。| [詳細][jinfonet_more] |
| ![連携][12] |**連携**<br>連携して、Logi Analytics と Microsoft Azure SQL Data Warehouse、組織の収集、分析、および、最大ですぐに機能を利用でき、パフォーマンスの高い組織を後押しの世界で最も多様なデータを設定します。 | [詳細][logianalytics_more] |
| ![Looker][13] |**Looker**<br>Looker は、会社のすべてのユーザーに探索、コントロールに柔軟で再利用可能なモデリング レイヤーは、データ アナリスト、同時にビジネスの推進データを理解してそのデータを監督する機能を利用します。  Looker を触媒として使用することによって、企業のカルチャが根本的に変革されます。 | [詳細][looker_more] |
| ![Qlik][14] |**Qlik**<br>Qlik は、Microsoft Azure SQL Data Warehouse の上にビジュアル分析にプラットフォーム ベースのアプローチを提供します。  Qlik は、Microsoft のエラスティックなデータ ウェアハウス内でエンタープライズ クラスの機能を活用して、自信を持って迅速な意思決定を行うことができるよう組織全体を支援します。 Qlik は、最も必要とされる決断ポイントに洞察と明確さをもたらします。 | [詳細][qlik_more] |
| ![グローバル Redpont][15] |**Redpoint Global**<br>RedPoint Interaction と RedPoint Data Management は、マーケティング担当者は、すべてのデータ構造の実行中にクロス チャネルでの顧客に惹き付け、非構造化データの管理を活用してを有効にします。 RedPoint ユーザーは Microsoft Azure SQL Data Warehouse を活用してすべての構造化または非構造化データの価値を最大化することで、今日のオムニ チャネルの顧客を引き込むために必要な高度にパーソナライズされ、状況に応じたやりとりを実現できます。 ドラッグ アンド ドロップ インターフェイスにより、データ管理プロセスを容易に設計および実行できます。 | [詳細][redpoint_global_more] |
| ![SiSense][16] |**SiSense**<br>SiSense は、フル スタック ビジネス インテリジェンスのソフトウェアを分析およびデータの視覚化に必要なツールに付属している: 高パフォーマンスの分析データベース、複数のソース、単純なデータの抽出 (ETL)、および web ベースのデータの視覚エフェクトを結合する機能です。 SiSense BI と Analytics を使用して、今すぐ大規模なデータ セットの分析と視覚化を開始できます。 | [詳細][sisense_more] |
| ![SnapLogic][17] |**SnapLogic**<br>SnapLogic の統合プラットフォーム (iPaaS) をサービスとしてにより、お客様は Microsoft Azure SQL Data Warehouse に出入りするデータをすばやく転送します。  SnapLogic は、クラウド、オンプレミス、またはハイブリッド モードで、数百ものアプリケーション、サービス、IoT を統合する機能を提供することで、迅速かつ効果的な洞察をもたらします。 | [詳細][snaplogic_more] |
| ![SQL Sentry][18] |**SQL Sentry**<br>SQL Sentry Performance Advisor は、パフォーマンスの傾向を表示することができるという、最も適切な Azure SQL Data Warehouse のパフォーマンス メトリックのリアルタイムと履歴の表示を提供し、詳細なより突っ込んだ分析とトラブルシューティング情報を簡単にアクセスします。 Performance Advisor は、任意の時点または日付の範囲について、パフォーマンス データを表示できます。 | [詳細][sql_sentry_more] |
| ![Tableau][19] |**Tableau**<br>Tableau のセルフ_サービス分析は、さまざまな種類のフラット ファイルからデータベースへのデータの間で、データを理解し、やすくだれでもが視覚化されます。 Tableau には、Microsoft Azure SQL Data Warehouse への、最適化されたネイティブのコネクタが備わっており、ライブ データ分析とメモリ内分析の両方がサポートされます。 | [詳細][tableau_more] |
| ![Targit][20] |**Targit**<br>Targit Decision Suite は、BI および分析プラットフォームでリアルタイムのダッシュ ボード、セルフ_サービス分析、わかりやすいレポート、魅力的なモバイル機能、および簡単なデータ検出テクノロジを 1 つのまとまりのあるソリューションにを提供します。 Targit は、行動を起こす勇気を企業にもたらします。 | [詳細][targit_more] |
| ![Yellowfin][21] |**Yellowfin**<br>Yellowfin は、BARC の The BI survey アドホック レポートとダッシュ ボードのクラウド BI ベンダーを評価します。 Azure SQL Data Warehouse に接続すると、受賞歴のあるコラボレーション BI 機能とロケーション インテリジェンス機能を備えた美しいレポートとダッシュボードが作成されて共有できるようになります。 | [詳細][yellowfin_more] |
