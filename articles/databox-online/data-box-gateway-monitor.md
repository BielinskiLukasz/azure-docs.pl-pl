# <a name="monitor-your-azure-data-box-gateway"></a>Monitorowanie usługi Azure Data Box bramy

W tym artykule opisano sposób monitorowania bramy pola danych platformy Azure. Aby monitorować urządzenie, można użyć witryny Azure portal lub lokalnego Interfejsu w przeglądarce. Użyj witryny Azure portal, aby wyświetlić zdarzenia urządzenia, konfigurowanie i Zarządzanie alertami i wyświetlanie metryk.

W tym artykule omówiono sposób wykonywania następujących zadań:

> [!div class="checklist"]
> * Wyświetl zdarzenia urządzenia i odpowiednie alerty
> * Wyświetlanie metryk pojemności i transakcji dla Twojego urządzenia
> * Skonfiguruj alerty i zarządzaj nimi
