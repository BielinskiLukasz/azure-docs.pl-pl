Nie można zaktualizować lokalizacji kotwicy po jej utworzeniu w usłudze — należy utworzyć nową kotwicę i usunąć starą, aby śledzić nową pozycję.

Jeśli nie trzeba zlokalizować zakotwiczenia, aby zaktualizować jego `GetAnchorPropertiesAsync()` właściwości, można `CloudSpatialAnchor` użyć metody, która zwraca obiekt z właściwościami.
