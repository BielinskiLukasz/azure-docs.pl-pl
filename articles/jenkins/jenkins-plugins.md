# <a name="jenkins-plugins-for-azure"></a>Wtyczki Jenkins dla platformy Azure

Następujące wtyczki usługi Jenkins obsługują różne funkcje do użytku z platformą Azure.
  
| Wtyczka Jenkins | Opis                                   |
|------------------------------------------------------------------------------|
| [Wtyczka usługi Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Wtyczka usługi Jenkins do wdrażania usługi Azure App Service (obecnie obsługuje tylko aplikację sieci Web). | 
| [Wtyczka usługi Azure AD](https://plugins.jenkins.io/azure-ad)                       | Wtyczka usługi Jenkins, która obsługuje autoryzację & uwierzytelniania za pośrednictwem usługi Azure Active Directory. | 
| [Wtyczka usługi Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | Wtyczka usługi Azure Artifact Manager to Menedżer artefaktów, który umożliwia przechowywanie artefaktów w usłudze Azure Blob Storage. Wtyczka usługi Azure Artifact Manager działa w sposób przejrzysty do usługi Jenkins i zadań, jest jak domyślny Menedżer artefaktów. | 
| [Wtyczka agentów kontenera platformy Azure](https://plugins.jenkins.io/azure-container-agents) | Wtyczka Azure Container Agents może pomóc w uruchomieniu kontenera jako agenta w usłudze Jenkins | 
| [Wtyczka zadania rejestru kontenerów platformy Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Wtyczka usługi Jenkins do wysyłania żądania docker-build do [rejestru kontenerów platformy Azure](/azure/container-registry/container-registry-tasks-overview). |
| [Wtyczka usługi kontenera platformy Azure](https://plugins.jenkins.io/azure-acs)       | Wtyczka usługi Jenkins do wdrażania konfiguracji usługi Azure Container Service (AKS). | 
| [Wtyczka poświadczeń platformy Azure](https://plugins.jenkins.io/azure-credentials)      | Wtyczka usługi Jenkins do zarządzania poświadczeniami platformy Azure. | 
| [Wtyczka funkcji platformy Azure](https://plugins.jenkins.io/azure-function)           | Wtyczka usługi Jenkins do wdrażania funkcji platformy Azure. | 
| [Wtyczka usługi Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | Wtyczka usługi Azure IoT Edge ułatwia konfigurowanie potoku ciągłej integracji/ciągłego wdrażania w celu opracowania rozwiązania usługi IoT Edge w usłudze Jenkins. | 
| [Wtyczka usługi Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Wtyczka Jenkins dla projektów sieci szkieletowej usługi Azure systemu Linux. |
| [Wtyczka usługi Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | wtyczka do przekazywania artefaktów kompilacji do magazynu obiektów Blob platformy Microsoft Azure lub pobierania ich z niej. | 
| [Wtyczka agentów platformy Azure VM](https://plugins.jenkins.io/azure-vm-agents)         | Wtyczka usługi Jenkins do tworzenia agentów usługi Jenkins na maszynach wirtualnych platformy Azure (za pomocą szablonu usługi Azure Resource Manager). | 
| [Wtyczka zestawu skalowania maszyny wirtualnej platformy Azure](https://plugins.jenkins.io/azure-vmss)           | Wtyczka usługi Jenkins do wdrażania obrazów maszyn wirtualnych w zestawach skalowania maszyny wirtualnej platformy Azure. |
