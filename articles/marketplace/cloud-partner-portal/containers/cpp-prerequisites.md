# <a name="container-publishing-prerequisites"></a>Wymagania wstępne dotyczące publikowania kontenera

> [!IMPORTANT]
> Od 13 kwietnia 2020 r. rozpoczniemy przenoszenie zarządzania ofertami usługi Azure Container do Centrum partnerów. Po migracji utworzysz oferty i zarządzasz nimi w Centrum partnerów. Postępuj zgodnie z instrukcjami w [tworzenie oferty usługi Azure Container](https://aka.ms/CreateContainerOffer) do zarządzania zmigrowanych ofert.

W tym artykule opisano wymagania wstępne dotyczące publikowania oferty kontenerów w portalu Azure Marketplace.  Jeśli jeszcze tego nie zrobiłeś, należy zapoznać się z [Przewodnikiem publikowania ofert kontenerów](../../marketplace-containers.md).

## <a name="publishing-prerequisites"></a>Wymagania wstępne publikowania

Aby opublikować nowy obraz kontenera, należy spełnić następujące wymagania wstępne:

- Dostęp do portalu cloud partnerów. Aby uzyskać więcej informacji, zobacz Przewodnik publikowania w portalu Azure Marketplace i AppSource.
- Umowa z warunkami portalu Azure Marketplace
- Hostuj zasób techniczny kontenera w rejestrze kontenerów platformy Azure.
- Przygotuj metadane kontenera do użycia. Na przykład następująca niewyczerpywna lista:
  - Tytuł
  - Opis (w formacie HTML)
  - Obraz logo (w formacie PNG) i w tych stałych rozmiarach obrazu: 40x40 px, 90x90 px, 115x 115 px i 255x115 px.
- *Warunki użytkowania* i oświadczenia dotyczące *polityki prywatności*
- Dokumentacja rozwiązania kontenerowego
- Kontakt z pomocą techniczną
