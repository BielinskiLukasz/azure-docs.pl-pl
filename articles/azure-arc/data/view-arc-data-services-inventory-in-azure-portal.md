# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Wyświetlanie spisu wystąpień w Azure Portal

Po przekazaniu [metryk, dzienników](upload-metrics-and-logs-to-azure-monitor.md)lub [użycia](view-billing-data-in-azure.md)można wyświetlić wystąpienie jako zasób platformy Azure z Azure Portal. Aby to zrobić, przejdź do specjalnego adresu URL [https://portal.azure.com](https://portal.azure.com) i postępuj zgodnie z poniższymi zrzutami ekranu

1. Przejdź do wszystkich usług
1. Wyszukaj typy wystąpień bazy danych
1. Dodaj do ulubionych
1. Kliknij typ wystąpienia na lewym panelu nawigacyjnym
1. Wyświetlaj wystąpienia w tym samym widoku co inne zasoby usługi Azure SQL lub PostgreSQL w skali platformy Azure (Użyj filtrów, aby uzyskać szczegółowy widok)
