Gdy usługa Microsoft Azure Traffic Manager umożliwia równoważenie obciążenia ruchem do witryny sieci Web platformy Azure, tej witryny sieci Web mają dostęp za pomocą  **\*. trafficmanager.net** nazwy domeny przypisany przez platformę Azure. Można również skojarzyć niestandardowej nazwy domeny, takich jak www\.contoso.com za pomocą witryny sieci Web w celu zapewnienia bardziej rozpoznawalną nazwę domeny dla użytkowników.
