# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Monitoruj i Zarządzaj usługą Wunderlist przy użyciu Azure Logic Apps

Od 6 maja 2020 [Usługa Wunderlist zostanie ZAMKNIĘTA](https://www.wunderlist.com/) i poprosiła klientów o przeprowadzenie [migracji danych i rozpoczęcie korzystania z firmy Microsoft w zamian](https://www.wunderlist.com/blog/goodbye-from-wunderlist/). Możesz użyć [łącznika firmy Microsoft do wykonania](https://docs.microsoft.com/connectors/todo/) , aby zastąpić przestarzały [Łącznik Wunderlist](https://docs.microsoft.com/connectors/wunderlist/).
