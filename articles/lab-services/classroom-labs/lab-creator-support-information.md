# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>Wyświetlanie informacji o pomocy technicznej (twórca laboratorium w usłudze Azure Lab Services)
W tym artykule wyjaśniono, w jaki sposób (jako twórca laboratorium) możesz wyświetlić następujące informacje pomocy technicznej:

- Adres URL
- Adres e-mail
- Telefon
- Dodatkowe instrukcje

Można użyć tych informacji, aby uzyskać pomoc, gdy napotkasz wszelkie problemy techniczne podczas tworzenia laboratorium na koncie laboratorium.

 
## <a name="view-support-information"></a>Wyświetlanie informacji o pomocy technicznej
1. Zaloguj się do [portalu usług Azure Lab Services](https://labs.azure.com).
2. Wybierz znak zapytania (**?**) w prawym górnym rogu strony. 
3. Upewnij się, że widzisz linki do **witryny pomocy technicznej widoku,** **obsługi poczty e-mail**i **numeru telefonu pomocy technicznej.**

    ![Wyświetlanie informacji o pomocy technicznej](../media/lab-creator-support-information/support-information.png)
