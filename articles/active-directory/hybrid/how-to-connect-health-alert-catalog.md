# <a name="azure-active-directory-connect-health-alert-catalog"></a>Azure Active Directory Connect Health katalog alertów 

Alerty wysyłania usług w usłudze Azure AD Connect Health wskazują, że swoją infrastrukturę tożsamości nie jest w dobrej kondycji. Ten artykuł zawiera alerty tytułów, opisów i czynności zaradcze dla każdego alertu. <br />
Błąd, ostrzeżenie i Prewarning są trzy etapy alerty, które są generowane na podstawie usługi Connect Health. Zdecydowanie zalecamy podjęcia akcji natychmiastowych alertów wyzwalanych. <br />
Alerty programu Azure AD Connect Health można rozwiązać, pod warunkiem powodzenia. Agentów Azure AD Connect Health wykrywanie i okresowo zgłoszenia warunków powodzenia do usługi. Kilka alertów pomijanie jest oparte na czasie. Oznacza to jeśli ten sam błąd nie zostanie wykryty w ciągu 72 godzin z generowania alertów, alert zostanie rozwiązany automatycznie.
