> [!NOTE]
> W tym artykule jest stosowana Najnowsza wersja rozszerzenia usługi Azure IoT o nazwie `azure-iot`. Starsza wersja jest nazywana `azure-iot-cli-ext`. W danym momencie powinna być zainstalowana tylko jedna wersja. Za pomocą polecenia `az extension list` można sprawdzić poprawność zainstalowanych rozszerzeń.
>
> Użyj `az extension remove azure-iot-cli-ext`, aby usunąć starszą wersję rozszerzenia.
> Użyj `az extension add -name azure-iot`, aby dodać nową wersję rozszerzenia. 
>
> Aby zobaczyć, jakie rozszerzenia zostały zainstalowane, użyj `az extension show `.
>
