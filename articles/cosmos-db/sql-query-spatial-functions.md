# <a name="spatial-functions-azure-cosmos-db"></a>Funkcje przestrzenne (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

Usługa Cosmos DB obsługuje następujące wbudowane funkcje OGC (Open Geospatial Consortium) w celu wykonywania zapytań dotyczących danych geoprzestrzennych. 

## <a name="functions"></a>Funkcje

Poniższe funkcje skalarne wykonują operacje na wartości wejściowej obiektu przestrzennego i zwracają wartość liczbową lub logiczną.  

* [ST_DISTANCE](sql-query-st-distance.md)
* [ST_INTERSECTS](sql-query-st-intersects.md)
* [ST_ISVALID](sql-query-st-isvalid.md)
* [ST_ISVALIDDETAILED](sql-query-st-isvaliddetailed.md)
* [ST_WITHIN](sql-query-st-within.md)
