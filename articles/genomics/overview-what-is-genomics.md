# <a name="what-is-microsoft-genomics"></a>Co to jest Microsoft Genomics?
Microsoft Genomics oferuje implementację w chmurze Burrows-Wheeler align (BWA) i genom Analysis Toolkit (GATK) do analizy pomocniczej. Usługa jest certyfikowana pod kątem zgodności z normą ISO i jest zgodna z przepisami HIPAA i oferuje przewidywalność cen dla potrzeb sekwencjonowania genomu. Dowiedz się, jak używać usługi Microsoft Genomics i zintegrować ją z naszym interfejsem API, czytając nasze Przewodniki Szybki Start, samouczki i dokumentację.

## <a name="support-your-most-demanding-sequencing-needs"></a>Obsługuj najbardziej wymagające potrzeby dotyczące sekwencjonowania
Zamiast zarządzać własnymi centrami danych, skorzystaj z zalet skalowania i doświadczeń firmy Microsoft w uruchamianiu obciążeń liczonych w eksabajtach. Nasze implementacje w chmurze BWA-GATK jest wysoce wydajny w przypadku potoku najlepszych rozwiązań z szeroką praktyką.


## <a name="keep-your-business-running"></a>Utrzymuj swoją działalność w działaniu
Microsoft Genomics oferuje umowę dotyczącą poziomu usług (SLA) dostępności na 99,99% na potrzeby otrzymywania żądań przepływu pracy. Dowiedz się więcej o Microsoft Genomics [umowie SLA](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/).


## <a name="secure-your-data"></a>Zabezpieczanie danych
Usługa Microsoft Genomics to ISO 27001, ISO 27018 i ISO 9001 certyfikowane i zgodne z przepisami HIPAA. Odwiedź [Centrum zaufania Azure](https://www.microsoft.com/trustcenter/security), aby uzyskać informacje o zabezpieczeniach platformy Azure.
