# <a name="reset-your-own-work-or-school-password-overview"></a>Zresetuj własne hasło służbowe — Omówienie

Jeśli nie pamiętasz hasła służbowego, nigdy nie otrzymasz hasła z Twojej organizacji lub nie masz zablokowanego konta, możesz użyć informacji zabezpieczających i urządzenia przenośnego, aby zresetować hasło służbowe.

Administrator musi włączyć tę funkcję, aby móc zarejestrować informacje i zresetować własne hasło. Jeśli nie widzisz opcji nie **pamiętam hasła** , oznacza to, że administrator nie włączył tej funkcji na potrzeby Twojej organizacji. Jeśli uważasz, że jest to nieprawidłowe, skontaktuj się z działem pomocy technicznej w celu uzyskania pomocy.

>[!Important]
>Ta zawartość jest przeznaczona dla użytkowników. Jeśli jesteś administratorem, więcej informacji na temat konfigurowania i zarządzania środowiskiem usługi Azure Active Directory (Azure AD) możesz znaleźć w [dokumentacji usługi Azure Active Directory](https://docs.microsoft.com/azure/active-directory).
