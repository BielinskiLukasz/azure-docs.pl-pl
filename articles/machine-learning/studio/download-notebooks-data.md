# <a name="download-notebookspreview-data"></a>Pobierz dane notesów (wersja zapoznawcza)

**dotyczy:** ![ tak ](../../../includes/media/aml-applies-to-skus/yes.png) Machine Learning Studio (klasyczny) ![ nie](../../../includes/media/aml-applies-to-skus/no.png)[Azure Machine Learning](../compare-azure-ml-to-studio-classic.md)  

Funkcja notesów (wersja zapoznawcza) została usunięta z *kwietnia 13 2020*. Usuwanie z karty notesów (wersja zapoznawcza) i plików notesu użytkownika jest obecnie wdrażany w regionach platformy Azure na całym świecie.

Ten artykuł zawiera instrukcje krok po kroku dotyczące pobierania danych notesów (wersja zapoznawcza).

1. Przejdź do karty **notesy** w Azure Machine Learning Studio (klasyczne).
    ![Przejdź do karty notesy.](./media/download-notebooks/notebooks-list.png)

1. Wybierz Notes, który chcesz pobrać. Zostanie on otwarty w Jupyter.
    ![Wybranie notesu ](./media/download-notebooks/select-notebook.png) pobierającego wiele notesów jednocześnie nie jest obsługiwane.

1. Przejdź do pozycji **pliki**  ->  **Pobierz jako**, a następnie wybierz opcję Format.
    ![Opcje pobierania notesu](./media/download-notebooks/download-options.PNG)

1. Po wybraniu opcji plik notesu rozpocznie się pobieranie. Zignoruj wszystkie błędy 500 lub 502, które są wyświetlane, nie wpłyną na pobieranie.
