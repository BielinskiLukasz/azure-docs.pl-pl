# <a name="overview-of-the-hipaa-hitrust-blueprint-sample"></a>Omówienie przykładu strategii HIPAA HITRUST

Przykład strategii HIPAA HITRUST zawiera zestaw zabezpieczeń w zakresie ładu opracowanych przy użyciu usługi [Azure Policy](../../../policy/overview.md), które ułatwiają zaświadczanie w ramach standardu [HIPAA HITRUST](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Przykład strategii

Ten przykład strategii ułatwia klientom wdrażanie podstawowego zestawu zasad dla dowolnej architektury wdrożonej na platformie Azure wymagającej akredytacji lub zgodności ze strukturą HIPAA HITRUST. Sekcja [mapowania mechanizmów kontroli](./control-mapping.md) zawiera szczegółowe informacje dotyczące zasad w ramach tej inicjatywy i tego, jak te zasady pomagają spełnić wymagania różnych mechanizmów kontroli zdefiniowanych w strukturze HIPAA HITRUST. Po przypisaniu do architektury zasoby są oceniane przez usługę Azure Policy pod względem zgodności z przypisanymi zasadami.
