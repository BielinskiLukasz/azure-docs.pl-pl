# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Jak zarejestrować się do interfejsu API tłumaczenia mowy

Wykonaj następujące kroki, aby się zarejestrować dla interfejsu API tłumaczenia mowy.

## <a name="1-sign-into-azure"></a>1: Zaloguj się do platformy Azure

- Nie masz jeszcze konta? [Załóż konto Microsoft Azure](http://azure.com/)
- Masz już konto? [Rejestrowanie](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: subskrybowanie usługi Microsoft Translator

- Po zalogowaniu się do platformy Azure, przejdź do [usług Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) sekcji
- W obszarze "Typ interfejsu API" Wybierz interfejs API mowy usługi Translator
- W sekcji warstwy cenowej wybierz warstwę cenową, która najlepiej spełnia Twoje potrzeby
    - Każda subskrypcja ma warstwę bezpłatną. Bezpłatna warstwa ma takie same funkcje i funkcje jako naszych płatnych planów i nie ma datę wygaśnięcia.
    - Tylko jedna bezpłatna subskrypcja może mieć na swoim koncie
- Wypełnij pozostałej części formularza, a następnie wybierz przycisk Utwórz
- Subskrypcja jest gotowa do użycia

## <a name="3-authentication-key"></a>3: klucz uwierzytelniania

- Pobierz klucz uwierzytelniania, przechodząc do menu wszystkie zasoby > kliknij w ramach subskrypcji
- Wartość klucza jest używany do uwierzytelniania, należy zachować wartość klucza jest poufny. Potrzebujesz wartości klucza, podczas opracowywania aplikacji

- [Przykłady kodu w serwisie GitHub](https://github.com/MicrosoftTranslator)
- [Forum pomocy technicznej usługi Microsoft Translator](http://www.aka.ms/TranslatorForum)
