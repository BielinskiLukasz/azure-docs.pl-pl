# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Jak skonfigurować serwer Proxy aplikacji aplikacji na używanie PingAccess

Nasza współpraca z PingAccess teraz umożliwia czerpać korzyści z serwera Proxy aplikacji w aplikacji przy użyciu uwierzytelniania opartego na nagłówka. Jeśli aplikacje nie należy używać nagłówków, zobacz nasze [rejestracji jednokrotnej dokumentacji](https://docs.microsoft.com/azure/active-directory/active-directory-application-proxy-sso-using-kcd) szczegółowe informacje na temat innych.

## <a name="overview-of-steps-and-recommended-documents"></a>Omówienie kroków i zalecane dokumentów

Aby skonfigurować aplikację z PingAccess, istnieją cztery kroki:

1.  Skonfiguruj Konektory serwera Proxy aplikacji

2.  Tworzenie aplikacji serwera Proxy aplikacji usługi Azure AD

3.  & Pobierz skonfigurować PingAccess

4.  Konfigurowanie aplikacji w PingAccess

Aby uzyskać szczegółowe informacje na każdym z tych kroków, zobacz nasze [rejestracji jednokrotnej z dokumentacją nagłówki](https://docs.microsoft.com/azure/active-directory/application-proxy-ping-access).
