# <a name="customize-your-azure-security-center-for-iot-solution"></a>Dostosowywanie rozwiązania Usługi Azure Security Center dla IoT

W tym przewodniku dowiesz się, jak dostosować różne ustawienia w usłudze Azure Security Center dla IoT.

> [!div class="checklist"]
> * Konfigurowanie rekomendacji rozwiązań
> * Zmienianie ustawień

## <a name="change-settings"></a>Zmienianie ustawień

Zarządzanie ustawieniem Usługi Azure Security Center dla IoT:

W centrum IoT Hub przejdź do bloku przeglądu zabezpieczeń w lewym górnym rogu, zobacz "ustawienia" Aby skonfigurować usługę Azure Security Center dla ustawień IoT, wykonaj następujące czynności:

1. Otwórz centrum **IoT Hub** w witrynie Azure portal.
1. Z lewego menu w obszarze **Zabezpieczenia**wybierz i otwórz **pozycję Przegląd**.
1. W obszarze Ustawienia wybierz ustawienie rozwiązania, które chcesz zmienić.
1. Pamiętaj, aby zawsze **klikać przycisk Zapisz** u góry dowolnego ekranu ustawień, aby zapisać zmiany ustawień.

## <a name="configure-solution-recommendations"></a>Konfigurowanie rekomendacji rozwiązań

Aby skonfigurować usługę Azure Security Center dla zaleceń dotyczących rozwiązań IoT, wykonaj następujące czynności:

1. Otwórz centrum **IoT Hub** w witrynie Azure portal.
1. Zaznacz i otwórz **pozycję Przegląd** w obszarze **Zabezpieczenia** w menu po lewej stronie.
1. W obszarze Ustawienia wybierz **polecenie Zalecana konfiguracja** z lewego menu.
1. Wyłącz/włącz zalecenia dotyczące rozwiązania odpowiednie dla twojej organizacji i przepływów pracy.
1. Wybierz **pozycję Zapisz** u góry ekranu, aby zapisać wybrane opcje.
