# <a name="scenario-hdp-upgrade-causes-no-data-in-apache-phoenix-views-in-azure-hdinsight"></a>Scenariusz: uaktualnienie HDP powoduje, że nie ma danych w widokach Apache Phoenix w usłudze Azure HDInsight

W tym artykule opisano kroki rozwiązywania problemów i możliwe rozwiązania problemów podczas interakcji z klastrami usługi Azure HDInsight.

## <a name="issue"></a>Problem

Widok Apache Phoenix nie zawiera daty po uaktualnieniu z HDP 2.4 do HDP 2.5.

## <a name="cause"></a>Przyczyna

Tabela indeksów widoków (wszystkie indeksy dla widoku są przechowywane w jednej fizycznej tabeli Apache HBase) jest obcinana podczas uaktualniania

## <a name="resolution"></a>Rozwiązanie

Upuść i ponownie utworzyć wszystkie indeksy widoku po uaktualnieniu.
