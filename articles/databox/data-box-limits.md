## <a name="data-box-limits"></a>Limity pól danych

- Data Box może przechowywać maksymalnie 500 milionów plików.
- Data Box obsługuje maksymalnie 512 kontenerów lub udziałów w chmurze. Katalogi najwyższego poziomu w ramach udziału użytkownika stają się kontenerami lub udziałami plików platformy Azure w chmurze.
