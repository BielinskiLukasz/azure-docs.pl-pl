

新しいモバイル サービスを作成するには、次のステップに従います。

1.  ログイン、 [Azure クラシック ポータル](https://manage.windowsazure.com/)します。 ナビゲーション ウィンドウの下部にあるをクリックして **+ 新規**します。 展開 **コンピューティング** と **Mobile Service**, 、順にクリックして **作成**します。

    ![](./media/mobile-services-create-new-service/mobile-create.png)

    これを表示、 **モバイル サービスを作成する** ダイアログ。

2.   **モバイル サービスを作成する** ダイアログで、 **無料の 20 MB SQL データベースの作成**, を選択 **JavaScript** で新しいモバイル サービスの名前を実行時、サブドメインの種類、 **URL** ] ボックスに貼り付けます。 右矢印ボタンをクリックして、次のページに進みます。

    ![](./media/mobile-services-create-new-service/mobile-create-page1.png)

    これを表示、 **データベース設定の指定** ページです。
    
    >[AZURE.NOTE]このチュートリアルの一環として、新しい SQL データベース インスタンスとサーバーを作成します。 この新しいデータベースは、再利用したり、その他の SQL データベース インスタンスと同様に管理したりすることができます。 既に、新しいモバイル サービスと同じリージョンにデータベースがある場合を選択した **既存のデータベースを使用して** しそのデータベースを選択します。 別のリージョンにあるデータベースを使用することは、帯域幅コストと待機時間が増加するため、お勧めしません。

3.   **名**, 、名前を入力の新しいデータベースを入力し、 **ログイン名**, 、管理者のログインであるという名前を次のように新しい SQL データベース サーバーの種類と、パスワードの確認し、プロセスを完了するチェック ボタンをクリックします。
    ![](./media/mobile-services-create-new-service/mobile-create-page2.png)

これで、モバイル アプリケーションで使用できる新しいモバイル サービスが作成されました。
