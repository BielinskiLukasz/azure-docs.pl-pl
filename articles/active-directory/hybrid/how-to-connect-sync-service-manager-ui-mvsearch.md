# <a name="sync-service-manager-metaverse-search"></a>Wyszukiwanie Metaverse Menedżera usługi synchronizacji

![Menedżer usługi synchronizacji](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Karta wyszukiwania metaverse jest przydatna do rozwiązywania problemów związanych z danymi. W górnej połowie można utworzyć kwerendę na podstawie kombinacji atrybutów. Po zakończeniu kwerendy kliknij przycisk **Wyszukaj**. Wynik jest widoczny w dolnej siatce. Można wybrać kolumny, które mają być widoczne w **ustawieniach kolumn**.

W wynikach wyszukiwania zaznacz obiekt i **właściwości,** aby wyświetlić [właściwości obiektu metaverse](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
