# <a name="api-reference"></a>Dokumentacja interfejsu API

Interfejs API rozpoznawania twarzy Azure jest oparta na chmurze interfejs API, który udostępnia algorytmy wykrywania twarzy i rozpoznawania. Interfejsy API rozpoznawania twarzy obejmują następujące kategorie:

- [Face API algorytm](https://docs.microsoft.com/rest/api/cognitiveservices/face/face): obejmuje podstawowe funkcje, takie jak [wykrywania](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/detectwithstream), [Znajdź podobne](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar), [weryfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/verifyfacetoface), [identyfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify), i [grupy](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/group).
- [Interfejsy API zarządzania FaceList](https://docs.microsoft.com/rest/api/cognitiveservices/face/facelist): używane do zarządzania FaceList dla [Znajdź podobne](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [Interfejsy API zarządzania osoby LargePersonGroup](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroupperson): używane do zarządzania LargePersonGroup twarzy osób [identyfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [Interfejsy API zarządzania LargePersonGroup](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroup): używane do zarządzania zestawu danych LargePersonGroup [identyfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [Interfejsy API zarządzania LargeFaceList](https://docs.microsoft.com/rest/api/cognitiveservices/face/largefacelist): używane do zarządzania LargeFaceList dla [Znajdź podobne](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [Interfejsy API zarządzania osoby grupie](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroupperson): używane do zarządzania w grupie twarzy osób [identyfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [Interfejsy API zarządzania w grupie](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroup): używane do zarządzania zestawu danych grupie [identyfikacji](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
