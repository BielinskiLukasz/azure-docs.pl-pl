# <a name="azure-status"></a>Stan platformy Azure
[Stan Azure](https://azure.microsoft.com/status/) oferuje globalne widok kondycji usług platformy Azure. O stanie Azure możesz uzyskać najbardziej aktualnych informacji o dostępności usługi. Wszyscy ma dostęp do stanu platformy Azure i widok wszystkich usług zgłosić ich kondycję usługi. 

![Strony stanu platformy Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Aktualizacje stanu platformy Azure
Na stronie stanu Azure jest aktualizowana w czasie rzeczywistym jako kondycji usług platformy Azure zmiany. Pozostawienie otwartych strony stanu platformy Azure można kontrolować szybkość odświeżeniu strony nowymi danymi. U góry spowoduje jej ostatniej strony zostało zaktualizowane.

![Odśwież stan Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Historia stanu platformy Azure
Gdy na stronie stanu Azure zawsze zawiera najnowsze informacje o kondycji, można wyświetlić starsze zdarzenia przy użyciu [stronie historii stanu Azure](https://azure.microsoft.com/status/history/).

Można przejrzeć zdarzenia się do 90 dni w przeszłości, w tym wstępne przyczynę, ograniczenie i kolejne kroki dla platformy Azure.

## <a name="rss-feed"></a>Źródło danych RSS
Stan Azure udostępnia również [źródła danych RSS](https://azure.microsoft.com/status/feed/) zmian kondycji usług platformy Azure, które można subskrybować.
