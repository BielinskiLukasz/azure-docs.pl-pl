# <a name="azure-maps-community---open-source-projects"></a>Społeczność usługi Azure Maps — projekty typu open source

Te inicjatywy oparte na otwartej społeczności są tworzone i obsługiwane przez zespół usługi Azure Maps. Nie są one częścią standardowej oferty produktów lub usług.

Poniżej wymieniono niektóre z najbardziej popularnych projektów i przykładów usługi Azure Maps typu open source.

| Nazwa projektu | Opis |
|-|-|
| [Przykłady kodu usługi Azure Maps](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Kolekcja przykładów kodu do korzystania z usługi Azure Maps w aplikacjach internetowych. |
| [Przykłady kodu w chmurze usługi Azure Maps Gov](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Kolekcja przykładów kodu do korzystania z usługi Azure Maps za pośrednictwem usługi Azure Government Cloud. |
| [Przykłady usługi Azure Maps & usługi Azure Active Directory](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Kolekcja przykładów, które pokazują, jak korzystać z usługi Azure Active Directory z usługą Azure Maps. | 
| [Wdrażanie analizy przestrzennej IoT przy użyciu usługi Azure Maps](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | Śledzenie i przechwytywanie istotnych zdarzeń, które występują w przestrzeni i czasie jest typowy scenariusz IoT. |
| [Przykłady notesu jupytera usługi Azure Maps](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Kolekcja przykładów języka python przy użyciu usług REST usługi Azure Maps. |
| [Bot Framework - Umiejętność punktu zainteresowania](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | Umiejętność punktu zainteresowania zapewnia możliwości związane z poi do wirtualnego asystenta przy użyciu usługi Azure Maps z usługą Azure Bot Service i Bot Framework. |

Dłuższa lista zawierająca projekty typu open source utworzone przez społeczność jest dostępna [tutaj](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)

## <a name="supportability-of-open-source-projects"></a>Wsparcie projektów typu open source

Następujące instrukcje mają zastosowanie we wszystkich mapach azure utworzonych i obsługiwanych projektów i przykładów typu open source:

- Projekty i przykłady usługi Azure Maps typu open source są tworzone przez firmę Microsoft i społeczność.
- Projekty i przykłady usługi Azure Maps typu open source są obsługiwane przez firmę Microsoft i społeczność.
- Projekty i przykłady usługi Azure Maps typu open source używają obsługiwanych i zalecanych technik.
- Projekty i próbki usługi Azure Maps typu open source to inicjatywa społecznościowa — osoby, które pracują nad inicjatywą z korzyścią dla innych osób i mają również normalną pracę dzienną.
- Projekty i przykłady usługi Azure Maps typu open source NIE są produktem i dlatego nie są obsługiwane przez pomoc techniczną Premier ani inne oficjalne kanały pomocy technicznej.
- Projekty i przykłady usługi Azure Maps typu open source są obsługiwane w podobny sposób, jak inne projekty typu open source wykonywane przez firmę Microsoft przy wsparciu społeczności przez społeczność.
