# Get-AzureRmFrontDoor
Pobierz konfigurację drzwi

## SKŁADNIA

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## OPIS
**Get AzureRmFrontDoor** cmdletGet pobiera wszystkie istniejące drzwi frontonu w bieżącej subskrypcji, te informacje pod warunkiem dopasowania.
