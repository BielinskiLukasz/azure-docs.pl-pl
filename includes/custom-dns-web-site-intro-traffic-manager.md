Gdy usługa Microsoft Azure Traffic Manager umożliwia równoważenie obciążenia ruchem do witryny sieci Web platformy Azure, tej witryny sieci Web mają dostęp za pomocą  **\*. trafficmanager.net** nazwy domeny przypisany przez platformę Azure. Niestandardową nazwę domeny, np. www.contoso.com, można również skojarzyć z witryny sieci Web w celu zapewnienia bardziej rozpoznawalną nazwę domeny dla użytkowników.
