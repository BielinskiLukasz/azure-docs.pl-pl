# <a name="manage-your-personal-application-preferences"></a>Zarządzanie osobistymi preferencjami aplikacji

*Ten artykuł dotyczy operatorów, konstruktorów i administratorów.*

IoT Central zapewnia elastyczność dostosowywania aplikacji do twoich potrzeb. Zapewniamy również pewną elastyczność dla użytkownika, aby dostosować własny widok. W tym artykule opisano różne opcje dostosowywania, które użytkownik może zastosować do swojego profilu.

## <a name="changing-language"></a>Zmiana języka

IoT Central jest obsługiwany w wielu językach. Preferowany język można zmienić za pomocą **selektora języka** na ikonie ustawień na górnym pasku nawigacyjnym. Po zmianie języka IoT Central zapamiętuje twój wybór i zastosuje go we wszystkich aplikacjach. Dostosowywanie w aplikacji takie obrazy pulpitu nawigacyjnego nie są zlokalizowane.

## <a name="changing-theme"></a>Zmiana motywu

Mamy wsparcie zarówno dla ciemnego tematu, jak i jasnego motywu. Podczas gdy motyw świetlny jest domyślny, można zmienić motyw, wybierając ikonę ustawień na górnym pasku nawigacyjnym.

![Selektor motywów IoT Central](media/howto-manage-preferences/settings.png)

> [!NOTE]
> Opcja wyboru między jasnymi i ciemnymi motywami nie jest dostępna, jeśli administrator skonfigurował niestandardowy motyw dla aplikacji.
