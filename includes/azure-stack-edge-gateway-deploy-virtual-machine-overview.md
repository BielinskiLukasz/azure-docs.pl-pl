Można tworzyć maszyny wirtualne i zarządzać nimi na Azure Stack urządzeniu brzegowym przy użyciu interfejsów API. Te interfejsy API są standardowymi interfejsami API Azure Resource Manager, które są wywoływane przy użyciu lokalnego punktu końcowego Azure Stack Edge. Azure Resource Manager interfejsy API zapewniają spójną warstwę zarządzania, która w tym przypadku umożliwia tworzenie, aktualizowanie i usuwanie maszyn wirtualnych w lokalnej subskrypcji, która istnieje na urządzeniu. Możesz połączyć się z Azure Resource Manager uruchomionym na Azure Stack Edge za pomocą poleceń cmdlet Azure PowerShell.
