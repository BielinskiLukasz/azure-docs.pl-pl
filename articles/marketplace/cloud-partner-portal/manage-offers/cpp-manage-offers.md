# <a name="manage-azure-and-appsource-marketplace-offers"></a>Zarządzanie ofertami platformy Azure i appsource marketplace

W tej sekcji przedstawiono ogólne operacje używane do zarządzania ofertami dla platformy Azure i witryn AppSource Marketplaces.  Na przykład wyjaśniono, jak za pomocą [portalu cloud partner do](https://cloudpartner.azure.com/) tworzenia i publikowania oferty marketplace.  W tej sekcji wyjaśniono również problemy techniczne, które są wspólne dla większości typów ofert, takich jak zasoby tekstowe i graficzne.


## <a name="standard-offer-operations"></a>Standardowe operacje ofertowe

Portal cloud partner umożliwia wydawcy wykonywanie następujących podstawowych operacji w ofercie.

|     Operacja      |  Opis                                           |
|     ---------      |  -----------                                           |
| [Tworzenie oferty](./cpp-create-offer.md)   | Tworzy oczekujący wpis oferty w repozytorium dołączania, ale nie publikuje go automatycznie w portalu Microsoft Marketplace | 
| [Wyświetlanie stanu oferty](./cpp-view-status-offer.md)   | Zapewnia stan publikowania oferty.  W portalu znajduje się wiele miejsc, w których są wyświetlane informacje o stanie. |
| [Publikowanie oferty](./cpp-publish-offer.md) | Sprawdza poprawność, a następnie księguje wypełniony wpis oferty w odpowiednim portalu microsoft marketplace.  Oferta jest następnie mówi *się, że na żywo*. |
| [Aktualizowanie oferty](./cpp-update-offer.md)   | Umożliwia modyfikację niektórych cech istniejącej oferty.  Dotyczy zarówno ofert oczekujących, jak i opublikowanych. |
| [Usuwanie oferty](./cpp-delete-offer.md)   | Usuwa istniejącą ofertę oczekującą lub na żywo.  | 
|  |  |
