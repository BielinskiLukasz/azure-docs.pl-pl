Poniższe limity mają zastosowanie tylko wtedy, gdy wykonywać operacje zarządzania za pomocą usługi Azure Resource Manager z usługą Azure Storage.

| Resource | Limit domyślny |
| --- | --- |
| Operacje zarządzania kontem magazynu (odczyt) |800 na 5 minut |
| Operacje zarządzania kontem magazynu (zapis) |200 za godzinę |
| Operacje zarządzania kontem magazynu (lista) |100 na 5 minut |
