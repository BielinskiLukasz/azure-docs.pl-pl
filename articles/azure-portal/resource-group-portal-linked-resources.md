# タイル ギャラリーの関連リソースとリンク済みリソース

タイル ギャラリーを使用すると、特定のリソースのタイルを見つけて、現在のブレードにドラッグできます。 
タイル ギャラリーを使用して、リソースにまたがる管理ビューを作成できます。

## Azure リソース マネージャーのリンク済みリソース

リンクは、Azure リソース マネージャーの機能です。 間のリレーションシップを宣言することができます。

## リソースをリンクする方法

リンクは、ポータルまたは Azure PowerShell または Azure CLI からテンプレートを展開することでリソースを作成するときに 
いくつかの依存リソースが自動的に作成します。 プログラムによってを使用してリソースをリンクすることができます、 
[リンクされたリソースの REST API](https://msdn.microsoft.com/library/azure/mt238499.aspx) テンプレート内のリレーションシップを宣言します。
