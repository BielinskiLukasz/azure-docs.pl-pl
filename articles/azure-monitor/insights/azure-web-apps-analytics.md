# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Wyświetlanie danych analitycznych dla metryk we wszystkich zasobach aplikacji Azure Web App

![Symbol aplikacji sieci Web](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Rozwiązanie usługi Azure Web App Analytics zostało przeniesione do pomocy technicznej społeczności.  
>- Rozwiązanie nie jest już dostępne w portalu Azure Marketplace, ale można je zainstalować za pomocą [szablonów szybki startu platformy Azure,](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) gdzie jest obsługiwane przez społeczność.
>- Klienci, którzy już zainstalowali rozwiązanie, mogą nadal używać go ze zmianami.  
>- Firma Microsoft zaleca monitorowanie aplikacji za pomocą [usługi Application Insights.](../../azure-monitor/app/app-insights-overview.md)
