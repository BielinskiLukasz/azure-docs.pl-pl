# <a name="getting-support-for-b2b-collaboration"></a>Uzyskiwanie wsparcia dla współpracy B2B

Przeczytałeś dokumentację, zrobiłeś właściwe rzeczy, ale nadal nie możesz dostać czegoś do pracy? Otwórz bilet pomocy technicznej (wymaga planu pomocy technicznej):

1. W [witrynie Azure portal](https://portal.azure.com)przejdź do bloku Pomoc i pomoc techniczna i wybierz pozycję Nowe żądanie pomocy technicznej:
   - Typ problemu: Techniczny
   - Subskrypcja: wybierz subskrypcję, na która ma wpływ
     - Usługa: Active Directory
     - Plan pomocy technicznej: wybierz odpowiedni plan pomocy technicznej

   ![Zrzut ekranu przedstawiający nowe żądanie pomocy technicznej](media/get-support/new-support-request.png)

2. Opisz swój problem:
   - Wybierz odpowiednią ważność, która odzwierciedla potrzebne.
   - Wybieranie typu problemu jako zarządzania użytkownikami i grupami
   - Wybierz kategorię jako dodawanie użytkowników (B2B)
   - Dołącz wszystkie komunikaty o błędach, takie jak CorrelationID, użytkownicy, których dotyczy problem, i tak dalej.

   ![Zrzut ekranu przedstawiający opis problemu z biletem pomocy technicznej](media/get-support/problem-description.png)

3. Aby przedstawiciel pomocy technicznej skontaktował się z Tobą w celu dalszego rozwiązywania problemów, dodaj informacje kontaktowe.
