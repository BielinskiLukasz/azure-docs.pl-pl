# <a name="view-and-change-your-cloud-solution-provider-reseller-incentive-in-the-azure-marketplace"></a>Wyświetlanie i zmiana programu Cloud Solution Provider "Odsprzedawcy Incentive" w portalu Azure Marketplace
15 marca 2016 r wybierzemy dostawców rozwiązań w chmurze (CSP) "Odsprzedawcy Incentive" dostępne dla wydawców Marketplace.  Domyślne ustawienie dla tej wartości motywacji **tak**.  Jako wydawca można zdecydować nie wyrazić zgody motywacji program.  W tym artykule pokazano, jak zmienić ustawienie profilu wydawcy, więc użytkownik może zrezygnować.

Aby wyświetlić lub zmienić swój stan "Odsprzedawcy Incentive", wykonaj następujące kroki:

1. Zaloguj się do Twojego [konto Microsoft Azure publikowania](https://publish.windowsazure.com/workspace)

   ![Profil docelowa ekranu][1]
2. Kliknij przycisk **wydawcy**

   ![Link wydawców ekranu docelowa profilu —][2]
3. Zwróć uwagę, informacje dotyczące dostawców usług kryptograficznych w Twoim profilu.  Brak **przełączyć się do powiedz tak lub nie** "Incentive odsprzedawcy" - domyślnego stanu jest "Tak".

   ![Profil - wydawcy][3]
4. Do zmiany z "Incentive odsprzedawcy" stan dostępu do Twojego profilu wydawcy, a następnie wybierz i Zapisz wybraną:

   * Wybierz **tak** wybrać zapewnienie rabatem 20% do dostawcy usług kryptograficznych, którzy są odsprzedaż aplikacje i usługi zostały wymienione w portalu Azure Marketplace.  Lub wybierz **nr** rezygnacji z tego programu motywacji.  W obu przypadkach wydawcy odbiera wypłaty tej samej.
   * Po dokonaniu wyboru upewnij się, że kliknij **zapisać** w prawym górnym rogu.

   ![Profil - wydawców — zmiany pola][4]
