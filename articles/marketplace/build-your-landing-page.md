# <a name="build-your-landing-page"></a>Zbuduj stronę docelową

Strona docelowa w witrynie firmy to brama od działań związanych z generowaniem popytu do aukcji w [portalu Azure Marketplace.](https://azuremarketplace.microsoft.com)

Zacznij od określenia swoich celów. W przypadku rozwiązań wybierz klienta docelowego i działania, które podejmie twój klient. Rozwiązanie może obejmować następujące akcje:
*   Test-drive rozwiązanie
*   Uzyskaj rozwiązanie już teraz

Możesz odwoływać się do strony docelowej w wielu taktykach marketingowych. Możesz skorzystać z następujących taktyk marketingowych: 
*   Zdarzenia
*   Audycje w sieci Web
*   Media społecznościowe
*   Oficjalne dokumenty
*   Szkolenia techniczne
*   Informacje prasowe

Im bardziej spójne jesteś w wiadomościach i wezwaniu do działania, tym łatwiej jest klientom znaleźć rozwiązanie.

## <a name="recommendations"></a>Zalecenia

Gdy masz gotowy plan kampanii, utwórz stronę docelową w Marketplace. W poniższej tabeli opisano sposoby maksymalizacji skuteczności strony docelowej: 

| Najlepsze rozwiązania | Unikaj |
|:--- |:--- |
| **Informuj o tym, jaki problem klienta rozwiązuje twoje rozwiązanie i jak można wykorzystać platformę Azure, aby to osiągnąć** | Nie komunikuje się ze sposobami, w jakie rozwiązanie współpracuje z platformą Azure. |
| **Tworzenie krótkiego, łatwego do zapamiętania adresu URL** | Korzystanie z długich adresów URL. Długie adresy URL nie są zapadawające w pamięć i trudno je zlokalizować. |
| **Dodawanie odpowiednich treści wizualnych, takich jak wideo z referencjami klientów lub architektura rozwiązania** | Używanie zbyt dużej ilości tekstu powoduje, że klienci ignorują zawartość i przestają eksplorować.|
| **Tworzenie wyraźnego wezwania do działania w celu kierowania odwiedzających do strony katalogu marketplace** | Mając zbyt wiele linków lub działań na stronie. |
| **Umieść wezwanie do działania w nagłówku lub wyraźnie wytyczonej sekcji** | Osadzanie linków do aukcji w Marketplace w akapicie tekstu. |
| **Zbadaj najlepsze słowa kluczowe i zoptymalizuj stronę pod kątem wyszukiwania** | Zakładając, że nazwa produktu generuje wyższe rankingi wyszukiwania. |
| **Używanie słów kluczowych z kampanii reklamowych** | Używanie wielu różnych słów kluczowych w swoich usługach internetowych osłabia inwestycje reklamowe. |
| **Umieść odpowiednie nazwy produktów i słowa kluczowe "nad fałdem"** | Dokonywanie użytkownicy przewijania, aby zobaczyć produkt lub rozwiązanie, które prezentujesz. |
| **Użyj obrazów marki, aby zweryfikować rozwiązanie zgodnie z wytycznymi firmy Microsoft dotyczącymi znakowania, takimi jak certyfikat platformy Azure** | Korzystanie z obrazów marki Microsoft, które nie zostały zatwierdzone. |
