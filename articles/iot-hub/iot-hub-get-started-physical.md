# <a name="azure-iot-hub-get-started-with-physical-devices-tutorials"></a>Usługa Azure IoT Hub wprowadzenie do samouczków dotyczących urządzeń fizycznych

Te samouczki przedstawiają usługi Azure IoT Hub i zestawów SDK urządzeń. Samouczki obejmują typowe scenariusze IoT, aby zademonstrować możliwości usługi IoT Hub. Samouczki ilustrują również sposób łączenia usługi IoT Hub z innymi usługami i narzędziami platformy Azure w celu tworzenia bardziej zaawansowanych rozwiązań IoT. Samouczki wymienione w poniższej tabeli pokazują, jak utworzyć fizyczne urządzenia IoT.

| Urządzenie IoT                       | Język programowania |
|---------------------------------|----------------------|
| Raspberry Pi                    | [Node.js](iot-hub-raspberry-pi-kit-node-get-started.md), [C](iot-hub-raspberry-pi-kit-c-get-started.md)  |
| Zestaw devektu IoT                      | [Arduino w vscode](iot-hub-arduino-iot-devkit-az3166-get-started.md)     |
