# <a name="discover-jupyter-notebooks-in-the-cortana-intelligence-gallery"></a>Odnajdywanie notesów Jupyter w Cortana Intelligence Gallery
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="jupyter-notebooks"></a>Notesy programu Jupyter
**[Notesów Jupyter](https://gallery.cortanaintelligence.com/notebooks)**  obejmują kod, wizualizacje danych i dokumentacji na kanwie jednej, interaktywnego.
Notesów w galerii Podaj samouczki i szczegółowe wyjaśnienia zaawansowane machine learning technik i rozwiązania.

## <a name="discover"></a>Wykrywanie
  Aby wyszukać notesów w galerii, otwórz [galerii](http://gallery.cortanaintelligence.com) i kliknij przycisk **notesów** w górnej części strony głównej galerii.

 **[Notesów Jupyter](https://gallery.cortanaintelligence.com/notebooks)**  zostanie wyświetlona lista najpopularniejszych notesów.
Kliknij przycisk **zobaczyć wszystkie** Aby wyświetlić wszystkie notesy.
Na tej stronie można przejrzeć wszystkie notesy w galerii. Można również przeszukać wybierając kryteria filtrowania po lewej stronie i wprowadzanie terminy wyszukiwania w górnej.

 Kliknij przycisk żadnych notesu, aby otworzyć stronę szczegółów notesu i uzyskania bardziej szczegółowych informacji. Na tej stronie można komentarz, wyrazić swoją opinię lub zadać pytania za pośrednictwem sekcji komentarzy. Można nawet udostępniać znajomym lub współpracowników za pomocą funkcji udziału LinkedIn i Twitter. Można również e-mail łącze do notesu z zaproszeniem dla innych użytkowników, aby wyświetlić stronę.

## <a name="download"></a>Do pobrania
Możesz pobrać kopię każdego dostępnego z galerii do obszaru roboczego usługi Machine Learning Studio.

### <a name="download-from-the-gallery"></a>Pobierz z galerii
Aby pobrać kopię Notes z galerii:

1. Otwórz stronę szczegółów notesu w galerii
2. Kliknij przycisk **Otwórz w Studio**
3. Wybierz Region i chcesz użyć obszaru roboczego
   
    ![Otwieranie notesu Jupyter z galerii](./media/gallery-jupyter-notebooks/open-notebook-from-gallery.png)

Po kliknięciu **Otwórz w Studio**, notesu jest ładowany do obszaru roboczego usługi Machine Learning Studio i otworzyć (jeśli jeszcze nie zalogowano Cię Studio, zostanie wyświetlony monit zalogowanie się przy użyciu konta Microsoft, zanim notesu jest kopiowany do swojego obszaru roboczego). Znajdź notesu ponownie później, klikając **notesów** w lewej części strony głównej Studio.

### <a name="download-in-machine-learning-studio"></a>Pobierz w usłudze Machine Learning Studio
Możesz również otworzyć dowolne z tym samym notesów galerii podczas pracy w usłudze Machine Learning Studio:

1. W usłudze Machine Learning Studio, kliknij przycisk **+ nowy**
2. Wybierz **notesu** — zostanie wyświetlona lista notebooków z galerii, można wybrać, czy można znaleźć określonego notesu przy użyciu pola wyszukiwania
3. Wskaż przy użyciu myszy notesu i wybierz **Otwórz notes**
   
    ![Otwórz galerię notesu z wewnątrz usługi Machine Learning Studio](./media/gallery-jupyter-notebooks/open-notebook-from-studio.png)

Kopię notesu jest pobierane i otworzyć w obszarze roboczym w sekcji notesów Jupyter w usłudze Machine Learning Studio.
Notesu, zostaną wyświetlone, wraz z notesów na **notesów** strony Studio (na [strony głównej Studio](https://studio.azureml.net/), kliknij przycisk **notesów** po lewej stronie).
