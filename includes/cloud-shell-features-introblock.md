Usługa Azure Cloud Shell to oparte na przeglądarce środowisko powłoki do tworzenia zasobów platformy Azure i zarządzania nimi.

Usługa Cloud Shell oferuje dostępne dla przeglądarki, wstępnie skonfigurowane środowisko powłoki do zarządzania zasobami platformy Azure bez konieczności samodzielnego instalowania, przechowywania wersji i utrzymywania komputera.

Cloud Shell aprobuje maszyny na podstawie żądania i w rezultacie stan komputera nie będzie się utrzymywać w sesjach. Ponieważ usługa Cloud Shell jest przeznaczona do sesji interaktywnych, powłoki automatycznie kończą się po 20 minutach braku aktywności powłoki.
