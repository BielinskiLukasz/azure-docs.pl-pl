# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Informacje o wersji platformy Azure krawędź pola danych i Azure Data Box bramy 1905

## <a name="overview"></a>Omówienie

W poniższych informacjach o Zidentyfikuj krytyczne nierozwiązane problemy i rozwiązane problemy w wersji 1905 dla usługi Azure Data Box Edge i bramy pola danych platformy Azure.

Informacje o wersji są ciągle aktualizowane, a ponieważ są wszystkie wykrywane krytyczne problemy wymagające obejścia są dodawane. Przed wdrożeniem bramy okno usługi Edge i dane pole danych, należy uważnie przeczytaj informacje zawarte w informacjach o wersji.

Ta wersja odpowiada wersji oprogramowania:

- **Brama pola danych 1905 (1.6.887.626)**
- **Data Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Aktualizację 1905 można zastosować tylko do urządzeń krawędź pola danych, systemem w wersji ogólnie dostępnej wersji oprogramowania.

## <a name="whats-new"></a>Co nowego

- **Ulepszenia logowania pole programowalny bramy tablicy (FPGA)** — w tej wersji wprowadzono rejestrowania i ulepszenia alertów związanych z FPGA. Jeśli używasz funkcji obliczeń brzegowych za pomocą FPGA jest wymagana aktualizacja do krawędzi ramki danych. Aby uzyskać więcej informacji, zobacz instrukcje [Przekształcanie danych za pomocą obliczeń brzegowych na krawędzi sieci pole danych](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Znane problemy w wersji ogólnie dostępnej wersji

Nie nowych problemów są wydania wymienionych w tej wersji. Wszystkie wersji zauważyć, że problemy zostały przeniesione z poprzednich wersji. Aby wyświetlić listę znanych problemów, przejdź do [znane problemy w wersji ogólnie dostępnej](data-box-gateway-release-notes.md#known-issues-in-ga-release).
