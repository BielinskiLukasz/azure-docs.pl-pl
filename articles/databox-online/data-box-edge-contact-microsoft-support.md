# <a name="open-a-support-ticket-for-azure-data-box-edge-and-azure-data-box-gateway"></a>Otwieranie biletu pomocy technicznej dla usługi Azure Data Box Edge i bramy usługi Azure Data Box

Ten artykuł dotyczy usługi Azure Data Box Edge i bramy usługi Azure Data Box Gateway, które są zarządzane przez usługę Azure Data Box Edge/Azure Data Box Gateway. Jeśli wystąpią jakiekolwiek problemy z usługą, możesz utworzyć żądanie usługi dla pomocy technicznej. W tym artykule otrzymasz:

* Jak utworzyć żądanie pomocy technicznej.
* Jak zarządzać cyklem życia żądania pomocy technicznej z poziomu portalu.

## <a name="create-a-support-request"></a>Tworzenie żądania obsługi

Wykonaj następujące kroki, aby utworzyć żądanie pomocy technicznej:

1. Przejdź do kolejności data box edge lub data box gateway. Przejdź do sekcji **Pomoc techniczna + rozwiązywanie problemów,** a następnie wybierz pozycję **Nowe żądanie pomocy technicznej**.
   
2. W **obszarze Nowe żądanie pomocy technicznej**na karcie **Podstawy** należy wykonać następujące czynności:
    
    1. Z listy rozwijanej **Typ problemu** wybierz pozycję **Techniczne**.
    2. Wybierz **subskrypcję**.
    3. W obszarze **Usługa**sprawdź **moje usługi**. Z listy rozwijanej wybierz pozycję **Data Box Edge i Data Box Gateway**.
    4. Wybierz **zasób**. Odpowiada to nazwie zamówienia.
    5. Podaj krótkie **podsumowanie** występującego problemu. 
    6. Wybierz **typ problemu**.
    7. Na podstawie wybranego typu problemu wybierz odpowiedni **podtyp Problem**.
    8. Wybierz **dalej: Rozwiązania >>**.

        ![Podstawy](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)

3. Na karcie **Szczegóły** należy wykonać następujące czynności:
    
    1. Podaj datę i godzinę rozpoczęcia problemu.
    2. Podaj **opis** problemu.
    3. W **folderze Przekazywanie plików**wybierz ikonę folderu, aby przeglądać inne pliki, które chcesz przekazać.
    4. Sprawdź **informacje diagnostyczne udostępniania**.
    5. Na podstawie subskrypcji **plan pomocy technicznej** jest automatycznie wypełniany.
    6. Z listy rozwijanej wybierz pozycję **Ważność**.
    7. Określ **preferowaną metodę kontaktu**.
    8. **Godziny odpowiedzi** są wybierane automatycznie na podstawie planu subskrypcji.
    9. Podaj język preferowany przez twoją preferencję w zakresie pomocy technicznej.
    10. W **informacjach kontaktowych**podaj swoje imię i nazwisko, adres e-mail, telefon, opcjonalny kontakt, kraj/region. Pomoc techniczna firmy Microsoft korzysta z tych informacji, aby uzyskać dalsze informacje, diagnozę i rozwiązanie problemu. 
    11. Wybierz **dalej: Recenzja + Utwórz >>**.

        ![Problem](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-2.png)

4. Na przegląd **+ Tworzenie** kartę, przejrzyj informacje związane z biletem pomocy technicznej. Wybierz **pozycję Utwórz**. 

    ![Problem](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-3.png)

    Po utworzeniu biletu pomocy technicznej, inżynier pomocy technicznej skontaktuje się z Tobą tak szybko, jak to możliwe, aby kontynuować twoją prośbę.

## <a name="get-hardware-support"></a>Uzyskaj pomoc techniczną dla sprzętu

Te informacje dotyczą tylko urządzenia Data Box Edge. Proces zgłaszania problemów ze sprzętem jest następujący:

1. Otwórz bilet pomocy technicznej w witrynie Azure portal dla problemu ze sprzętem. W obszarze **Typ problemu**wybierz pozycję Sprzęt krawędzi **pola danych**. Wybierz **podtyp Problem** jako **błąd sprzętu**. 

    ![Problem ze sprzętem](./media/data-box-edge-contact-microsoft-support/data-box-edge-hardware-issue-1.png)

    Po utworzeniu biletu pomocy technicznej, inżynier pomocy technicznej skontaktuje się z Tobą tak szybko, jak to możliwe, aby kontynuować twoją prośbę. 

2. Jeśli pomoc techniczna firmy Microsoft stwierdzi, że jest to problem ze sprzętem, występuje jedna z następujących czynności: 

    - Wysyłana jest jednostka wymiany pola (FRU) dla nieudanej części sprzętowej. Obecnie zasilacz jest jedynym obsługiwanym FRU. 
    - W przypadku awarii innych części firma Microsoft wykonuje pełną wymianę systemu (FSR) lub zamianę urządzenia.

3. Jeśli bilet pomocy technicznej zostanie podniesiony przed godziną 16:30 czasu lokalnego (od poniedziałku do piątku), następnego dnia roboczego do lokalizacji zostanie wysłany technik na miejscu, aby wykonać FRU lub pełną wymianę urządzenia.

## <a name="manage-a-support-request"></a>Zarządzanie żądaniem pomocy technicznej

Po utworzeniu biletu pomocy technicznej możesz zarządzać jego cyklem życia z poziomu portalu.

#### <a name="to-manage-your-support-requests"></a>Aby zarządzać żądaniami pomocy technicznej

1. Aby przejść do strony pomocy i pomocy technicznej, przejdź do **sekcji Przeglądaj > Pomoc + pomoc + pomoc .**

    ![Zarządzanie żądaniami pomocy technicznej](./media/data-box-edge-contact-microsoft-support/data-box-edge-manage-support-request-1.png)   

2. Tabelaryczny wykaz **ostatnich żądań pomocy technicznej** jest wyświetlany w Pomocy + pomocy **technicznej**.

    <!--[Manage support requests](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)--> 

3. Wybierz i kliknij żądanie pomocy technicznej. Można wyświetlić stan i szczegóły tego żądania. Kliknij **+ Nowa wiadomość,** jeśli chcesz wykonać następujące żądanie.
