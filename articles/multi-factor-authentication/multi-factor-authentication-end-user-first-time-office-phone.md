# Azure Multi-Factor Authentication での連絡方法として会社電話を使用する

主要な連絡方法として会社電話を使用する場合は、この記事を使用できます。  通話の連絡方法として会社電話を使用できるよう多要素認証を設定する方法について説明します。

## 連絡方法として会社電話を使用するには

<ol>
<li>ドロップダウン リストから [会社電話] 選択します。</li>

<center>![セットアップ](./media/multi-factor-authentication-end-user-first-time-office-phone/office.png)</center>


<li>ドロップ-お住まいの国を指定し、携帯電話番号を入力します</li>
<li>。携帯電話のテキストまたは通話で使用する方がモードを選択します</li>
<li>。[次へ] をクリックします</li>
<li>。今すぐ確認] ボタンをクリックします。 これによって、会社電話への通話が開始されます。  必ず電話のそばにいるようにしてください。 
<li>[次へ] をクリックして</li>
<li>この時点であるセットアップの連絡方法と Outlook 2010 などの非ブラウザー アプリの時間はアプリ パスワードを設定またはそれ以前になっています。 これらのアプリ] をクリックを使用しない場合 **実行**します。  使用している場合は、次の手順に進みます。
<li>これらのアプリを使用している場合は、指定されたアプリ パスワードをコピーします。</li>

<li>クリップボードにコピーされたパスワードを、非ブラウザー アプリケーションに貼り付けます。 Outlook および Lync など、個々のアプリケーションでの手順については、「電子メールのパスワードをアプリ パスワードに変更する」と「アプリケーションのパスワードをアプリ パスワードに変更する」を参照してください。</li>
<li>[Done] をクリックします。</li>
