# <a name="cloud-partner-portal-tour"></a>Przewodnik po portalu partnerów w chmurze

Ten artykuł zawiera wprowadzenie do [portalu cloud partner:](https://cloudpartner.azure.com)jego zawartość, mechanizmy nawigacji i funkcje.   Ten portal umożliwia zarejestrowanym partnerom tworzenie nowych ofert i zarządzanie istniejącymi ofertami dla [portalu Azure Marketplace](https://azuremarketplace.microsoft.com) i [AppSource Marketplace.](https://azuremarketplace.microsoft.com)  Cloud Partner Portal jest dostępny tylko dla zarejestrowanych partnerów; Aby uzyskać więcej informacji, zobacz [Zostań wydawcą cloud marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher).

## <a name="section-contents"></a>Zawartość sekcji

Po wyjaśnieniu mechanizmów [nawigacji](./cpp-portal-navigation.md) portalu, zbadamy podstawowe wpisy w lewym okienku nawigacji w odpowiednim artykule: 
- [Menu Nowa oferta](./cpp-new-offer-menu.md) umożliwia użytkownikom tworzenie nowych ofert określonego typu.
- [Wszystkie oferty](./cpp-all-offers-page.md) strony umożliwia partnerom przeglądanie wszystkich przesłanych ofert i niektórych z ich podstawowych cech. 
- Strona [Zatwierdzanie](./cpp-approvals-page.md) umożliwia partnerom wyświetlanie wszystkich aktualnie zatwierdzonych ofert.
- [Strona Wszyscy wydawcy](./cpp-all-publishers-page.md) zawiera listę wszystkich bieżących zarejestrowanych wydawców partnerów w chmurze.
- [Strona Użytkownicy](./cpp-users-page.md) zawiera listę wszystkich bieżących kont użytkowników powiązanych z zarejestrowanymi partnerami w chmurze.
- [Strona historia](./cpp-history-page.md) zawiera listę zdarzeń publikowania i modyfikacji oferty dla wszystkich wydawców. 
- Strona [Statystyk](./cpp-insights-page.md) zawiera różne pulpity nawigacyjne, które składają się na funkcję *Statystyk sprzedawcy* w tym portalu.
