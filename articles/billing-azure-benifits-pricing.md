# Azure サービスの利点と価格情報はどこで確認できますか。

Microsoft Azure を使用すると、インフラストラクチャとサービスを素早くデプロイしてすべてのビジネス ニーズを満たすことができます。 Windows や Linux ベースのアプリケーションは 19 の Azure データセンター リージョンで実行でき、エンタープライズ レベル SLA が適用されます。 Azure サービスの利点については、次を参照してください。 [Microsoft Azure プランの詳細](https://azure.microsoft.com/support/legal/offer-details/)します。 料金情報については、次を参照してください。、 [料金のページが Azure](https://azure.microsoft.com/pricing/)します。
