# <a name="sign-up-your-organization-to-use-azure-active-directory"></a>Zarejestruj się w organizacji, aby korzystać z usługi Azure Active Directory
Zarejestruj się w usłudze Azure Active Directory (Azure AD) lub nowej subskrypcji platformy Microsoft Azure przy użyciu:

- **konto Microsoft.** Użyj osobistego konta Microsoft, aby uzyskać dostęp do platformy Azure i wszystkich zorientowanych na konsumenta produktów i usług w chmurze, takich jak Outlook (Hotmail), Messenger, OneDrive, MSN, Xbox LIVE lub Office 365. Zarejestrowanie się w Outlook.com skrzynce pocztowej automatycznie tworzy konto Microsoft. Aby uzyskać więcej informacji, zobacz [omówienie konta Microsoft](https://account.microsoft.com/account).

- **Konto służbowe.** Użyj konta służbowego lub szkolnego, aby uzyskać dostęp do wszystkich małych, średnich i korporacyjnych usług w chmurze firmy Microsoft, takich jak Azure, Microsoft Intune lub Office 365. Po zarejestrowaniu się w jednej z tych usług jako organizacja usługa Azure AD automatycznie udostępnia katalog oparty na chmurze, który reprezentuje twoją organizację. Aby uzyskać więcej informacji, zobacz [Zarządzanie katalogami usługi Azure AD](active-directory-administer.md).

  > [!Note]
  >   Zaleca się korzystanie z konta służbowego, jeśli masz już dostęp do usługi Azure AD. Należy jednak użyć dowolnego typu konta skojarzonego z subskrypcją platformy Azure.
