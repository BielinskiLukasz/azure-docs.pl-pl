W poniższych przykładach `YOUR_MANAGEMENT_API_URL` odwołuje się do identyfikatora URI cyfrowego Twins interfejsów API:

```plaintext
https://YOUR_INSTANCE_NAME.YOUR_LOCATION.azuresmartspaces.net/management/api/v1.0
```

| Name (Nazwa) | Zamień na |
| --- | --- |
| YOUR_INSTANCE_NAME | Nazwa wystąpienia usługi Azure cyfrowego bliźniaczych reprezentacji |
| YOUR_LOCATION | Region, który wystąpienie usługi jest hostowana na |
