# <a name="view-dashboard"></a>Wyświetl pulpit nawigacyjny #

Kliknij przycisk **pulpitu nawigacyjnego** kartę, aby wyświetlić kluczowe metryki dla wszystkich przeglądu zawartości wykonać w ramach narzędzia. Widać przeglądami całkowity, została zakończona i w toku dla obrazy i tekst. Widać metryki dla zespołu domyślnego i subteams Twojego.
