> [!IMPORTANT]
> Obsługa interfejsu wiersza polecenia platformy Azure do celów zarządzania zasobami usługi HDInsight przy użyciu usługi Azure Service Manager (ASM) jest **przestarzała** i została usunięta z dniem 1 stycznia 2017 r. W czynnościach opisanych w niniejszym dokumencie są używane nowe polecenia interfejsu wiersza polecenia platformy Azure współpracujące z usługą Azure Resource Manager.
> 
> Wykonaj czynności opisane w temacie [Instalowanie i konfigurowanie interfejsu wiersza polecenia platformy Azure](../articles/cli-install-nodejs.md), aby zainstalować najnowszą wersję interfejsu wiersza polecenia platformy Azure. Jeśli masz skrypty wymagające modyfikacji w celu użycia nowych poleceń współpracujących z usługą Azure Resource Manager, zobacz temat [Migrowanie do narzędzi programistycznych opartych na usłudze Azure Resource Manager w celu obsługi klastrów usługi HDInsight](../articles/hdinsight/hdinsight-hadoop-development-using-azure-resource-manager.md), aby uzyskać więcej informacji.
