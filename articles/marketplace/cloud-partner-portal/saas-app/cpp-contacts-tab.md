# <a name="saas-application-contacts-tab"></a>Karta kontaktów aplikacji SaaS

W tym artykule opisano, jak na potrzeby identyfikacji kontaktów dla klientów korzystających z oferty na karcie kontaktów.


## <a name="identify-contacts"></a>Identyfikacji kontaktów

Umożliwia następnym zrzucie ekranu jako wskazówki Podaj wymagane informacje. Wszystkie pola oznaczone gwiazdką (*) muszą być wypełnione.

![Karta kontaktów](./media/saas-offer-contacts.png)

1. W polu kontakt techniczny Podaj informacje dotyczące następujących pól:

   - **Nazwa\***  — nazwa kontakt techniczny dla swojej aplikacji. Kontakt otrzyma komunikacji techniczne firmy Microsoft.
   - **Adres e-mail\***  — kontakt techniczny adres E-mail.
   - **Telefon\***  — kontakt techniczny numer telefonu. Notacji numeru telefonu ISO są obsługiwane. Aby uzyskać więcej informacji, zobacz [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. W obszarze skontaktuj się z pomocą techniczną Podaj informacje dotyczące następujących pól:

   - **Nazwa\***  — nazwę kontaktu, pomocy technicznej dla aplikacji. Kontakt będą otrzymywać informacji dotyczących pomocy technicznej firmy Microsoft.
   - **Adres e-mail\***  — skontaktuj się z pomocą techniczną aplikacji adres E-mail.
   - **Telefon\***  — numer pomocy technicznej skontaktuj się z telefonu. Notacji numeru telefonu ISO są obsługiwane. Aby uzyskać więcej informacji, zobacz [E.123](https://en.wikipedia.org/wiki/E.123)
   - **Adres URL pomocy technicznej\***  — adres URL do strony pomocy technicznej.

3. Wybierz **Zapisz** po zakończeniu formularzy.
