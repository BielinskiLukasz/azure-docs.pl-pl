# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Oprogramowanie innych firm zgodne z usługami domenowymi usługi Azure AD

## <a name="one-identity-authentication-services"></a>Jeden serwis uwierzytelniania tożsamości

![Jedna tożsamość](./media/compatible-software-icons/OneIdentity.png)

Usługi uwierzytelniania tożsamości umożliwiają systemom Unix, Linux i Mac OS X korzystanie z dostępu, uwierzytelniania i autoryzacji istniejącej infrastruktury usługi Active Directory (AD) organizacji. Usługi uwierzytelniania obsługują teraz usługi domenowe Active Directory platformy Azure, umożliwiając zasoby inne niż Windows korzystanie z tej samej platformy nowej generacji, z której już korzystają istniejące rozwiązania SaaS.

[Jeden serwis uwierzytelniania tożsamości](https://www.quest.com/products/authentication-services/)

[Artykuł KB — używanie usług uwierzytelniania tożsamości z usługami domenowymi usługi Azure AD](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux to wiodąca na świecie platforma Linux dla przedsiębiorstw zbudowana w celu zaspokojenia potrzeb współczesnego przedsiębiorstwa. Red Hat Enterprise Linux jest preferowanym wyborem dla obciążeń maszyn wirtualnych systemu Linux dla przedsiębiorstw na platformie Microsoft Azure. Red Hat Enterprise Linux to otwarta, niezawodna i bezpieczna platforma przeznaczona dla klientów, którzy chcą elastyczności wdrażania dla swoich krytycznych obciążeń biznesowych — od centrum danych po chmurę platformy Azure — wspierana przez ściśle zintegrowaną pomoc techniczną klasy korporacyjnej. Red Hat i Microsoft.
