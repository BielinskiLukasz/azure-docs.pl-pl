> [AZURE.NOTE] Azure PowerShell は、2 つのリリースの 1.0 および 0.9.8 で現在使用できます。 既存のスクリプトがあって今は変更したくない場合、0.9.8 リリースをそのまま使用してかまいません。 1.0 リリースを使用する場合は、予期しない影響を防ぐために、運用前の環境でスクリプトを慎重にテストしてから運用環境で使用してください。
>
> 1.0 のコマンドレットが {動詞} の名前付けパターンに従う-AzureRm {名詞};一方、名は含まれません 0.9.8 **Rm** (たとえば、新規の AzureRmResourceGroup New-azureresourcegroup ではなく)。 Azure PowerShell 0.9.8 を使用する場合を実行して、リソース マネージャー モードをさせる必要があります、 **Switch-azuremode AzureResourceManager** コマンドです。 このコマンドは、1.0 では必要ありません。
>
> 新機能が追加されるのは、1.0 リリースのみです。 1.0 のリリースでは、インストールし、リリースをアンインストールする方法などに関する情報を参照してください。 [Azure PowerShell 1.0](https://azure.microsoft.com/blog/azps-1-0/)します。
