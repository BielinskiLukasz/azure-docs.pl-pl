# <a name="azure-marketplace-listing-guidelines"></a>Wskazówki dotyczące wyświetlania w portalu Azure Marketplace

Ten dokument zawiera wskazówki dotyczące wymagań oraz listę kontrolną dla recenzentów nowych ofert i usług wymienionych w witrynie Azure Marketplace.

Wszystkie oferty muszą spełniać [wszystkie wymagania](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#all-listing-requirements) wymienione poniżej. Dodatkowe wymagania i listy kontrolne są dostępne dla konkretnych typów list, w tym [](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#trial-offer-requirements): oferty z dołączonymi wersjami próbnymi, [aplikacjami SaaS](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#saas-apps-requirements), kontenerami i [ofertami konsultacji](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#consulting-offer-requirements). [](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#container-offer-requirements)
