# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Nadchodzące zmiany usługi Azure Application Insights (wersja zapoznawcza) 

Usługa Application Insights często wydaje nowe funkcje. Jeśli chcesz zobaczyć wersje zapoznawcze ulepszeniami, możesz zarejestrować się w bloku usługi Application Insights w wersji zapoznawczej.  Zespół opracowujący sprawia, że wersje zapoznawcze nowych funkcji dostępnych w ograniczonym zakresie przed ich wszystkim użytkownikom. 

Na poniższym obrazie przedstawiono sposób ustawiania preferencji użytkownika dotyczących wersji zapoznawczej.

![Application Insights w wersji zapoznawczej bloku](media/previews/preview.png)

## <a name="set-preferences"></a>Ustawianie preferencji

W bloku (wersja zapoznawcza) możesz wybrać spośród następujących opcji dla po wyświetleniu podglądu.

- Zawsze: Zobaczysz podglądu środowiska, jak są one dostępne.
- Automatycznie: Zobaczysz podglądu środowiska, że firma Microsoft zaleca się do konta. 
- Nigdy: Widoczne są tylko procesy (wersja zapoznawcza), które wybierzesz.
