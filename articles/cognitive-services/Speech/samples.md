# <a name="microsoft-speech-client-samples"></a>Przykłady klienta mowy firmy Microsoft

Usługa rozpoznawania mowy firmy Microsoft zapewnia przykłady end-to-end przedstawiający sposób użycia interfejsu API rozpoznawania mowy firmy Microsoft w innych przypadków użycia, na przykład polecenie rozpoznawania, ciągłe rozpoznawanie i wykrywanie konwersji. Wszystkie przykłady są dostępne w serwisie GitHub i mogą być pobierane przez następujących łączy: README.md w każdym repozytorium, jak również [bibliotek klienckich](GetStarted/GetStartedClientLibraries.md) strony zawierają szczegółowe informacje o sposobie tworzenia i uruchamiania przykładów.

- [Przykłady interfejsu API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Przykłady kodu JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Przykłady klienta pulpitu C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Przykłady usługi C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Java na próbkach systemu Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [ObjectiveC na próbkach systemu iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Wszystkie zestawy SDK usługi kognitywnych firmy Microsoft i przykłady, jest udzielana z licencją MIT. Aby uzyskać więcej informacji, zobacz [licencji](https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript/blob/master/LICENSE.md).
