Usługa Azure Cloud Shell to oparte na przeglądarce środowisko powłoki do tworzenia zasobów platformy Azure i zarządzania nimi.

Cloud Shell oferuje dostępne dla przeglądarki, wstępnie skonfigurowane środowisko powłoki do zarządzania zasobami platformy Azure bez konieczności samodzielnego instalowania, przechowywania i obsługi maszyn.

Cloud Shell Inicjuje obsługę administracyjną maszyn na żądanie i w związku z tym Kondycja komputera nie będzie trwała między sesjami. Ponieważ Cloud Shell została skompilowana dla sesji interakcyjnych, powłoki są automatycznie kończyć się po 20 minutach braku aktywności powłoki.
