# <a name="azure-cli-samples-for-azure-cache-for-redis"></a>Przykłady interfejsu wiersza polecenia platformy Azure dla pamięci podręcznej platformy Azure dla redis

Poniższa tabela zawiera linki do skryptów bash utworzonych za pomocą interfejsu wiersza polecenia platformy Azure.

| | |
|---|---|
|**Tworzenie pamięci podręcznej**||
| [Tworzenie pamięci podręcznej](./scripts/create-cache.md) | Tworzy grupę zasobów i podstawową warstwę Usługi Azure Cache dla redis. |
| [Tworzenie pamięci podręcznej premium z klastrem](./scripts/create-premium-cache-cluster.md) | Tworzy grupę zasobów i pamięć podręczną warstwy premium z włączonym klastrem.|
| [Pobierz szczegóły pamięci podręcznej](./scripts/show-cache.md) | Pobiera szczegóły usługi Azure Cache dla wystąpienia Redis, w tym stan inicjowania obsługi administracyjnej. |
| [Pobierz nazwy hosta, porty i klucze](./scripts/cache-keys-ports.md) | Pobiera nazwy hosta, portów i kluczy dla pamięci podręcznej azure dla wystąpienia Redis. |
|**Aplikacja internetowa plus pamięć podręczna**||
| [Łączenie aplikacji internetowej z pamięcią podręczną Azure Cache for Redis](./../app-service/scripts/cli-connect-to-redis.md) | Tworzy aplikację sieci web platformy Azure i pamięć podręczną Azure cache dla firmy Redis, a następnie dodaje szczegóły połączenia redis do ustawień aplikacji. |
|**Usuwanie pamięci podręcznej**||
| [Usuwanie pamięci podręcznej](./scripts/delete-cache.md) | Usuwa pamięć podręczną platformy Azure dla wystąpienia Redis  |
| | |
