# <a name="iot-edge-module-support-tab"></a>Karta Obsługa modułu IoT Edge

Użyj karty **Pomoc techniczna** na stronie **Nowa oferta,** aby zapewnić zasoby pomocy technicznej i pomocy technicznej dla swojej oferty.  W formularzu Pomocy technicznej znajdują się następujące obszary wejściowe: **Kontakt inżynierski,** **Obsługa klienta**i **Adresy URL pomocy technicznej.** Gwiazdka (*) dołączona do nazwy pola wskazuje, że jest ona wymagana.

![Karta Obsługa w formularzu Nowa oferta dla modułów IoT Edge](./media/iot-edge-module-support-tab.png)

## <a name="support-form-fields"></a>Pola formularza pomocy technicznej

W poniższej tabeli opisano pola formularza i dane wejściowe wymagane dla każdego pola. Wymagane pola są oskarżane gwiazdką (*).

|  **Pole**                |     **Opis**                                                          |
|  ---------                |     ---------------                                                          |
| ***Kontakt techniczny***  |  *Służy jako kontakt techniczny między firmą Microsoft a twoją organizacją* | 
| **Nazwa\***                  | Nazwisko osoby lub grupy, która służy jako wsparcie techniczne/inżynieryjne     |
| **Adres e-mail\***                 | Adres e-mail tego kontaktu technicznego                                      |
| **Telefon\***                 | Numer telefonu do pomocy technicznej                                           |
| ***Obsługa klienta***  | *Odbiera bilety pomocy technicznej otwierane przez klientów na platformie Azure* |
| **Nazwa\***                  | Imię i nazwisko osoby lub grupy, która służy jako obsługa klienta                  |
| **Adres e-mail\***                 | Adres e-mail działu obsługi klienta                                            |
| **Telefon\***                 | Numer telefonu do obsługi klienta                                            |
| ***Adresy URL pomocy technicznej***  | *Witryny pomocy technicznej, z których będzie korzystać firma Microsoft, gdy klienci otworzą bilety pomocy technicznej* |
| **Publiczna platforma Azure**          | Adres URL publicznej witryny pomocy technicznej w Internecie                                         |
| **Chmura dla instytucji rządowych platformy Azure**| Adres URL witryny pomocy technicznej chmury rządowej                                        |
|   |   |
