## <a name="select-subscription-and-authoring-resource"></a>Wybieranie zasobu subskrypcji i tworzenia

W portalu usługi LUIS, aby wyświetlić listę aplikacji lub utworzyć aplikację, wybierz zasób subskrypcji i tworzenia. Aplikacje są dostępne dla zasobów tworzenia usługi LUIS.
