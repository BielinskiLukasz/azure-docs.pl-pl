## <a name="more-information-about-bots"></a>Więcej informacji na temat botów

Aby uzyskać więcej informacji na temat korzystania z tej usługi z botami zaczynają się od następujących zasobów:

|Resource|Przeznaczenie|
|--|--|
|[Usługa Azure Bot service](https://dev.botframework.com/)|Usługa Azure Bot service udostępnia usługę pełną sieci web, które są hostowane w chmurze z punktem końcowym botów. Zastosowań usługi [platformy Bot framework](https://github.com/Microsoft/botframework), która jest dostępna w kilku językach.|
|[Bot Framework](https://github.com/Microsoft/botframework)|Microsoft Bot Framework jest kompleksowej platformy do tworzenia konwersacji środowisk sztucznej Inteligencji przeznaczonych dla przedsiębiorstw.|
|[Bot Framework emulatora](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Bot Framework Emulator to aplikacja komputerowa dla wielu platform, umożliwiający deweloperom testowanie i debugowanie Boty utworzone przy użyciu Bot Framework SDK botów. Do przetestowania Boty uruchomiony lokalnie na Twoim komputerze lub połączyć się Boty działa zdalnie, można użyć emulatora usługi Bot Framework.|
|[Narzędziami BOT](https://github.com/Microsoft/botbuilder-tools)|Narzędzia platformy Bot Framework to zbiór narzędzi wiersza polecenia dla wielu platform przeznaczonych do przepływu pracy tworzenia oprogramowania bot end-to-end obejmować. |
|[Bot builder przykłady](https://github.com/Microsoft/BotBuilder-Samples)|Przykłady opracowany pełny bot są przeznaczone do zilustrowania scenariusze, które będą potrzebne do zaimplementowania do tworzenia wspaniałych botów.|
