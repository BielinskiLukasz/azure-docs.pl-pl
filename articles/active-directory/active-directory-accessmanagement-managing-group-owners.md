# グループの所有者の管理

## グループへの所有者の割り当て

**Azure ポータルでグループに所有者を追加するには**

1. Azure ポータルで **[Active Directory]** をクリックし、該当する組織のディレクトリ名をクリックします。
2. **[グループ]** タブをクリックします。
2. **[グループ]** ページで、所有者の追加先となるグループの名前をクリックします。
3. そのグループのページで、**[所有者の追加]** をクリックします。
4. **[所有者の追加]** ページで、このグループの所有者として追加するユーザーの名前をクリックし、その名前が **[選択済み]** ウィンドウに追加されていることを確認します。


**Azure ポータルでグループから所有者を削除するには**

1. Azure ポータルで **[Active Directory]** をクリックし、該当する組織のディレクトリ名をクリックします。
2. **[グループ]** タブをクリックします。
3. **[グループ]** ページで、所有者を削除するグループの名前をクリックします。
4. そのグループのページで、**[所有者]** タブをクリックします。
5. グループから削除する所有者の名前をそのグループのページでクリックし、**[削除]** をクリックします。
