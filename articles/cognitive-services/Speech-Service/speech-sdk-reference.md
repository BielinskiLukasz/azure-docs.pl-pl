# <a name="speech-sdk-api-reference"></a>Dokumentacja interfejsu API zestawu SDK rozpoznawania mowy

Mowy Software Development Kit (SDK) daje dostęp aplikacji do funkcji usługi rozpoznawania mowy, co ułatwia tworzenie oprogramowania dla komputerów z obsługą mowy. Obecnie zestawy SDK zapewniają dostęp do **zamiana mowy na tekst**, **tłumaczenia mowy**, i **rozpoznawanie intencji**. Ogólne omówienie dotyczące możliwości i obsługiwanych platform można znaleźć w dokumentacji [strony wprowadzania](http://aka.ms/csspeech).
