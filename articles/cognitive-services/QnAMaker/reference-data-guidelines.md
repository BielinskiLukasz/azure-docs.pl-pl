# <a name="import-and-export-data-reference"></a>Importowanie i eksportowanie danych referencyjnych

Przejrzyj to odwołanie do importu i eksportu, aby uzyskać najlepsze wyniki dla kopii zapasowej, magazynu i wymiany bazy wiedzy.

## <a name="import-and-export-knowledge-base"></a>Baza wiedzy o imporcie i eksporcie

**Pliki TSV i XLS**, z wyeksportowanych baz wiedzy, mogą być używane tylko przez importowanie plików ze strony **Ustawienia** w portalu QnA Maker. Nie można ich używać jako źródeł danych podczas tworzenia bazy wiedzy ani z funkcji **+ Dodaj plik** lub + Dodaj adres **URL** na stronie **Ustawienia.**
