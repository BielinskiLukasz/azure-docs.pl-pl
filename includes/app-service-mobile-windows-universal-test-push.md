1. Kliknij prawym przyciskiem myszy projekt Windows Store, kliknij przycisk **Ustaw jako projekt startowy**, naciśnij klawisz F5, aby uruchomić aplikację Windows Store.

    Po uruchomieniu aplikacji, urządzenie jest zarejestrowane na potrzeby obsługi powiadomień wypychanych.
2. Zatrzymaj aplikację Windows Store i powtórz poprzedni krok dla aplikacji Windows Phone Store.

    W tym momencie oba urządzenia są zarejestrowane do odbierania powiadomień wypychanych.

3. Ponownie uruchom aplikację Windows Store, a następnie wpisz tekst w **Wstaw czynność do wykonania**, a następnie kliknij przycisk **Zapisz**.

    Należy pamiętać, że po ukończeniu Wstaw Store Windows i aplikacje Windows Phone otrzymaj powiadomienie wypychane z usługi WNS. Powiadomienie jest wyświetlane na Windows Phone, nawet wtedy, gdy aplikacja nie jest uruchomiona.
