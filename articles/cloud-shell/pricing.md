# <a name="pricing"></a>Cennik
Powłoka bash w usłudze Cloud Shell i PowerShell w usłudze Cloud Shell podlegają poniższe informacje.

## <a name="compute-cost"></a>Koszt zasobów obliczeniowych
Usługa Azure Cloud Shell jest uruchamiana na komputerze, za darmo udostępnianych przez platformę Azure, ale wymaga udziału plików platformy Azure do użycia. 

## <a name="storage-cost"></a>Koszt usługi Storage
Usługa cloud Shell wymaga nowego lub istniejącego udziału plików platformy Azure ma zostać zainstalowany, aby utrwalać pliki między sesjami. Magazyn jest naliczana opłata regularne kosztów.

Sprawdź [tutaj, aby uzyskać szczegółowe informacje na temat usługi Azure Files koszty](https://azure.microsoft.com/pricing/details/storage/files/).
