# Visual Studio を使用してクラウド サービス コンテナーを作成する方法

表示する **サーバー エクスプ ローラー** Visual Studio のメニュー バーで選択 **ビュー**, 、**サーバー エクスプ ローラー**します。 

1.  展開、 **Azure** ノード

2.  ユーザーがサインインしていない場合は、自動的にサインイン ダイアログ ボックスが表示されます。 指示に従って Azure サブスクリプションにログインします。

3.  右クリックして **クラウド サービス** ] ノードを選択 **クラウド サービスを作成しています.**

4.  A **クラウド サービスの作成** ダイアログが表示されます。 記入 **サブスクリプション**, 、**名前**, 、**リージョンまたはアフィニティ グループ** フィールドとクリック **作成** クリックすると、空のクラウド サービス コンテナーを作成

## 次のステップ

クラウド サービス コンテナーがある場合は、これを実行できます [を作成および展開](cloud-services-how-to-create-deploy.md) クラウド サービスです。
