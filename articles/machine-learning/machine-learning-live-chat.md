# 機械学習のライブ チャット サポートを利用する

[Azure Machine Learning Studio](machine-learning-what-is-ml-studio.md) 機械学習モデルを構築するための直感的なインターフェイスを提供します。  [ギャラリー](machine-learning-gallery-how-to-use-contribute-publish.md) と [フォーラム](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) するうえで役立つにします。

ただし、短い質問をするだけで問題が解決する場合もあります。  [Studio](machine-learning-what-is-ml-studio.md) の上部ナビゲーションにチャット アイコンを探します。 このアイコンが表示されているときは、製品チームのメンバーによるリアルタイムのサポートがオンラインで受けられることを意味します。

![Azure ML チャット](./media/machine-learning-live-chat/AzureMLChatNavBar.png)

質問を入力して、回答を得ましょう。

![Azure ML チャット ダイアログ](./media/machine-learning-live-chat/AzureMLChat.png)

## ライブ チャット アイコンが表示されない場合

チームのメンバーが実際に質問に回答しています。 ライブ チャット アイコンが表示されていないのは、業務時間外には表示されないためです。
