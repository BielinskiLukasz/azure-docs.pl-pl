# <a name="discover-jupyter-notebooks-in-the-azure-ai-gallery"></a>Odnajdywanie aplikacji Jupyter Notebooks w galerii Azure AI

**[Notesy Jupyter notebook](https://gallery.cortanaintelligence.com/notebooks)**  w [Galeria sztucznej Inteligencji platformy](http://gallery.cortanaintelligence.com) obejmują kodu, wizualizacje danych i dokumentacji na kanwie pojedynczego, interaktywne.
Notesy w galerii zapewniają samouczków i uzyskać bardziej szczegółowe objaśnienia zaawansowanych technik i rozwiązań uczenia maszynowego.

## <a name="discover"></a>Wykrywanie
  Aby przejść do notesów w galerii, otwórz [galerii](http://gallery.cortanaintelligence.com) i kliknij przycisk **notesów** w górnej części strony głównej galerii.

 **[Notesów programu Jupyter](https://gallery.cortanaintelligence.com/notebooks)** stronie wyświetlana jest lista najpopularniejszymi notesami.
Kliknij przycisk **holograficznych** Aby wyświetlić wszystkie notesy.
Na tej stronie można przeglądać wszystkie notesy w galerii. Również można przeszukiwać, wybierając kryteria filtrowania w lewej części strony i wprowadzanie wyszukiwanych terminów u góry.

 Kliknij pozycję Wszystkie Notes, aby otworzyć stronę szczegółów Notes i Dowiedz się więcej. Na tej stronie, którą możesz przekształcić w komentarz przesłać opinię lub zadać pytania przy użyciu sekcji komentarzy. Można nawet udostępniać znajomym lub współpracowników, korzystając z możliwości udziału usługi LinkedIn i Twitter. Możesz również wysłać wiadomość e-mail łącze do notesu można zaprosić innych użytkowników, aby wyświetlić stronę.

![Udostępnij ten element ze znajomymi](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Dodać swoje własne komentarze](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Do pobrania
Możesz pobrać kopię każdego dostępnego w galerii, do obszaru roboczego usługi Machine Learning Studio.

### <a name="download-from-the-gallery"></a>Pobierz z galerii
Aby pobrać kopię notesu z galerii:

1. Otwórz stronę szczegółów Notes w galerii
2. Kliknij przycisk **Otwórz w Studio**
3. Wybierz Region i obszar roboczy, którego chcesz użyć
   
    ![Otwieranie notesu programu Jupyter, z galerii](./media/gallery-jupyter-notebooks/open-notebook-from-gallery.png)

Po kliknięciu **Otwórz w programie Studio**, Notes jest ładowany do obszaru roboczego usługi Machine Learning Studio i otworzyć (jeśli jeszcze nie zostało to zrobione Studio, zostanie wyświetlony monit logować się za pomocą konta Microsoft, zanim będzie notesu skopiowany do swojego obszaru roboczego). Znajdź Notes później, klikając **notesów** w lewej części strony głównej Studio.

### <a name="download-in-machine-learning-studio"></a>Pobierz w usłudze Machine Learning Studio
Możesz również otworzyć żadnego z tych samych notesów galerii podczas pracy w usłudze Machine Learning Studio:

1. W usłudze Machine Learning Studio, kliknij przycisk **+ nowy**
2. Wybierz **notesu** -zostanie wyświetlona lista notesów z galerii, że możesz wybrać spośród lub można znaleźć określonego Notes, korzystając z pola wyszukiwania
3. Wskaż przy użyciu myszy w notesie i wybierz pozycję **Otwieranie notesu**
   
    ![Otwórz galerię Notes z wewnątrz usługi Machine Learning Studio](./media/gallery-jupyter-notebooks/open-notebook-from-studio.png)

Kopię Notes jest pobrany i otwarty w obszarze roboczym w sekcji notesów programu Jupyter w usłudze Machine Learning Studio.
Notes, zostaną wyświetlone, wraz z notesów innych na **notesów** strony Studio (na [stronę główną Studio](https://studio.azureml.net/), kliknij przycisk **notesów** po lewej stronie).
