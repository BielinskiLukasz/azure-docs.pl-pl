## <a name="test"></a>Weryfikowanie funkcji na platformie Azure

Użyj Zwinięciea, aby zweryfikować wdrożoną funkcję. Przy użyciu adresu URL, w tym klucza funkcji skopiowanego z poprzedniego kroku, dołącz ciąg `&name=<yourname>` zapytania do adresu URL.

![Używanie zwinięcie do wywołania funkcji na platformie Azure.](./media/functions-test-function-code/functions-azure-cli-function-test-curl.png) 

Możesz także wkleić skopiowany adres URL, w tym klucz funkcji, na pasku adresu przeglądarki sieci Web. Ponownie Dołącz ciąg `&name=<yourname>` zapytania do adresu URL przed wykonaniem żądania.

![Używanie przeglądarki sieci Web do wywoływania funkcji.](./media/functions-test-function-code/functions-azure-cli-function-test-browser.png)
