# <a name="view-dashboard"></a>Wyświetl pulpit nawigacyjny

Kliknij przycisk **pulpit nawigacyjny** kartę w celu wyświetlenia kluczowych metryk dla wszystkich Przegląd zawartości w narzędziu. Możesz zobaczyć Suma, ukończone i w trakcie przeglądy dla aplikacji dotyczących obrazów i tekstu. Dla Twojego zespołu domyślnego i Twoje zespoły podrzędne, widzą metryki.
