# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Wskazówki dotyczące usługi Azure Data Lake magazynu Gen1 klientów

Azure Data Lake Podgląd Gen2 magazynu oferty natywnego obiektu Blob magazynu funkcje, w tym warstwy danych, Zarządzanie cyklem życia i większe bezpieczeństwo w sieci, oprócz dostępnych w Data Lake magazynu Gen1 systemu plików Hadoop zgodny.

Jeśli masz istniejący klient Data Lake magazynu Gen1, firma Microsoft zachęca do oceny wszystkich nowych i ulepszonych funkcji Data Lake magazynu Gen2 i wyrazić swoją opinię przy [uczestniczących w podglądzie](https://aka.ms/adlsgen2signup). Ponadto Data Lake magazynu Gen1 będzie w dalszym ciągu obsługiwana. Data Lake magazynu Gen1 i Gen2 mogą współistnieć, co z istniejących zasobów i aplikacji korzystających z danych Lake magazynu Gen1 kontynuować pracę bez aktualizacji. Możesz utworzyć plan migracji Data Lake magazynu Gen2 wygodne.

Jeśli oceniasz obecnie Data Lake magazynu Gen1, firma Microsoft zaleca uczestniczyć w wersji zapoznawczej Gen2 i ocena nowych i ulepszonych funkcji oferowanych przez Gen2 magazynu Lake danych. Podobnie, jeśli masz istniejący klient magazynu obiektów Blob, firma Microsoft zachęca do oceny, czy system plików Hadoop zgodny jest odpowiednie do potrzeb analitycznych.
