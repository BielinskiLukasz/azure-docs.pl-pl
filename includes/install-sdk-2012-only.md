<h2><a name="setupdevenv"></a>開発環境を設定する</h2>

最初に、Azure SDK for .NET をインストールして、開発環境を設定します。

1. SDK をインストールするには、次のリンクをクリックします。 このチュートリアルには、Visual Studio 2012 が必要です。 Visual Studio がインストールされていない場合は、SDK と共に Visual Studio Express for Web がインストールされます。

    - [Azure SDK for Visual Studio 2012][]

    **注:** に応じて、コンピューターに既にある SDK の依存関係の数、SDK をインストールすることが長時間かかる場合、数分から 30 分以上にします。

2. 実行するか、実行可能ファイルをクリックしてインストールを保存するメッセージが表示されたら **実行**します。

3. Web Platform Installer のウィンドウで [ **インストール** し、インストールを実行します。

    ![Web Platform Installer - Azure SDK for .NET][WebPIAzureSdk]

インストールが完了すると、開発に必要なツールがすべて揃います。

[Azure SDK for Visual Studio 2012]: http://go.microsoft.com/fwlink/?LinkID=324323
