# <a name="bing-visual-search-sdk"></a>Visual wyszukiwania usługi Bing zestawu SDK

Visual wyszukiwania usługi Bing przekazuje temu klientowi następujących zestawów SDK, które służy do wykonywania Visual API wyszukiwania.

* [Przykłady dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) definicje i zależności
* [Przykłady dla platformy Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Biblioteki środowiska node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) definicje i zależności
* [Przykłady w języku Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Biblioteki języka Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) definicje i zależności
* [Przykłady w języku Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) definicje i zależności

Przykłady zestawu SDK dla każdego języka obejmują plik ReadMe z wymagań wstępnych dotyczących używania zestawu SDK oraz kroki dotyczące instalowania i uruchamiania przykładów.
