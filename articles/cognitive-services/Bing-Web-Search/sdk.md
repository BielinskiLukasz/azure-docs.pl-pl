# <a name="bing-web-search-sdk-samples"></a>Przykłady zestawu SDK wyszukiwania sieci Web Bing

Przykłady zestawu SDK wyszukiwania sieci Web Bing obejmują scenariusze który:

* Zapytania pojedynczego wyrazu i Drukuj nazwy i adresu URL strony sieci web pierwsze obraz, artykułu z wiadomościami i wideo, dołączony do odpowiedzi.
* Zapytania frazę, sprawdź liczbę wyników, a następnie wydrukuj nazwy i adresu URL pierwszego wyniku.
* Zapytanie dotyczące wyszukiwany termin odpowiedzi filtruje zestaw do `news` i wydrukować szczegóły wyniki wiadomości.
* Zapytanie dotyczące wyszukiwany termin `answerCount` i `promote` parametrów, a następnie wydrukować szczegóły wyników.

Zestaw SDK wyszukiwania w sieci Web Bing jest dostępna w języku Python, Node.js, C# i Java.

* Rozpoczynanie pracy z usługą [przykłady dla platformy .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
  * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.WebSearch/1.2.0)
  * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingWebSearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykłady dla platformy Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples)
  * Zobacz też [biblioteki technologii Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/webSearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykładów w języku Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples)
  * Zobacz też [biblioteki Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search/BingWebSearch) dla definicji i zależności.
* Rozpoczynanie pracy z usługą [przykłady w języku Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples)
  * Zobacz też [biblioteki języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-websearch) dla definicji i zależności.
