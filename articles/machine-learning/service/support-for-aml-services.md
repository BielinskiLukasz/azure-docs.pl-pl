## <a name="get-support-for-azure-machine-learning-services"></a>Uzyskaj pomoc techniczną dla usługi Azure Machine Learning

Możesz uzyskać pomoc techniczną dla usługi Azure Machine Learning, w następujący sposób:

+ **Pomoc techniczna**: Azure skontaktuj się z pomocą techniczną, odwiedzając stronę [pomocy technicznej systemu Azure](https://azure.microsoft.com/support/options/) i wybierając opcję Machine Learning. 

+ **Forum użytkowników MSDN**: zadawaj pytania, odpowiadać na pytania i połącz się z innymi użytkownikami w [forum pomocy technicznej usługi Azure Machine Learning w witrynie MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Stack Overflow**: Odwiedź społeczność usługi Azure Machine Learning na [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) oznakowane za pomocą "Azure-Machine-Learning".

+ **Udostępnianie sugestie dotyczące produktów** oraz funkcji żądań w naszym [kanału sprzężenia zwrotnego usługi Azure Machine Learning](https://feedback.azure.com/forums/257792-machine-learning), która jest również dostępna przy użyciu linku w dolnej części każdego artykułu.
