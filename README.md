# <a name="microsoft-azure-documentation"></a>Dokumentacja platformy Microsoft Azure

Witamy w "open source" [dokumentacji](https://docs.microsoft.com/azure) z [Microsoft Azure](https://azure.microsoft.com). Przejrzyj ten plik README, aby zrozumieć, jak mogą pomóc w Współtworzenie dokumentacji firmy Microsoft Azure.

## <a name="getting-started"></a>Wprowadzenie

Współtworzenie "open source" więcej niż tylko dostarczanie aktualizacji, również jest umożliwienie nam znać, jeśli występuje problem. Przeczytaj nasze [współtworzenia wskazówki](.github/CONTRIBUTING.md) Aby dowiedzieć się więcej.

### <a name="prerequisites"></a>Wymagania wstępne

Decydujesz się zapewnienie wkładu, która doskonale nadaje się! Istnieje kilka narzędzi, które są potrzebne do współtworzenia dokumentacji.

Współtworzenie dokumentacji wymaga konta usługi GitHub, jeśli nie masz konta, postępuj zgodnie z instrukcjami dotyczącymi [Konfigurowanie konta usługi GitHub](https://docs.microsoft.com/contribute/get-started-setup-github) z naszego przewodnika współautora.

#### <a name="download"></a>Do pobrania

Zainstaluj następujące narzędzia:

* [Usługa Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* [Docs Authoring Pack](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) rozszerzenia programu Visual Studio Code

#### <a name="install"></a>Instalowanie

Postępuj zgodnie z instrukcjami podanymi w [zawartości instalowania narzędzi do tworzenia](https://docs.microsoft.com/contribute/get-started-setup-tools) z naszego przewodnika współautora.
