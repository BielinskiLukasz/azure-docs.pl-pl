# <a name="build-real-time-applications-with-azure-signalr"></a>Tworzenie aplikacji w czasie rzeczywistym za pomocą usługi Azure Signal

[Usługa Azure sygnalizująca](https://azure.microsoft.com/services/signalr-service/) pozwala łatwo dodawać do aplikacji funkcje w czasie rzeczywistym. Ta funkcja w czasie rzeczywistym umożliwia usłudze wypychanie aktualizacji zawartości do połączonych aplikacji mobilnych. W związku z tym aplikacje są aktualizowane bez konieczności sondowania serwera ani przesyłania nowych żądań HTTP do aktualizacji.

## <a name="azure-signalr-features"></a>Funkcje usługi Azure Signal
- **Szeroka obsługa klientów** — współpracuje z szeroką gamą klientów, takimi jak przeglądarki internetowe i mobilne, aplikacje klasyczne, proces serwera, urządzenia IoT i konsole gier.
- Usługa sygnalizująca umożliwia współpracę **wielu wystąpień** w celu skalowania do milionów połączeń aplikacji mobilnych. Obsługuje także wiele regionów globalnych na potrzeby fragmentowania, wysokiej dostępności lub odzyskiwania po awarii.
- **Oferuje zaawansowane interfejsy API dla różnych wzorców obsługi komunikatów** w celu wysyłania komunikatów do określonego połączenia, wszystkich połączeń lub podzbioru połączeń, które należą do określonego użytkownika lub zostały umieszczone w dowolnej grupie.
