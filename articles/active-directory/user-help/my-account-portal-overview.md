# <a name="what-is-the-my-account-portal"></a>Co to jest portal Moje konto?

Portal **Moje konto** ułatwia zarządzanie kontem służbowym, konfigurując informacje zabezpieczające, zarządzając połączonymi organizacjami i urządzeniami, wyświetlając sposób, w jaki organizacja wykorzystuje dane.

Możesz przejść do portalu Mojehttps://myprofile.microsoft.com) **konto** ( , z bieżącej wersji dowolnej z następujących przeglądarek:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

![Portal Moje konto, strona Przegląd](media/my-account-portal/my-account-portal-overview.png)

>[!Important]
>Ten artykuł jest przeznaczony dla użytkowników próbujących uzyskać kontakt z portalem Moje konto w celu zaktualizowania informacji zabezpieczających, informacji o urządzeniu, hasła, połączonych organizacji, prywatności lub poprzednich informacji logowania. Jeśli jesteś administratorem szukającym informacji na temat włączania uwierzytelniania i innych funkcji usługi Azure Active Directory (Azure AD) dla pracowników i innych zastosowań, zobacz [dokumentację usługi Azure AD dla administratorów.](https://docs.microsoft.com/azure/active-directory/)
