# <a name="manage-your-azure-maps-account"></a>Zarządzaj swoim kontem usługi Azure Maps

Możesz zarządzać kontem usługi Azure Maps w witrynie Azure portal. Po utworzeniu konta usługi, można zaimplementować interfejsów API w witrynie sieci Web lub aplikacji mobilnej.

Jeśli nie masz subskrypcji platformy Azure, przed rozpoczęciem utwórz [bezpłatne konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F).

## <a name="create-a-new-account"></a>Utwórz nowe konto

1. Zaloguj się w witrynie [Azure Portal](https://portal.azure.com).

2. Wybierz **Utwórz zasób** w lewym górnym rogu witryny Azure portal.

3. Wyszukaj i wybierz pozycję **mapy**. Następnie wybierz przycisk **Utwórz**.

4. Wprowadź informacje dla nowego konta.

![Wprowadź informacje o koncie w portalu](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="delete-an-account"></a>Usuwanie konta

Możesz usunąć konto w witrynie Azure portal. Przejdź do strony Przegląd konta i wybierz **Usuń**.

![Usunąć swoje konto w portalu](./media/how-to-manage-account-keys/account-delete-portal.png)

Następnie zobaczysz stronę potwierdzenia. Użytkownik potwierdzi usunięcie Twojego konta, wpisując jej nazwę.
