## <a name="sap-dbms-on-azure"></a>Systemu SAP DBMS na platformie Azure
Tytuł: Systemu SAP DBMS w przewodniku wdrażania na platformie Azure

Podsumowanie: W tym dokumencie opisano zagadnienia dotyczące systemów DBMS, które powinny być uruchamiane w połączeniu z oprogramowaniem SAP planowania i implementacji. W pierwszej części wymieniono i przedstawiono zagadnienia ogólne. Kolejne części dokumentu dotyczą wdrożeń różnych systemów DBMS na platformie Azure obsługiwanych przez oprogramowanie SAP. Różne przedstawione systemy DBMS obejmują program SQL Server, SAP ASE, Oracle, SAP MaxDB i IBM DB2 dla systemów Linux, Unix i Windows. W tych konkretnych częściach zostały omówione zagadnienia, które należy uwzględnić w przypadku uruchamiania systemów SAP na platformie Azure w połączeniu z tymi systemami DBMS. Tematy takie jak metody tworzenia kopii zapasowej i zapewniania wysokiej dostępności obsługiwane przez różne systemy DBMS na platformie Azure są przedstawione do użycia z aplikacjami SAP.

## <a name="sap-netweaver-on-azure"></a>SAP NetWeaver na platformie Azure
Tytuł: Oprogramowanie SAP NetWeaver — tworzenie platformy Azure na podstawie rozwiązanie odzyskiwania po awarii

Podsumowanie: ten dokument zawiera wskazówki krok po kroku dla platformy Azure do tworzenia opartych na rozwiązanie odzyskiwania po awarii dla oprogramowania SAP NetWeaver. Rozwiązanie przedstawione przyjęto założenie, że środowisko SAP jest uruchomiony zwirtualizowany lokalnych oparte na funkcji Hyper-V. W pierwszej części dokumentu odzyskiwania lokacji platformy Azure (ASR) usług są wprowadzane w jego składniki. Druga część dokumentu opisano charakterystyki dla oprogramowania SAP NetWeaver na podstawie zapewniały realizację niezbędnych zadań. Możliwości przy użyciu usługi ASR za pomocą oprogramowania SAP NetWeaver wystąpień aplikacji i usług SAP Central Services są omówione i opisem. Fokus części drugiej polega na wykorzystaniu usługi ASR dla SAP Central Services, które są chronione za pomocą konfiguracji klastra trybu Failover systemu windows.

Aktualizacja: Września 2015 r.
