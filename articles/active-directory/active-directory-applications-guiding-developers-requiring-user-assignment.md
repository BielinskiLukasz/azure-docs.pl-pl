# <a name="azure-ad-and-applications-require-user-assignment"></a>Usługa Azure AD i aplikacji: wymaganie przypisania użytkownika
## <a name="requiring-user-assignment"></a>Wymaganie przypisania użytkownika
1. Zaloguj się do witryny Azure portal przy użyciu konta administratora.
2. Kliknij pozycję **wszystkich usług** element w menu głównym.
3. Wybierz katalog, którego używasz dla aplikacji.
4. Kliknij pozycję **aplikacje dla przedsiębiorstw** kartę.
5. Wybierz aplikację z listy aplikacji skojarzonych z tym katalogiem.
6. Kliknij przycisk **właściwości** kartę.
7. Zmiana **wymagane przypisanie użytkownika?** Przełącz Yes (tak).
8. Kliknij przycisk **Zapisz** znajdujący się u góry ekranu.

Będą teraz mieć przypisać użytkowników i/lub grup do aplikacji. Zobacz [przypisywanie użytkowników do aplikacji](active-directory-applications-guiding-developers-assigning-users.md) i [Przypisywanie grup do aplikacji](active-directory-applications-guiding-developers-assigning-groups.md).
