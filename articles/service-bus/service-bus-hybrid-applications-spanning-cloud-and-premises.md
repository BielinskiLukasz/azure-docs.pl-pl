# クラウドと社内にまたがったハイブリッド アプリケーションの有効化

最新の IT 環境ではクラウド/オンプレミスのハイブリッド アプリケーションが現実になっています。 Microsoft Azure Service Bus により、既存のオンプレミス アセットを利用して、信頼性の高い安全なソリューションを作成し、現在のインフラストラクチャを強化、拡張できます。
