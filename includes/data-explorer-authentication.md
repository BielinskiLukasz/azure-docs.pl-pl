## <a name="authentication"></a>Uwierzytelnianie

Aby uruchomić poniższy przykład, potrzebujesz aplikacji i jednostki usługi Azure Active Directory (Azure AD), która może uzyskiwać dostęp do zasobów. Aby utworzyć bezpłatną aplikację usługi Azure AD i dodać przypisanie roli na poziomie subskrypcji, zobacz [Tworzenie aplikacji usługi Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal). Potrzebny jest również identyfikator katalogu (dzierżawy), identyfikator aplikacji i klucz tajny klienta.
