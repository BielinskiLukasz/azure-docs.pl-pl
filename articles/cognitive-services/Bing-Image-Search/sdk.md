# <a name="bing-search-sdk"></a>Wyszukiwania usługi Bing zestawu SDK
Przykłady interfejsu API Bing obraz wyszukiwania obejmują senarios który:
1.  Zapytanie dla prostego obrazu.
2.  Ograniczenie do określonego obrazu typów i proporcje przedstawiający obraz możliwości filtrowania wyszukiwania.
3.  Wyszukiwanie obrazów umożliwia analizę trendów w różnych sieciach społecznościowych w tej chwili.
4.  Zapytanie dla sformatowanego szczegóły obrazu, takie jak źródło obrazu, tekst zapytania reprezentatywny itp.

Zestawy SDK wyszukiwania usługi Bing należy funkcji wyszukiwania w sieci web łatwo dostępne w następujących językach programowania:
* Rozpoczynanie pracy z [próbek .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Pakiet NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.ImageSearch/1.2.0)
    * Zobacz też [bibliotek .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch) definicje i zależności.
* Rozpoczynanie pracy z [Node.js — przykłady](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Zobacz też [bibliotek Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Zobacz też [bibliotek Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch) definicje i zależności.
* Rozpoczynanie pracy z [próbek Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Zobacz też [bibliotek języka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) definicje i zależności.

Przykłady zestawu SDK dla każdego języka obejmują plik ReadMe o szczegółowe informacje o wymaganiach wstępnych oraz instalowania/uruchomiony próbek.
