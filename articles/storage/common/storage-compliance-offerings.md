# <a name="azure-storage-compliance-offerings"></a>Oferty zgodności usługi Azure Storage

Aby pomóc organizacjom w spełnieniu krajowych, regionalnych i branżowych wymagań dotyczących gromadzenia i wykorzystywania danych osób fizycznych, microsoft Azure & Azure Storage oferują najbardziej kompleksowy zestaw certyfikatów i certyfikatów dowolnego dostawcy usług w chmurze.

Poniżej znajdziesz oferty zgodności w usłudze Azure Storage, aby zapewnić, że twoja usługa jest regulowana przy użyciu usługi Azure Storage. Mają one zastosowanie do następujących ofert usługi Azure Storage: obiekty Blobs, pliki, kolejki, tabele, dyski, chłodny magazyn i magazyn w warstwie Premium.
