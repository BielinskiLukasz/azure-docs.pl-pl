## <a name="get-support-for-azure-machine-learning-services"></a>Uzyskać pomoc techniczną dla usługi Azure Machine Learning

Można uzyskać pomoc techniczną dla usługi uczenie maszynowe Azure w następujący sposób:

+ **Pomoc techniczna**: Azure skontaktuj się z działem pomocy technicznej, przechodząc [Azure obsługuje](https://azure.microsoft.com/support/options/) i wybierając uczenia maszynowego. 

+ **Forum użytkowników MSDN**: zadawać pytania, odpowiedzi na pytania i Połącz z innymi użytkownikami w [forum pomocy technicznej usługi Azure Machine Learning w witrynie MSDN](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Przepełnienie stosu**: odwiedź stronę społeczności usługi Azure Machine Learning na [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) oznaczone tagiem "Azure-Machine-Learning".

+ **Udostępnianie sugestie produktu** oraz funkcji żądań w naszym [Azure Machine Learning opinii kanału](https://feedback.azure.com/forums/257792-machine-learning), która jest również dostępna przy użyciu łącza w dolnej części każdego artykułu.
