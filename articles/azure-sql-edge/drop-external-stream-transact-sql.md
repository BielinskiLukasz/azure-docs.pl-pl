# <a name="drop-external-stream-transact-sql"></a>Usuń strumień zewnętrzny (Transact-SQL)

Odrzuca zadanie przesyłania strumieniowego. 

## <a name="syntax"></a>Składnia

```sql
DROP EXTERNAL STREAM {external_stream_name}  
```
