# <a name="azure-cosmos-db-documentation-experiment"></a>Eksperymentu dokumentacji w usłudze Azure DB rozwiązania Cosmos

Azure DB rozwiązania Cosmos jest globalnie rozproszone i wiele modeli bazy danych. Ponieważ możliwość tak wiele z bazy danych rozwiązania Cosmos Azure — Tworzenie wykresu do tworzenia bazy danych dokumentów lub migrację danych z magazynu bazy danych MongoDB, Cassandra lub tabel Azure — organizowanie dokumentacji może być trudne. Dlatego chcemy pomocy.

Testujemy organizacji bazy danych Azure rozwiązania Cosmos spis treści i [strony docelowej](https://docs.microsoft.com/en-us/azure/cosmos-db/), jak pokazano na poniższych ilustracjach i chcemy uzyskać swoją opinię na temat tych proponowane zmiany.

## <a name="to-load-both-views"></a>Aby załadować zarówno widoków

1. Aby otworzyć obu widokach, obok siebie, przejdź do [strony docelowej](https://docs.microsoft.com/en-us/azure/cosmos-db/) z ulubionej przeglądarce internetowej. 
2. Obrazy powyżej, aby ustalić, która wersja zostanie wyświetlony, oryginalnej lub eksperymentalne. 
3. Otwórz okno inprivate lub incognito przeglądarki i obciążenia [strony docelowej](https://docs.microsoft.com/en-us/azure/cosmos-db/) w nowym oknie przeglądarki i ponownie użyj obrazy, aby ustalić, która wersja zostanie wyświetlony. 

   Jeśli nie masz obu widokach obecne, zamknij i ponownie otwórz nowe okno inprivate lub incognito i obciążenia [strony docelowej](https://docs.microsoft.com/en-us/azure/cosmos-db/) ponownie. Może upłynąć kilka próbuje pobrać obie wersje załadowany w oddzielnych okien. Odświeżanie okna incognito nie zostanie załadowany innych wersji, należy otworzyć nowe okno. 

4. Następnie można zmienić rozmiar systemu windows, aby naprawić obok siebie, lub przełączać się między widokami do porównania. 

5. Po już wcześniej szansę, aby zapoznać się z organizacji i układu eksperymentalne środowisko, kliknij przycisk **eksperymentu ankiety** poniższy przycisk, aby wyrazić swoją opinię. Nie później niż północ 4 kwietnia 2018 przesłać formularza tak, aby opinii można przejrzeć i włączona.
