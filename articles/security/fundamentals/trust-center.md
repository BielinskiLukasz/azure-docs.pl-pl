# <a name="microsoft-trust-center"></a>Centrum zaufania firmy Microsoft
Witryna informacji o zabezpieczeniach platformy Azure w systemie Azure.com zawiera informacje potrzebne do bezpiecznego planowania, projektowania, wdrażania i konfigurowania rozwiązań w chmurze oraz zarządzania nimi. Centrum zaufania firmy Microsoft zawiera również informacje niezbędne do zagwarantowania, że platforma Azure, w której są uruchamiane usługi, jest bezpieczna.

Wiemy, że po powierzeniu aplikacji i danych na platformie Azure masz pytania. Gdzie jest? Kto może uzyskać do niego dostęp? Co robią firma Microsoft w celu jej ochrony? Jak można sprawdzić, czy firma Microsoft robi to?

Mamy odpowiedzi. Ponieważ jest to Twoje dane, użytkownik decyduje o tym, kto ma dostęp, i pracuje z nami, aby zdecydować, gdzie się znajdują. Aby chronić dane, korzystamy z najnowocześniejszej technologii zabezpieczeń i kryptografii światowej klasy. Nasza zgodność jest niezależnie poddawana inspekcji i jest przejrzysta na wielu poziomach — od tego, jak obsługujemy wymagania prawne dla danych klienta w zakresie bezpieczeństwa naszego kodu.

Oto co znajdziesz w centrum zaufania firmy Microsoft:

* [Zabezpieczenia](https://aka.ms/tcsecurity) — Dowiedz się, w jaki sposób wszystkie usługi Microsoft Cloud są zabezpieczone.
* [Prywatność](https://aka.ms/tcprivacy) — informacje o tym, jak firma Microsoft gwarantuje prywatność danych w chmurze firmy Microsoft.
* [Zgodność](https://aka.ms/tccompliance) — Odkryj, w jaki sposób firma Microsoft pomaga organizacjom w zgodności z krajowymi, regionalnymi i branżowymi wymaganiami dotyczącymi zbierania i używania danych indywidualnych.
* [Przezroczystość](https://aka.ms/tctransparency) — Zobacz, w jaki sposób firma Microsoft uważa, że dane są kontrolowane w chmurze, oraz jak firma Microsoft pomaga znać, jak to możliwe, jak te dane są obsługiwane.
* [Produkty i usługi](https://aka.ms/tcproductsservices) — zobacz wszystkie Microsoft Cloud produkty i usługi w jednym miejscu
* [Portal zaufania usługi](https://aka.ms/tcservicetrportal) — uzyskiwanie kopii niezależnych raportów inspekcji usług w chmurze firmy Microsoft, ocen ryzyka, najlepszych rozwiązań w zakresie zabezpieczeń i powiązanych materiałów.
* [Co nowego](https://aka.ms/tcwhatsnew) — Dowiedz się, co nowego w Microsoft Cloud zaufania
* [Zasoby](https://aka.ms/tcresources) — Zbadaj oficjalne dokumenty, wideo i analizy przypadków w chmurze firmy Microsoft

[Centrum zaufania firmy Microsoft](https://www.microsoft.com/trustcenter) zawiera informacje potrzebne do zapoznania się z tym, co należy zrobić, aby zabezpieczyć Microsoft Cloud.
