### <a name="azure-active-directory-authentication"></a>Uwierzytelnianie usługi Azure Active Directory

Kotwice przestrzenne platformy Azure umożliwiają również aplikacjom uwierzytelnianie za pomocą tokenów użytkownika usługi Azure AD (Active Directory). Można na przykład użyć tokenów usługi Azure AD do integracji z zakotwiczeniami przestrzennymi platformy Azure. Jeśli firma utrzymuje użytkowników w usłudze Azure AD, możesz podać token użytkownika usługi Azure AD w zestawie SDK kotwice przestrzenne platformy Azure. Dzięki temu można uwierzytelniać się bezpośrednio w usłudze kotwic Azure w ramach konta należącego do tej samej dzierżawy usługi Azure AD.
