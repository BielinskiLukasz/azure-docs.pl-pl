Najprościej jest utworzyć pierwsze konto usługi Azure Storage przy użyciu witryny [Azure Portal](https://portal.azure.com). Więcej informacji można znaleźć w temacie [Tworzenie konta magazynu](../articles/storage/common/storage-quickstart-create-account.md).

Możesz utworzyć konto usługi Azure Storage przy użyciu programu [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md) lub [interfejsu wiersza polecenia platformy Azure](../articles/storage/common/storage-azure-cli.md).

Jeśli nie chcesz teraz tworzyć konta magazynu, możesz także użyć emulatora usługi Azure Storage do uruchomienia i testowania kodu w środowisku lokalnym. Więcej informacji można znaleźć w temacie [Use the Azure Storage Emulator for Development and Testing](../articles/storage/common/storage-use-emulator.md) (Używanie emulatora usługi Azure Storage do programowania i testowania).
