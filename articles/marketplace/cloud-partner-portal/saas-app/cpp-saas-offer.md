# <a name="azure-saas-application-offer"></a>Oferta aplikacji Azure SaaS

[Portal Cloud partner](https://cloudpartner.azure.com/) nie obsługuje już tworzenia ofert aplikacji SaaS i zarządzania nimi.  Zamiast tego ta funkcja została przeniesiona do [portalu Centrum partnerskiego](https://partner.microsoft.com/pcv/)firmy Microsoft.  Aby uzyskać więcej informacji, zobacz [Tworzenie nowej oferty SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Następne kroki

Przed utworzeniem nowej oferty aplikacji SaaS należy przeczytać [Podręcznik publikowania aplikacji SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) i zapoznać się z instrukcjami portalu Centrum partnerskiego wymienionymi na liście [kontrolnej tworzenia oferty](../../partner-center-portal/offer-creation-checklist.md).
