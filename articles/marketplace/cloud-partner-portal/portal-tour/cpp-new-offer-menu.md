# <a name="new-offer-menu"></a>Menu nowej oferty

Menu **Nowa oferta** umożliwia partnerom tworzenie wystąpień typów ofert obsługiwanych przez portal Cloud Partner [Portal](https://cloudpartner.azure.com).  Wybranie pozycji menu **Nowa oferta** na pasku menu nawigacji po lewej stronie powoduje wyświetlenie następującej listy typów ofert.  Wybranie jednego z tych typów rozpoczyna proces tworzenia i publikowania oferty.

### <a name="next-steps"></a>Następne kroki

Jeśli tworzysz nową ofertę, możesz użyć poprzedniej tabeli, aby przejść do sekcji oferty, aby uzyskać wskazówki dotyczące tworzenia i publikowania oferty.  W przeciwnym razie przewodnik po portalu będzie kontynuowany z badaniem [strony Wszystkie oferty](./cpp-all-offers-page.md).
