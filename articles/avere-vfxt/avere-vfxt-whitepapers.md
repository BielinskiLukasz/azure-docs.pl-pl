## <a name="deploy-agisoft-photoscan-on-azure"></a>Wdrażanie Agisoft PhotoScan na platformie Azure

**Link:** [Wdrażanie Agisoft PhotoScan na platformie Azure przy użyciu Avere vFXT dla platformy Azure lub BeeGFS](https://azure.microsoft.com/mediahandler/files/resourcefiles/deploy-agisoft-photoscan-on-azure-with-azere-vfxt-for-azure-or-beegfs/AgiSoft%20PhotoScan%20on%20Azure%20using%20Avere%20vFXT%20or%20BeeGFS.pdf) 

Ten przewodnik zawiera wskazówki krok po kroku dotyczące instalowania Agisoft PhotoScan oprogramowania photogrammetry objęta Avere vFXT magazynu lub BeeGFS równoległy system plików. Autor Paulo Marques da Kostaryka z AzureCAT, pokazuje, jak skonfigurować PhotoScan w usłudze Azure Virtual Machines (VMs). Magazyn o wysokiej wydajności przyspiesza czas przetwarzania, a wyniki jego testach porównawczych są uwzględniane. To środowisko można skalować w górę i w dół odpowiednio do potrzeb i obsługuje TB miejsca do magazynowania bez obniżania oczekiwanego poziomu wydajności.
