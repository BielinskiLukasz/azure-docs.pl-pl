## <a name="use-azure-cloud-shell"></a>Używanie usługi Azure Cloud Shell

Azure obsługuje usługę Azure Cloud Shell, środowisko interaktywnej powłoki, które można użyć za pośrednictwem przeglądarki. Usługa cloud Shell pozwala użyć `bash` lub `PowerShell` do pracy z usługami platformy Azure. Aby uruchomić kod w tym artykule, bez konieczności instalowania żadnych narzędzi na środowisku lokalnym, można użyć poleceń wstępnie zainstalowane w usłudze Cloud Shell.

Aby uruchomić usługę Azure Cloud Shell:

| Opcja | Przykład/łącze |
|-----------------------------------------------|---|
| Wybierz pozycję **Wypróbuj** w prawym górnym rogu bloku kodu. Wybieranie **wypróbuj** nie kopiuje automatycznie kod do usługi Cloud Shell. | ![Przykład Wypróbuj usługę Azure Cloud Shell](./media/cloud-shell-try-it/cli-try-it.png) |
| Przejdź do [ https://shell.azure.com ](https://shell.azure.com) lub wybierz **Launch Cloud Shell** przycisk, aby otworzyć usługę Cloud Shell w przeglądarce. | <a href="https://shell.azure.com" title="Uruchom usługę Azure Cloud Shell"><img name="launch-cloud-shell" src="https://docs.microsoft.com/azure/includes/media/cloud-shell-try-it/launchcloudshell.png" /></a> |
| Wybierz **Cloud Shell** przycisk na pasku menu w prawym górnym rogu w [witryny Azure portal](https://portal.azure.com). | ![Przycisk Cloud Shell w witrynie Azure Portal](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Aby uruchomić kod, w tym artykule w usłudze Azure Cloud Shell:

1. Uruchom usługę Cloud Shell.
1. Wybierz **kopiowania** przycisku w bloku kodu, aby skopiować kod. 
1. Wklej kod do sesji usługi Cloud Shell za pomocą **Ctrl**+**Shift**+**V** systemach Windows i Linux, lub **Cmd** + **Shift**+**V** w systemie macOS. 
1. Naciśnij klawisz **Enter** do uruchomienia kodu.
