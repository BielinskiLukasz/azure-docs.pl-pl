## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious Logo](./media/graph-visualization/linkurious-logo.jpg)

Linkurious przedsiębiorstwa używa wykres technologii i wizualizację danych, aby przekształcić złożone zestawy danych w interaktywne sieci visual. Platforma nawiązuje połączenie ze źródłami danych i umożliwia badaczy bezproblemowo przejść między miliardów jednostek i relacji. Wynik jest nowe możliwości w zakresie wykrywania podejrzanych relacje bez miejscami z zapytania lub tabel.

Interfejs interaktywna wizualizacja Linkurious Enterprise oferuje prosty sposób zbadać złożonych połączonych danych. Wyszukaj dla konkretnych jednostek, rozwiń połączeń odkryć ukryte relacje i stosowanie układów wybranym związanych z porządkowaniem złożonych sieci. Linkurious jest teraz zgodna z usługi Azure Cosmos DB — interfejs API Gremlin. Jest ona odpowiednia dla scenariuszy wizualizacji wykresu end-to-end obsługuje odczytywać i zapisywać możliwości z poziomu interfejsu użytkownika. 

![Pokaz Linkurious](./media/graph-visualization/linkurious-demo.gif)

**Następne kroki:** Możesz poprosić o [pokaz Linkurious za pomocą usługi Azure Cosmos DB](https://linkurio.us/contact/).
