# <a name="connect-with-the-web-client"></a>Łączenie się z klientem internetowym

Klient sieci Web umożliwia dostęp do zasobów pulpitu wirtualnego systemu Windows z przeglądarki sieci Web bez długiego procesu instalacji.

>[!NOTE]
>Klient internetowy nie ma obecnie obsługi mobilnego systemu operacyjnego.

## <a name="supported-operating-systems-and-browsers"></a>Obsługiwane systemy operacyjne i przeglądarki

Podczas gdy każda przeglądarka obsługująca HTML5 powinna działać, oficjalnie obsługujemy następujące systemy operacyjne i przeglądarki.

| Przeglądarka           | Obsługiwany system operacyjny                     | Uwagi               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows, macOS, Linux            | Wersja 55 lub nowsza |
| Google Chrome     | Windows, macOS, Linux, Chrome OS |                     |

## <a name="access-remote-resources-feed"></a>Dostęp do kanału informacyjnego zasobów zdalnych

W przeglądarce przejdź do [klienta sieci Web pulpitu wirtualnego systemu Windows](https://rdweb.wvd.microsoft.com/webclient) i zaloguj się za pomocą konta użytkownika.

>[!NOTE]
>Jeśli zalogowano się już przy użyciu innego konta usługi Azure Active Directory niż to, którego chcesz użyć na pulpicie wirtualnym systemu Windows, należy wylogować się lub użyć prywatnego okna przeglądarki.

Po zalogowaniu się powinna zostać wyświetlona lista zasobów. Zasoby można uruchamiać, wybierając je tak, jak w normalnej aplikacji na karcie **Wszystkie zasoby.**
