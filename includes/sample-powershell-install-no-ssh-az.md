Ten przykładowy skrypt wymaga programu Azure PowerShell. Uruchom polecenie `Get-Module -ListAvailable Az`, aby dowiedzieć się, jaka wersja jest używana.
Jeśli konieczna będzie instalacja lub uaktualnienie, zobacz [Instalowanie modułu Azure PowerShell](/powershell/azure/install-az-ps). 

Uruchom polecenie cmdlet [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) , aby nawiązać połączenie z platformą Azure.
