# <a name="supported-languages-for-speech-service"></a>Obsługiwane języki dla usługi mowy

## <a name="speech-translation"></a>Tłumaczenie mowy

**Tłumaczenia mowy** interfejs API obsługuje inne języki do tłumaczenia mowy mowy i mowy na tekst. Język źródłowy musi być zawsze z poniższej tabeli języka mowy. Dostępne języki docelowych zależy od tego, czy docelowym translacji jest tekst lub mowy.

Można uzyskać listę języków albo programowo lub przy użyciu punktu końcowego języków interfejsu API REST. Na liście przedstawiono każdy kod języka, a także nazwę języka w języku angielskim — lub w innych obsługiwanych językach. Ta lista jest aktualizowana przez usługę Microsoft Translator zawsze, gdy nowy język staje się dostępny.
