# <a name="authoring-api-changes"></a>Tworzenie zmian interfejsu API 

Interfejsy API tworzenia są aktualizowane dla wersji v3, aby umożliwić [dekompozycję modelu z jednostkami](luis-concept-data-extraction.md), które są związane z maszyną. 

Możesz tworzyć w interfejsie API w wersji 2 lub Preview v3. Ta sama aplikacja może również korzystać z odpowiedzi punktu końcowego przewidywania w wersji 2 lub v3. Nie trzeba mieć tej samej wersji interfejsu API na potrzeby tworzenia i przewidywania odpowiedzi punktu końcowego.

Portal LUIS używa wyłącznie interfejsów API tworzenia w wersji 3.
