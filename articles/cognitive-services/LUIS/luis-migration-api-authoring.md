# <a name="authoring-api-changes"></a>Tworzenie zmian interfejsu API 

Interfejsy API tworzenia są aktualizowane dla wersji 3, aby umożliwić [rozkład modelu z jednostkami nauczanych maszynowo](luis-concept-data-extraction.md). 

Można tworzyć w V2 lub podgląd V3 APIS. Ta sama aplikacja może również wybrać użycie odpowiedzi punktu końcowego przewidywania V2 lub V3. Nie trzeba mieć tej samej wersji interfejsu API do tworzenia i przewidywania odpowiedzi punktu końcowego.

Portal usługi LUIS używa wyłącznie interfejsów API tworzenia wersji 3.

## <a name="next-steps"></a>Następne kroki
