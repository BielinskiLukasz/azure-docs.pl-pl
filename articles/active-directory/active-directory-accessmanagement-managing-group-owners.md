# グループの所有者の管理

## グループへの所有者の割り当て

**Azure ポータルでグループに所有者を追加するには**

1. Azure ポータルで、クリックして **Active Directory**, 、組織のディレクトリの名前をクリックします。
2. クリックして、 **グループ** ] タブをクリックします。
2.  **グループ** ] ページで、所有者を追加するグループの名前をクリックします。
3. そのグループのページでクリックして **[所有者の追加**します。
4.  **所有者追加** ] ページで、このグループの所有者として追加し、この名前を追加するかどうかを確認するユーザーの名前をクリックして、 **選択された** ウィンドウです。


**Azure ポータルでグループから所有者を削除するには**

1. Azure ポータルで、クリックして **Active Directory**, 、組織のディレクトリの名前をクリックします。
2. クリックして、 **グループ** ] タブをクリックします。
3.  **グループ** ] ページで、所有者を削除するグループの名前をクリックします。
4. そのグループのページをクリックして、 **所有者** ] タブをクリックします。
5. そのグループのページで、クリックして、このグループから削除する所有者の名前をクリックして **削除**します。
