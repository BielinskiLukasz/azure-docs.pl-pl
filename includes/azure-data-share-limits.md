Usługa Azure Data Share umożliwia organizacjom łatwe i bezpieczne udostępnianie danych swoim klientom i partnerom.

| **Zasobów** | **Limit** |
| --- | --- |
| Maksymalna liczba zasobów udziału danych na subskrypcję platformy Azure |50 |
| Maksymalna liczba wysłanych udziałów na zasób udziału danych |100 |
| Maksymalna liczba odebranych udziałów na zasób udziału danych |100 |
| Maksymalna liczba zaproszeń na jedną wysłano jedną akcję |100 |
| Maksymalna liczba subskrypcji akcji na jedną wysłano jedną akcję |100 |
| Maksymalna liczba zestawów danych na jedną akcję |100 |
| Maksymalna liczba harmonogramów migawek na jedną akcję |1 |
